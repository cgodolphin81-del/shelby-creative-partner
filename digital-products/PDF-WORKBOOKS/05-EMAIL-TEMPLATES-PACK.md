## CHAPTER 1: Cold Outreach Templates {#chapter-1}

### Template 1: The Value-First Cold Email

**Best For:** B2B outreach, service offers

```
Subject: Quick question about [Company Name]

Hi [First Name],

I noticed [specific observation about their company - 
recent funding, new hire, product launch, etc.].

We recently helped [similar company] achieve 
[specific result] using [your solution].

[Client name] was able to [quantifiable result] 
in [timeframe].

Would you be open to a 15-minute call to explore 
if something similar could work for [Company Name]?

Best,
[Your Name]

P.S. Here's the case study: [link]
```

**Why It Works:**
- Shows research
- Social proof immediately
- Specific results
- Low-pressure ask

## CHAPTER 8: Template Customization Guide {#chapter-8}

### How to Use These Templates

**DO:**
```
✓ Personalize for each recipient
✓ Adapt to your voice and style
✓ Test different variations
✓ Track what works
✓ Update regularly
```

**DON'T:**
```
✗ Send without customization
✗ Use for spam
✗ Ignore context
✗ Forget to proofread
✗ Sound robotic
```

### Customization Checklist

Before sending any template:

```
□ Recipient's name is correct
□ Company name is correct
□ Specific details are accurate
□ Tone matches your brand
□ Links work
□ No placeholder text remains
□ Proofread for errors
□ Mobile-friendly formatting
```

### A/B Testing Framework

**Test One Element at a Time:**

```
Week 1-2: Subject Lines
• Version A: Question format
• Version B: Statement format

Week 3-4: Opening Lines
• Version A: Direct
• Version B: Story-based

Week 5-6: Call to Action
• Version A: Specific ask
• Version B: Open-ended

Week 7-8: Email Length
• Version A: Short (under 100 words)
• Version B: Long (200+ words)
```

### Metrics to Track

**For Each Template:**
```
□ Open rate
□ Reply rate
□ Click-through rate
□ Conversion rate
□ Unsubscribe rate
□ Time to response
```

**Benchmarks by Type:**

| Email Type | Good Open Rate | Good Reply Rate |
|------------|---------------|-----------------|
| Cold Outreach | 20-30% | 5-10% |
| Follow-Up | 30-40% | 10-20% |
| Newsletter | 25-35% | 1-5% |
| Sales | 30-50% | 15-25% |
| Client Comms | 60-80% | 30-50% |

## 📎 APPENDIX: Quick Reference

### Subject Line Formulas

```
• Question: "Quick question about [Company]?"
• Curiosity: "I noticed something about [Company]"
• Benefit: "How to [achieve result] in [timeframe]"
• Social Proof: "[Similar Company] just did [result]"
• Urgency: "[Deadline] approaching for [opportunity]"
• Personalization: "[Mutual Contact] suggested we connect"
```

### Email Timing Best Practices

```
BEST DAYS TO SEND:
• Tuesday, Wednesday, Thursday

BEST TIMES:
• 9-11 AM (local time of recipient)
• 1-3 PM (after lunch)

AVOID:
• Monday mornings (inbox overflow)
• Friday afternoons (checking out)
• Weekends (unless expected)
• Holidays (obviously)
```

### Follow-Up Cadence

```
Day 1: Initial email
Day 3: Follow-up #1 (add value)
Day 7: Follow-up #2 (case study)
Day 14: Follow-up #3 (different angle)
Day 30: Follow-up #4 (breakup)
Day 60: Re-engagement (new insight)
```
