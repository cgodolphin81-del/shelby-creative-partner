## CHAPTER 1: Modern CV Fundamentals {#chapter-1}

### The New Rules of CV Writing

**Old Rules (Ignore These):**
```
✗ Keep it to one page
✗ Use objective statements
✗ List duties and responsibilities
✗ Include "References available upon request"
✗ Use generic templates
✗ Same CV for every application
```

**New Rules (Follow These):**
```
✓ 2 pages is standard (3+ for senior roles)
✓ Start with professional summary
✓ Focus on achievements and impact
✓ Omit references section (provide when asked)
✓ Customize for each application
✓ Design matters (clean, modern, readable)
```

### What Recruiters Actually Want

**The 6-Second Test:**
Recruiters spend an average of 6 seconds on initial CV screening. They look for:

```
1. Current/most recent role
2. Previous role
3. Education
4. Skills match
5. Career progression
6. Employment gaps
```

**Key Questions They're Asking:**
```
□ Can this person do the job?
□ Have they done similar work before?
□ Will they fit the culture?
□ Are they ambitious/growing?
□ Is there anything concerning?
```

### CV vs. Resume: Know the Difference

| Aspect | CV | Resume |
|--------|-----|--------|
| Length | 2+ pages | 1-2 pages |
| Focus | Comprehensive | Targeted |
| Use | Academic, Medical, International | Corporate, US jobs |
| Updates | Add everything | Curate for role |
| Sections | Detailed, extensive | Concise, relevant |

**For this guide:** We'll use "CV" to mean a comprehensive professional document (2 pages).

## CHAPTER 2: CV Structure & Format {#chapter-2}

### The Modern CV Structure

```
┌─────────────────────────────────────────┐
│              HEADER                     │
│         Name + Contact Info             │
├─────────────────────────────────────────┤
│         PROFESSIONAL SUMMARY            │
│         (3-4 lines, punchy)             │
├─────────────────────────────────────────┤
│            KEY SKILLS                   │
│         (6-10 bullet points)            │
├─────────────────────────────────────────┤
│          WORK EXPERIENCE                │
│        (Reverse chronological)          │
│                                         │
│    Current Role (most detail)           │
│    Previous Role                        │
│    Earlier Roles (less detail)          │
├─────────────────────────────────────────┤
│             EDUCATION                   │
│         (Degree, School, Year)          │
├─────────────────────────────────────────┤
│      ADDITIONAL SECTIONS (Optional)     │
│    • Certifications                     │
│    • Projects                           │
│    • Publications                       │
│    • Languages                          │
│    • Volunteer Work                     │
└─────────────────────────────────────────┘
```

### Formatting Best Practices

**Font & Size:**
```
✓ Font: Calibri, Arial, Helvetica, or Garamond
✓ Size: 10-12pt for body, 14-16pt for headers
✓ Name: 18-24pt
✓ Consistent throughout
```

**Margins & Spacing:**
```
✓ Margins: 0.5" to 1" on all sides
✓ Line spacing: 1.0 to 1.15
✓ Space between sections: 6-12pt
✓ Bullet points: Standard round or square
```

**Length Guidelines:**
```
Entry Level (0-3 years): 1 page
Mid-Level (3-10 years): 2 pages
Senior (10+ years): 2-3 pages
Executive: 3+ pages acceptable
Academic/Medical: As long as needed
```

### File Format & Naming

**Save As:**
```
✓ PDF (unless specifically asked for Word)
✓ Preserves formatting
✓ Professional appearance
✓ ATS-compatible (modern ATS read PDFs)
```

**File Naming:**
```
✓ FirstName_LastName_CV.pdf
✓ FirstName_LastName_CV_Role.pdf
✓ FirstName_LastName_Resume.pdf

✗ resume.pdf
✗ CV_2026.pdf
✗ document1.pdf
```

## CHAPTER 3: Writing Each Section {#chapter-3}

### 1. Header/Contact Information

**Required:**
```
• Full Name (as you use professionally)
• Phone Number
• Email Address (professional)
• LinkedIn URL (customized)
• Location (City, Country - full address not needed)
```

**Optional:**
```
• Portfolio/Website URL
• GitHub (for developers)
• Twitter (if professional/relevant)
```

**Example:**
```
JANE SMITH
San Francisco, CA | (555) 123-4567
jane.smith@email.com | linkedin.com/in/janesmith
janesmith.com
```

### 2. Professional Summary

**The Formula:**
```
[Adjective] [Current Role] with [X] years of experience 
in [Specialty/Industry]. Proven track record of 
[Key Achievement 1] and [Key Achievement 2]. 
Seeking to [Goal] at [Target Company Type].
```

**Examples:**

**Marketing Manager:**
```
Results-driven Marketing Manager with 8+ years of 
experience leading B2B SaaS marketing teams. 
Increased pipeline generation by 340% and reduced 
CAC by 45% at TechCorp. Seeking to drive growth 
as Senior Marketing Director at a scaling startup.
```

**Software Engineer:**
```
Full-stack Software Engineer with 5 years building 
scalable web applications. Led migration serving 
2M+ users with 99.9% uptime. Passionate about 
clean code and user-centric design. Looking to 
contribute to innovative products at a mission-driven company.
```

**Career Changer:**
```
Former Finance Manager transitioning to Data 
Science. Completed intensive data science 
bootcamp with capstone project predicting 
customer churn at 94% accuracy. Combining 
10 years of business acumen with new technical 
skills to drive data-informed decisions.
```

### 3. Key Skills Section

**Format Options:**

**Option A - Categorized:**
```
KEY SKILLS

Technical: Python, SQL, Tableau, AWS, Git
Marketing: SEO, SEM, Content Strategy, Email Marketing
Leadership: Team Management, Strategic Planning, Budgeting
```

**Option B - Simple List:**
```
KEY SKILLS

• Strategic Planning    • Team Leadership
• Data Analysis         • Project Management
• Budget Management     • Stakeholder Communication
• Process Improvement   • Change Management
```

**Option C - With Proficiency:**
```
KEY SKILLS

Advanced: Python, SQL, Machine Learning, Tableau
Intermediate: R, AWS, Docker, Git
Familiar: Kubernetes, TensorFlow, Spark
```

### 4. Work Experience

**Standard Format:**
```
JOB TITLE                                    Dates
Company Name, Location

• Achievement-focused bullet point with metric
• Another accomplishment with specific result
• Third bullet showing scope/impact
• Fourth bullet if highly relevant

• Optional second paragraph for context
```

**Example:**
```
SENIOR MARKETING MANAGER                    2022-2026
TechCorp Inc., San Francisco, CA

• Grew MQLs by 340% in 18 months through 
  integrated demand generation campaigns
• Reduced customer acquisition cost by 45% 
  while increasing lead volume 2x
• Led team of 6 marketers; 3 promoted 
  during tenure
• Launched ABM program targeting Fortune 500, 
  resulting in $2.3M pipeline
• Managed $1.2M annual marketing budget
```

### 5. Education Section

**Standard Format:**
```
DEGREE NAME                                 Graduation Year
University Name, Location

Optional additions:
• GPA (if 3.5+ and recent grad)
• Relevant coursework
• Honors/awards
• Study abroad
```

**Examples:**
```

Stanford University, Stanford, CA
Magna Cum Laude, GPA: 3.8/4.0

MASTER OF BUSINESS ADMINISTRATION           2022

```

```
University of California, Berkeley, CA
Expected GPA: 3.7/4.0
Relevant Coursework: Econometrics, Statistics, Data Analysis
```

## CHAPTER 4: Achievement Statements {#chapter-4}

### The Achievement Formula

```
ACTION VERB + TASK + RESULT (with metric)
```

**Weak → Strong Examples:**

```
✗ "Responsible for sales"
✓ "Increased regional sales by 47% in 12 months"

✗ "Managed social media accounts"
✓ "Grew Instagram following from 5K to 50K in 6 months"

✗ "Handled customer service"
✓ "Achieved 98% customer satisfaction rating across 500+ interactions"

✗ "Wrote code for application"
✓ "Developed feature reducing page load time by 60%"
```

### Power Verbs by Category

**Leadership:**
```
Led, Directed, Managed, Supervised, Coordinated, 
Mentored, Trained, Developed, Hired, Built
```

**Achievement:**
```
Achieved, Exceeded, Surpassed, Delivered, Accomplished, 
Attained, Secured, Won, Earned, Generated
```

**Improvement:**
```
Improved, Enhanced, Optimized, Streamlined, Reduced, 
Increased, Maximized, Minimized, Accelerated
```

**Creation:**
```
Created, Developed, Designed, Built, Launched, 
Established, Founded, Initiated, Pioneered
```

**Analysis:**
```
Analyzed, Evaluated, Assessed, Investigated, 
Researched, Identified, Diagnosed, Audited
```

**Communication:**
```
Presented, Negotiated, Persuaded, Influenced, 
Authored, Published, Spoke, Wrote, Communicated
```

### Quantifying Your Impact

**What to Measure:**

| Category | Metrics to Include |
|----------|-------------------|
| Revenue | $ amount, % growth, deals closed |
| Costs | $ saved, % reduced, efficiency gains |
| Time | Hours saved, % faster, deadlines met |
| Volume | Units, users, customers, transactions |
| Quality | Accuracy %, satisfaction scores, ratings |
| Growth | % increase, new markets, expansion |
| Team | Team size, promotions, retention |

**If You Don't Have Exact Numbers:**

```
✓ Use approximations: "approximately," "about," "~"
✓ Use ranges: "15-20%," "$50K-75K"
✓ Use frequency: "daily," "weekly," "for 50+ clients"
✓ Use comparisons: "top 10%," "highest in region"
✓ Estimate based on available data
```

## CHAPTER 6: ATS Optimization {#chapter-6}

### What is ATS?

**Applicant Tracking Systems** are software used by 95%+ of large companies to:
```
• Parse and store CVs
• Screen for keywords
• Rank candidates
• Manage hiring workflow
```

**Common ATS Platforms:**
- Workday
- Taleo
- Greenhouse
- Lever
- iCIMS
- BambooHR

### ATS Best Practices

**Formatting:**
```
✓ Use standard section headings
✓ Simple, clean layout
✓ Standard fonts (Arial, Calibri, Times)
✓ No graphics, images, or tables
✓ No headers/footers for critical info
✓ Use .docx or PDF (check job posting)
```

**Keywords:**
```
✓ Mirror language from job description
✓ Include both acronyms and full terms
✓ Use standard job titles
✓ Include relevant skills explicitly
✓ Don't keyword stuff (write naturally)
```

**Section Headings (Use These Exact Terms):**
```
✓ Work Experience
✓ Professional Experience
✓ Education
✓ Skills
✓ Certifications
✓ Projects
```

### ATS Checklist

```
□ Standard section headings used
□ No columns or tables
□ No graphics or images
□ Simple bullet points
□ Standard fonts only
□ Contact info in body (not header)
□ Keywords from job description included
□ Skills listed explicitly
□ File saved as PDF or .docx
□ File name is professional
```

### Keyword Extraction Exercise

**From Job Description:**

```
1. Copy the job description
2. Highlight all hard skills
3. Circle all software/tools mentioned
4. Note all qualifications required
5. Identify key responsibilities

Incorporate these naturally into your CV.
```

**Example:**

Job Description says:
```
"Looking for a Marketing Manager with 
experience in SEO, SEM, and marketing 
automation. Must be proficient in Google 
Analytics, HubSpot, and have 5+ years 
in B2B SaaS."
```

Your CV should include:
```
• "5+ years of B2B SaaS marketing experience"
• "SEO and SEM campaign management"
• "Marketing automation expertise"
• "Proficient in Google Analytics, HubSpot"
```

## CHAPTER 7: Cover Letters {#chapter-7}

### When to Write a Cover Letter

**Always Write One:**
```
✓ When explicitly required
✓ When you have a referral
✓ When you're career changing
✓ When you have employment gaps
✓ When you're very excited about the role
```

**Optional:**
```
• When not required but you want to stand out
• When you have unique context to share
• When applying to smaller companies
```

**Skip It:**
```
✗ When it says "optional" and you're 
  a perfect match on paper
✗ When applying through employee referral 
  (referral note is enough)
✗ When the company clearly doesn't read them
```

### Cover Letter Structure

```
┌─────────────────────────────────────────┐
│  Header (your info + their info)        │
│  Dear [Hiring Manager Name],            │
├─────────────────────────────────────────┤
│  PARAGRAPH 1: Hook + Role              │
│  (Why you're excited + what you bring) │
├─────────────────────────────────────────┤
│  PARAGRAPH 2-3: Evidence               │
│  (2-3 specific achievements relevant   │
│   to the role)                          │
├─────────────────────────────────────────┤
│  PARAGRAPH 4: Company Connection       │
│  (Why THIS company, not any company)   │
├─────────────────────────────────────────┤
│  PARAGRAPH 5: Call to Action           │
│  (Next steps + thank you)              │
│  Sincerely, [Your Name]                │
└─────────────────────────────────────────┘
```

### Cover Letter Template

```
[Your Name]
[Your Address]
[Your Email]
[Your Phone]
[Date]

[Hiring Manager Name]
[Company Name]
[Company Address]

Dear [Mr./Ms./Mx. Last Name],

[HOOK] I'm writing to express my strong interest 
in the [Role] position at [Company]. When I saw 
[specific thing about company/role], I knew I 
had to apply. With [X] years of experience in 
[relevant area] and a track record of [key 
achievement], I'm excited about the opportunity 
to [contribute specific thing].

[EVIDENCE 1] At [Current/Previous Company], I 
[specific achievement with metric]. This required 
[relevant skill] and resulted in [specific outcome].

[EVIDENCE 2] Previously, I [another relevant 
achievement]. This experience taught me [relevant 
lesson/skill] that I'd bring to [Company].

[COMPANY CONNECTION] I've followed [Company]'s 
work in [specific area] and particularly admire 
[specific thing]. Your mission to [company mission] 
resonates with me because [personal connection].

[CALL TO ACTION] I'd welcome the opportunity to 
discuss how my experience in [key area] can help 
[Company] [achieve specific goal]. Thank you for 
your consideration.

Sincerely,
[Your Name]
```

## 📎 APPENDIX: Quick Checklists

### Final CV Checklist

```
CONTENT:
□ No spelling or grammar errors
□ Consistent formatting throughout
□ All dates are accurate
□ Contact information is current
□ LinkedIn profile is updated
□ All claims can be backed up

FORMAT:
□ Clean, professional appearance
□ Easy to scan in 6 seconds
□ Consistent bullet point style
□ Proper margins and spacing
□ Saved as PDF with proper name

CUSTOMIZATION:
□ Tailored to specific role
□ Keywords from job description included
□ Most relevant experience emphasized
□ Company research reflected
```

### Common Mistakes to Avoid

```
✗ Typos and grammatical errors
✗ Unprofessional email address
✗ Too long or too short
✗ Vague, generic statements
✗ Listing duties instead of achievements
✗ Including irrelevant information
✗ Unexplained employment gaps
✗ Lying or exaggerating
✗ Including photo (US/UK)
✗ Using "I" or "me" (first person)
```
