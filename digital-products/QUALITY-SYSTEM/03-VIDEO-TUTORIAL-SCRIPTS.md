# Video Tutorial Scripts

**Complete Walkthrough Scripts for Each Product Type**

---

## 📹 VIDEO STANDARDS

### Technical Requirements:
- **Length:** 3-10 minutes (varies by product)
- **Resolution:** 1080p minimum
- **Format:** MP4 (H.264)
- **Audio:** Clear, no background noise
- **Captions:** Required (auto-generated + reviewed)
- **Hosting:** Loom, YouTube (unlisted), or Gumroad native

### Style Guidelines:
- **Tone:** Friendly, encouraging, professional
- **Pace:** Moderate (not rushed, not slow)
- **Visuals:** Screen recording with cursor highlights
- **Intro:** Consistent branding (5 seconds max)
- **Outro:** Clear next steps + support info

## 📋 PRODUCTION CHECKLIST

### Pre-Recording:
```
□ Script reviewed and timed
□ Template is final version (no placeholder text)
□ Screen is clean (close unnecessary tabs/apps)
□ Audio setup tested (mic levels, no background noise)
□ Lighting is good (if on camera)
□ Water nearby (for voice)
```

### Recording:
```
□ Record in quiet environment
□ Speak clearly and at moderate pace
□ Smile while talking (it comes through in voice)
□ Pause between sections (easier editing)
□ Re-record any mistakes immediately
```

### Post-Production:
```
□ Trim dead air at start/end
□ Add intro/outro branding (5 sec max)
□ Add captions (auto-generate then review)
□ Export in 1080p MP4
□ Upload to hosting platform
□ Test video plays correctly
□ Add to product delivery
```
