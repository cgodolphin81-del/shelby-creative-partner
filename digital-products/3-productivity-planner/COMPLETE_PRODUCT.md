## Production Notes

### Creating the PDF

**Tools:**
- Canva Pro (easiest)
- Adobe InDesign (professional)
- Affinity Publisher (one-time purchase)
- Google Slides (free, hacky but works)

**Design Tips:**
- Use consistent fonts throughout
- Leave margin for printing/binding
- Use light gray lines (not black) for writing areas
- Include page numbers
- Add your branding (subtle)
- Test print before finalizing

**Fonts:**
- Headers: Montserrat, Poppins, or Playfair Display
- Body: Open Sans, Lato, or Inter
- Handwriting (optional): Caveat or Dancing Script

**Colors:**
- Keep it minimal (black, gray, 1 accent color)
- Ensure good contrast for printing
- Test in grayscale

### File Sizes
- Keep each PDF under 10MB
- Compress images
- Use "Save for Web" or similar

### Testing
- Print test pages (check margins)
- Test on iPad (GoodNotes/Notability)
- Check hyperlinks work
- Verify all pages render correctly
