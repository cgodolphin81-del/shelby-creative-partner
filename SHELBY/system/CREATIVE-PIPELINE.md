## 🔄 Batch Workflow (Efficient)

### WRITING BLOCK (2-3 hours)
```
1. Episode research (15 min)
2. Episode writing (45 min)
3. Fashion design (15 min)
4. Image prompts (5 min)
5. Song lyrics (10 min)
6. Learning log (3 min)
→ REPEAT for next episode
```

### AUDIO BLOCK (1 hour)
```
1. TTS generation for all episodes
2. 4 segments per episode
3. Log all audio files
```

### POETRY BLOCK (30 min)
```
1. Write 3-5 poems
2. Theme-based (destination-inspired)
3. Log to poetry folder
```

### ADMIN BLOCK (15 min)
```
1. Journal entry
2. Git commit
3. Update task list
```

## ⚡ Efficiency Rules

1. **Batch similar tasks** — All fashion designs together, all songs together
2. **No context switching** — Writing block, then audio block, not mixed
3. **Template everything** — Fashion design template, song template, poem template
4. **Time-box each task** — 15 min per fashion design, not "until perfect"
5. **Ship before refining** — Complete > perfect
