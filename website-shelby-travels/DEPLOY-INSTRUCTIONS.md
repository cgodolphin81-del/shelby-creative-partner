## 🚀 Deploy to GitHub Pages

### Step 1: Create GitHub Repository

1. Go to https://github.com/new
2. Repository name: `shelby-travels`
3. Make it **Public**
4. **Do NOT** initialize with README
5. Click "Create repository"

### Step 2: Push Code

```bash
cd /mnt/data/openclaw/workspace/.openclaw/workspace/website-shelby-travels

# Add remote (replace USERNAME with your GitHub username)
git remote add origin https://github.com/USERNAME/shelby-travels.git

# Push to GitHub
git push -u origin main
```

### Step 3: Enable GitHub Pages

1. Go to your repo: `https://github.com/USERNAME/shelby-travels`
2. Click **Settings** → **Pages**
3. Under "Source", select: **Deploy from a branch**
4. Branch: **main**
5. Folder: **/(root)**
6. Click **Save**

### Step 4: Get Your Live URL

After 1-2 minutes, your site will be live at:
```
https://USERNAME.github.io/shelby-travels/
```
