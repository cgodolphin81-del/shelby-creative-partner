## 🔧 TECHNICAL STACK

### **Recommended:**

**Option 1: VitePress (Vue-based)**
- ✅ Fast, modern
- ✅ Markdown-based
- ✅ Built-in search
- ✅ Dark mode
- ✅ Easy deployment
- Used by: Vite, Vue, many OSS projects

**Option 2: Docusaurus (React-based)**
- ✅ Mature, stable
- ✅ Great docs features
- ✅ Blog included
- ✅ Versioning
- Used by: Meta, many large projects

**Option 3: Next.js + MDX**
- ✅ Maximum flexibility
- ✅ React ecosystem
- ✅ Custom components
- ✅ Vercel deployment
- More work, more control

**Option 4: Astro**
- ✅ Fastest performance
- ✅ Islands architecture
- ✅ Markdown support
- ✅ Modern
- Growing popularity

### **Hosting:**
- Vercel (free for OSS)
- Netlify (free for OSS)
- Cloudflare Pages (free)
- Self-hosted (OpenClaw node!)

### **Search:**
- Algolia DocSearch (free for OSS)
- Pagefind (static, self-hosted)
- FlexSearch (client-side)

### **Analytics:**
- Plausible (privacy-focused)
- Fathom (simple, privacy)
- Google Analytics (traditional)
