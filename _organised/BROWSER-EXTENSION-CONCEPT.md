# 🌐 AI Content Creator Browser Extension
*Chrome Extension Concept*
*For: Content Creators using AI tools*

---

## 🎯 WHAT IT DOES

One-click access to all your AI tools while browsing:

- **Quick Script Generator** - Generate scripts for any topic
- **Auto-Caption** - Generate captions for any text
- **Voice Clip** - Instant TTS for selected text
- **Hashtag Finder** - Generate hashtags for any post
- **Content Ideas** - Get viral content ideas

## 📦 INSTALLATION

**Step 1: Load Extension**
```bash
# Clone repo
git clone https://github.com/your-repo/ai-content-creator

# Load in Chrome
1. Go to chrome://extensions/
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select extension folder
```

**Step 2: Configure**
```json
{
  "apiKey": "YOUR_OPENCLAW_KEY",
  "ollamaUrl": "http://localhost:11434",
  "ttsVoice": "en-US-AriaNeural"
}
```
