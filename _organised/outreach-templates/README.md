## How to Use

1. **Personalize placeholders** - Replace `[BRACKETED TEXT]` with specific details
2. **Research first** - Always research the prospect before sending
3. **Test and iterate** - Track response rates and refine messaging
4. **Timing matters** - Send emails Tue-Thu, 10am-2pm prospect timezone

## Objection Handler Quick Reference

See each vertical's section for specific objection handlers. Common themes:
- **Price objections** → ROI framing, payment plans, pilot programs
- **Time objections** → Done-for-you emphasis, minimal lift required
- **Trust objections** → Case studies, testimonials, guarantees
- **Competition objections** → Differentiation, unique methodology
