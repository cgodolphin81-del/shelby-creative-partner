## 🔧 OPENCLAW CONFIGURATION & SETUP

### **OpenClaw Tools Configuration** ⭐⭐⭐⭐⭐
**Source:** Steps to Enable Automation Tools

**Configuration File:** `~/.openclaw/openclaw.json`

**Enable All Tools:**
```json
{
  "tools": {
    "allow": [
      "group:fs",      // Filesystem (read, write, edit, apply_patch)
      "exec",          // Shell execution
      "process",       // Process management
      "browser"        // Browser automation
    ]
  }
}
```

**Security Notes:**
- Use `approval: "always"` for exec in production
- Run in sandboxed/isolated environment
- Use `openclaw doctor` to verify setup
- Install browser extension: `openclaw browser extension install`

**When:** Configure this weekend  
**Cost:** Free
