# 🎨 CV Preview Image Creation Guide

**Quick method using Canva (Free)**

---

## ⚡ FASTEST METHOD: Canva Templates

### **Step 1: Open Canva**
1. Go to https://canva.com
2. Sign in (free account is fine)
3. Search for "Resume Mockup" or "CV Preview"

### **Step 2: Choose a Template**
Search terms:
- "Resume mockup"
- "CV template preview"
- "Document mockup"
- "Paper mockup"

### **Step 3: For Each CV Template (15 total)**

**Create 7 Images Per Listing:**

#### **Image 1: Main Preview** (Most Important!)
- Full first page of CV
- Clean, professional
- Shows overall design
- **Size:** 2000 x 2667 px (3:4 ratio - Etsy standard)

**How:**
1. Upload your CV as PDF to Canva
2. Place on clean background
3. Add subtle shadow
4. Export as PNG

---

#### **Image 2: Header Detail**
- Close-up of name/contact section
- Shows typography quality
- **Size:** 2000 x 1500 px (4:3 ratio)

---

#### **Image 3: Experience Section**
- Shows work history layout
- Demonstrates bullet points
- **Size:** 2000 x 1500 px

---

#### **Image 4: Color Variations** (if applicable)
- Show 2-3 color options side-by-side
- **Size:** 2000 x 1500 px

---

#### **Image 5: Cover Letter** (for bundles)
- Matching cover letter design
- **Size:** 2000 x 2667 px

---

#### **Image 6: How to Edit**
- Simple 3-step instructions:
  1. "Download template"
  2. "Edit in Word/Google Docs"
  3. "Save and apply!"
- **Size:** 2000 x 2667 px

---

#### **Image 7: Features/Benefits**
- Bullet points:
  - ✅ ATS-Friendly
  - ✅ Instant Download
  - ✅ Easy to Edit
  - ✅ Professional Design
  - ✅ Free Fonts
- **Size:** 2000 x 2667 px

---

## 🎯 BATCH WORKFLOW (Do All 15 at Once)

### **Session 1: Main Images (1 hour)**
1. Open Canva
2. Create design (2000 x 2667 px)
3. Upload ALL 15 CVs as PDFs
4. Create main preview for each
5. Export all 15 at once

### **Session 2: Detail Shots (1 hour)**
1. Same Canva project
2. Create header close-ups
3. Create experience sections
4. Export all

### **Session 3: Info Graphics (30 min)**
1. Create "How to Edit" template
2. Create "Features" template
3. Reuse for all 15 listings
4. Export

**Total Time: 2.5 hours**

---

## 📸 CANVA SHORTCUTS

### **Use Canva's Built-in Mockups:**
1. Upload your CV PDF
2. Click "Edit image"
3. Select "Mockups"
4. Choose "Paper" or "Document"
5. Instant professional mockup!

### **Batch Export:**
1. Create one page per CV
2. Click "Share" → "Download"
3. Select "All pages"
4. Choose PNG format
5. Download all at once!

## 🎨 DESIGN TIPS

### **Colors That Convert:**
- Navy blue (#1E3A5F) - Trust, professional
- Teal (#2D8B8B) - Modern, fresh
- Burgundy (#800020) - Premium, executive
- Black/Gold - Luxury, high-end

### **Typography:**
- Use large, clear fonts
- Minimum 24pt for body text in mockups
- Headlines 40pt+

### **Backgrounds:**
- White or very light gray
- Subtle shadows for depth
- No busy patterns
