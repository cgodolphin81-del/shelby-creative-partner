## 📁 Folder Structure

| Folder | Purpose | Examples |
|--------|---------|----------|
| `01-QUALIFROST/` | Qualifrost business projects | Sales scripts, client work, deliverables |
| `02-COURSES/` | All courses we create | AI marketing, content creation, automation |
| `03-GUIDES/` | All guides & documentation | How-to guides, manuals, playbooks |
| `04-BUSINESS/` | Business plans & strategy | Business plans, financial models, pitches |
| `05-CREATIVE/` | Creative projects | The Infinite Walk, writing, media |
| `06-PLATFORMS/` | Platforms & apps | Websites, SaaS tools, digital products |
| `07-COMMUNICATION/` | Internal comms | Daily status, meeting notes, announcements |

## 📝 Required Files Per Project

Every project folder must include:
- `README.md` — Project overview
- `BUILD-COMPLETE.md` — Status & completion notes

## 🔗 Related Files

- `MIGRATION-TRACKER.md` — Track what's been migrated/created
- `MIGRATION-MANIFEST.md` — Full structure reference
- `CLEANUP-REPORT.md` — Cleanup & reorganization log
