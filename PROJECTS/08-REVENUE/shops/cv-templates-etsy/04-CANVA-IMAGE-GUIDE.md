# Canva Image Guide - Etsy CV Template Listings

**Tool:** Canva (Free version works perfectly!)  
**Goal:** Create professional listing images that convert browsers into buyers

---

## Image Specifications

### Main Image (First/Thumbnail)
- **Size:** 1000 x 1000 pixels (square)
- **Purpose:** Thumbnail in search results - must stand out!
- **Format:** PNG or JPG

### Additional Images (2-5)
- **Size:** 1000 x 1333 pixels (4:5 ratio)
- **Purpose:** Show details, features, examples
- **Format:** PNG or JPG

### Etsy Requirements:
- Minimum: 1 image (but use all 5!)
- Maximum: 5 images per listing
- File size: Under 10MB each
- Background: White or light recommended

## Canva Quick Start Guide

### Step 1: Create Account
- Go to canva.com
- Sign up for free account
- No payment needed for basic features

### Step 2: Create First Design
1. Click "Create a design"
2. Click "Custom size"
3. Enter 1000 x 1000 (or 1000 x 1333)
4. Click "Create new design"

### Step 3: Upload CV Template
1. Click "Uploads" in left sidebar
2. Click "Upload files"
3. Select your CV template PDF or screenshot
4. Wait for upload to complete

### Step 4: Add to Canvas
1. Click uploaded image
2. Drag onto canvas
3. Resize and position as needed
4. Right-click → "Set image as background" if desired

### Step 5: Add Text
1. Click "Text" in left sidebar
2. Click "Add a heading" or "Add a subheading"
3. Type your text
4. Adjust font, size, color using top toolbar
5. Drag to position

### Step 6: Add Elements (Optional)
1. Click "Elements" in left sidebar
2. Search for: "stars", "checkmark", "banner", "badge"
3. Click to add to canvas
4. Resize and position

### Step 7: Download
1. Click "Share" button (top right)
2. Click "Download"
3. Select file type: PNG (recommended) or JPG
4. Click "Download"
5. Save to your computer

## Pro Tips

1. **Consistency is Key:** Use same colors, fonts, and style across all 10 listings
2. **Mobile First:** Most Etsy browsing is on mobile - make sure text is large enough
3. **Test Readability:** Zoom out to 25% - can you still read the main text?
4. **Brand Your Images:** Add your shop name or logo to each image
5. **Use Templates:** Once you create one great image, duplicate and modify for others
6. **Save as Brand Kit:** In Canva, save your colors and fonts as a Brand Kit for quick access
7. **Batch Create:** Make all 5 images for one listing before moving to the next
