## 📋 Pre-Launch QC (Must Pass All)

### Basics
- [ ] Product solves a real pain point
- [ ] Target customer is clearly defined
- [ ] Price is competitive (researched 3+ competitors)
- [ ] No typos or errors

### Digital Products (Etsy/Gumroad)

**File Quality:**
- [ ] Files are properly formatted (.pdf, .docx, .notion, etc.)
- [ ] All links work (if applicable)
- [ ] Files open correctly on multiple devices
- [ ] Instructions included (if needed)

**Listing Quality:**
- [ ] Title is SEO-optimized (includes main keywords)
- [ ] Description is compelling (benefits, not just features)
- [ ] Tags are relevant (13 for Etsy, maxed out)
- [ ] Category is correct
- [ ] Price is set correctly
- [ ] Digital download is attached and working

**Images/Mockups:**
- [ ] Main image is eye-catching (1000x1000px minimum)
- [ ] 5+ images showing different angles/uses
- [ ] Images are high resolution
- [ ] Text on images is readable
- [ ] Mockups look professional

### Print-on-Demand Products

**Design Quality:**
- [ ] Design is high resolution (300 DPI minimum)
- [ ] Design is properly sized for product
- [ ] Colors are vibrant and accurate
- [ ] No pixelation or blurriness
- [ ] Design placement is correct

**Product Selection:**
- [ ] Product quality is good (ordered sample if unsure)
- [ ] Product colors match design intent
- [ ] Pricing allows for good profit margin (50%+ after costs)

**Listing Quality:**
- [ ] Title includes product type + design theme
- [ ] Description explains design + product details
- [ ] Size chart included (for apparel)
- [ ] Care instructions included
- [ ] Shipping times are clear

**Mockups:**
- [ ] Mockups show product in use (lifestyle shots)
- [ ] Multiple angles shown
- [ ] Mockups are realistic and professional

### Content Articles

**Content Quality:**
- [ ] Article is 1,500+ words (for SEO)
- [ ] Title is compelling + includes keyword
- [ ] Introduction hooks the reader
- [ ] Content is well-organized (headings, subheadings)
- [ ] Includes personal insights/examples (not just generic info)
- [ ] Conclusion includes call-to-action
- [ ] No grammatical errors

**SEO Optimization:**
- [ ] Primary keyword in title
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in H1 heading
- [ ] Secondary keywords used naturally
- [ ] Meta description written (150-160 chars)
- [ ] Internal links included (2-3)
- [ ] External links to authoritative sources (1-2)
- [ ] Image alt text includes keywords

**Readability:**
- [ ] Short paragraphs (2-4 sentences)
- [ ] Bullet points and numbered lists used
- [ ] Images/screenshots included (1 per 300 words)
- [ ] Flesch Reading Ease score: 60+

**Monetization:**
- [ ] Affiliate links included (where relevant)
- [ ] Product mentions (our own products, if relevant)
- [ ] Email signup CTA included
- [ ] Social share buttons enabled

### Social Media Content

**Content Quality:**
- [ ] Hook in first 3 seconds (video) or first line (text)
- [ ] Clear value proposition
- [ ] Call-to-action included
- [ ] Branding consistent (colors, fonts, voice)

**Platform Optimization:**
- [ ] Correct dimensions for platform
- [ ] Hashtags included (platform-appropriate number)
- [ ] Caption is engaging
- [ ] Posting time optimized for audience
