**Created:** March 25, 2026  
**Purpose:** Create high-converting Etsy listing images  
**Tool:** Canva (free or Pro)

---

## 📐 ETSY IMAGE REQUIREMENTS

### Required Specifications
- **Size:** 1000 x 1000 pixels (square)
- **Format:** PNG or JPG
- **Minimum:** 72 DPI (150+ DPI recommended)
- **First image:** Most important (thumbnail in search)
- **Max images:** 10 per listing

### Etsy Image Order Strategy
1. **Main Hero Image** - Template preview + key benefits
2. **Full Template View** - Complete CV layout
3. **Features/Benefits** - What's included
4. **Use Cases** - Who it's for
5. **Before/After** - Customization example
6-10. **Additional previews, variations, testimonials**

```

│  ✓ Instant Download     │
│                         │
│  $12.99  ⭐ INSTANT     │  ← Price + CTA
└─────────────────────────┘
```

**Text Elements:**
- Main headline: "PROFESSIONAL CV TEMPLATE" or "EXECUTIVE RESUME"

**Canva Tips:**
- Search "minimalist product showcase" templates
- Use frames for template preview
- Add subtle shadow under preview
- Keep text large and readable on mobile

## 🛠️ CANVA SETUP GUIDE

### Step 1: Create Account
1. Go to canva.com
2. Sign up (free account works fine)
3. Upgrade to Pro if you want premium elements (optional)

### Step 2: Set Custom Dimensions
1. Click "Create a design"
2. Select "Custom size"
3. Enter: **1000 x 1000 pixels**
4. Click "Create new design"

### Step 3: Choose Template (Optional)
Search for:
- "Product showcase"
- "Minimalist presentation"
- "Social media post square"
- "Etsy listing"

### Step 4: Design Your Images
Follow the 5 image concepts above, or use these quick templates:

**Quick Template Search Terms in Canva:**
- "Minimalist product"
- "Clean presentation"
- "Professional service"
- "Digital product"
- "Template showcase"

### Step 5: Export
1. Click "Share" → "Download"
2. Select **PNG** (best quality) or **JPG**
3. Quality: 100%
4. Click "Download"

## 🎨 DESIGN BEST PRACTICES

### Do's ✅
- Use high contrast for readability
- Keep text large (minimum 24pt for body, 40pt+ for headers)
- Maintain consistent branding across all images
- Use whitespace effectively
- Show actual template previews
- Include price in at least one image
- Use arrows and callouts to highlight features
- Keep mobile viewing in mind (most Etsy browsing is mobile)

### Don'ts ❌
- Don't clutter with too much text
- Don't use low-resolution images
- Don't use copyrighted fonts or graphics
- Don't make text too small to read on mobile
- Don't hide important information
- Don't use more than 3 fonts per image
- Don't ignore Etsy's image guidelines

## 📱 MOBILE OPTIMIZATION

**60%+ of Etsy browsing happens on mobile!**

### Mobile Checklist:
- [ ] Text readable at 50% zoom
- [ ] Main message visible in thumbnail crop
- [ ] Price visible without clicking
- [ ] Key benefits scannable in 3 seconds
- [ ] No tiny text or fine details
- [ ] High contrast colors
- [ ] Simple, uncluttered layouts

### Test Your Images:
1. Download images to your phone
2. View at thumbnail size
3. Can you read the main message?
4. Is it clear what's being sold?
5. Would you click on this?

## ⚡ QUICK CANVA WORKFLOW

### Batch Create All 5 Images for One Template:
1. **Create first image** (Hero) - 10-15 minutes
2. **Duplicate the page** - Click "Duplicate page"
3. **Modify for Image 2** - Change content, keep style - 5-7 minutes
4. **Repeat** for images 3-5 - 5-7 minutes each
5. **Download all** - Export as PNG

**Total time per template:** ~30-40 minutes
**Total time for 10 templates:** 5-7 hours

### Batch Create for Multiple Templates:
1. Design master template style
2. Duplicate and customize for each template
3. Swap preview images and text
4. Export all at once

## 📊 IMAGE PERFORMANCE TIPS

### A/B Test Your First Image:
- Try different headlines
- Test price display vs. no price
- Compare template preview styles
- Check which gets more clicks

### Update Based on Data:
- If views are low → Improve first image
- If clicks but no sales → Improve description/pricing
- If favorites but no sales → Consider price adjustment

### Seasonal Updates:
- "2026 Resume" → Update to "2027 Resume" in January
- Add holiday promotions
- Refresh with new testimonials
