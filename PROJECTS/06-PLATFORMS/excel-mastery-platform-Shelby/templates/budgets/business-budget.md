### Key Formulas

```excel
// Gross Profit (Row 10)
=B5-B9

// Gross Margin % (Row 11)
=IF(B5>0,B10/B5,0)

// Total Operating Expenses (Row 24)
=SUM(B13:B23)

// Operating Income (Row 26)
=B10-B24

// Net Income (Row 32)
=B26+B28-B29-B30

// Net Margin % (Row 33)
=IF(B5>0,B32/B5,0)

// Annual Total (Column M)
=SUM(B2:L2)
```

```excel

=IF(ABS(D2/C2)<0.1,"On Track",IF(D2<0,"Over Budget","Under Budget"))

// Highlight Over Budget (>10%)
=AND(D2/C2<-0.1,C2>0)
```

## KPI Dashboard Metrics

| Metric | Formula | Target |
|--------|---------|--------|
| Gross Margin | Gross Profit / Revenue | >40% |
| Operating Margin | Operating Income / Revenue | >15% |
| Net Margin | Net Income / Revenue | >10% |
| Revenue Growth | (Current - Previous) / Previous | >10% YoY |
| Expense Ratio | Total Expenses / Revenue | <70% |

## Usage Instructions

1. Input historical data for baseline
2. Set realistic monthly targets
3. Update actuals weekly
4. Review variance reports monthly
5. Adjust forecasts based on trends
