**Prerequisites**: Beginner Certification or equivalent knowledge

- What-if analysis
- Basic macros

### 🥇 Expert Certification
**Prerequisites**: Intermediate Certification  
**Exam Duration**: 90 minutes  
**Questions**: 40 multiple choice + 3 practical exercises + 1 case study  
**Passing Score**: 85%  
**Valid For**: 3 years (renewable with continuing education)  

**Skills Validated**:
- Complex nested formulas
- Array formulas and dynamic arrays
- Power Query and data modeling
- Advanced dashboard design
- VBA/Apps Script programming
- Optimization and best practices

**Exam Topics**:
| Topic | % of Exam |
|-------|-----------|
| Advanced Formulas | 25% |
| Data Modeling | 20% |
| Dashboard Design | 20% |
