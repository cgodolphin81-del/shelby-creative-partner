## 📖 Lesson Content

### 1.1 What is a Spreadsheet?

A **spreadsheet** is a digital tool for organizing, analyzing, and storing data in a tabular format.

**Common Uses**:
- Budget tracking
- Data analysis
- Inventory management
- Project planning
- Grade books
- Business reports

**Key Benefits**:
- Automatic calculations
- Easy data organization
- Visual representations (charts)
- Data sorting and filtering
- Collaboration capabilities

---

### 1.2 Excel vs. Google Sheets

| Feature | Microsoft Excel | Google Sheets |
|---------|-----------------|---------------|
| **Platform** | Desktop application | Web-based |
| **Cost** | Paid (Office 365) | Free |
| **Collaboration** | Good | Excellent |
| **Offline Access** | Yes | Limited |
| **Advanced Features** | More powerful | Simpler |
| **Storage** | Local/OneDrive | Google Drive |
| **Best For** | Complex analysis | Collaboration |

**Which Should You Use?**
- **Choose Excel** if: You need advanced features, work offline, handle large datasets
- **Choose Sheets** if: You collaborate frequently, need free access, work across devices

**Key Terms**:
- **Workbook**: The entire file
- **Worksheet/Sheet**: Individual tabs within a workbook
- **Cell**: Intersection of a row and column (e.g., A1)
- **Row**: Horizontal line of cells (numbered 1, 2, 3...)
- **Column**: Vertical line of cells (lettered A, B, C...)
- **Range**: Group of cells (e.g., A1:B10)
- **Formula Bar**: Where you enter/edit formulas

### 1.4 Creating Your First Spreadsheet

#### In Excel:
1. Open Excel
2. Click "Blank Workbook"
3. Save: File > Save As > Choose location > Name file > Save

#### In Google Sheets:
1. Go to sheets.google.com
2. Click "+ Blank" or "+" button
3. Auto-saves to Google Drive
4. Rename: Click "Untitled spreadsheet" at top

---

### 1.5 Basic Navigation

| Action | Excel Shortcut | Google Sheets Shortcut |
|--------|---------------|----------------------|
| Move right | Tab | Tab |
| Move left | Shift+Tab | Shift+Tab |
| Move down | Enter | Enter |
| Move up | Shift+Enter | Shift+Enter |
| Go to cell | Ctrl+G | Ctrl+G |
| Go to start | Ctrl+Home | Ctrl+Home |
| Go to end | Ctrl+End | Ctrl+End |
| Next sheet | Ctrl+PgDn | Ctrl+PgDn |
| Previous sheet | Ctrl+PgUp | Ctrl+PgUp |

---

### 1.6 Entering Data

**Types of Data**:
1. **Text**: Names, labels, descriptions
2. **Numbers**: Values, amounts, quantities
3. **Dates**: 1/1/2026, Jan-26, etc.
4. **Formulas**: Start with = (e.g., =A1+B1)

**Tips**:
- Press Enter to move down
- Press Tab to move right
- Double-click cell to edit
- Press F2 to edit cell
- Press Esc to cancel edit
