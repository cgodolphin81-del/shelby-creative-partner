#### Color Usage Guidelines

**Do:**
- Use purple as the dominant brand color
- Maintain 4.5:1 contrast ratio for accessibility
- Use coral sparingly for CTAs and important actions
- Use teal for success/positive messages

**Don't:**
- Use more than 3 colors in a single composition
- Place text over busy backgrounds
- Use colors that clash with the palette
- Change logo colors (except monochrome versions)

### Typography

#### Primary Font: Inter

**Usage:** All digital interfaces, web, presentations

```
Inter Bold (700)     — Headlines, H1-H3
Inter SemiBold (600) — Subheadings, H4-H6
Inter Medium (500)   — Buttons, labels
Inter Regular (400)  — Body text
Inter Light (300)    — Large display text
```

**Web Font:**
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

#### Secondary Font: Space Grotesk

**Usage:** Logo, special headlines, creative projects

```
Space Grotesk Bold (700)   — Special headlines
Space Grotesk Medium (500) — Accent text
```

#### Typography Hierarchy

| Element | Font | Size | Weight | Line Height |
|---------|------|------|--------|-------------|
| H1 | Inter | 48px | 700 | 1.2 |
| H2 | Inter | 36px | 700 | 1.3 |
| H3 | Inter | 28px | 600 | 1.3 |
| H4 | Inter | 24px | 600 | 1.4 |
| H5 | Inter | 20px | 600 | 1.4 |
| H6 | Inter | 18px | 600 | 1.5 |
| Body | Inter | 16px | 400 | 1.6 |
| Small | Inter | 14px | 400 | 1.5 |
| Caption | Inter | 12px | 400 | 1.4 |

#### Typography Guidelines

**Do:**
- Maintain consistent hierarchy
- Use maximum 2 fonts per design
- Ensure readable line lengths (50-75 characters)
- Use appropriate line height for readability

**Don't:**
- Use all caps for long text
- Stretch or distort fonts
- Use light fonts on dark backgrounds (reduced readability)
- Mix too many font weights

## 📝 Brand Voice

### Tone & Voice

**We sound:**

| Quality | Description | Example |
|---------|-------------|---------|
| **Confident** | Know our stuff, no hedging | "We deliver studio-quality animation" |
| **Friendly** | Warm, approachable | "Let's create something amazing together" |
| **Clear** | Simple, jargon-free | "Your video will be ready in 10 days" |
| **Enthusiastic** | Excited about possibilities | "This is going to look incredible!" |
| **Professional** | Reliable, trustworthy | "Here's what happens next..." |

### Writing Guidelines

#### Do:
- Use active voice
- Address the reader as "you"
- Keep sentences concise (15-20 words avg)
- Use contractions (we're, you'll, let's)
- Lead with benefits, not features
- Be specific with numbers and timelines

#### Don't:
- Use industry jargon without explanation
- Sound robotic or overly formal
- Make promises we can't keep
- Use passive voice excessively
- Write walls of text

## 🎬 Content Guidelines

### Video Content

#### Showreel Style
- **Length:** 60-90 seconds
- **Music:** Upbeat, modern, non-intrusive
- **Pacing:** Quick cuts (2-4 seconds each)
- **Text overlays:** Minimal, brand fonts
- **End card:** Logo + CTA + website

#### Tutorial Videos
- **Intro:** 5 seconds (logo + topic)
- **Content:** Clear, step-by-step
- **Voice:** Friendly, expert
- **Outro:** CTA + subscribe

#### Client Testimonials
- **Format:** Talking head + B-roll
- **Length:** 30-60 seconds
- **Key message:** Problem → Solution → Result
- **Include:** Name, title, company

### Presentations

#### Template Structure

1. **Title Slide**
   - Logo
   - Presentation title
   - Date

2. **Agenda**
   - 3-5 bullet points

3. **Content Slides**
   - One idea per slide
   - Visual > text
   - Brand colors

4. **Case Study Slides**
   - Problem
   - Solution
   - Results (with numbers)

5. **Pricing Slide**
   - Clear tiers
   - What's included

6. **Next Steps**
   - Timeline
   - Action items

7. **Contact Slide**
   - Logo
   - Contact info
   - CTA

#### Design Rules
- Use brand fonts only
- Maintain consistent margins
- High-contrast text
- Maximum 6 lines per slide
- Include logo on every slide (small, corner)

## 🚫 Brand Misuse

### Logo Misuse

**Don't:**
- Stretch or distort the logo
- Change the colors
- Add effects (shadows, gradients, outlines)
- Place on busy backgrounds
- Use outdated versions
- Recreate the logo from scratch

### Color Misuse

**Don't:**
- Use non-brand colors
- Create new color combinations
- Use low-contrast text
- Apply filters that change brand colors

### Voice Misuse

**Don't:**
- Sound overly corporate or stiff
- Use excessive jargon
- Make exaggerated claims
- Sound robotic or AI-generated
- Be negative about competitors
