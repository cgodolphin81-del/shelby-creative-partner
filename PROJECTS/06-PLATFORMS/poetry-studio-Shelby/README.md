# 🎭 AI Poetry Studio

A complete poetry generation platform with voice narration, visual cards, print-on-demand, and monetization.

## 📁 Project Structure

```
poetry-studio/
├── generators/          # AI poem generation (haiku, sonnet, free verse, rap)
├── voice/               # ElevenLabs voice narration system
├── visuals/             # Canva templates for Instagram cards
├── print/               # Printful POD integration
├── pricing/             # Pricing strategy & tiers
├── website/             # Landing page & checkout
├── social/              # 30 social media posts
├── email/               # Capture & nurture sequences
├── collaboration/       # Custom poem system (events, weddings)
├── financials/          # Revenue projections
└── README.md            # This file
```

## 🚀 Quick Start

1. **Generate Poems**: Use `/generators/poem-generator.py`
2. **Add Voice**: Run `/voice/narrate.py` with ElevenLabs API
3. **Create Cards**: Use Canva templates in `/visuals/`
4. **Sell Prints**: Connect Printful via `/print/integration.md`
5. **Launch Site**: Deploy `/website/landing.html`

## 💰 Business Model

- **Free Tier**: 5 poems/day
- **Premium**: $9.99/month unlimited
- **Prints**: $19-49 (posters, books)
- **Custom**: $50-500 (events, weddings, memorials)
