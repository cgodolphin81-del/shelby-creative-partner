## 🛠️ CANVA AUTOMATION

### Using Canva API (Bulk Creation)

```python
# Canva API integration for bulk card creation
import requests

CANVA_API_KEY = "your-api-key"
CANVA_TEAM_ID = "your-team-id"

def create_poetry_card(poem_text, style, template_id):
    """Create poetry card using Canva API"""
    
    url = "https://api.canva.com/v1/designs"
    
    payload = {
        "templateId": template_id,
        "elements": {
            "poem_text": {"text": poem_text},
            "style_tag": {"text": style},
            "handle": {"text": "@aipoetrystudio"}
        }
    }
    
    headers = {
        "Authorization": f"Bearer {CANVA_API_KEY}",
        "Content-Type": "application/json"
    }
    
    response = requests.post(url, json=payload, headers=headers)
    return response.json()
```
