### Remote Recording Best Practices

1. **Send guests a tech guide** before recording
2. **Require wired connection** (no WiFi if possible)
3. **Use headphones** (prevent echo)
4. **Record local backup** (ask guest to record locally)
5. **Test before recording** (5-minute sound check)
6. **Close all other apps** (reduce CPU load)

## 🏠 STUDIO SETUP

### Room Treatment (Budget: $100-300)

1. **Bass traps** (corners) - $50-100
2. **Acoustic panels** (first reflection points) - $50-100
3. **Carpet/rug** (reduce floor reflection) - $30-50
4. **Heavy curtains** (cover windows) - $40-80
5. **Bookshelves** (diffusion) - existing furniture

### Room Treatment (Professional: $500-2000)

1. **Professional bass traps** - $200-400
2. **Acoustic panels (8-12)** - $200-500
3. **Diffusers** - $150-300
4. **Floating floor** (optional) - $300-800
5. **Sound isolation booth** - $500-1000

### DIY Treatment Tips

- **Blankets work!** Heavy moving blankets reduce echo
- **Closets are great** - clothes = natural sound treatment
- **Mattresses behind you** - reduces wall reflection
- **Carpets on walls** - temporary acoustic treatment
- **Bookshelves** - natural diffusion

## 💻 COMPUTER REQUIREMENTS

### Minimum Specs (Basic Recording)

- **CPU:** Intel i5 / AMD Ryzen 5 (8th gen+)
- **RAM:** 8GB
- **Storage:** 256GB SSD
- **OS:** Windows 10 / macOS 10.15+

### Recommended Specs (Professional)

- **CPU:** Intel i7 / AMD Ryzen 7 (10th gen+)
- **RAM:** 16-32GB
- **Storage:** 512GB+ SSD + external drive
- **OS:** Windows 11 / macOS 12+

### Storage Needs

- **Raw audio:** ~500MB per hour (WAV)
- **Project files:** 1-5GB per episode
- **Music/SFX library:** 10-50GB
- **Recommendation:** 1TB external SSD for archives

## 🔧 MAINTENANCE

### Regular Maintenance

- **Clean microphone** monthly (soft cloth)
- **Replace pop filters** every 6-12 months
- **Check cables** for wear quarterly
- **Update software** monthly
- **Backup recordings** after each session

### Troubleshooting Common Issues

| Problem | Solution |
|---------|----------|
| Hum/buzz | Check ground loop, use different outlet |
| Hissing | Increase gain at interface, not software |
| Plosives | Adjust mic angle, use pop filter |
| Echo | Add room treatment, move closer to mic |
| Clipping | Reduce gain, watch levels (-12 to -6dB) |
| Latency | Reduce buffer size, close other apps |
