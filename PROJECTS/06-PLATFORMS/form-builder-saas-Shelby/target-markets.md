# FormFlow Target Markets

## Market Overview

The online form builder market is valued at $3.5B (2024) and growing at 12% CAGR. FormFlow targets four primary market segments with tailored messaging and features.

---

## 1. Small & Medium Businesses (SMBs)

### Profile
- **Company Size:** 1-100 employees
- **Revenue:** $500K - $50M annually
- **Decision Maker:** Owner, Operations Manager, Marketing Manager
- **Technical Skill:** Low to moderate
- **Budget:** $25-100/month for tools

### Pain Points
- Forms are ugly and don't match brand
- Too complicated to set up
- Can't collect payments easily
- Data ends up in spreadsheets manually
- No integrations with existing tools
- Expensive enterprise solutions

### Use Cases
| Use Case | Description | Value Prop |
|----------|-------------|------------|
| Contact Forms | Website lead capture | Beautiful, high-converting forms |
| Order Forms | Product/service orders | Built-in payments, no coding |
| Quote Requests | Service pricing inquiries | Conditional logic for accurate quotes |
| Booking Forms | Appointment scheduling | Calendar integration |
| Feedback Forms | Customer satisfaction | Easy analysis and reporting |
| Registration Forms | Events, webinars | Automated confirmations |

### Messaging
> "Create beautiful forms that convert visitors into customers. No coding required. Start collecting payments today."

### Acquisition Channels
- Google Ads (form builder keywords)
- Facebook/Instagram ads
- Content marketing (SEO)
- Shopify App Store
- WordPress plugin directory
- Small business forums

### Pricing Sensitivity
- **High** - Price is a key decision factor
- **Sweet Spot:** $25-50/month
- **Preferred:** Monthly billing, easy cancellation

## 2. Digital Agencies

### Profile
- **Company Size:** 5-50 employees
- **Services:** Web design, marketing, development
- **Decision Maker:** Agency Owner, Technical Director
- **Technical Skill:** High
- **Budget:** $100-500/month (billable to clients)

### Pain Points
- Managing forms for multiple clients
- Client wants to edit forms themselves
- Need white-label solution
- Forms break when clients touch them
- No centralized management
- Hard to track form performance across clients

### Use Cases
| Use Case | Description | Value Prop |
|----------|-------------|------------|
| Client Projects | Forms for client websites | White-label, client editing access |
| Lead Gen Campaigns | Landing page forms | High conversion, analytics |
| Internal Forms | Team processes | Templates, standardization |
| Client Reporting | Performance dashboards | Shareable analytics |
| Onboarding Forms | New client intake | Automated workflows |

### Messaging
> "Manage forms for all your clients in one place. White-label branding, client access controls, and centralized analytics. Bill clients for form usage."

### Acquisition Channels
- Agency partnerships program
- Industry conferences (Smashing Conf, etc.)
- Agency-focused content
- Referral program
- LinkedIn advertising
- Web design communities

### Pricing Sensitivity
- **Medium** - Value features over price
- **Sweet Spot:** $50-200/month (Professional + add-ons)
- **Preferred:** Annual billing, agency discounts

### Agency Program
- **Agency Partner:** 20% commission on referrals
- **White Label:** Resell FormFlow under your brand
- **Bulk Discount:** 30% off for 10+ seats
- **Dedicated Support:** Agency success manager

- **Size:** 50-10,000+ students
- **Decision Maker:** IT Director, Administrator, Teacher

- **Budget:** Varies (grant-funded, departmental)

### Pain Points

- Need FERPA compliance
- Budget constraints
- Multiple stakeholders
- Accessibility requirements (WCAG)
- Integration with student systems

| Feedback Forms | Course evaluations | Anonymous responses, analytics |

| Event Registration | School events, sports | Payment collection |

- Grant writing assistance
- Education partnerships
- Teacher communities

## 4. Healthcare & Wellness

### Profile
- **Types:** Clinics, private practices, telehealth, wellness centers
- **Size:** 1-100 providers
- **Decision Maker:** Practice Manager, Provider
- **Technical Skill:** Low to moderate
- **Budget:** $100-500/month

### Pain Points
- HIPAA compliance is complex
- Patient intake is paper-based
- Need digital signatures
- Insurance verification
- Appointment scheduling
- Payment collection

### Use Cases
| Use Case | Description | Value Prop |
|----------|-------------|------------|
| Patient Intake | New patient forms | HIPAA-compliant, digital signatures |
| Consent Forms | Treatment consent | Legally binding e-signatures |
| Symptom Checkers | Pre-visit questionnaires | Conditional logic |
| Feedback Forms | Patient satisfaction | Anonymous responses |
| Insurance Forms | Verification, updates | Secure data handling |
| Telehealth Forms | Virtual visit prep | Integrated with video platforms |

### Messaging
> "HIPAA-compliant forms for healthcare practices. Secure patient intake, digital consent forms, and integrated payments. BAA included."

### Acquisition Channels
- Healthcare conferences (HIMSS, etc.)
- Medical practice management software
- Healthcare publications
- Direct sales
- EHR/EMR integrations
- Medical associations

### Pricing Sensitivity
- **Low-Medium** - Compliance is worth paying for
- **Sweet Spot:** $100-300/month (Professional + HIPAA)
- **Preferred:** Annual contracts, clear compliance docs

### Healthcare Program
- **HIPAA Compliance:** BAA included
- **Enhanced Security:** Audit logs, encryption
- **EHR Integrations:** Epic, Cerner, etc. (Enterprise)
- **Priority Support:** Healthcare-trained team
- **Compliance Documentation:** Ready for audits
