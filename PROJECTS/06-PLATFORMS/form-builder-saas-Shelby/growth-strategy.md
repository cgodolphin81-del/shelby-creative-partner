## 1. Freemium Strategy

### Free Tier Design
**Goal:** Get users to experience core value, then hit natural limits.

**Generous Enough To:**
- Build unlimited forms
- Test all features
- Get real value
- Share forms publicly

**Limited Enough To:**
- Hit submission limit (100/mo) at ~500 form views
- See FormFlow branding
- Lack integrations
- Need upgrade for payments

### Conversion Triggers
| Trigger | Timing | Message | Conversion Rate |
|---------|--------|---------|-----------------|
| Submission limit warning | 80 submissions | "You're almost at your limit" | 8% |
| Submission limit reached | 100 submissions | "Upgrade to continue collecting" | 15% |
| Branding removal prompt | After 10 form views | "Remove FormFlow branding" | 5% |
| Payment feature access | When adding payment | "Upgrade to collect payments" | 12% |
| Integration attempt | When clicking integrations | "Upgrade for integrations" | 10% |

### Upgrade Paths
```
Free User
    │
    ├──→ Starter ($25) - Most common, self-serve
    │       └──→ Professional ($50) - After team growth
    │               └──→ Enterprise ($299+) - Sales-assisted
    │
    └──→ Professional ($50) - Direct, power users
            └──→ Enterprise ($299+) - Sales-assisted
```

### Freemium Metrics
- **Free → Paid Conversion:** Target 5-8%
- **Time to Upgrade:** Median 21 days
- **Free User Retention (30-day):** Target 40%
- **Paid User Retention (annual):** Target 85%

## 2. Viral Growth Loops

### Embedded Forms (Primary Viral Loop)

**Mechanism:**
```
User creates form → Embeds on website → Visitors see form → 
Some visitors sign up → Create their own forms → Embed → Repeat
```

**Viral Coefficient Target:** 0.3-0.5
- Each user brings 0.3-0.5 new users through form embeds

**Optimization Tactics:**
1. **"Powered by FormFlow" badge** on free tier (clickable)
2. **Form footer** with subtle branding
3. **Shareable results pages** (quizzes, surveys)
4. **Email signatures** on notifications
5. **Social sharing** buttons on thank you pages

### Viral Features

#### Quiz Results Sharing
- Personality quiz results with share buttons
- "Create your own quiz" CTA on results
- Branded result images for social media

#### Survey Results Pages
- Public results pages (optional)
- "Take this survey" link
- "Create your own survey" CTA

#### Form Templates Gallery
- Public template showcase
- "Use this template" → signup required
- Template creator attribution

#### Referral Program
```
Refer a friend → Both get 1 month free (Professional)
Refer 5 friends → Get 6 months free
Refer 10 friends → Get 1 year free
```

**Referral Mechanics:**
- Unique referral link
- Dashboard tracking
- Automated rewards
- Email reminders

## 6. Product-Led Growth Tactics

### In-App Upgrade Prompts

**Contextual Triggers:**
1. **Usage-based:** "You've used 80% of your submissions"
2. **Feature-gating:** "Upgrade to use this feature"
3. **Success-moment:** "Love FormFlow? Upgrade for more"
4. **Pain-point:** "Tired of FormFlow branding?"

**Prompt Design:**
- Non-intrusive
- Value-focused messaging
- Clear upgrade path
- Easy dismissal

### Onboarding Optimization

**Goal:** Get users to "aha moment" quickly

**Aha Moment Definition:** User creates and publishes their first form

**Onboarding Flow:**
1. Sign up (email only, no credit card)
2. Choose template or start blank
3. Quick tutorial (optional, skippable)
4. Customize form
5. Publish and share
6. First submission celebration 🎉

**Metrics:**
- Time to first form: Target < 5 minutes
- Activation rate: Target > 60%
- Day 7 retention: Target > 40%

### Email Nurturing

**Free User Sequence:**
```
Day 0: Welcome + get started
Day 1: Template suggestions
Day 3: Feature highlight (logic jumps)
Day 7: Success stories
Day 14: Upgrade offer (20% off first month)
Day 21: Last chance reminder
Day 30: Win-back offer
```

**Trial User Sequence:**
```
Day 0: Trial started + setup guide
Day 2: Feature deep-dive
Day 5: Integration showcase
Day 10: Upgrade reminder
Day 13: Last day reminder
Day 14: Trial ended + special offer
```
