## Target Platforms & Endpoints

### 1. Google AI Overviews

| Property | Value |
|----------|-------|
| **Base URL** | `https://www.google.com/search` |
| **Trigger** | Queries that trigger AI Overview (question-based, complex) |
| **Detection** | Look for `div[data-sokro]` or `.AIOverview` container |
| **Frequency** | Daily for tracked keywords |
| **Priority** | CRITICAL |

**Data Fields to Extract:**

```typescript
interface GoogleAIOverview {
  // Metadata
  query: string;                    // Search query
  timestamp: Date;                  // Crawl timestamp
  location: string;                 // Geo location (US, UK, etc.)
  language: string;                 // Language code (en, es, etc.)
  device: 'desktop' | 'mobile';     // Device type
  
  // AI Overview Content
  overview_text: string;            // Full AI-generated answer
  overview_html: string;            // Raw HTML for parsing
  
  // Citations/Sources
  citations: Array<{
    position: number;               // 1, 2, 3...
    title: string;                  // Page title
    url: string;                    // Source URL
    domain: string;                 // Extracted domain
    snippet: string;                // Citation snippet
    favicon?: string;               // Site favicon URL
  }>;
  
  // SERP Context
  organic_results_count: number;    // Number of organic results
  has_ai_overview: boolean;         // Whether AI Overview appeared
  ai_overview_position: 'top' | 'middle' | 'bottom';
  
  // Our Site Tracking
  our_site_cited: boolean;          // Is our tracked site in citations?
  our_site_position?: number;       // Position if cited (1-indexed)
  our_site_snippet?: string;        // Snippet if cited
}
```

**Crawl Parameters:**

```yaml
headers:
  User-Agent: "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
  Accept-Language: "en-US,en;q=0.9"
  
query_params:
  q: "{KEYWORD}"                    # The search query
  gl: "us"                          # Country code
  hl: "en"                          # Language
  udm: "14"                         # Force AI Overview (experimental)

pinchtab_settings:
  stealth: true
  wait_for_selector: "div[data-sokro]"
  timeout_ms: 30000
  viewport: { width: 1920, height: 1080 }
```

## Data Extraction Pipeline

### Step 1: Navigate & Wait

```python
async def crawl_page(pinchtap_client, url, wait_selector):
    # Navigate
    await pinchtab.navigate(url)
    
    # Wait for AI content to load
    await pinchtab.wait_for_selector(wait_selector, timeout=30000)
    
    # Additional wait for dynamic content
    await asyncio.sleep(3)
    
    return True
```

### Step 2: Extract Accessibility Tree

```python
async def extract_content(pinchtap_client):
    # Get filtered accessibility tree (token-efficient)
    snapshot = await pinchtab.snapshot(interactive=True, refs="aria")
    
    # Extract text content
    text = await pinchtab.text()
    
    # Get full HTML for citation parsing
    html = await pinchtab.evaluate("document.documentElement.outerHTML")
    
    return snapshot, text, html
```

### Step 3: Parse Citations

```python
def parse_citations(html: str, platform: str) -> List[Citation]:
    citations = []
    
    if platform == "google":
        # Parse Google AI Overview citations
        soup = BeautifulSoup(html, 'html.parser')
        citation_divs = soup.select('div[data-ved] a[href]')
        
        for i, div in enumerate(citation_divs):
            citations.append(Citation(
                position=i + 1,
                title=div.get_text(strip=True),
                url=div['href'],
                domain=extract_domain(div['href']),
            ))
    
    elif platform == "perplexity":
        # Parse Perplexity source items
        soup = BeautifulSoup(html, 'html.parser')
        source_items = soup.select('.source-item')
        
        for i, item in enumerate(source_items):
            citations.append(Citation(
                position=i + 1,
                title=item.select_one('.source-title')?.get_text(strip=True),
                url=item.select_one('a')['href'],
                domain=extract_domain(item.select_one('a')['href']),
            ))
    
    return citations
```

### Step 4: Generate Embeddings

```python
async def generate_embedding(text: str) -> List[float]:
    """Generate embedding for semantic search."""
    response = await openai.embeddings.create(
        model="text-embedding-3-small",
        input=text[:8191],  # Token limit
    )
    return response.data[0].embedding
```

### Step 5: Store Results

```python
async def store_crawl_result(result: CrawlResult):
    # Store in PostgreSQL
    await db.crawl_results.insert(result)
    
    # Store in Weaviate for semantic search
    await weaviate.data.create(
        class_name="CrawlResult",
        vector=result.embedding,
        properties={
            "query": result.query,
            "platform": result.platform,
            "overview_text": result.overview_text,
            "timestamp": result.timestamp.isoformat(),
            "citations_json": json.dumps(result.citations),
        }
    )
```

## Error Handling

| Error Type | Handling Strategy |
|------------|------------------|
| **Timeout** | Retry 2x with exponential backoff |
| **CAPTCHA** | Pause crawls for 1 hour, alert admin |
| **Rate Limited (429)** | Backoff 15 min, reduce rate |
| **Selector Not Found** | Update parser, flag for review |
| **Network Error** | Retry 3x, then mark as failed |
| **Auth Required** | Skip, flag for manual crawl |
