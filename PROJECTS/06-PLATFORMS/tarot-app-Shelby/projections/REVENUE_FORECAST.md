## Unit Economics

### Customer Acquisition Cost (CAC)
```
Marketing Spend / New Paying Customers

Month 1-6: $20,000 / 1,875 = $10.67 per paying user
Month 7-12: $15,000 / 2,775 = $5.41 per paying user

Blended CAC (Year 1): ~$7.50 per paying user
```

### Lifetime Value (LTV)
```
Average Revenue per User (ARPU): $7.50/month
Average Lifespan: 12.5 months (1 / 8% churn)
LTV = $7.50 × 12.5 = $93.75

LTV:CAC Ratio = $93.75 / $7.50 = 12.5:1
(Healthy! Target is 3:1 or better)
```

### Contribution Margin
```
Revenue per Premium User: $7.99/month
Variable Cost per User:
  - AI Processing: $0.40 (8 readings × $0.05)
  - Payment Fees: $0.24 (3%)
  - Support: $0.50 (allocated)
Total Variable: $1.14

Contribution Margin: $7.99 - $1.14 = $6.85 per user/month
Margin %: 86% (excellent for SaaS)
```

## Risks & Mitigation

### Risk 1: Low Conversion Rate
**Risk**: Free users don't upgrade to premium
**Mitigation**: 
- A/B test paywall messaging
- Extend trial period (7 → 14 days)
- Add more premium features

### Risk 2: High Churn
**Risk**: Users cancel after 1-2 months
**Mitigation**:
- Improve onboarding experience
- Add engagement features (challenges, community)
- Win-back email campaigns

### Risk 3: AI Cost Overruns
**Risk**: AI processing costs exceed projections
**Mitigation**:
- Cache common readings
- Optimize prompt length
- Negotiate volume discounts

### Risk 4: Competition
**Risk**: Major player launches similar app
**Mitigation**:
- Build strong community (moat)
- Focus on niche (AI + journaling)
- Speed to market advantage
