# 🌙 Mystic AI - Tarot & Oracle Platform

A complete AI-powered tarot reading platform with mobile-first design, community features, and freemium monetization.

## Project Structure

```
tarot-app/
├── deck/           # 78-card digital tarot deck specifications
├── engine/         # AI reading engine (spreads & interpretations)
├── journal/        # Journaling system & prompt library
├── design/         # Mobile-first wireframes & user flows
├── pricing/        # Freemium pricing model
├── landing/        # Landing page design & copy
├── social/         # 30-day social media content calendar
├── email/          # 7-email nurture sequence
├── community/      # Community features (daily readings, moon phases)
└── projections/    # 12-month revenue forecast
```

## Quick Start

1. **Deck**: Review `deck/` for complete 78-card specifications
2. **Engine**: AI interpretation logic in `engine/`
3. **Design**: Mobile wireframes in `design/`
4. **Launch**: Landing page copy in `landing/`

## Monetization

- **Free Tier**: 3 readings/month
- **Premium**: $7.99/month unlimited
- **Target**: 5% conversion rate

## Tech Stack Recommendations

- **Frontend**: React Native (mobile), Next.js (web)
- **Backend**: Node.js + PostgreSQL
- **AI**: Gensee API for interpretations
- **Payments**: Stripe
- **Hosting**: Vercel + Supabase
