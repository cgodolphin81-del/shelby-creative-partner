## 🛒 Etsy Internal Marketing

### 1. Etsy SEO (Search Engine Optimization)

**How Etsy Search Works:**
```
Customer searches → Etsy matches:
1. Listing titles
2. Tags (13 max)
3. Categories
4. Attributes
5. Shop relevance
6. Customer behavior (clicks, favorites, purchases)
```

**SEO Optimization Checklist:**

#### Title Optimization
**Formula:**
```
[Primary Keyword] + [Secondary Keyword] + [Use Case] + [Style] + [Feature]
```

**Examples:**
```
✅ Good:
"Digital Wedding Planner PDF, Printable Wedding Organizer, 
Bride-to-Be Checklist, Wedding Budget Template, Instant Download"

❌ Bad:
"My Awesome Planner"

✅ Good:
"SVG Birthday Cake Cut File, Cricut Birthday Design, 
Party Decorations, Vinyl Cutting File, Instant Download"

❌ Bad:
"Cake SVG"
```

**Title Best Practices:**
- [ ] First 40 characters are most important (mobile truncation)
- [ ] Include primary keyword first
- [ ] Use all 140 characters
- [ ] Natural language, not keyword stuffing
- [ ] Include style descriptors (modern, vintage, minimalist)

#### Tag Strategy (13 Tags)

**Tag Types to Include:**
1. **Primary keyword:** 2-3 tags
2. **Long-tail keywords:** 4-5 tags
3. **Use cases:** 2-3 tags
4. **Style/aesthetic:** 2-3 tags
5. **Audience:** 1-2 tags

**Examples by Niche:**

**Planner Shop:**
```
1. printable planner
2. digital planner
3. wedding planner
4. pdf planner
5. wedding checklist
6. printable checklist
7. wedding organizer
8. bride to be
9. wedding planning
10. instant download
11. modern minimalist
12. wedding templates
13. pdf organizer
```

**SVG Shop:**
```
1. svg cut file
2. cricut svg
3. silhouette svg
4. birthday svg
5. cake svg
6. vinyl cutting file
7. cricut design
8. party decorations
9. birthday party
10. svg bundle
11. digital download
12. cricut explore
13. silhouett studio
```

**Tag Best Practices:**
- [ ] Use all 13 tags
- [ ] Multi-word phrases (2-4 words)
- [ ] No repeats of same word
- [ ] Match customer search behavior
- [ ] Update quarterly based on performance

#### Category Selection

**Best Practices:**
- [ ] Most specific category possible
- [ ] Don't guess - research competitors
- [ ] Check what top sellers use
- [ ] Can change if not performing

**How to Research:**
1. Search your main keyword
2. Note categories of top 10 results
3. Choose the most relevant
4. Test for 30 days, adjust if needed

#### Attributes

**Fill out ALL applicable attributes:**
- Occasion
- Style
- Color
- Theme
- Holiday
- Audience

**Why:** Attributes filter in search = more visibility

### 2. Etsy Ads (Pay-Per-Click)

**Etsy Ads Overview:**
- Pay only when someone clicks your listing
- Set daily budget ($1-10 recommended for starters)
- Ads appear in Etsy search results
- Can turn on/off anytime

**When to Use Ads:**
- [ ] Listings with good conversion rate (>3%)
- [ ] New listings (to get initial traction)
- [ ] Promoting best sellers
- [ ] Seasonal promotions

**When NOT to Use Ads:**
- [ ] Listings with poor conversion (<1%)
- [ ] New shop with no reviews
- [ ] Low-quality listings (poor photos, SEO)

**Ad Budget Strategy:**

| Shop Stage | Daily Budget | Goal |
|------------|--------------|------|
| Launch (0-50 sales) | $1-2/day | Test listings |
| Growth (50-200 sales) | $3-5/day | Scale winners |
| Established (200+ sales) | $5-10/day | Maximize profit |

**Ad Optimization:**

**Week 1-2:**
- Run ads on 5-10 best listings
- Monitor clicks and impressions
- Don't adjust daily

**Week 3-4:**
- Identify top performers (high CTR, conversions)
- Increase budget on winners
- Pause underperformers

**Monthly:**
- Review ad spend vs. revenue
- Calculate ROAS (Return on Ad Spend)
- Adjust strategy

**ROAS Calculation:**
```
ROAS = Revenue from Ads / Ad Spend

Example:
Ad Spend: $100
Revenue: $500
ROAS: 5.0 (or 5:1)

Target ROAS: 3:1 or higher
```

### 3. Sales & Promotions

**Types of Sales:**

#### Percentage Off
```
Example: "20% OFF All Printables"
Best for: Clearing inventory, seasonal sales
Timing: Holidays, shop anniversaries
```

#### BOGO (Buy One Get One)
```
Example: "Buy 2 SVG Bundles, Get 1 Free"
Best for: Increasing average order value
Timing: Slow periods, new product launches
```

#### Free Shipping
```
Example: "Free Digital Download on Orders $25+"
Best for: Encouraging larger orders
Note: Digital = instant, so "free" is automatic
```

#### Bundle Discounts
```
Example: "Buy 3 Planners, Save 30%"
Best for: Moving multiple items
Auto-applied in cart
```

**Sale Calendar:**

| Holiday | When | What to Promote |
|---------|------|-----------------|
| New Year | Jan 1 | Planners, goal-setting |
| Valentine's | Feb 14 | Romantic themes, cards |
| Mother's Day | May | Family planners, cards |
| Summer | June-Aug | Beach, vacation themes |
| Back to School | Aug-Sep | Student planners, worksheets |
| Halloween | Oct 31 | Spooky designs, party |
| Thanksgiving | Nov | Gratitude journals |
| Christmas | Dec | Holiday planners, cards |
| Shop Anniversary | Your launch date | Celebration sale |

**Sale Best Practices:**
- [ ] Announce 1 week in advance
- [ ] Use Etsy's sale tool (Shop Manager → Marketing → Sales)
- [ ] Set clear start/end dates
- [ ] Promote on social media
- [ ] Update banner to reflect sale

## 🌐 External Traffic Sources

### 1. Pinterest (Best for Digital Products)

**Why Pinterest:**
- Visual search engine (like Google for images)
- Long content lifespan (months vs. hours)
- High purchase intent
- Free to use

**Pinterest Strategy:**

**Setup:**
- [ ] Business account (free)
- [ ] Claim your Etsy shop
- [ ] Enable Rich Pins
- [ ] Create boards by category

**Pin Creation:**
- [ ] Vertical pins (1000x1500px)
- [ ] Clear, readable text overlay
- [ ] Brand colors and logo
- [ ] Link directly to listing

**Pin Frequency:**
- [ ] 5-10 pins/day (scheduled)
- [ ] Mix of new and repinned content
- [ ] Use Tailwind or Pinterest native scheduler

**Pin Templates:**
1. Product mockup
2. Before/after
3. Step-by-step guide
4. Tips/how-to
5. User-generated content

**SEO:**
- [ ] Keyword-rich pin titles
- [ ] Detailed descriptions (2-3 sentences)
- [ ] Relevant hashtags (3-5)
- [ ] Link to specific listing

## ⚡ Quick Win Marketing Tactics

### 1. Optimize Top 10 Listings
- Improve titles with better keywords
- Add more tags
- Refresh main image
- Update descriptions

### 2. Run a Limited Sale
- 24-48 hour flash sale
- Promote heavily on social
- Create urgency

### 3. Leverage Customer Reviews
- Share 5-star reviews on social
- Create review graphics
- Use in ads

### 4. Collaborate with Micro-Influencers
- Find accounts in your niche (5k-50k followers)
- Offer free product for feature
- Track results

### 5. Join Etsy Teams
- Connect with other sellers
- Share tips and support
- Cross-promote (if allowed)
