## 🎯 Key Metrics to Track

### Traffic Metrics

#### Views vs. Visits
```
Views = Times listing appeared in search/results
Visits = Times someone clicked to your listing

Conversion Rate = Visits / Views × 100

Target: 50-70% (higher = better relevance)
```

#### Traffic Sources
**Breakdown:**
- Etsy Search (organic)
- Etsy Ads
- Social media
- Direct (bookmarked)
- External (Pinterest, Instagram, etc.)

**Goal:** Diversify traffic sources
- Etsy Search: 40-50%
- Social/External: 20-30%
- Direct: 10-20%
- Ads: 10-20%

### Sales Metrics

#### Conversion Rate
```
Conversion Rate = Orders / Visits × 100

Industry Average: 2-3%
Good: 3-5%
Excellent: 5%+

Low conversion (<1%)?
→ Improve photos
→ Optimize price
→ Enhance description
→ Add more reviews
```

#### Average Order Value (AOV)
```
AOV = Total Revenue / Number of Orders

Target: $15-50 for digital products

Increase AOV by:
- Bundling products
- Offering upgrades
- Free shipping threshold
- Cross-selling
```

#### Revenue Metrics
**Daily:**
- Total revenue
- Number of orders
- AOV

**Weekly:**
- Week-over-week growth
- Best-selling products
- Traffic trends

**Monthly:**
- Month-over-month growth
- Customer retention rate
- Profit margins

## 📊 Analyzing Listing Performance

### Listing Scorecard

Rate each listing (1-10) on:

| Metric | Score | Notes |
|--------|-------|-------|
| Views (last 30 days) | ___/10 | |
| Conversion rate | ___/10 | |
| Favorites (last 30 days) | ___/10 | |
| Revenue generated | ___/10 | |
| SEO optimization | ___/10 | |
| Photo quality | ___/10 | |
| Price competitiveness | ___/10 | |

**Total: ___/70**

**Action:**
- 60-70: Keep as-is, minor tweaks
- 40-59: Optimize photos, SEO, price
- <40: Major overhaul needed

### Listing Optimization Framework

**Step 1: Diagnose the Problem**

```
LOW VIEWS?
→ SEO issue (titles, tags, categories)
→ Poor main image
→ Competitor pricing
→ Seasonal demand

LOW CONVERSION?
→ Price too high
→ Photos don't show value
→ Missing information
→ No reviews

LOW FAVORITES?
→ Not compelling enough
→ Price not attractive
→ Not saving for later
→ Seasonal item
```

**Step 2: Implement Changes**

**SEO Optimization:**
1. Research new keywords (use eRank or Marmalead)
2. Update title with better keywords
3. Refresh all 13 tags
4. Check category selection
5. Add/update attributes

**Photo Optimization:**
1. Test new main image
2. Add more lifestyle shots
3. Include size/dimension reference
4. Show what's included
5. A/B test for 30 days

**Price Optimization:**
1. Check competitor pricing
2. Test price increase/decrease
3. Add bundle options
4. Create tiered pricing

**Description Optimization:**
1. Add more details
2. Include use cases
3. Add FAQs
4. Improve readability
5. Add keywords naturally

**Step 3: Monitor Results**

**Track for 30 days:**
- Views (should increase)
- Conversion rate (should improve)
- Favorites (should grow)
- Sales (ultimate goal)

**If no improvement after 30 days:**
→ Try different approach
→ Consider discontinuing listing
→ Pivot to new product

## 🔍 Search Term Analysis

### Finding Your Best Keywords

**Etsy Search Terms Report:**
1. Shop Manager → Stats → Search Terms
2. Sort by "Orders" or "Visits"
3. Identify top performers

**Keyword Categories:**

**High-Volume, High-Conversion:**
- Use in all listings
- Create new products around these
- Feature in titles prominently

**High-Volume, Low-Conversion:**
- Optimize listing to convert better
- Adjust price or photos
- Consider if keyword is wrong fit

**Low-Volume, High-Conversion:**
- Niche opportunities
- Less competition
- Worth targeting

**Low-Volume, Low-Conversion:**
- Remove from tags
- Not worth targeting

### Keyword Research Tools

**Free Tools:**
- Etsy search bar (autocomplete)
- eRank (free tier)
- Marmalead (free trial)

**Paid Tools:**
- eRank Pro ($7-30/month)
- Marmalead Pro ($29-79/month)
- EverBee ($39-99/month)

**Keyword Research Process:**

1. **Start with seed keyword:**
   ```
   Example: "planner"
   ```

2. **Use Etsy autocomplete:**
   ```
   Type "planner" in Etsy search
   Note suggestions:
   - planner printable
   - planner digital
   - planner wedding
   - planner 2024
   ```

3. **Check search volume:**
   ```
   Use eRank to check:
   - Monthly searches
   - Competition level
   - Seasonal trends
   ```

4. **Analyze competition:**
   ```
   Search the keyword
   - How many results?
   - What are top sellers charging?
   - What's their review count?
   ```

5. **Select target keywords:**
   ```
   Primary: 1-2 keywords (title)
   Secondary: 3-4 keywords (tags)
   Long-tail: 4-6 keywords (tags)
   ```

## 📈 A/B Testing Guide

### What to Test

**Priority 1: Main Image**
- Different mockup style
- Different background
- Different text overlay
- Different angle/position

**Priority 2: Title**
- Keyword order
- Length (short vs. long)
- Style descriptors

**Priority 3: Price**
- $9.97 vs $10.97
- Bundle vs. individual
- Tiered pricing

**Priority 4: Tags**
- Different keyword combinations
- Long-tail vs. short-tail

### A/B Test Setup

**Test Variables:**
```
Product: [Name]
Variable: [Main Image]
Test Duration: 30 days
Success Metric: Conversion rate
```

**Test Process:**
1. **Baseline:** Record current performance (30 days)
2. **Change ONE thing:** Only test one variable
3. **Run test:** Keep other factors constant
4. **Measure:** Compare results
5. **Decide:** Keep winner or revert

**Example Test:**

```
Product: Wedding Planner Printable

Test A (Current):
- Main image: White background mockup
- Views: 500
- Visits: 350
- Sales: 10
- Conversion: 2.86%

Test B (New):
- Main image: Lifestyle mockup (planner on desk)
- Views: 520
- Visits: 380
- Sales: 18
- Conversion: 4.74%

Result: Test B wins (+65% conversion)
Action: Keep new image
```

### Testing Schedule

**Month 1:**
- Week 1-2: Test main images (2-3 listings)
- Week 3-4: Test titles (2-3 listings)

**Month 2:**
- Week 1-2: Test prices (2-3 listings)
- Week 3-4: Test tags (2-3 listings)

**Month 3:**
- Review all tests
- Implement winners
- Plan next round

## 📊 Customer Analytics

### Customer Segmentation

**New Customers:**
- First-time buyers
- Acquired through search/social
- Conversion rate: 1-2%

**Returning Customers:**
- 2nd-3rd purchase
- Acquired through email/favorites
- Conversion rate: 5-10%

**VIP Customers:**
- 4+ purchases
- High lifetime value
- Conversion rate: 15%+

### Customer Lifetime Value (CLV)

**Formula:**
```
CLV = Average Order Value × Purchase Frequency × Customer Lifespan
```

**Example:**
```
AOV: $12
Purchases per year: 3
Lifespan: 2 years

CLV = $12 × 3 × 2 = $72
```

**Increase CLV by:**
- Email marketing
- Bundle offers
- Loyalty discounts
- New product notifications
- Excellent customer service

### Repeat Purchase Rate

**Formula:**
```
Repeat Rate = (Customers with 2+ purchases) / (Total customers) × 100
```

**Target:** 20-30% for digital products

**Improve repeat rate by:**
- Email list building
- Cross-selling in downloads
- New product launches
- Exclusive customer sales

## 📉 Declining Performance? Here's What to Do

### Symptom: Views Decreasing

**Causes:**
- Seasonal demand drop
- Increased competition
- SEO decay (Etsy algorithm changes)
- Poor main image

**Solutions:**
1. Research new keywords
2. Update title and tags
3. Refresh main image
4. Create new seasonal listings
5. Increase ad spend

### Symptom: Conversion Rate Decreasing

**Causes:**
- Price too high
- Photos not compelling
- Negative reviews
- Competitor undercutting

**Solutions:**
1. A/B test price
2. Improve photos
3. Address review concerns
4. Add value (more files, better support)
5. Create bundles

### Symptom: Favorites Decreasing

**Causes:**
- Not saving for later
- Price not attractive
- Listing not memorable
- Seasonal item

**Solutions:**
1. Add urgency (limited time offer)
2. Improve main image
3. Add "save for later" CTA
4. Create seasonal content
