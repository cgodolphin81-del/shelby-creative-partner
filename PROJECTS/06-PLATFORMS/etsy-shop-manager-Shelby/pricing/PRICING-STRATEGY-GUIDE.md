## 🎯 Psychological Pricing Strategies

### 1. Charm Pricing (End in .97 or .99)

**Why it works:**
- Left-digit effect (4.97 feels like $4 not $5)
- Perceived as "sale price"
- Standard for online shopping

**Examples:**
- $4.97 instead of $5.00
- $12.97 instead of $13.00
- $24.97 instead of $25.00

**Best for:** Impulse purchases, lower-priced items

### 2. Anchor Pricing

**How it works:**
- Show higher "regular" price
- Display discounted price
- Create perceived value

**Example:**
```
✨ BUNDLE DEAL ✨
Regular Price: $47.00
Today's Price: $27.97
You Save: 40%!
```

**Best for:** Bundles, limited-time offers

### 3. Tiered Pricing

**Create 3 price points:**
```
BASIC: $7.97 (entry-level)
STANDARD: $14.97 (best value - highlight this)
PREMIUM: $24.97 (comprehensive)
```

**Psychology:**
- Middle option feels like "smart choice"
- Premium justifies standard price
- Basic attracts budget shoppers

### 4. Odd-Number Pricing

**End in 7 vs 9:**
- .97 = deeper discount perception
- .99 = standard sale
- .95 = psychological sweet spot

**Research shows:**
- $4.97 converts better than $4.99
- $7.97 outperforms $7.99
- $12.97 is gold standard

### 5. Bundle Pricing

**Bundle discount formula:**
```
Individual Total: $45.00
Bundle Price: $27.97
Savings: 38%
```

**Bundle types:**
- **Complementary:** Related products used together
- **Volume:** Same product, multiple versions
- **Thematic:** All products in a category
- **Seasonal:** Holiday-specific bundles

**Positioning Decision:**
- [ ] Undercut market (70-80% of average)

## 📈 Pricing by Shop Stage

### Phase 1: Launch (Months 1-3)

**Goal:** Get first sales and reviews

**Strategy:**
- Price 20-30% below market average
- Focus on value, not profit
- Build social proof first

**Example:**
```
Market Average: $12.00
Your Launch Price: $7.97
Goal: 10-20 sales, 5-10 reviews
```

### Phase 2: Growth (Months 4-6)

**Goal:** Optimize for profit

**Strategy:**
- Raise prices 10-20%
- Test different price points
- Introduce tiered pricing

**Example:**
```
Launch Price: $7.97
Growth Price: $9.97
Test: $8.97 vs $10.97
```

### Phase 3: Established (Months 7-12)

**Goal:** Maximize profit

**Strategy:**
- Price at or above market average
- Focus on value propositions
- Premium tiers available

**Example:**
```
Market Average: $12.00
Your Price: $12.97-$14.97
Justification: Better quality, more files, superior support
```

### Phase 4: Premium (Year 2+)

**Goal:** Brand positioning

**Strategy:**
- Premium pricing (150%+ market average)
- Emphasize exclusivity
- Limited editions

## 🎁 Bundle Pricing Calculator

### Bundle Value Formula

```
Individual Prices:
Product A: $9.97
Product B: $12.97
Product C: $7.97
Total: $30.91

Bundle Price (30% off): $21.97
Savings: $8.94 (29%)
```

### Bundle Types

**1. Collection Bundles**
```
All products in category
Discount: 25-35%
Example: "100 SVG Bundle - $27.97 (vs $47 individually)"
```

**2. Complementary Bundles**
```
Products that work together
Discount: 20-30%
Example: "Planner + Stickers Bundle - $14.97"
```

**3. Tiered Bundles**
```
Small (3 items): $12.97
Medium (6 items): $19.97
Large (12 items): $29.97
```

### Bundle Pricing Psychology

**Do:**
- ✓ Show individual total vs. bundle price
- ✓ Calculate % saved
- ✓ Highlight best value
- ✓ Use "limited time" for urgency

**Don't:**
- ✗ Bundle too few items (doesn't feel like value)
- ✗ Make bundle harder to buy than individual
- ✗ Hide what's included

## 🔄 Price Testing Framework

### A/B Testing Prices

**Test Variables:**
1. **Price Point**
   - $7.97 vs $8.97 vs $9.97
   - Test for 30 days each

2. **Anchor Price**
   - "Was $15, Now $9.97" vs no anchor

3. **Bundle Discount**
   - 20% off vs 30% off vs 40% off

4. **Charm Ending**
   - .97 vs .99 vs .95

**Tracking:**
```
Product: [Name]
Test Period: [Dates]
Views: [Number]
Conversions: [Number]
Revenue: [Amount]
```

### Price Increase Strategy

**When to raise prices:**
- [ ] Consistent sales for 60+ days
- [ ] 50+ reviews with 4.8+ stars
- [ ] New features added
- [ ] Market demand increased

**How to raise:**
1. Raise 10-15% at a time
2. Monitor conversion rate
3. If conversion drops <10%, revert
4. If stable, raise again in 30 days

**Communication:**
```
"Great news! We've added [new feature] to this design.
To reflect the enhanced value, the price will increase
from $X to $Y on [date]. Lock in current price now!"
```

## ⚠️ Pricing Mistakes to Avoid

**1. Underpricing**
- ✗ Pricing too low devalues product
- ✗ Attracts difficult customers
- ✗ Hard to raise later
- ✓ Solution: Start reasonable, increase gradually

**2. Overpricing**
- ✗ No sales = no reviews
- ✗ Killing momentum
- ✗ Hard to justify value
- ✓ Solution: Research market, test lower first

**3. Inconsistent Pricing**
- ✗ Same product, different prices
- ✗ Confuses customers
- ✗ Looks unprofessional
- ✓ Solution: Standardize pricing tiers

**4. Ignoring Fees**
- ✗ Forgetting Etsy fees
- ✗ Miscalculating profit
- ✗ Losing money on sales
- ✓ Solution: Use profit calculator

**5. Not Testing**
- ✗ Setting price and never changing
- ✗ Missing optimization opportunities
- ✗ Leaving money on table
- ✓ Solution: A/B test regularly
