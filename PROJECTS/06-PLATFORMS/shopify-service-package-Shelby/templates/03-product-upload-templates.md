## 📝 PRODUCT TITLE TEMPLATES

### Template 1: Basic Formula
```
[Brand] + [Product Type] + [Key Feature] + [Size/Color/Variant]
```
**Examples:**
- Nike Air Max 270 Running Shoes - Men's Size 10 - Black/White
- Organic Cotton T-Shirt - Unisex - Navy Blue - Medium
- Stainless Steel Water Bottle - 32oz - Insulated - Rose Gold

---

### Template 2: Benefit-Driven
```
[Benefit] + [Product] + [Key Spec] + [Audience]
```
**Examples:**
- Ultra-Comfortable Memory Foam Pillow - Queen Size - Side Sleepers
- Fast-Charging Wireless Earbuds - 24hr Battery - For Athletes
- Wrinkle-Release Travel Dress - Packable - Business Women

---

### Template 3: SEO-Optimized
```
[Primary Keyword] + [Secondary Keyword] + [Brand] + [Differentiator]
```
**Examples:**
- Running Shoes for Women - Athletic Sneakers - Nike - Lightweight Design
- Organic Face Moisturizer - Natural Skincare - Glow Co - Hyaluronic Acid
- Yoga Mat Non-Slip - Exercise Mat - ZenFit - Extra Thick 6mm

---

### Template 4: Luxury/Premium
```
[Brand] + [Collection] + [Product] + [Material/Quality Indicator]
```
**Examples:**
- Tiffany & Co. Return Collection Sterling Silver Bracelet
- Louis Vuitton Monogram Canvas Neverfull MM Handbag
- Rolex Submariner Date 41mm Stainless Steel Watch

---

### Template 5: Bundle/Multi-Pack
```
[Product] + [Quantity] + [Bundle Type] + [Savings/Benefit]
```
**Examples:**
- Organic Cotton T-Shirts - 3 Pack - assorted Colors - Save 20%
- Vitamin C Serum Set - 4 Piece Kit - Complete Skincare Routine

## 🔍 SEO TEMPLATE FOR PRODUCTS

### SEO Title Template (50-60 characters)
```
[Primary Keyword] - [Secondary Keyword] | [Brand Name]
```
**Examples:**
- Running Shoes for Women - Lightweight Athletic Sneakers | Nike
- Organic Face Serum - Anti-Aging Vitamin C | Glow Co
- Wireless Bluetooth Earbuds - Noise Cancelling | SoundMax

---

### Meta Description Template (150-160 characters)
```
[Compelling hook]. [Key benefit 1] + [Key benefit 2]. [Social proof/urgency]. [CTA]
```
**Examples:**
- Experience ultimate comfort with our bestselling running shoes. Lightweight design + superior cushioning. 10,000+ 5-star reviews. Shop now with free shipping!
- Transform your skin with organic vitamin C serum. Reduce wrinkles + brighten complexion. Dermatologist recommended. Order today - 30-day guarantee!

---

### URL Handle Template
```
/primary-keyword-secondary-keyword
```
**Examples:**
- /womens-running-shoes-lightweight
- /organic-vitamin-c-serum
- /wireless-bluetooth-earbuds-noise-cancelling

**Rules:**
- Use lowercase only
- Use hyphens (not underscores)
- Remove stop words (the, a, an, and, or)
- Keep under 60 characters
- Include primary keyword

---

## 📊 PRODUCT UPLOAD CHECKLIST

### Before Uploading
- [ ] Product title optimized (under 70 characters)
- [ ] SEO title created (50-60 characters)
- [ ] Meta description written (150-160 characters)
- [ ] URL handle optimized
- [ ] Product description complete
- [ ] All images ready (min 5 per product)
- [ ] Image alt text written
- [ ] Pricing calculated (cost + margin)
- [ ] Inventory SKU assigned
- [ ] Variants configured (sizes, colors, etc.)
- [ ] Product tags added
- [ ] Collection assigned

### Images Required
- [ ] Main product shot (white background)
- [ ] Alternative angles (left, right, back)
- [ ] Detail/close-up shots
- [ ] Lifestyle/in-use shot
- [ ] Size/scale reference
- [ ] Packaging shot (if relevant)
- [ ] Video (optional but recommended)

### Image Specifications
- Format: JPG or PNG (WebP for performance)
- Size: Min 1024x1024px (2048x2048px recommended)
- File size: Under 500KB each
- Color profile: sRGB
- Naming: product-name-color-view.jpg

## 📈 PRODUCT DESCRIPTION BEST PRACTICES

### Do's ✅
- Write for your target customer
- Focus on benefits, not just features
- Use sensory language
- Include social proof
- Add urgency/scarcity when appropriate
- Answer common questions preemptively
- Use formatting (bullets, bold, headers)
- Include clear CTAs
- Optimize for SEO naturally
- Keep paragraphs short (2-3 sentences)

### Don'ts ❌
- Don't use manufacturer copy (duplicate content)
- Don't keyword stuff
- Don't make false claims
- Don't use jargon without explanation
- Don't write walls of text
- Don't forget mobile readability
- Don't ignore your brand voice
- Don't skip proofreading
