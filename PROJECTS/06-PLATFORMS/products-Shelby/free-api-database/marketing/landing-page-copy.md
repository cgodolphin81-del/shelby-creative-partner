## FAQ Section

### Frequently Asked Questions

**Q: Is this really 100+ APIs?**
A: Yes! The database currently includes 125 verified APIs across 10 categories. New APIs are added monthly.

**Q: Are these APIs actually free?**
A: All APIs listed have a free tier or are completely free. We clearly label each API as "Free," "Freemium," "Trial," or "Pay-per-use" so you know exactly what to expect.

**Q: How often is the database updated?**
A: We verify and update the database weekly. Lifetime members get notified of all updates. Pro members receive monthly update emails.

**Q: What format is the database?**
A: The database is provided as a CSV file that imports directly into Notion. We also provide a Notion template link for instant duplication.

**Q: Can I use this for commercial projects?**
A: Yes! The Lifetime license includes commercial use rights. You can use the database to build client projects, SaaS products, or internal tools.

**Q: What if an API changes or shuts down?**
A: We monitor all APIs and update the database when changes occur. Deprecated APIs are marked and alternatives are suggested.

**Q: Do the affiliate links work for me?**
A: The affiliate links in the database are for the database creator. However, the Affiliate Integration guide shows you how to sign up for your own affiliate programs and monetize your usage.

**Q: Is there a refund policy?**
A: Yes! We offer a 30-day money-back guarantee. If you're not satisfied, email us for a full refund. No questions asked.

**Q: Can I share this with my team?**
A: The Pro license is for individual use. The Lifetime license includes a team license for up to 5 team members.
