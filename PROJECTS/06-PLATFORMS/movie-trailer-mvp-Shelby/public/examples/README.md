## Usage

These examples are displayed on the landing page in the Examples section. Each example card shows:

- Thumbnail image
- Style badge
- Play button overlay
- Title
- Style name + duration

### To Add New Examples

1. Generate trailer using the platform
2. Export thumbnail (frame at 5 seconds)
3. Save video to `public/examples/`
4. Add metadata JSON to this file
5. Update homepage HTML with new example card

---

## Marketing Usage

These examples can be used for:
- Landing page demos
- Social media content
- Paid ad creatives
- Blog post illustrations
- Email marketing

**Note:** Always get permission from users before using their trailers in marketing.
