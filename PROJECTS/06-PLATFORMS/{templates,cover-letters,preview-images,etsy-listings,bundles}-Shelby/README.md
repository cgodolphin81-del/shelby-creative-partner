# 📦 CV Templates Asset Collection

Organized asset collection for CV template Etsy shop including templates, cover letters, preview images, and listing materials.

## 🎯 What It Is

A working directory containing all assets needed for the CV templates Etsy shop, organized by category for easy management and upload.

## 📁 Directory Structure

This directory is organized to contain:

### templates/
- Master CV template files
- Multiple format versions
- Customizable source files

### cover-letters/
- Matching cover letter templates
- Industry-specific versions
- General and targeted formats

### preview-images/
- Product mockups
- Listing images
- Social media graphics
- Pinterest pins

### etsy-listings/
- Listing descriptions
- SEO titles and tags
- Pricing information
- Category assignments

### bundles/
- Pre-packaged template bundles
- Discount combinations
- Upsell packages

## 🚀 Workflow

1. **Select Templates:** Choose templates for a bundle or listing
2. **Create Previews:** Generate preview images
3. **Write Listings:** Create SEO-optimized descriptions
4. **Price Strategically:** Set competitive pricing
5. **Upload to Etsy:** Publish listings
6. **Automate Delivery:** Set up instant download

## 📊 Organization System

Files should be organized with clear naming:
```
[Category]_[Description]_[Version].[ext]
```

Example:
- `CV_Professional_Modern_v1.docx`
- `CL_Tech_Industry_v2.docx`
- `Preview_CV_Bundle_01.png`

## 💡 Best Practices

- Keep source files organized
- Version control for updates
- Consistent naming conventions
- Regular backups
- Quality check before upload
