## NICHE SELECTION FRAMEWORK

### Score Your Niche (1-10 scale)

| Factor | Weight | Your Score | Weighted |
|--------|--------|------------|----------|
| Audience Size | 15% | | |
| Willingness to Pay | 25% | | |
| Competition Level (inverse) | 15% | | |
| Your Expertise | 20% | | |
| Monetization Options | 15% | | |
| Content Sustainability | 10% | | |
| **TOTAL** | **100%** | | |

### Red Flags to Avoid
- ❌ Oversaturated without differentiation
- ❌ No clear path to monetization
- ❌ Audience has no budget (students, hobbyists without spending power)
- ❌ Requires daily content you can't sustain
- ❌ Legal/regulatory minefield without expertise
- ❌ Trend-based (will die in 12 months)

### Green Lights to Pursue
- ✅ You have unique expertise/angle
- ✅ Clear B2B or high-income B2C audience
- ✅ Multiple monetization paths
- ✅ Evergreen + timely content mix possible
- ✅ Sponsorship market exists
- ✅ You can create content for 2+ years without burning out
