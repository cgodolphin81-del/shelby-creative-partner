## PROMPT USAGE NOTES

**Aspect Ratios:**
- `--ar 16:9` - Standard video/YouTube format
- `--ar 3:2` - Traditional photography
- `--ar 4:3` - Social media, presentations
- `--ar 21:9` - Cinematic panoramas

**Best Practices:**
- Use `--v 6` for Midjourney v6 photorealism
- Adjust `--stylize` parameter for more/less artistic interpretation
- For DALL-E 3, remove Midjourney parameters and use natural language
- For Stable Diffusion, add negative prompts for quality control

**Negative Prompts (for SD):**
```
low quality, blurry, distorted, deformed, ugly, text, watermark, signature, cartoon, anime, 3d render, plastic, artificial
```

**Seed Consistency:**
- Use same seed number across related prompts for visual consistency
- Recommended seed range: 1000-9999
