## Quick Reference

### Key Locations
1. **Pena Palace** - Colorful Romantic palace, fairy-tale architecture
2. **Quinta da Regaleira** - Mystical estate, Initiation Well, Templar symbolism
3. **Moorish Castle** - 8th-9th century fortress, Atlantic views
4. **Monserrate Palace** - Neo-Moorish, exotic gardens
5. **Sintra National Palace** - Royal residence, twin chimneys
6. **Cabo da Roca** - Westernmost point of continental Europe

### Color Palette
- **Palace Yellow:** #F4D03F
- **Pena Red:** #C41E3A
- **Garden Green:** #2E8B57
