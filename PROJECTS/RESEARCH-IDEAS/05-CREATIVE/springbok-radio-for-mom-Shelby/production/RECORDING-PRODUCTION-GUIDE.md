### Pre-Recording Checklist

Before recording each episode:

- [ ] Read through entire script aloud
- [ ] Mark breathing points and pauses
- [ ] Identify character voices needed
- [ ] Set up recording space (quiet, no echo)
- [ ] Test microphone levels
- [ ] Record 30 seconds of "room tone" (silence)
- [ ] Have water nearby (stay hydrated)
- [ ] Turn off phones, fans, AC (reduce noise)

## PHASE 2: RECORDING

### Recording Environment

#### Finding the Best Space:
**Best Options:**
- Walk-in closet (clothes absorb sound)
- Bedroom with carpets and curtains
- Under a blanket fort (seriously!)

**Avoid:**
- Kitchens and bathrooms (hard surfaces = echo)
- Rooms with windows facing streets (traffic noise)
- Spaces with fans or AC (constant hum)

#### Quick Acoustic Treatment:
- Hang blankets on walls
- Place pillows behind you
- Record under a thick duvet (professional trick!)
- Face into a corner (reduces echo)

### Microphone Technique

#### Positioning:
```
        [Microphone]
            |
            | 6-8 inches
            |
        [Your Mouth]
            |
            | Slightly off-axis (not directly in front)
            |
        [Pop Filter]
```

**Distance:** 6-8 inches (fist and a half)  
**Angle:** Slightly off-center (reduces plosives)  
**Height:** At mouth level or slightly above

#### Speaking Technique:
- **Volume:** Conversational, not shouting
- **Clarity:** Over-enunciate slightly (radio requires it)
- **Pacing:** Slower than normal conversation
- **Pauses:** Allow space for SFX and music
- **Consistency:** Stay same distance from mic throughout

#### Recording Settings (Audacity):
```
Sample Rate: 44100 Hz
Bit Depth: 24-bit (or 32-bit float)
Channels: Mono (for dialogue)
Format: WAV (not MP3 during editing)
```

### Recording Workflow

#### Option A: Record All Voices Yourself
**Pros:** Complete creative control, free, flexible timing  
**Cons:** Time-consuming, all voices sound similar

**Technique:**
1. Record each character separately
2. Leave 2-3 seconds of silence between lines
3. Label tracks immediately: "Margaret_Scene1"
4. Save after each recording session

#### Option B: Use AI Voices (ElevenLabs)
**Pros:** Distinct character voices, fast, consistent  
**Cons:** Cost (£22/month), less emotional nuance

**Workflow:**
1. Generate all dialogue in ElevenLabs
2. Download as WAV files
3. Import into Audacity
4. Organize by character and scene

#### Option C: Hire Voice Actors
**Pros:** Professional quality, authentic accents  
**Cons:** Expensive, scheduling coordination

**Platforms:**
- Fiverr.com (search "South African voice actor")
- Upwork.com
- Voices.com (professional, pricey)
- Local theater groups (contact directly)

### Recording Best Practices

#### DO:
✅ Record in short segments (easier to edit)  
✅ Leave 2-3 seconds of silence at start/end  
✅ Clap once at start (visual marker for syncing)  
✅ Save every 10 minutes (prevent data loss)  
✅ Record multiple takes of important lines  
✅ Label files immediately and clearly

#### DON'T:
❌ Record in one long take (nightmare to edit)  
❌ Rustle papers near the mic  
❌ Drink from glass bottles (use open cups)  
❌ Record when tired (voice shows fatigue)  
❌ Edit while recording (do separately)  
❌ Use MP3 for recording (use WAV)

## PHASE 3: EDITING

### Importing & Organizing

#### Create Session Template:
```
Track 1: Narrator/Announcer
Track 2: Character 1 (e.g., Margaret)
Track 3: Character 2 (e.g., David)
Track 4: Character 3 (e.g., Susan)
Track 5: Sound Effects
Track 6: Music Bed
Track 7: Master Output
```

#### Import Order:
1. Room tone (bottom track, very quiet)
2. Dialogue tracks (center)
3. Sound effects (below dialogue)
4. Music (bottom, under everything)

### Editing Techniques

#### 1. **Cutting Silence**
- Remove long pauses between lines
- Leave natural breathing space (0.5-1 second)
- Don't over-edit (sounds robotic)

**How-To (Audacity):**
```
1. Select silent section
2. Press Delete
3. Add 0.5s silence if needed (Generate > Silence)
```

#### 2. **Crossfading**
- Smooth transitions between edits
- Prevents clicks and pops

**How-To:**
```
1. Select edit point (0.1 seconds)
2. Effect > Crossfade In/Out
3. Or manually fade (Effect > Fade In/Out)
```

#### 3. **Timing & Pacing**
- Radio drama is slower than modern podcasts
- Allow pauses for emotional moments
- Let sound effects breathe (don't talk over them)

**Pacing Guide:**
```
Normal conversation: 150 words/minute
Radio drama: 130 words/minute
Emotional scenes: 110 words/minute
```

#### 4. **Noise Reduction**
- Remove background hum and hiss
- Use sparingly (over-processing sounds artificial)

**How-To (Audacity):**
```
1. Select section of "room tone" (silence)
2. Effect > Noise Reduction > Get Noise Profile
3. Select entire track
4. Effect > Noise Reduction > OK (use default settings)
```

### Episode Assembly

#### Scene Structure:
```
[00:00-00:15] Opening music & announcer
[00:15-02:00] Scene 1 setup (establish location)
[02:00-08:00] Scene 2-3 (main story)
[08:00-12:00] Scene 4-5 (climax)
[12:00-14:00] Resolution
[14:00-15:00] Closing music & announcer
```

#### Transition Techniques:
- **Music bridge:** 3-5 seconds of theme music
- **Sound effect:** Door close, train whistle, etc.
- **Fade out/in:** Gradual volume change
- **Announcer:** "Meanwhile, back at the ranch..."

## PHASE 4: MIXING

### Volume Balancing

#### Target Levels (dB):
```
Dialogue:          0 dB (reference)
Music (under):    -18 dB
Music (featured):  -6 dB
Sound Effects:    -12 dB
Ambience:         -24 dB
```

#### How to Adjust (Audacity):
```
1. Select track
2. Effect > Amplify
3. Enter dB value (negative = quieter)
4. Preview, then Apply
```

### EQ (Equalization)

#### Vintage Broadcast EQ:
The goal: Make voices sound like 1950s radio

**Settings:**
```
High-Pass Filter: 100 Hz (remove low rumble)
Low Shelf: -3 dB at 200 Hz (reduce boominess)
Mid Boost: +3 dB at 2 kHz (presence)
High Shelf: -6 dB at 8 kHz (reduce modern crispness)
```

**How-To (Audacity):**
```
1. Select dialogue track
2. Effect > Filter Curve EQ
3. Draw curve matching above settings
4. Preview, then Apply
```

#### Preset: "Vintage Radio"
```
Frequency    Gain
31 Hz       -12 dB
62 Hz        -6 dB
125 Hz       -3 dB
250 Hz        0 dB
500 Hz       +2 dB
1 kHz        +3 dB
2 kHz        +4 dB
4 kHz        +2 dB
8 kHz        -4 dB
16 kHz       -8 dB
```

### Compression

#### Why Compress?
- Evens out volume differences
- Makes quiet words audible
- Gives "radio announcer" sound

#### Settings for Dialogue:
```
Threshold:    -18 dB
Ratio:        3:1
Attack:       5 ms
Release:      50 ms
Makeup Gain:  +6 dB
```

**How-To (Audacity):**
```
1. Select dialogue track
2. Effect > Compressor
3. Enter settings above
4. Preview, then Apply
```

### Reverb & Space

#### Vintage Radio = DRY (Minimal Reverb)
1950s radio was recorded in dead studios (no echo)

**Settings:**
```
Room Size:    Small (2-3 meters)
Damping:      High (80%)
Dry/Wet:      10-15% wet
```

**When to Use:**
- Phone calls (more reverb, EQ like telephone)
- Flashback scenes (slightly more space)
- Henry's voice in Episode 5 (otherworldly)

**How-To Create Phone Voice:**
```
1. EQ: Cut below 500 Hz, cut above 3 kHz
2. Add reverb: 20% wet, small room
3. Add slight distortion (Effect > Distortion)
```

### Slapback Delay (THE VINTAGE SOUND!)

This is the secret sauce for 1950s broadcast sound.

#### What is Slapback Delay?
- Single echo, 75-150 milliseconds
- Creates "doubling" effect
- Used on vocals, announcers, some dialogue

#### Settings:
```
Delay Time:    100 ms (adjust to taste)
Feedback:      10-20% (one repeat only)
Mix:           15-25% wet
```

**How-To (Audacity):**
```
1. Duplicate dialogue track
2. On duplicate: Effect > Delay
3. Settings: 100ms delay, 10% feedback
4. Lower volume of delayed track (-15 dB)
5. Mix with original
```

**When to Use:**
✅ Announcer introductions  
✅ Narration  
✅ Emotional monologues  
✅ Title announcements  

**When NOT to Use:**
❌ Regular dialogue (sounds weird)  
❌ Fast-paced scenes  
❌ Already has reverb

## PHASE 5: MASTERING

### What is Mastering?
Final polish to make episode sound:
- Consistent volume throughout
- Professional quality
- Ready for distribution

### Mastering Chain

Apply these effects to your **Master Track** (final output):

#### 1. **EQ (Final Polish)**
```
High-Pass: 40 Hz (remove sub-bass rumble)
Low Shelf: -2 dB at 100 Hz
High Shelf: -3 dB at 12 kHz (smooth the top)
```

#### 2. **Compression (Gentle)**
```
Threshold: -12 dB
Ratio: 2:1
Attack: 10 ms
Release: 100 ms
Makeup Gain: +3 dB
```

#### 3. **Limiter (Prevent Clipping)**
```
Ceiling: -1 dB (prevents distortion)
Threshold: -6 dB
```

**How-To (Audacity):**
```
Effect > Limiter
Settings: Type "Hard Limit", Ceiling -1 dB
```

#### 4. **Normalize (Final Volume)**
```
Target: -3 dB peak
```

**How-To:**
```
Effect > Normalize
Set to -3 dB
```

### Loudness Standards

#### For Podcast/Online:
- **Target:** -16 LUFS (integrated)
- **Peak:** -1 dB maximum
- **Dynamic Range:** 8-12 dB

#### For CD/Physical:
- **Target:** -14 LUFS
- **Peak:** -0.3 dB maximum

#### For This Project:
Since it's a personal gift, aim for:
- **Consistent volume** across all 10 episodes
- **No distortion** (never hit 0 dB)
- **Clear dialogue** (most important!)

### Quality Control Checklist

Before exporting each episode:

- [ ] Listen from start to finish (no skipping)
- [ ] Check volume consistency (no sudden jumps)
- [ ] Verify all SFX are present and timed correctly
- [ ] Ensure music doesn't overpower dialogue
- [ ] Check for clicks, pops, or audio artifacts
- [ ] Test on different speakers (headphones, phone, car)
- [ ] Get feedback from someone else (fresh ears)
- [ ] Compare to reference (actual Springbok Radio recording)

## VINTAGE SOUND PRESETS

### Quick Reference: "1950s Radio" Chain

Apply to dialogue tracks:

```
1. EQ (Filter Curve):
   - Cut below 100 Hz
   - Boost 1-3 kHz (+3 dB)
   - Cut above 8 kHz (-6 dB)

2. Compressor:
   - Threshold: -18 dB
   - Ratio: 3:1
   - Attack: 5 ms
   - Release: 50 ms

3. Slapback Delay (announcer only):
   - Delay: 100 ms
   - Feedback: 10%
   - Mix: 20%

4. Reverb (minimal):
   - Room: Small
   - Wet: 10%
```

### Quick Reference: "Telephone Voice"

For phone call scenes:

```
1. EQ:
   - High-Pass: 500 Hz
   - Low-Pass: 3 kHz
   - Boost: 2 kHz (+6 dB)

2. Reverb:
   - Room: Small
   - Wet: 20%

3. Distortion (slight):
   - Drive: 5%
```

## TROUBLESHOOTING

### Problem: Dialogue sounds muddy
**Solution:** Cut frequencies below 100 Hz, boost 2-3 kHz

### Problem: SFX overpower dialogue
**Solution:** Lower SFX track by 6-12 dB, use EQ to carve space

### Problem: Music is distracting
**Solution:** Lower music to -20 dB under dialogue, use high-pass filter

### Problem: Voice sounds too modern/clean
**Solution:** Add slapback delay, reduce high frequencies, add slight noise

### Problem: Background noise is audible
**Solution:** Use noise reduction, record in quieter space, use directional mic

### Problem: Episodes have inconsistent volume
**Solution:** Use same mastering chain on all episodes, check LUFS meter

### Problem: File sizes are too large
**Solution:** Export as MP3 192 kbps instead of WAV

## FINAL TIPS

### The 80/20 Rule:
- 80% of the quality comes from 20% of the effort
- Don't obsess over perfection
- Mom will love it because it's from you, not because it's flawless

### Take Breaks:
- Your ears fatigue after 45-60 minutes
- Take 10-minute breaks every hour
- Don't master when tired (you'll make bad decisions)

### Reference Constantly:
- Listen to actual Springbok Radio recordings
- Compare your mix to professional old-time radio
- Aim for "in the spirit of" not "exact copy"

### Have Fun:
- This is a labor of love
- Laugh at mistakes (they're memories)
- Enjoy the creative process
- Mom will hear the love in every word
