## HOOKS & OPENING LINES

### Cold Opens (First 10 seconds)

**Hook 1: The Road**
```
There is a place where the road ends.
```
*Simple, mysterious, makes you lean in*

**Hook 2: The Map**
```
The maps lie. They suggest that Alaska is a place you can visit.
```
*Contrarian, intriguing, challenges assumptions*

**Hook 3: The Refusal**
```
The wilderness is not untamed because no one has tried. It is untamed because it refuses.
```
*Powerful, personifies nature, memorable*

**Hook 4: The Smallness**
```
There is a particular kind of humility that only Alaska can teach you.
```
*Contemplative, promises transformation*

**Hook 5: The Time**
```
Glaciers are made of time. This is not poetry. It is geology.
```
*Unexpected, educational, poetic*

**Hook 6: The Bear**
```
A grizzly bear does not care about your itinerary.
```
*Humorous, true, immediately grounding*

**Hook 7: The Light**
```
Alaska is a way the light falls across snow at three in the afternoon in December.
```
*Visual, specific, evocative*

**Hook 8: The Ending**
```
This is not the kind of ending you find on a cul-de-sac. This is where the map itself surrenders.
```
*Builds curiosity, contrasts mundane with sublime*
