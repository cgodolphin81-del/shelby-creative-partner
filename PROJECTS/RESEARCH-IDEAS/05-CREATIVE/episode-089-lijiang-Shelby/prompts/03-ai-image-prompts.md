## PROMPT USAGE NOTES

### For Midjourney:
- Add `--ar 16:9` or `--ar 4:5` or `--ar 1:1` for aspect ratios
- Add `--v 6` or current version
- Add `--style raw` for more photographic results
- Add `--q 2` for higher quality

### For DALL-E 3:
- Prompts work as-is
- Specify aspect ratio in conversation
- Can request variations on successful images

### For Stable Diffusion:
- Add negative prompts: `ugly, deformed, noisy, blurry, distorted, out of focus, bad anatomy, extra limbs, poorly drawn hands, poorly drawn faces, mutation, mutated, extra fingers, bad proportions, missing arms, missing legs, extra legs, extra arms, fused fingers, too many fingers, long neck, username, watermark, signature`
- Use appropriate checkpoints for photorealistic or artistic styles

### Recommended Workflow:
1. Generate landscape establishing shots first (prompts 1-5, 13-15, 25, 47)
2. Create character/cultural images (prompts 6, 26-36, 55-60)
3. Generate historical/atmospheric images (prompts 37-46, 61-65)
4. Review and regenerate any that don't match mood board
5. Create variations of strongest images for social media
