# Episode 005: Zhangjiajie — Visual Mood Board

## Visual Direction & Design Guide

*Misty greens, grays, towering verticality, ethereal atmosphere*

---

## CORE AESTHETIC

### Visual Keywords
```
ETHEREAL    MISTY    VERTICAL    ANCIENT    SUBLIME
CONTEMPLATIVE    LUMINOUS    TIMELESS    SACRED    FLOATING
```

### Emotional Tone
- **Primary:** Awe, wonder, humility
- **Secondary:** Solitude, peace, introspection
- **Tertiary:** Adventure, discovery, courage

### Visual Philosophy
> "The void is as important as the form."

Chinese scroll painting aesthetic — negative space (mist, sky) carries as much meaning as the pillars themselves. Less is more. Restraint over excess.

## COLOR GRADING PRESETS

### For Photography (Lightroom/Capture One)

#### "Floating Peaks" Base Grade
```
Exposure: -0.3
Contrast: -10
Highlights: -25
Shadows: +15
Whites: -10
Blacks: +5

Temperature: -5 (slightly cool)
Tint: +3 (slight magenta for warmth)

Vibrance: -10
Saturation: -5

Texture: +10
Clarity: -5 (soften for ethereal feel)
Dehaze: -15 (enhance mist)

HSL Adjustments:
- Greens: Shift toward yellow, reduce saturation
- Blues: Reduce saturation, increase luminance
- Oranges: Increase luminance (skin tones, stone warmth)
```

#### "Golden Hour" Variation
```
Temperature: +15 (warmer)
Highlights: +10 (enhance sun)
Shadows: +25 (lift for detail)
Vibrance: +15

Split Toning:
- Highlights: Warm orange (#D4A574), 15%
- Shadows: Cool blue-gray, 10%
```

#### "Misty Morning" Variation
```
Exposure: +0.5
Contrast: -20
Dehaze: -25 (maximum mist effect)
Clarity: -15
Highlights: -30
Whites: -20

Blacks: +15 (lift for dreamy feel)
```

### For Video (LUT Recommendations)

#### Recommended LUTs
1. **Koji Advance SI** — Cinematic, soft contrast
2. **FilmConvert Kodak 2383** — Filmic, warm highlights
3. **Custom "Zhangjiajie Mist"** — See below

#### Custom LUT Notes
```
- Lift blacks slightly (dreamy, not harsh)
- Soft highlight roll-off (protect sky detail)
- Green shift in midtones (vegetation)
- Cool shadows, warm highlights (complementary)
- Reduce overall saturation 10-15%
- Add subtle film grain (0.5-1.0%)
```

## TYPOGRAPHY

### Primary Typeface: **Cormorant Garamond**
**Style:** Elegant, literary, timeless  
**Weights:** Light, Regular, Semi-Bold  
**Usage:** Titles, pull quotes, episode numbers

**Why:** Evokes book pages, literary quality, pairs beautifully with landscape imagery

### Secondary Typeface: **Inter**
**Style:** Clean, modern, highly readable  
**Weights:** Regular, Medium, Semi-Bold  
**Usage:** Body text, captions, UI elements

**Why:** Neutral, doesn't compete with imagery, excellent legibility

### Accent Typeface: **Cinzel**
**Style:** Classical, monumental, epic  
**Weights:** Regular, Semi-Bold  
**Usage:** Episode titles, dramatic moments

**Why:** Carves like stone, matches the monumental subject matter

### Typography Hierarchy

```
EPISODE TITLE (Cinzel Semi-Bold, 48pt)
"Floating Peaks"

SUBTITLE (Cormorant Garamond Italic, 24pt)
"Episode 005 of The Infinite Walk"

BODY TEXT (Inter Regular, 16pt)
Main narrative content, show notes

CAPTIONS (Inter Light, 14pt)
Image descriptions, credits

PULL QUOTE (Cormorant Garamond Semi-Bold, 28pt)
"Part of me stayed on the mountain."
```

### Text Treatment

#### On Light Backgrounds (Mist/Sky)
```
Color: #2D3748 (dark gray, not pure black)
Shadow: None or very subtle
Tracking: +25 (slight breathing room)
```

#### On Dark Backgrounds (Stone/Forest)
```
Color: #F7FAFC (off-white, not pure white)
Shadow: 0 2px 4px rgba(0,0,0,0.5)
Tracking: 0 (standard)
```

#### Overlay on Images
```
Background: Gradient overlay (transparent to 60% black)
Text: White with subtle drop shadow
Position: Rule of thirds, avoid covering key subjects
```

## COMPOSITION GUIDELINES

### Framing Principles

#### 1. Vertical Dominance
- Emphasize height over width
- Use 4:5, 9:16, or 21:9 aspect ratios
- Pillars should feel impossibly tall

#### 2. Layered Depth
- Foreground: Trees, railings, human element
- Midground: Primary pillars
- Background: Distant peaks, mist, sky
- Create 3+ distinct layers

#### 3. Negative Space
- Leave 40-60% of frame as "void" (mist, sky)
- Don't fear emptiness
- The absence is as important as the presence

#### 4. Human Scale
- Include human figures sparingly
- Make them small against the landscape
- Emphasizes the sublime scale

### Rule of Thirds — Zhangjiajie Edition

```
┌─────────┬─────────┬─────────┐
│   Sky   │  Sky    │  Sky    │
│         │         │         │
├─────────┼─────────┼─────────┤
│ Pillar  │ Pillar  │ Pillar  │  ← Horizon line LOW (bottom third)
│         │         │         │
├─────────┼─────────┼─────────┤
│ Mist    │ Mist    │ Mist    │
│ Ground  │ Ground  │ Ground  │
└─────────┴─────────┴─────────┘
```

**Why:** Emphasizes the vertical reach of pillars, gives them "room to grow"

### Leading Lines

- **Natural:** Pillar edges, tree lines, mist layers
- **Built:** Boardwalks, cable cars, bridge railings
- **Purpose:** Guide eye through frame, create journey

## IMAGERY STYLES

### Photography Style

#### Characteristics
- **Lighting:** Soft, diffused, golden hour preferred
- **Contrast:** Low to medium (harsh contrast kills mist)
- **Saturation:** Muted, natural (no oversaturated greens)
- **Sharpness:** Moderate (some softness enhances ethereal quality)
- **Grain:** Subtle film grain adds texture and warmth

#### Shot Types

| Type | Description | Usage |
|------|-------------|-------|
| **Establishing** | Wide panorama of pillar forest | Openings, transitions |
| **Hero** | Single iconic pillar (Hallelujah) | Cover art, key visuals |
| **Detail** | Rock texture, vegetation, water | Intimate moments |
| **Human** | Small figure against landscape | Emotional connection |
| **Action** | Walking, climbing, Glass Bridge | Adventure elements |
| **Atmospheric** | Mist, clouds, weather | Mood, transitions |

### Illustration Style (if used)

#### Characteristics
- **Medium:** Digital watercolor or ink wash
- **Influence:** Chinese scroll painting (shan shui)
- **Palette:** Restricted (grays, greens, gold accents)
- **Detail:** Suggestive, not photorealistic
- **Space:** Generous negative space

#### When to Use
- Chapter/section breaks
- Abstract concepts (time, floating)
- When photography unavailable
- Stylized social content

## MOTION & ANIMATION

### Video Motion Principles

#### 1. Slow & Deliberate
- **Pace:** Slower than typical travel content
- **Transitions:** Gentle fades, dissolves (no hard cuts unless dramatic)
- **Movement:** Slow pans, gradual push-ins

#### 2. Mist as Character
- Capture mist movement (time-lapse or real-time)
- Use mist for natural transitions
- Let mist reveal/hide subjects

#### 3. Human Moments
- Brief moments of stillness (person looking out)
- Natural movement (walking, not posing)
- Authentic reactions (Glass Bridge)

### Animation Elements

#### Subtle Animations
```
- Gentle fade-ins (1-2 seconds)
- Slow scale (100% → 105% over 10 seconds)
- Mist overlay (semi-transparent, slow drift)
- Light leak (warm, subtle, brief)
```

#### Text Animation
```
- Fade + slight upward drift
- Letter spacing animation (subtle)
- Never animate more than 2 elements at once
```

#### Transitions
```
- Mist wipe (custom mask)
- Light leak fade
- Match cut (similar shapes)
- Slow cross-dissolve (1-2 seconds)
```

## DO'S & DON'TS

### ✅ DO
- Embrace negative space
- Keep colors muted and natural
- Include human scale (sparingly)
- Let mist be a character
- Use vertical compositions
- Maintain series consistency
- Prioritize emotion over perfection
- Leave room for text (if needed)

### ❌ DON'T
- Oversaturate greens
- Use harsh contrast
- Fill every inch of frame
- Add too many text elements
- Use generic travel photography clichés
- Ignore series branding
- Rush the editing process
- Forget mobile viewing (check at small sizes)

## FINAL NOTES

### The Zhangjiajie Feeling

> "When you look at these images, you should feel:
> - Small (in a good way)
> - Quiet (even with sound on)
> - Like you're breathing slower
> - Like time is moving differently
> - Like you're somewhere sacred"

### Consistency Across Episode 005

Every visual element should feel like it belongs to the same world:
- Same color grade
- Same typography
- Same compositional approach
- Same emotional tone

**Exception:** Episode 006 (Antarctica) should feel like a COMPLETE contrast — cold vs. warm, white vs. green, horizontal vs. vertical. This intentional shift reinforces the "Infinite Walk" concept.
