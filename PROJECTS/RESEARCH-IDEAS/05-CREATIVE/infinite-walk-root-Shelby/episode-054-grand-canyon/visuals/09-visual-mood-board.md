## TYPOGRAPHY

### Primary Font: Headlines
```
Font: Montserrat (Google Fonts)
Weights: 700 (Bold), 800 (Extra Bold)
Usage: Episode titles, section headers, social graphics

Why: Clean, modern, highly readable, works well with natural imagery
Alternatives: Oswald, Bebas Neue, Playfair Display
```

### Secondary Font: Body Text
```
Font: Lato (Google Fonts)
Weights: 400 (Regular), 700 (Bold)
Usage: Body copy, captions, descriptions

Why: Friendly, readable at small sizes, pairs well with Montserrat
Alternatives: Open Sans, Source Sans Pro, Roboto
```

### Accent Font: Quotes & Poetry
```
Font: Cormorant Garamond (Google Fonts)
Weights: 400 (Regular), 600 (Semi Bold)
Usage: Quote cards, poetic sections, special emphasis

Why: Elegant, literary, evokes timelessness and depth
Alternatives: Playfair Display, Merriweather, Crimson Text
```

### Font Pairing Examples
```
HEADLINE: Montserrat Bold, 48px, #2C3E50
Body: Lato Regular, 18px, #2C3E50
Quote: Cormorant Garamond Semi Bold, 24px, #C1440E
```

## IMAGERY DIRECTION

### Photography Style
```
Mood: Cinematic, dramatic, warm
Lighting: Golden hour preferred (sunrise/sunset)
Composition: Wide landscapes with human element for scale
Color Grade: Warm tones enhanced, blues deepened, shadows lifted

Key Shots:
- Grand Canyon from Mather Point at sunrise
- Delicate Arch framing sunset
- Bryce hoodoos with long shadows
- Zion's Great White Throne from below
- Colorado River from above (drone)
- Human silhouette at canyon rim (scale)
```

### Human Element
```
Approach: Small figures in vast landscapes
Purpose: Show scale, evoke emotion, invite viewer into scene
Placement: Rule of thirds, often silhouetted
Shelby Character: Friendly, curious, appropriately scaled

AI Prompt Examples:
- "Tiny hiker silhouette at Grand Canyon rim, vast canyon below, sunrise, epic scale"
- "Person standing beneath Delicate Arch, looking up, sense of wonder, sunset"
- "Shelby robot character at Bryce Canyon viewpoint, hoodoos stretching to horizon, golden hour"
```

### Detail Shots
```
Subjects:
- Rock layer close-ups (show geology)
- Hoodoo textures
- Arch details
- Desert plants (cacti, juniper)
- Water reflections
- Shadows and light patterns

Style: Macro or tight framing, sharp focus, rich colors
Usage: Social media carousels, blog illustrations, section breaks
```

## VIDEO MOTION DIRECTION

### Transitions
```
Fade Through White: Clean, hopeful, new sections
Slow Zoom In: Drawing viewer into scene, intimacy
Pan Across Landscape: Establishing scale, journey feeling
Layer Reveal: Mimicking geologic layers, educational

Timing: Slow (2-3 seconds), contemplative, not rushed
```

### Text Animation
```
Entrance: Fade in + slight scale up (subtle)
Emphasis: Gentle pulse or color shift
Exit: Fade out (never slide out—too harsh)

Timing: Match narration pace, never distract
```

### B-Roll Suggestions
```
- Drone flyover of canyon (establishing)
- Time-lapse of clouds over rim (passing of time)
- Water flowing in river (patience, persistence)
- Shadows moving across formations (time passing)
- Hiker's feet on trail (journey, walking)
- Sunrise/sunset time-lapse (transformation)
- Raven soaring (wildlife, freedom)
```

## ACCESSIBILITY GUIDELINES

### Color Contrast
```
Minimum Ratios (WCAG AA):
- Normal text: 4.5:1 contrast ratio
- Large text (18px+): 3:1 contrast ratio
- UI elements: 3:1 contrast ratio

Tools to Check:
- WebAIM Contrast Checker
- Stark (Figma/Sketch plugin)
- Color Oracle (color blindness simulator)

Safe Combinations:
✅ #2C3E50 (Twilight Blue) on #FFFFFF (White) = 12.5:1
✅ #FFFFFF (White) on #8B3103 (Rust Red) = 8.2:1
✅ #2C3E50 (Twilight Blue) on #F4A460 (Desert Orange) = 5.1:1

Avoid:
❌ #D97B2F (Sandstone Orange) on #FFFFFF (White) = 2.1:1 (too low)
❌ #9CAF88 (Desert Sage) on #BCA88E (Taupe) = 1.3:1 (too low)
```

### Alt Text Guidelines
```
Format: [Subject] + [Action/Context] + [Setting] + [Mood/Color]

Examples:
- "Grand Canyon at sunrise from Mather Point, golden light on red rock layers, Colorado River visible in distance"
- "Delicate Arch glowing orange at sunset, La Sal Mountains in background, deep blue sky"
- "Bryce Canyon hoodoos with long shadows at sunset, orange and red sandstone spires"
- "Shelby robot character standing at canyon rim, looking out over vast landscape, warm light"

Avoid:
- "Image of canyon" (too vague)
- "Beautiful sunset" (subjective)
- Starting with "Image of..." (redundant)
```

### Font Sizes (Minimum)
```
Social Media Graphics:
- Headlines: 36px minimum
- Body text: 18px minimum
- Captions: 14px minimum

Video:
- On-screen text: 24px minimum (for mobile)
- Subtitles: 18px minimum, high contrast

Web:
- Headlines: 24px minimum
- Body: 16px minimum
- Small text: 12px minimum
```
