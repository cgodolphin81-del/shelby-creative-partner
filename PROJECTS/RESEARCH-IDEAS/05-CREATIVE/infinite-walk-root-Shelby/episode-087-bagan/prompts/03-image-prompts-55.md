## Usage Notes

### Aspect Ratios
- **16:9** - YouTube, presentations, wide landscapes
- **3:2** - Standard photography, Instagram landscape
- **4:5** - Instagram portrait, mobile-first
- **1:1** - Instagram square, profile images
- **2:1** - Cinematic panoramas, hero images
- **3:1** - Ultra-wide banners, headers

### Best For
- **Prompts 1-12:** Sunrise/hero shots, episode thumbnails
- **Prompts 13-24:** Temple details, educational content
- **Prompts 25-32:** Adventure/balloon content, premium experiences
- **Prompts 33-40:** River scenes, cultural context
- **Prompts 41-50:** Shelby host imagery, personal connection
- **Prompts 51-60:** Details, atmosphere, social media fill
- **Prompts 61-65:** Epic vistas, video backgrounds
- **Prompts 66-70:** Seasonal variety, special content

### Style Notes
- All prompts optimized for Midjourney v6
- Adjust `--v` parameter for different versions
- Add `--stylize 250` for more artistic interpretation
- Add `--quality 2` for highest detail
- Use `--chaos 20` for more variation in results
