## USAGE NOTES

### For Midjourney v6:
- Add `--ar 16:9` or appropriate aspect ratio
- Add `--v 6.0` for version
- Add `--style raw` for photorealistic results
- Consider `--s 250` for stylization balance

### For DALL-E 3:
- Prompts work as-is
- Specify "photorealistic" or "illustration" as needed
- Request specific aspect ratios in follow-up

### For Stable Diffusion XL:
- Add negative prompts: "blurry, low quality, distorted"
- Use appropriate checkpoints for landscapes
- Consider ControlNet for composition

### Batch Processing:
- These 100 prompts can be processed in batches of 10
- Maintain consistent style across each location's set
- Consider creating a montage image combining all 10 locations
