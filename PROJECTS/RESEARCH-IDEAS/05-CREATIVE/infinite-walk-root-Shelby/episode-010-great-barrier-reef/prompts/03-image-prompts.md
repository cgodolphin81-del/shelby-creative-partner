## USAGE NOTES

**Aspect Ratios:**
- `--ar 16:9` for landscape (essays, web, video thumbnails)
- `--ar 4:3` for standard photography feel
- `--ar 1:1` for social media squares

**Style Keywords:**
- "National Geographic style" for documentary realism
- "Cinematic" for dramatic lighting
- "Macro photography" for close-ups
- "Wide angle" for environmental shots

**Quality Settings (Midjourney):**
- `--v 6` for latest version
- `--style raw` for more photographic results
- `--q 2` for higher quality (uses more GPU)

**Variations:**
- Generate 4 variations per prompt
- Select best, then create variations of that
- Mix elements between prompts for custom scenes
