## USAGE NOTES

**Aspect Ratios:**
- `--ar 16:9` - Standard video/cinematic
- `--ar 3:4` - Social media portrait
- `--ar 4:5` - Instagram portrait
- `--ar 9:16` - Stories/vertical video
- `--ar 2:3` - Classic photo ratio

**Style Parameters:**
- `--style raw` - More photorealistic, less stylized
- Remove `--style raw` for more artistic interpretation

**Best Practices:**
- Use consistent seed values for character consistency (Shelby prompts)
- Generate multiple variations per prompt
- Upscale final selections for print quality
- Consider Midjourney v6 or DALL-E 3 for best textural detail

**Color Palette Reference:**
- Rose reds: #C45B5B, #B8545B, #A64A4A
- Sand golds: #D4A574, #C9A26F, #B8956A
- Canyon oranges: #D97B4E, #CC6E42, #BF6238
- Sky blues: #5B8DB8, #4A7DA8, #3D6F99
