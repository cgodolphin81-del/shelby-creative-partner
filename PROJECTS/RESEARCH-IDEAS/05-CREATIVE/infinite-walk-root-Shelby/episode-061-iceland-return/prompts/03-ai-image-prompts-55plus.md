## USAGE NOTES

### Aspect Ratios
- **16:9**: YouTube, blog headers, landscape prints
- **3:2**: Instagram landscape, standard photography
- **4:3**: Traditional photography, presentations
- **1:1**: Instagram square, profile images
- **9:16**: Instagram Stories, TikTok, mobile vertical

### Style Consistency
- Maintain volcanic color palette (lava reds, black sands, ocean blues, sky greys)
- Keep Shelby character consistent (friendly, curious, wearing signature scarf)
- Balance dramatic landscapes with intimate wildlife moments
- Include both historical/educational and artistic/emotional images

### Prompt Customization
- Adjust `--v 6` to your preferred Midjourney version
- Modify `--ar` for different platform requirements
- Add `--stylize` values for more/less artistic interpretation
- Use `--chaos` for more variation in results
