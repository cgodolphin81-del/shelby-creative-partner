**Hook B (Statistic):**
> "Two and a half million birds. Seventy-five percent of an entire species. One lake. Zero backup plans."

**Hook C (Question):**
> "What would it be like to have no backup plan? To be so perfectly adapted to one place that nowhere else would do?"

## A/B TESTING STRATEGY

### Week 1: Title Testing
- **Platform:** Instagram (posts + stories)
- **Method:** Post same image with different titles in caption
- **Metric:** Saves + shares (deeper engagement than likes)
- **Sample Size:** Minimum 3 titles, 48 hours each

### Week 2: Hook Testing
- **Platform:** TikTok + Reels
- **Method:** Same footage, different opening hooks
- **Metric:** Watch time + completion rate
- **Sample Size:** 3 hooks, track drop-off points

### Week 3: Thumbnail Testing
- **Platform:** YouTube + Blog
- **Method:** Same content, different featured images
- **Metric:** Click-through rate
- **Sample Size:** 2-3 thumbnails per platform
