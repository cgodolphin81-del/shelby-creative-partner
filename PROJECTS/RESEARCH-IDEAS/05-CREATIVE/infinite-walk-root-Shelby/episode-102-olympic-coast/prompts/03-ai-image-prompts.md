## PROMPT USAGE NOTES

**Recommended Aspect Ratios:**
- `--ar 16:9` - Standard landscape, YouTube thumbnails, blog headers
- `--ar 9:16` - Vertical for Stories, Reels, TikTok
- `--ar 1:1` - Instagram feed posts
- `--ar 4:5` - Portrait orientation, Pinterest
- `--ar 21:9` - Ultra-wide panoramas
- `--ar 3:2` - Traditional photography ratio

**Style Modifiers:**
- Add `--v 6` for Midjourney v6
- Add `--stylize 250` for more artistic interpretation
- Add `--chaos 20` for more variety in results

**Quality Enhancements:**
- Add "8K ultra detailed" for maximum resolution
- Add "professional photography" for realistic style
- Add "fine art" for gallery-quality aesthetics
