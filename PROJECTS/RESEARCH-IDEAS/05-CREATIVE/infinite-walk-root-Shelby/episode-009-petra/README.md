## 🚀 Quick Start Guide

### To Publish This Episode:

**Step 1: Finalize Audio**
- Use `audio/04-audio-narration-script.md`
- Record with TTS or human voice
- Add music/sound design per notes
- Export as MP3 (15 minutes)

**Step 2: Generate Images**
- Use `prompts/03-image-prompts.md`
- Generate 10-20 key images (Treasury, Siq, Monastery, etc.)
- Apply color grading from `mood-board/09-visual-mood-board.md`

**Step 3: Upload Podcast**
- Title: "Episode 009: The Rose-Red City"
- Show notes: Use `essay/02-rose-red-city-essay.md` + travel guide link
- Distribute to all platforms (Apple, Spotify, etc.)

**Step 4: Create YouTube Video**
- Audio + image slideshow or video montage
- Title: "Petra: The Rose-Red City | Full Audio Documentary + Travel Guide"
- Description: Timestamps + links
- Thumbnail: Treasury at golden hour

**Step 5: Social Media Launch**
- Follow `08-publishing-strategy.md` timeline
- Use `social/06-social-media-snippets.md` for captions/scripts
- Post across Instagram, TikTok, Twitter/X, LinkedIn
- Send email sequence to list

**Step 6: Monitor & Engage**
- Track metrics per `08-publishing-strategy.md`
- Respond to comments/DMs
- Adjust strategy based on performance

**Step 7: Tease Episode 010**
- 3-4 days after launch, release tease from `tease/10-episode-010-tease.md`
- Build anticipation for Great Barrier Reef episode
