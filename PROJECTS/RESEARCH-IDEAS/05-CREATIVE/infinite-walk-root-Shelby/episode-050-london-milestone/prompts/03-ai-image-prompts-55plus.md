## USAGE NOTES

### Image Specifications
- **Primary Format**: 16:9 for episode thumbnails, social media headers
- **Square (1:1)**: Instagram posts, profile images
- **Vertical (9:16)**: Instagram Stories, TikTok, Pinterest
- **Resolution**: Generate at maximum available, upscale if needed

### Style Consistency
- All Shelby character images should use consistent design
- Maintain Pixar-style 3D render aesthetic for character shots
- Photorealistic for landmark photography
- Consistent color grading across London scenes (grey, red, blue, green palette)

### Recommended Generation Order
1. Landmark photography first (establishes visual tone)
2. Shelby character scenes (builds on established London aesthetic)
3. Mood board/abstract (for supplementary materials)

### File Naming Convention
```
ep050_[location]_[description]_[ratio].png
Example: ep050_bigben_goldenhour_16x9.png
Example: ep050_shelby_tower_3x2.png
```
