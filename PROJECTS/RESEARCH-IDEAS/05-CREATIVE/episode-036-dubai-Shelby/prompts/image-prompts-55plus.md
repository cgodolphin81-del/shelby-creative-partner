## USAGE NOTES

**Aspect Ratios:**
- 16:9 = YouTube, presentations, desktop
- 9:16 = TikTok, Reels, Stories (vertical)
- 1:1 = Instagram feed, profile
- 4:3 = Traditional photography, some social
- 21:9 = Cinematic, panoramic
- 3:2 = Classic photography ratio

**Style Tags:**
- `--v 6` = Midjourney version 6
- `--ar` = Aspect ratio
- Add `--style raw` for more photographic results
- Add `--q 2` for higher quality (uses more GPU)

**Customization:**
- Replace "Shelby" with specific character details as needed
- Adjust lighting times for different moods
- Mix and match elements from different prompts
- Use negative prompts (`--no`) to exclude unwanted elements
