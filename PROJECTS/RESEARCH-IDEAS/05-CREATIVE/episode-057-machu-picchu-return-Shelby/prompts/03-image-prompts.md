## 🎯 USAGE NOTES

### Prompt Customization

**For Midjourney:**
- Use `--v 6` for latest version
- Adjust `--ar` for your platform (16:9 for video, 1:1 for Instagram, 4:5 for portrait)
- Add `--style raw` for more photorealistic results
- Use `--chaos 20` for more variety in results

**For DALL-E 3:**
- Prompts work as-is
- Can add "photorealistic" or "painterly" based on desired style
- Specify "no text" if text appears unintentionally

**For Stable Diffusion:**
- May need negative prompts: "blurry, distorted, text, watermark"
- Adjust CFG scale (7-9 recommended)
- Use appropriate checkpoint for landscape/architecture

### Color Palette Reference

All prompts should maintain the Episode 057 mood board colors:
- **Stone Greys:** #3D3D3D, #6B6B6B, #8B8B8B
