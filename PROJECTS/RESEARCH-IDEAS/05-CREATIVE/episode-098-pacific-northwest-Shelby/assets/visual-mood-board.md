## COMPOSITION GUIDELINES

### Rule of Thirds
```
Apply to all photography and video
Place key elements (trees, person, focal points) at intersections
Horizon on upper or lower third, never center
```

### Leading Lines
```
Use trails, rivers, tree trunks to guide eye
Create depth and journey feeling
Lines should lead TO something, not off frame
```

### Framing
```
Use tree branches, ferns to frame subjects
Creates natural vignette
Adds depth and intimacy
```

### Scale
```
Include person or known object for scale
Ancient trees need reference to show immensity
Small human = vast forest (humility theme)
```

### Negative Space
```
Allow for silence in composition
Don't fill every inch
Let the forest breathe
```

## PHOTOGRAPHY STYLE

### Lighting
```
Preferred: Soft, diffused, overcast
Golden Hour: Warm, directional, magical
Blue Hour: Cool, mysterious, peaceful
Avoid: Harsh midday sun (creates harsh shadows)
```

### Mood
```
Contemplative, reverent, peaceful
Not: Adventurous, extreme, adrenaline
Yes: Slow, mindful, present
```

### Color Grading
```
Push greens toward emerald/jade
Keep browns warm, not orange
Desaturate slightly for timeless feel
Add subtle teal to shadows
Protect highlights from clipping
```

### Focus
```
Deep focus for landscapes (everything sharp)
Shallow focus for details (moss macro)
Focus stacking for critical shots
```

---

## VIDEO STYLE

### Movement
```
Slow, deliberate camera movements
No quick cuts or jarring transitions
Gentle pans, slow push-ins
Let shots breathe (hold for 5-10 seconds)
```

### Transitions
```
Cross-dissolves (1-2 seconds)
Fade to white (for section breaks)
Match cuts (similar shapes/movement)
Avoid: Whip pans, zoom transitions, flashy effects
```

### Frame Rate
```
24fps for cinematic feel
60fps for slow-motion (moss, water, leaves)
Variable: Match mood of scene
```

### Aspect Ratios
```
16:9 for YouTube/main video
9:16 for TikTok/Reels/Stories
1:1 for Instagram feed (when needed)
4:5 for Instagram portrait posts
```

## AI IMAGE GENERATION GUIDELINES

### Prompt Structure
```
[Subject] + [Environment] + [Lighting] + [Mood] + [Style] + [Color Palette] + [Technical]

Example:
"Ancient Sitka spruce tree (subject) in Hoh Rainforest (environment) with soft filtered morning light (lighting) creating reverent atmosphere (mood) photorealistic nature photography (style) forest greens and moss greens (color palette) 8k detail, cinematic composition (technical)"
```

### Consistency Tips
```
- Use same seed for related images
- Specify "Olympic National Park" for geographic accuracy
- Include "temperate rainforest" to avoid tropical confusion
- Add "Pacific Northwest" for regional specificity
- Use color hex codes in prompts when possible
```

### Common Mistakes to Avoid
```
- Tropical rainforest elements (palms, vines)
- Incorrect tree species (no pines, use spruce/fir/cedar)
- Too much direct sunlight (should be filtered)
- Over-saturated colors (keep natural)
- People too prominent (forest is the subject)
```

## SOCIAL MEDIA TEMPLATES

### Instagram Post (1080x1080)
```
Layout Options:
A) Full-bleed image with text overlay (bottom third)
B) Image with solid color bar (bottom 20%) for text
C) Carousel: Image + quote + call-to-action

Text: White or light grey on dark areas
Font: Cormorant Garamond for title, Inter for body
Logo: Small, bottom corner (optional)
```

### Instagram Story (1080x1920)
```
Layout: Vertical, full-bleed
Text: Top or bottom third (avoid center for tap-through)
Interactive: Polls, questions, link stickers
Duration: 3-5 seconds per frame minimum
```

### TikTok/Reels (1080x1920)
```
Safe Zones:
- Top 20%: Avoid (UI elements)
- Bottom 20%: Avoid (captions, buttons)
- Center 60%: Safe for key content

Text: Large, high contrast, short duration
Captions: Always include (80% watch without sound)
```

### YouTube Thumbnail (1280x720)
```
Text: Large, high contrast, 3-5 words max
Image: High impact, clear focal point
Branding: Consistent logo placement
Colors: Push contrast for small preview
```

### Pinterest Pin (1000x1500)
```
Ratio: 2:3 (vertical)
Text: Clear, readable at small size
Branding: Website/handle at bottom
Colors: Bright enough to stand out in feed
```
