## Common Pitfalls to Avoid

### ❌ Pricing Too Low
```
Problem: Undercutting to win work
Solution: Price on value, not hours. Minimum $50K projects.
```

### ❌ Taking Any Client
```
Problem: Working with difficult/misaligned clients
Solution: Ideal client profile. Say no to bad fits.
```

### ❌ Over-Hiring Too Soon
```
Problem: Fixed costs before revenue stability
Solution: Use freelancers first. Hire at 70%+ utilization.
```

### ❌ Ignoring Cash Flow
```
Problem: Profitable but no cash
Solution: 30% upfront, milestone billing, watch AR closely.
```

### ❌ Chasing Every Trend
```
Problem: Diluted positioning
Solution: Pick 2-3 specialties, become known for those.
```

### ❌ Skipping Contracts
```
Problem: Scope creep, payment issues
Solution: Always use signed SOW with clear terms.
```

## Your First 10 Clients

### Where to Find Them

**Warm Network (Highest Conversion)**
```
□ Former colleagues
□ Past clients
□ Industry contacts
□ LinkedIn connections
□ Friends/family (for intros, not work)
```

**Cold Outreach (Requires Volume)**
```
□ LinkedIn decision-makers
□ Company email addresses
□ Event attendees
□ Community members
□ Referral partners
```

**Inbound (Best Long-Term)**
```
□ Content marketing
□ SEO
□ Speaking engagements
□ Social media presence
□ PR/media coverage
```

### Ideal First Client Profile

```
✓ Budget: $50K-150K (can afford you, not too big)
✓ Industry: You have relevant experience
✓ Decision Maker: Accessible, decisive
✓ Timeline: 60-90 days (not rushed)
✓ Expectations: Realistic, collaborative
✓ Reference Potential: Will give testimonial
```

## Pricing Your First Projects

### Don't Undersell Yourself

```
Minimum Project: $50K
├── Anything less isn't worth the overhead
├── Exception: Strategic portfolio pieces
└── Exception: Nonprofit/education (discount ok)

Target Average: $100K-150K
├── Sustainable business model
├── Allows for quality delivery
└── Room for profit and growth
```

### Payment Terms

```
Standard Terms:
├── 30% upon signing
├── 30% at design approval
├── 30% at development complete
└── 10% at launch

For New Clients:
├── Consider 50% upfront
├── Or require credit card on file
└── Or use escrow for large projects
```

## Your First Team

### Hire in This Order

1. **Technical Lead/CTO** (if not you)
   - Can architect and build
   - Can manage other devs
   - Equity + salary

2. **3D Artist** (generalist first)
   - Can model, texture, optimize
   - Blender or Maya expert
   - Portfolio required

3. **Project Manager**
   - Keeps projects on track
   - Client communication
   - Agency experience preferred

4. **Business Development**
   - Fills the pipeline
   - Commission-heavy comp
   - Existing network a plus

### Freelance First Strategy

```
Before Full-Time Hire:
├── Use freelancers for overflow
├── Test working relationship
├── Prove consistent demand
└── Then convert to full-time

Benefits:
├── Lower fixed costs
├── Flexibility
├── Access to specialized skills
└── Easier to scale up/down
```

## When Things Go Wrong

### Client Wants to Cancel
```
→ Review contract terms
→ Document work completed
→ Invoice for work done
→ Offer pause option if appropriate
→ Learn why and fix process
```

### Project Going Over Budget
```
→ Identify cause immediately
→ Communicate with client
→ Submit change order
→ Adjust future estimates
→ Improve scoping process
```

### Can't Make Payroll
```
→ Talk to clients about early payment
→ Use line of credit if available
→ Negotiate payment plans with vendors
→ Be transparent with team
→ Never miss payroll if possible
```

### Key Person Leaves
```
→ Have documentation in place
→ Cross-train team members
→ Maintain freelancer bench
→ Start recruiting immediately
→ Conduct exit interview
```
