# Email Marketing Implementation Checklist

30-day rollout plan for launching or optimizing email marketing programs.

---

## Overview

**Timeline:** 30 days (4 weeks)  
**Goal:** Full email marketing program launch  
**Success Criteria:** All systems operational, first campaigns sent, tracking active

---

## Week 1: Foundation & Setup

### Day 1-2: Discovery & Planning

**Kickoff Meeting**
- [ ] Introduction meeting with client team
- [ ] Review completed onboarding form
- [ ] Discuss goals, KPIs, and expectations
- [ ] Identify key stakeholders and approval process
- [ ] Review brand guidelines and assets

**Account Setup**
- [ ] Create project in project management tool
- [ ] Set up shared folder (Google Drive/Dropbox)
- [ ] Create communication channels (Slack/email)
- [ ] Schedule recurring meetings (weekly check-ins)
- [ ] Share contact information and escalation paths

**Deliverables:**
- [ ] Meeting notes documented
- [ ] Project timeline shared
- [ ] Communication plan established
- [ ] Asset request list sent to client

---

### Day 3-4: Technical Setup

**ESP Account Setup**
- [ ] ESP account created or accessed
- [ ] User permissions configured
- [ ] Billing information entered
- [ ] Account settings reviewed and optimized

**Domain Authentication**
- [ ] Sending domain identified
- [ ] SPF record created in DNS
- [ ] DKIM keys generated and DNS record created
- [ ] DMARC record created (monitoring mode)
- [ ] Authentication verified in ESP
- [ ] Wait 24-48 hours for DNS propagation

**Tracking Setup**
- [ ] Google Analytics account accessed
- [ ] UTM parameters template created
- [ ] Conversion goals configured in GA
- [ ] Facebook Pixel installed (if applicable)
- [ ] Custom conversion tracking planned

**Deliverables:**
- [ ] DNS records documented
- [ ] Authentication verification screenshots
- [ ] Tracking plan documented
- [ ] UTM parameter template

---

### Day 5: List Preparation

**List Audit**
- [ ] Current list exported and analyzed
- [ ] List size confirmed
- [ ] Engagement segments identified
- [ ] Hard bounces removed
- [ ] Unsubscribes processed
- [ ] Spam complaints reviewed

**List Import**
- [ ] List formatted for ESP import
- [ ] Custom fields mapped (name, preferences, etc.)
- [ ] Tags/segments planned
- [ ] Test import with small sample
- [ ] Full list imported
- [ ] Import verification completed

**Compliance Check**
- [ ] Opt-in method documented
- [ ] Privacy policy reviewed
- [ ] Unsubscribe process tested
- [ ] Physical address added to account
- [ ] GDPR compliance verified (if applicable)

**Deliverables:**
- [ ] List audit report
- [ ] Import confirmation
- [ ] Compliance checklist completed

## Risk Mitigation

| Risk | Mitigation |
|------|------------|
| Low deliverability | Warm-up domain, authenticate properly, start with engaged segments |
| Poor engagement | A/B test subject lines, segment appropriately, provide value |
| High unsubscribes | Review frequency, content relevance, and targeting |
| Technical issues | Test thoroughly before launch, have backup plans |
| Client delays | Build buffer time into timeline, communicate early |
