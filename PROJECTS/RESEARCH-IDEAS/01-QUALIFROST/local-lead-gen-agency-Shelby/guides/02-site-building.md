## 📦 Step 1: Domain & Hosting

### Domain Selection

**Best Practices:**
- Include niche + city: `DenverRoofingPros.com`, `PhoenixHVACExperts.com`
- Keep it under 15 characters if possible
- Use .com only (avoid .net, .org for local business)
- No hyphens or numbers
- Easy to spell and remember

**Domain Ideas by Niche:**
| Niche | Pattern Examples |
|-------|------------------|
| HVAC | [City]HVACPros.com, [City]HeatingAndAir.com |
| Plumbing | [City]PlumbingExperts.com, [City]PlumberNow.com |
| Roofing | [City]RoofingCo.com, [City]RoofRepair.com |
| Tree Service | [City]TreeService.com, [City]TreeExperts.com |

**Tools:**
- Namecheap (domains)
- GoDaddy (domains - use for domains only)
- LeanDomainSearch (domain ideas)

## ✍️ Step 4: Content Guidelines

### Homepage Copy Template

**Headline Formula:**
```
[Emergency/Best] [Service] in [City] | [Unique Value Proposition]
```
*Examples:*
- "Emergency Plumbing in Denver | 30-Minute Response Time"
- "Best Roofing Company in Phoenix | 25-Year Warranty"

**Subheadline:**
```
[Adjective] [service] for [target customer]. [Credibility statement]. [CTA].
```
*Example:*
- "Professional plumbing services for Denver homeowners. Licensed, insured, and available 24/7. Call now for immediate service!"

### Service Page Template

Each service page should include:

1. **H1**: Service Name + City
2. **Introduction**: 100-150 words about the service
3. **Common Problems**: What issues does this service solve?
4. **Our Process**: How you handle the service
5. **Pricing**: Starting at $X or "Free estimates"
6. **FAQ**: 3-5 service-specific questions
7. **CTA**: Phone + form

**Word Count**: 800-1,500 words per service page

### Content Best Practices

✅ **DO:**
- Write for humans first, Google second
- Include city/neighborhood names naturally
- Answer common customer questions
- Use bullet points for readability
- Include clear CTAs throughout
- Add schema markup (RankMath handles this)

❌ **DON'T:**
- Keyword stuff
- Use generic AI content without editing
- Copy competitor content
- Forget mobile optimization
- Skip proofreading

## 🗺️ Step 5: Local SEO Setup

### On-Page SEO Checklist

**Every Page Needs:**

- [ ] Target keyword in H1
- [ ] Keyword in first 100 words
- [ ] Keyword in URL slug
- [ ] Meta title (55-60 characters)
- [ ] Meta description (150-160 characters)
- [ ] 2-3 internal links
- [ ] 1-2 external links to authority sites
- [ ] Optimized images (compressed, alt text)
- [ ] Schema markup (via RankMath)
- [ ] Mobile-responsive design
- [ ] Fast load time (<3 seconds)

### Meta Title Templates

```
[Service] in [City] | [Company Name]
[Emergency Service] [City] | 24/7 Available
Best [Service] [City] | Free Estimates
```

### Meta Description Templates

```
Looking for [service] in [City]? [Company] provides [key benefit]. 
Licensed, insured, [unique value]. Call [phone] for free estimate!

Emergency [service] in [City]? We're available 24/7 with [unique value]. 
Trusted by 500+ customers. Call now: [phone]
```

### Local Schema Markup

Use RankMath's Local Business schema:

```
Business Type: [Specific type, e.g., "PlumbingService"]
Name: [Your site name]
Address: [Service area - can be general]
Phone: [Tracking number]
Email: [Contact email]
Hours: [Business hours]
Service Areas: [List of cities]
Price Range: $$
```

## 📍 Step 6: Google Business Profile (GBP)

### GBP Setup Process

**Important**: You can create a GBP for a lead gen site, but be aware of Google's guidelines. Many lead gen operators use "service area business" settings.

### Step-by-Step Setup:

1. **Go to**: google.com/business
2. **Click**: "Manage now"
3. **Enter business name**: Use your site/brand name
4. **Business type**: Select "Service Area Business"
5. **Service areas**: Add cities you're targeting
6. **Contact info**: 
   - Phone: Use your tracking number
   - Website: Your lead gen site
   - Email: Professional email
7. **Business hours**: Set accurate hours (or 24/7 for emergency)
8. **Verification**: Usually by postcard (takes 5-7 days)

### GBP Optimization Checklist

- [ ] Complete ALL profile fields
- [ ] Add 10+ high-quality photos
- [ ] Write detailed business description (750 characters)
- [ ] Add all services with descriptions
- [ ] Set accurate business hours
- [ ] Enable messaging
- [ ] Add products (if applicable)
- [ ] Create weekly posts
- [ ] Respond to all reviews
- [ ] Add Q&A (seed common questions)

### Photo Requirements

| Type | Quantity | Notes |
|------|----------|-------|
| Logo | 1 | Professional, square |
| Cover | 1 | 1080x608px, showcase work |
| Interior | 3-5 | Office/showroom if applicable |
| Exterior | 3-5 | Building/vehicle with branding |
| Team | 3-5 | Uniformed staff (stock OK initially) |
| Work | 10+ | Before/after, in-progress shots |

### GBP Posts Strategy

**Post weekly with these content types:**

1. **Update Posts**: Company news, service updates
2. **Offer Posts**: Special discounts, seasonal promos
3. **Event Posts**: Community involvement
4. **Product Posts**: Featured services

**Template:**
```
🔧 [Service] Special in [City]!

This month only: [Offer - e.g., 15% off AC tune-ups]

✅ Licensed & Insured
✅ 5-Star Rated
✅ Free Estimates

Call [phone] or visit [website]

#CityService #LocalBusiness
```
