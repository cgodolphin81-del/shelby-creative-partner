# SOP: Campaign Launch Checklist

**Standard Operating Procedure**

---

## Overview

**SOP Name:** Campaign Launch Checklist  
**Department:** Paid Media  
**Version:** 1.0  
**Last Updated:** March 2026  
**Owner:** Media Buying Team

---

## Purpose

Ensure all campaigns are properly set up, tracked, and optimized before going live to maximize performance and minimize errors.

---

## Scope

**Who This Applies To:** Media Buyers, Account Managers  
**When to Use:** Before launching any new paid media campaign  
**Frequency:** Per campaign launch

---

## Pre-Launch Checklist

### Strategy & Planning

- [ ] Campaign objective defined
- [ ] Target audience documented
- [ ] Budget allocated
- [ ] KPIs established (ROAS, CPA, CTR targets)
- [ ] Launch date confirmed with client
- [ ] Creative brief approved

### Creative Assets

- [ ] All ad creatives delivered
- [ ] Correct formats/sizes for each placement
- [ ] Copy reviewed and approved
- [ ] CTAs clear and compelling
- [ ] Compliance check completed (no prohibited claims)
- [ ] Landing pages match ad messaging

### Technical Setup

- [ ] Pixel installed and firing
- [ ] Conversion events configured
- [ ] UTM parameters created
- [ ] Tracking links tested
- [ ] Landing pages load correctly
- [ ] Mobile experience verified

### Campaign Structure

- [ ] Campaign name follows convention
- [ ] Ad set structure logical
- [ ] Audience targeting configured
- [ ] Placements selected
- [ ] Budget & schedule set
- [ ] Bid strategy chosen
- [ ] Optimization event selected

### Client Approval

- [ ] Campaign preview sent to client
- [ ] Approval received (email/Slack)
- [ ] Any revisions implemented
- [ ] Final approval confirmed

---

## Launch Day

### Pre-Launch (1 Hour Before)

- [ ] Final creative review
- [ ] Budget confirmation
- [ ] Tracking verification
- [ ] Team notified of launch

### Launch

- [ ] Campaign activated
- [ ] Spend tracking enabled
- [ ] Initial delivery check (15 min after)
- [ ] No delivery issues

### Post-Launch (Day 1)

- [ ] 24-hour performance check
- [ ] Spend pacing on track
- [ ] CTR within expected range
- [ ] No disapproved ads
- [ ] Client notified of launch

---

## First Week Monitoring

### Daily Checks

- [ ] Spend pacing
- [ ] ROAS/CPA trends
- [ ] CTR/CVR performance
- [ ] Creative performance
- [ ] Any errors/issues

### Optimization Actions

- [ ] Kill underperformers (ROAS < target by 50%+)
- [ ] Scale winners (increase budget 20-50%)
- [ ] Pause low CTR creatives (<1%)
- [ ] Adjust bids if needed
- [ ] Document learnings

### Client Communication

- [ ] Day 1 launch notification sent
- [ ] Day 3 check-in update
- [ ] Day 7 first performance report

## Campaign Naming Convention

```
[Platform]_[Objective]_[Audience]_[Creative]_[Date]

Examples:
META_PROS_BROAD_UGC1_JAN26
GOOGLE_PMAX_ALLPRODUCTS_FEB26
TIKTOK_RETARGETING_ATC_MAR26
```

---

## UTM Parameter Template

```
utm_source=[platform]
utm_medium=[paid_social/paid_search]
utm_campaign=[campaign_name]
utm_content=[ad_set_name]
utm_term=[audience/keyword]

Example:
utm_source=meta&utm_medium=paid_social&utm_campaign=prospecting_broad&utm_content=interest_stack&utm_term=beauty_25-44
```

## Troubleshooting

| Problem | Possible Cause | Solution |
|---------|---------------|----------|
| No delivery after 24h | Budget too low, audience too narrow | Increase budget, expand audience |
| High CPM | Audience too small, creative fatigue | Broaden targeting, refresh creative |
| Low CTR | Weak hook, wrong audience | Test new hooks, review targeting |
| High CPA | Landing page issues, wrong offer | Optimize LP, test different offers |
| Ads disapproved | Policy violation | Review policies, edit creative/copy |
| Tracking not working | Pixel not firing, wrong event | Check pixel, verify event setup |

---

## Quality Standards

**Success Criteria:**
- Campaign launches on scheduled date
- All tracking working correctly
- No policy violations
- Spend pacing on target
- Performance within 20% of projections in first 7 days

**Common Mistakes to Avoid:**
1. Launching without client approval
2. Forgetting UTM parameters
3. Wrong conversion event selected
4. Budget typos (extra zero!)
5. Not monitoring first 24 hours
