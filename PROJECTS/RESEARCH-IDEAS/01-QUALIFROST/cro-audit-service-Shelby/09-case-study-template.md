## 📋 HOW TO USE THIS TEMPLATE

This template helps you create compelling case studies that:
- Demonstrate real results from CRO work
- Build credibility with prospects
- Showcase your methodology
- Provide social proof

**Best Practices:**
- Get client permission before publishing
- Use real numbers (or ranges if confidential)
- Include specific tactics, not just results
- Add client quotes and testimonials
- Use visuals (screenshots, graphs, charts)
