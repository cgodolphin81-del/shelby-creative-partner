## 3. Sensor Networks

### Overview
Design and deployment of custom sensor networks for specific monitoring needs.

### Network Types

#### Wireless Sensor Networks (WSN)
- LoRaWAN deployments (long-range, low power)
- Zigbee mesh networks
- Z-Wave home networks
- Wi-Fi based sensors
- Cellular IoT (LTE-M, NB-IoT)

#### Wired Sensor Networks
- Industrial protocols (Modbus, BACnet, CAN bus)
- PoE sensor deployments
- RS-485 networks
- Ethernet-based sensors

### Sensor Categories

| Category | Examples | Use Cases |
|----------|----------|-----------|
| Environmental | Temp, humidity, air quality, CO2 | Buildings, agriculture, warehouses |
| Motion/Presence | PIR, radar, ultrasonic | Security, occupancy, automation |
| Contact/Open | Door/window sensors, reed switches | Security, access control |
| Water | Leak detectors, flow meters, level sensors | Flood prevention, irrigation |
| Energy | Current sensors, power meters, voltage monitors | Energy management, billing |
| Vibration | Accelerometers, piezoelectric sensors | Predictive maintenance |
| Optical | Light sensors, cameras, LiDAR | Security, automation, mapping |
| Gas | Smoke, CO, methane, refrigerant | Safety, compliance |

### Custom Solutions
- Prototype development
- PCB design for custom sensors
- Firmware development
- Gateway development
- Cloud integration

### Pricing
- Network design: $5K-$20K
- Per sensor deployment: $200-$2,000 each
- Gateway/hub: $500-$5,000 each
- Integration: $10K-$50K

## 4. Connectivity Solutions

### Overview
Network infrastructure design and implementation for IoT deployments.

### Services

#### Network Design
- Site surveys and RF analysis
- Network topology design
- Bandwidth requirements analysis
- Redundancy planning
- Future scalability planning

#### Implementation
- Wi-Fi 6/6E/7 deployments
- Dedicated IoT VLANs
- LoRaWAN gateway installation
- Cellular backup systems
- Mesh network deployment
- 5G private network consulting

#### Network Types Comparison

| Technology | Range | Power | Bandwidth | Best For |
|------------|-------|-------|-----------|----------|
| Wi-Fi | 50-100m | High | High | Cameras, high-bandwidth |
| Zigbee | 10-100m | Low | Low | Sensors, switches |
| Z-Wave | 30-100m | Low | Low | Home automation |
| LoRaWAN | 2-15km | Very Low | Very Low | Agriculture, city-scale |
| LTE-M | Cellular | Low | Medium | Mobile assets, backup |
| NB-IoT | Cellular | Low | Low | Static sensors, deep indoor |
| Bluetooth LE | 10-50m | Very Low | Low | Beacons, wearables |
| Thread | 10-30m | Low | Medium | Matter-compatible devices |

### Security
- Network segmentation
- Firewall configuration
- VPN setup for remote access
- Certificate management
- Regular security audits

### Pricing
- Network design: $5K-$25K
- Infrastructure installation: $10K-$100K+
- Ongoing network management: $1K-$10K/month
