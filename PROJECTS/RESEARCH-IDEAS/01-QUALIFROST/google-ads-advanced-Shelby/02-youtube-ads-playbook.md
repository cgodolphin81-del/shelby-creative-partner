## 1. YouTube Ad Formats Overview

### 1.1 Skippable In-Stream Ads (TrueView)

**Characteristics:**
- Play before, during, or after other videos
- Skippable after 5 seconds
- Advertiser pays when viewer watches 30+ seconds OR interacts
- Max length: 6 minutes (recommended: 2-3 minutes)
- Best for: Brand awareness, consideration, conversions

**When to Use:**
```
✓ Broad reach campaigns
✓ Storytelling opportunities
✓ Product demonstrations
✓ Brand building
✓ Retargeting engaged viewers
```

**Pricing:**
- CPV (Cost Per View): $0.10 - $0.30 average
- Pay only for engaged views (30+ sec or interaction)

### 1.2 Non-Skippable In-Stream Ads

**Characteristics:**
- Must watch before video content
- Max length: 15 seconds (some 20-second placements)
- Advertiser pays per impression (CPM)
- Best for: Brand awareness, time-sensitive promotions

**When to Use:**
```
✓ Short, punchy messages
✓ Brand awareness campaigns
✓ Product launches
✓ Event promotions
✓ When message can't be skipped
```

**Pricing:**
- CPM (Cost Per Thousand): $10 - $30 average
- Guaranteed views (no skip option)

### 1.3 Bumper Ads

**Characteristics:**
- Non-skippable, 6 seconds maximum
- CPM pricing
- Must convey message in 6 seconds
- Best for: Reach, frequency, brand recall

**When to Use:**
```
✓ Brand awareness at scale
✓ Reinforcing campaign messages
✓ Mobile-first campaigns
✓ High-frequency exposure
✓ Complement to longer formats
```

**Pricing:**
- CPM: $5 - $15 average
- Lowest cost per reach

### 1.4 Video Discovery Ads (In-Feed)

**Characteristics:**
- Appear in YouTube search results
- Show alongside related videos
- Appear on YouTube homepage
- User clicks to watch (not auto-play)
- Best for: Consideration, engaged viewers

**When to Use:**
```
✓ Users actively searching
✓ Consideration stage
✓ Educational content
✓ Longer-form content promotion
✓ Driving channel subscriptions
```

**Pricing:**
- CPV: $0.15 - $0.40 average
- Pay only when user chooses to watch

### 1.5 Masthead Ads

**Characteristics:**
- Premium homepage placement
- 24-hour exclusive placement
- Auto-play, muted by default
- Best for: Major brand campaigns, product launches

**When to Use:**
```
✓ Product launches
✓ Major announcements
✓ Seasonal campaigns
✓ Maximum reach in 24 hours
✓ Premium brand positioning
```

**Pricing:**
- CPM: $20 - $50+ (premium placement)
- Reserved buy (contact Google Sales)

### 1.6 Shorts Ads

**Characteristics:**
- Vertical, full-screen ads
- Appear between Shorts videos
- Skippable, up to 60 seconds
- Best for: Mobile-first, younger audiences

**When to Use:**
```
✓ Gen Z and Millennial targeting
✓ Mobile-first creative
✓ Trending content integration
✓ Quick, engaging messages
✓ App install campaigns
```

**Pricing:**
- CPV: $0.05 - $0.20 average
- Lower cost, high mobile reach

## 2. Video Ad Script Templates

### 2.1 The Problem-Agitate-Solve (PAS) Framework

**Structure:**
```
0-5 sec:  HOOK - State the problem
5-15 sec: AGITATE - Make it hurt (emotional connection)
15-25 sec: SOLVE - Introduce your solution
25-30 sec: CTA - Tell them what to do
```

**Template Script (30 seconds):**

```
[SCENE 1: 0-5 seconds]
VISUAL: Person struggling with [problem]
AUDIO: "Tired of [specific pain point]?"

[SCENE 2: 5-15 seconds]
VISUAL: Show frustration, wasted time/money
AUDIO: "You've tried everything. [Product category] 
        that promises results but delivers nothing. 
        You're wasting [time/money] and still dealing 
        with [problem]."

[SCENE 3: 15-25 seconds]
VISUAL: Product reveal, transformation
AUDIO: "Introducing [Product Name]. The [unique 
        differentiator] that [key benefit]. See how 
        [specific customer] got [specific result] in 
        [timeframe]."

[SCENE 4: 25-30 seconds]
VISUAL: CTA with offer
AUDIO: "Try [Product] risk-free for 30 days. 
        Click below and get [offer] today."

TEXT OVERLAY: [Website] | [Offer] | [Guarantee]
```

**Example (Fitness App):**
```
[0-5s] "Still can't stick to your workout routine?"
[5-15s] "You buy the gym membership. Download the apps. 
         Buy the equipment. And quit within weeks. 
         Again."
[15-25s] "FitTrack adapts to YOUR schedule. 10-minute 
         workouts that fit your life. 50,000 members 
         who finally see results."
[25-30s] "Start your free trial. No equipment needed. 
         Download now."
```

### 2.2 The Before-After-Bridge (BAB) Framework

**Structure:**
```
0-5 sec:  BEFORE - Show the current state
5-15 sec: AFTER - Show the transformed state
15-25 sec: BRIDGE - Explain how to get there
25-30 sec: CTA - Call to action
```

**Template Script (30 seconds):**

```
[SCENE 1: 0-5 seconds]
VISUAL: "Before" state (problem visible)
AUDIO: "This is [Customer] before [Product/Solution]..."

[SCENE 2: 5-15 seconds]
VISUAL: "After" state (transformation shown)
AUDIO: "And this is [Customer] after just [timeframe] 
        with [Product]. [Specific improvement 1]. 
        [Specific improvement 2]. [Emotional benefit]."

[SCENE 3: 15-25 seconds]
VISUAL: Product/method reveal
AUDIO: "The difference? [Product/Solution]. 
        [How it works in one sentence]. No [common 
        obstacle]. No [common obstacle]. Just results."

[SCENE 4: 25-30 seconds]
VISUAL: CTA, website, offer
AUDIO: "Start your transformation today. 
        [Website/CTA]. [Special offer if applicable]."
```

### 2.3 The Storytelling Framework

**Structure:**
```
0-5 sec:  CHARACTER - Introduce relatable person
5-15 sec: CONFLICT - Show their struggle
15-25 sec: RESOLUTION - How they overcame it
25-30 sec: LESSON + CTA - Moral and invitation
```

**Template Script (60 seconds):**

```
[SCENE 1: 0-10 seconds]
VISUAL: Introduce character in their world
AUDIO: "[Name] is a [role]. Every day, they 
        [routine]. But there was one problem..."

[SCENE 2: 10-20 seconds]
VISUAL: Show the struggle, the pain point
AUDIO: "[Specific challenge]. They tried [solution 1], 
        [solution 2], [solution 3]. Nothing worked. 
        [Emotional impact of problem]."

[SCENE 3: 20-35 seconds]
VISUAL: Discovery moment, trying your solution
AUDIO: "Then they discovered [Product]. At first, 
        [skepticism]. But after [timeframe], 
        [first result]. After [longer timeframe], 
        [transformation]."

[SCENE 4: 35-50 seconds]
VISUAL: Full transformation, new normal
AUDIO: "Now, [Name] [new capability]. [Emotional 
        benefit]. [Life improvement]. 'I wish I had 
        found this sooner.'"

[SCENE 5: 50-60 seconds]
VISUAL: Product shot, CTA
AUDIO: "[Product]. For anyone who [target audience 
        descriptor]. Start your story at [Website]."
```

### 2.4 The Demonstration Framework

**Structure:**
```
0-5 sec:  HOOK - Dramatic demo or claim
5-20 sec: DEMO - Show it working
20-25 sec: PROOF - Evidence/results
25-30 sec: CTA - What to do next
```

**Template Script (30 seconds):**

```
[SCENE 1: 0-5 seconds]
VISUAL: Dramatic product demonstration
AUDIO: "Watch this..." or "We can clean [X] in 
        [impossibly short time]."

[SCENE 2: 5-20 seconds]
VISUAL: Full demo, step by step
AUDIO: "[Product Name] uses [technology/method]. 
        Just [simple step 1], [simple step 2]. 
        And look at that. [Result achieved]."

[SCENE 3: 20-25 seconds]
VISUAL: Testimonial or data point
AUDIO: "[Number] customers. [Rating] stars. 
        [Specific result statistic]."

[SCENE 4: 25-30 seconds]
VISUAL: Product, offer, CTA
AUDIO: "Get yours today at [Website]. 
        [Guarantee/Offer]. [Product Name]."
```

### 2.5 The Testimonial Framework

**Structure:**
```
0-5 sec:  CREDIBILITY - Who is speaking
5-15 sec: PROBLEM - What they struggled with
15-25 sec: TRANSFORMATION - Results achieved
25-30 sec: ENDORSEMENT + CTA
```

**Template Script (30 seconds):**

```
[SCENE 1: 0-5 seconds]
VISUAL: Customer on camera, name/title overlay
AUDIO: "I'm [Name]. I'm a [title/role]."

[SCENE 2: 5-15 seconds]
VISUAL: B-roll of their struggle (before)
AUDIO: "For [time period], I struggled with 
        [problem]. I tried [solutions]. Nothing 
        worked. I was [emotional state]."

[SCENE 3: 15-25 seconds]
VISUAL: B-roll of success (after), results
AUDIO: "Then I found [Product]. In [timeframe], 
        I saw [result 1]. In [timeframe], 
        [result 2]. Now, [current positive state]. 
        It completely changed [aspect of life]."

[SCENE 4: 25-30 seconds]
VISUAL: Customer + product, CTA
AUDIO: "If you're dealing with [problem], try 
        [Product]. It worked for me. [Website/CTA]."
```

**The Question Hook:**
```

"What if you could [desirable outcome]?"
```

**The Statement Hook:**
```
"Most people [common mistake]."

"[Number] seconds to [desirable outcome]."
```

**The Visual Hook:**
```
[Dramatic transformation shown immediately]
[Unexpected visual that creates curiosity]
[Before/after split screen]
```

**The Story Hook:**
```
"This is [Name]. In [timeframe], they went from 
 [before] to [after]. Here's how..."
```

**The Controversial Hook:**
```
"Everything you know about [topic] is wrong."
```


### 3.5 Placement Targeting

**Placement Types:**
```
SPECIFIC CHANNELS:
→ Target entire YouTube channels
→ Good for: Consistent audience, brand alignment

SPECIFIC VIDEOS:
→ Target individual videos
→ Good for: Contextual relevance, trending content

WEBSITES & APPS:
→ Target Display Network placements
→ Good for: Extended reach beyond YouTube
```

**Placement Selection Strategy:**
```
PLACEMENT RESEARCH PROCESS:

1. IDENTIFY RELEVANT CHANNELS:
   □ Competitor channels
   □ Industry influencer channels
   □ Review channels in your category
   □ Educational channels related to product

2. ANALYZE CHANNEL METRICS:
   □ Subscriber count (reach potential)
   □ Average views per video (engagement)
   □ Upload frequency (consistency)
   □ Audience demographics (match to target)
   □ Content quality (brand safety)

3. PRIORITIZE PLACEMENTS:
   Tier 1: Direct competitors, high relevance
   Tier 2: Industry leaders, adjacent categories
   Tier 3: Broad reach, brand-safe content

4. MONITOR & OPTIMIZE:
   □ Review placement reports weekly
   □ Exclude underperforming placements
   □ Add new relevant placements
   □ Check brand safety regularly
```

**Placement Bid Adjustments:**
```
PLACEMENT PERFORMANCE TIERS:

Tier 1 (Top Performers):
├── ROAS > 400%
├── CPV < $0.15
├── CTR > 2%
└── Action: +25% bid, increase budget

Tier 2 (Solid Performers):
├── ROAS 250-400%
├── CPV $0.15-0.25
├── CTR 1-2%
└── Action: Maintain bids, monitor

Tier 3 (Underperformers):
├── ROAS < 250%
├── CPV > $0.25
├── CTR < 1%
└── Action: -50% bid or exclude
```

## 4. Bidding & Budget Optimization

### 4.1 Bidding Strategies Overview

| Strategy | Best For | Payment Model | Requirements |
|----------|----------|---------------|--------------|
| Maximize Clicks | Traffic, awareness | CPC | None |
| Target CPV | Controlled view cost | CPV | None |
| Maximize Conversions | Conversion volume | CPC/CPV | 15+ conv/month |
| Target CPA | Cost-per-acquisition | CPC/CPV | 30+ conv/month |
| Target ROAS | Revenue efficiency | CPC/CPV | 50+ conv/month |
| CPM | Brand awareness | CPM | None |

### 4.2 CPV Bidding (TrueView)

**When to Use:**
- Skippable in-stream ads
- Video discovery ads
- When you want engaged viewers

**Typical CPV Ranges:**
```
INDUSTRY CPV BENCHMARKS:

Industry                  | Avg CPV   | Good CPV
--------------------------|-----------|----------
Apparel & Fashion         | $0.18     | <$0.12
Beauty & Personal Care    | $0.22     | <$0.15
Consumer Electronics      | $0.25     | <$0.18
Finance & Insurance       | $0.35     | <$0.25
Food & Beverage           | $0.15     | <$0.10
Health & Fitness          | $0.20     | <$0.14
Home & Garden             | $0.17     | <$0.12
Retail (General)          | $0.19     | <$0.13
Travel & Hospitality      | $0.23     | <$0.16
B2B Services              | $0.30     | <$0.22
```

**CPV Optimization Tips:**
```
LOWER YOUR CPV:

1. IMPROVE VIDEO QUALITY:
   □ Hook viewers in first 5 seconds
   □ Keep content engaging throughout
   □ Use professional production
   □ Add captions for sound-off viewing

2. REFINE TARGETING:
   □ Narrow to most relevant audiences
   □ Use in-market for higher intent
   □ Layer demographics for precision
   □ Exclude poor-performing placements

3. OPTIMIZE VIDEO LENGTH:
   □ 15-30 seconds for direct response
   □ 2-3 minutes for storytelling
   □ Match length to campaign goal
   □ Test different lengths

4. IMPROVE CTR:
   □ Compelling thumbnails
   □ Strong title/description
   □ Clear value proposition
   □ A/B test creative elements
```

### 4.3 CPM Bidding (Non-Skippable, Bumper)

**When to Use:**
- Non-skippable in-stream ads
- Bumper ads
- Masthead ads
- Brand awareness focus

**Typical CPM Ranges:**
```
FORMAT CPM BENCHMARKS:

Format                  | Avg CPM    | Good CPM
------------------------|------------|----------
Non-Skippable (15s)     | $15-25     | <$18
Bumper (6s)             | $8-15      | <$10
Masthead                | $25-50+    | Varies
In-Feed Video           | $10-20     | <$15
```

**CPM Optimization Tips:**
```
MAXIMIZE CPM VALUE:

1. FREQUENCY MANAGEMENT:
   □ Cap at 3-5 impressions per user per week
   □ Avoid ad fatigue
   □ Rotate creative every 2-4 weeks
   □ Monitor frequency metrics

2. REACH OPTIMIZATION:
   □ Broad targeting for maximum reach
   □ Affinity audiences for scale
   □ Demographic expansion
   □ Geographic expansion if performing

3. CREATIVE QUALITY:
   □ Professional production value
   □ Clear brand messaging
   □ Memorable creative
   □ Consistent brand elements
```

### 4.4 Budget Allocation

**Campaign Budget Framework:**
```
BUDGET ALLOCATION BY FUNNEL STAGE:

AWARENESS (Top of Funnel):
├── Budget: 30-40% of total
├── Formats: Bumper, Non-skippable, Masthead
├── Goal: Reach, impressions, brand recall
├── KPI: CPM, Reach, Frequency, Brand Lift

CONSIDERATION (Middle of Funnel):
├── Budget: 30-40% of total
├── Formats: Skippable In-Stream, Discovery
├── Goal: Engagement, views, consideration
├── KPI: CPV, View Rate, CTR, Engagement

CONVERSION (Bottom of Funnel):
├── Budget: 20-30% of total
├── Formats: Skippable, Discovery, Retargeting
├── Goal: Conversions, sales, leads
├── KPI: CPA, ROAS, Conversion Rate
```

**Daily Budget Guidelines:**
```
BUDGET RECOMMENDATIONS BY GOAL:

BRAND AWARENESS:
├── Minimum: $50/day
├── Recommended: $200-500/day
└── Aggressive: $1,000+/day

CONSIDERATION:
├── Minimum: $30/day
├── Recommended: $100-300/day
└── Aggressive: $500+/day

CONVERSIONS:
├── Minimum: $20/day
├── Recommended: $50-200/day
└── Aggressive: $300+/day

RULE OF THUMB:
Daily budget should allow for at least:
→ 10-15 conversions per day (conversion campaigns)
→ 500+ views per day (view campaigns)
→ 10,000+ impressions per day (awareness campaigns)
```

## 6. Creative Best Practices

### 6.1 Video Production Guidelines

**Technical Specifications:**
```
VIDEO SPECIFICATIONS:

Resolution:
├── Minimum: 640x360 (360p)
├── Recommended: 1920x1080 (1080p)
└── Ideal: 3840x2160 (4K)

Aspect Ratios:
├── 16:9 (Standard YouTube)
├── 9:16 (Shorts/Vertical)
├── 1:1 (Square)
└── 4:5 (Vertical Feed)

File Format:
├── .MP4 (recommended)
├── .MOV
├── .AVI
└── .WMV

File Size:
└── Maximum: 256 GB

Frame Rate:
├── 24 fps (cinematic)
├── 25 fps (PAL regions)
├── 30 fps (standard)
└── 60 fps (smooth motion)

Audio:
├── Stereo or Mono
├── 48 kHz sample rate
└── Clear, professional quality
```

### 6.2 First 5 Seconds (The Hook)

**Hook Best Practices:**
```
THE CRITICAL FIRST 5 SECONDS:

DO:
✓ Lead with the most compelling visual
✓ State the problem immediately
✓ Show transformation preview
✓ Ask a provocative question
✓ Use pattern interrupt
✓ Match thumbnail promise
✓ Front-load value proposition

DON'T:
✗ Long brand intro/logo
✗ Slow build-up
✗ Generic stock footage
✗ Vague opening statements
✗ Misleading thumbnails
✗ Delay the main message
```

**Hook Examples:**
```
VISUAL HOOKS:
→ Dramatic before/after split screen
→ Surprising demonstration
→ Emotional customer moment
→ Bold text overlay with stat
→ Unexpected visual contrast

VERBAL HOOKS:
→ "Stop doing [common mistake]..."
→ "What if you could [desirable outcome]..."
→ "The truth about [industry myth]..."
→ "I tried [solution] for 30 days..."
→ "This changed everything..."
```

**Thumbnail Best Practices:**
```
THUMBNAIL CHECKLIST:

COMPOSITION:

COLOR:
```

## 7. Measurement & Analytics

### 7.1 Key Metrics Dashboard

| Metric | Definition | Good Benchmark | Action Threshold |
|--------|------------|----------------|------------------|
| Views | Number of times video was viewed | Varies by goal | Track trend |
| View Rate | Views / Impressions | 20-30% | < 15%: Optimize |
| Average View Duration | Average time watched | 30-50% of video | < 25%: Improve hook |
| CPV | Cost / Views | $0.10-0.30 | > $0.35: Optimize |
| CTR | Clicks / Views | 0.5-2% | < 0.3%: Improve CTA |
| Conversion Rate | Conversions / Clicks | 2-5% | < 1%: Check landing page |
| CPA | Cost / Conversions | < 30% of AOV | > 40% of AOV: Adjust |
| ROAS | Revenue / Ad Spend | 300%+ | < 200%: Restructure |

### 7.2 YouTube Analytics Metrics

**Engagement Metrics:**
```
ENGAGEMENT ANALYSIS:

WATCH TIME:
→ Total watch time (minutes)
→ Average view duration
→ Audience retention graph

KEY MOMENTS:
→ Where viewers drop off
→ Where viewers re-watch
→ Where viewers click away

ACTION:
→ If drop-off at 5s: Improve hook
→ If drop-off at 15s: Tighten middle
→ If drop-off before CTA: Move CTA earlier
→ If re-watch spikes: More of that content
```

**Audience Retention Graph Interpretation:**
```
RETENTION GRAPH ANALYSIS:

FLAT LINE (Good):
→ Content is engaging
→ Viewers staying throughout
→ Action: More content like this

GRADUAL DECLINE (Normal):
→ Natural viewer drop-off
→ Expected for longer videos
→ Action: Tighten content if too steep

SHARP DROP (Problem):
→ Something turned viewers off
→ Check content at that timestamp
→ Action: Edit or remove that section

SPIKE UP (Excellent):
→ Viewers re-watching section
→ Highly engaging content
→ Action: Analyze what worked, replicate
```

### 7.3 Attribution & Conversion Tracking

**Setting Up Conversion Tracking:**
```
CONVERSION TRACKING SETUP:

STEP 1: DEFINE CONVERSIONS
□ Purchase
□ Lead form submission
□ Sign up / Registration
□ Add to cart
□ Initiate checkout
□ Phone call
□ App install
□ Key page view

STEP 2: IMPLEMENT TRACKING
□ Google Ads tag on website
□ Google Tag Manager (recommended)
□ Enhanced conversions (for privacy)
□ Offline conversion import (if applicable)

STEP 3: CONFIGURE IN GOOGLE ADS
□ Set conversion value
□ Set conversion window (30 days typical)
□ Set attribution model (Data-driven recommended)
□ Set counting method (Every vs. One)

STEP 4: VERIFY & TEST
□ Use Tag Assistant
□ Test conversion path
□ Verify data appearing in Google Ads
□ Allow 24-48 hours for data
```

**Attribution Models:**
```
ATTRIBUTION MODEL COMPARISON:

LAST CLICK:
→ 100% credit to last touchpoint
→ Use: Simple reporting
→ Limitation: Ignores assist interactions

FIRST CLICK:
→ 100% credit to first touchpoint
→ Use: Understanding awareness drivers
→ Limitation: Ignores closing interactions

LINEAR:
→ Equal credit to all touchpoints
→ Use: Full-funnel understanding
→ Limitation: Doesn't weight important touches

TIME DECAY:
→ More credit to recent touches
→ Use: Short sales cycles
→ Limitation: Undervalues awareness

POSITION-BASED:
→ 40% first, 40% last, 20% middle
→ Use: Balanced view
→ Limitation: Arbitrary weighting

DATA-DRIVEN (Recommended):
→ Algorithmic based on actual data
→ Use: Most accurate (when available)
→ Requirement: 600+ conversions/month
```
