### 1.2 E-commerce KPIs

**Key Metrics:**
```
PRIMARY METRICS:

ROAS (Return on Ad Spend):
→ Formula: Revenue / Ad Spend × 100
→ Target: 400%+ (varies by margin)
→ Industry benchmark: 350-500%
→ Most important e-commerce metric

CPA (Cost Per Acquisition):
→ Formula: Ad Spend / Conversions
→ Target: < 25% of AOV
→ Industry benchmark: Varies widely
→ Important for margin protection

CONVERSION RATE:
→ Formula: Conversions / Clicks × 100
→ Target: 2-5% (e-commerce)
→ Industry benchmark: 2.5-3.5%
→ Varies by price point

AOV (Average Order Value):
→ Formula: Revenue / Orders
→ Target: Increase over time
→ Industry benchmark: Varies by category
→ Critical for profitability

SECONDARY METRICS:

CTR (Click-Through Rate):
→ Shopping: 1-3%
→ Search - Brand: 5-15%
→ Search - Non-Brand: 2-5%
→ Display: 0.5-1.5%

CPC (Cost Per Click):
→ Shopping: $0.30-1.00
→ Search - Brand: $0.50-2.00
→ Search - Non-Brand: $1.00-5.00
→ Display: $0.50-2.00

CART ABANDONMENT RATE:
→ Formula: (Carts - Purchases) / Carts × 100
→ Industry average: 70%
→ Target: < 65%
→ Remarketing opportunity

CUSTOMER LIFETIME VALUE (CLV):
→ Formula: AOV × Purchase Frequency × Lifespan
→ Target: 3x+ CAC
→ Critical for bidding strategy
→ Long-term profitability
```

### 1.3 Profitability Framework

**Calculating Target ROAS:**
```
STEP 1: KNOW YOUR MARGINS

GROSS MARGIN CALCULATION:
Gross Margin = (Revenue - COGS) / Revenue × 100

EXAMPLE:
Product Price: $100
COGS: $40
Gross Margin: ($100 - $40) / $100 = 60%

STEP 2: ACCOUNT FOR OTHER COSTS

OPERATING COSTS (as % of revenue):
├── Shipping: 5-15%
├── Payment Processing: 2-3%
├── Returns: 5-10%
├── Overhead: 10-20%
└── Total Other: 22-48%

EXAMPLE:
Gross Margin: 60%
Other Costs: 35%
Net Margin Before Ads: 25%

STEP 3: CALCULATE BREAK-EVEN ROAS

BREAK-EVEN ROAS = 1 / Net Margin Before Ads

EXAMPLE:
Net Margin Before Ads: 25%
Break-Even ROAS: 1 / 0.25 = 4.0 (400%)

AT 400% ROAS:
$1 ad spend = $4 revenue
$4 revenue × 25% margin = $1 profit
Profit = Ad Spend (break-even)

STEP 4: SET TARGET ROAS

PROFITABLE ROAS = Break-Even ROAS × Profit Factor

PROFIT FACTORS:
1.5x = Moderate profit
2.0x = Good profit
3.0x = Excellent profit

EXAMPLE:
Break-Even ROAS: 400%
Target Profit Factor: 1.5x
Target ROAS: 400% × 1.5 = 600%

AT 600% ROAS:
$1 ad spend = $6 revenue
$6 revenue × 25% margin = $1.50 profit
Profit after ads: $1.50 - $1.00 = $0.50
Profit margin on ad spend: 50%

INDUSTRY ROAS BENCHMARKS:

| Category              | Break-Even | Good   | Excellent |
|-----------------------|------------|--------|-----------|
| Apparel               | 250%       | 400%   | 600%+     |
| Electronics           | 200%       | 350%   | 500%+     |
| Home & Garden         | 300%       | 450%   | 650%+     |
| Beauty                | 350%       | 500%   | 700%+     |
| Sports & Outdoors     | 300%       | 450%   | 600%+     |
| Toys & Games          | 250%       | 400%   | 550%+     |
| Food & Beverage       | 200%       | 350%   | 500%+     |
| Luxury Goods          | 400%       | 600%   | 800%+     |
```

## 2. ROAS Optimization

### 2.1 ROAS Improvement Levers

**Four Levers of ROAS:**
```
ROAS = (Clicks × CTR × CVR × AOV) / (Clicks × CPC)

SIMPLIFIED:
ROAS = (CVR × AOV) / CPC

FOUR LEVERS:

1. INCREASE CONVERSION RATE (CVR)
   → Landing page optimization
   → Better product pages
   → Trust signals
   → Reviews and ratings
   → Clear CTAs
   → Mobile optimization
   → Page speed
   → Impact: 10-50% ROAS improvement

2. INCREASE AVERAGE ORDER VALUE (AOV)
   → Product bundles
   → Cross-sells
   → Upsells
   → Free shipping threshold
   → Volume discounts
   → Gift with purchase
   → Impact: 10-40% ROAS improvement

3. DECREASE COST PER CLICK (CPC)
   → Improve Quality Score
   → Better ad relevance
   → Landing page experience
   → Expected CTR
   → Negative keywords
   → Bid optimization
   → Impact: 10-30% ROAS improvement

4. INCREASE CLICK-THROUGH RATE (CTR)
   → Better ad copy
   → More relevant keywords
   → Ad extensions
   → Product feed optimization
   → Promotions
   → Impact: 5-20% ROAS improvement

PRIORITIZATION:

HIGHEST IMPACT (Start Here):
1. Conversion Rate Optimization
2. Product Feed Optimization
3. AOV Optimization

MEDIUM IMPACT:
4. Quality Score Improvement
5. Negative Keyword Expansion
6. Bid Strategy Optimization

LOWER IMPACT (Still Important):
7. Ad Copy Refinement
8. Extension Optimization
9. Audience Refinement
```

IMPLEMENTATION:
→ Set threshold 20-30% above current AOV

→ "Only $X away from free shipping!"

→ Test different thresholds

IMPACT:
→ Typical AOV increase: 15-25%

       Breathable Mesh - Black/White - Size 10"

BEST PRACTICES:
→ Front-load important keywords
→ Include brand (increases CTR 15-25%)
→ Add key features
→ Include size/color/variant
→ Stay under 150 characters
→ Use title case
→ No promotional text

DESCRIPTION OPTIMIZATION:

STRUCTURE:
Paragraph 1: Product overview (100-150 chars)

→ 500-1000 characters total
→ Include primary keyword

IMAGE OPTIMIZATION:

TECHNICAL REQUIREMENTS:
→ Minimum: 100x100 pixels

BEST PRACTICES:
→ White or neutral background

CUSTOM LABELS:

### 5.2 Feed Rules

**Automating Optimization:**
```
COMMON FEED RULES:

RULE 1: TITLE OPTIMIZATION

IF brand EXISTS AND product_type EXISTS
THEN title = brand + " " + product_type + " - " + 
             original_title

RESULT:
Before: "Comfortable athletic footwear"
After: "Nike - Athletic Shoes - Comfortable athletic footwear"

RULE 2: CATEGORY MAPPING

IF custom_category CONTAINS "running"
THEN google_product_category = "187 - Shoes - Athletic Shoes"

IF custom_category CONTAINS "dress"
THEN google_product_category = "166 - Clothing - Dresses"

RULE 3: CUSTOM LABEL ASSIGNMENT

IF price > 100
THEN custom_label_0 = "high-value"
ELSE IF price > 50
THEN custom_label_0 = "mid-value"
ELSE
THEN custom_label_0 = "budget"

RULE 4: IMAGE ENHANCEMENT

IF image_link IS EMPTY
THEN image_link = "https://site.com/default-product.jpg"

RULE 5: AVAILABILITY UPDATE

IF inventory_quantity = 0
THEN availability = "out of stock"
ELSE IF inventory_quantity < 10
THEN availability = "in stock"
ELSE
THEN availability = "in stock"

RULE 6: SALE PRICE

IF sale_start_date <= TODAY AND sale_end_date >= TODAY
THEN sale_price = original_price × 0.8
AND sale_price_effective_date = sale_start_date/sale_end_date

## 7. Customer Lifetime Value

### 7.1 CLV Calculation

**Understanding CLV:**
```
CLV FORMULA:

CLV = AOV × Purchase Frequency × Customer Lifespan

EXAMPLE CALCULATION:

AOV: $85
Purchase Frequency: 4 times/year
Customer Lifespan: 3 years

CLV = $85 × 4 × 3 = $1,020

CLV BY SEGMENT:

FIRST-TIME BUYERS:
→ Lower initial CLV
→ Potential for growth
→ Focus: Second purchase

REPEAT BUYERS:
→ Higher CLV
→ Proven loyalty
→ Focus: Retention

VIP CUSTOMERS:
→ Highest CLV
→ Brand advocates
→ Focus: Exclusivity, retention

CLV TO CAC RATIO:

HEALTHY RATIO: 3:1 or higher
→ CLV: $300
→ Maximum CAC: $100

CALCULATING MAX CAC:

Max CAC = CLV × Target CAC %

EXAMPLE:
CLV: $1,020
Target CAC %: 30%
Max CAC: $1,020 × 0.30 = $306

BIDDING IMPLICATIONS:

FIRST PURCHASE:
→ May be unprofitable
→ Accept higher CPA
→ Focus on acquisition
→ Target: Break-even or slight loss

SECOND PURCHASE:
→ Becomes profitable
→ Lower CPA acceptable
→ Focus on retention
→ Target: 200-300% ROAS

ONGOING:
→ Highly profitable
→ Efficient spend
→ Focus on loyalty
→ Target: 500%+ ROAS
```

**Bid Strategy by Customer Type:**
```

BID STRATEGY: Maximize Conversions

## 8. Attribution & Measurement

### 8.1 Attribution Models

**Understanding Attribution:**
```
ATTRIBUTION MODELS:

LAST CLICK:
→ 100% credit to last touchpoint
→ Google Ads default (historically)
→ Undervalues upper funnel
→ Simple but incomplete

FIRST CLICK:
→ 100% credit to first touchpoint
→ Values acquisition
→ Ignores nurturing
→ Rarely used alone

LINEAR:
→ Equal credit to all touchpoints
→ Fair but simplistic
→ Good for full-funnel view
→ May overvalue some touches

TIME DECAY:
→ More credit to recent touches
→ Values closing interactions
→ Good for short sales cycles
→ Undervalues awareness

POSITION-BASED (U-SHAPED):
→ 40% first click
→ 40% last click
→ 20% middle touches
→ Balanced approach
→ Good for most e-commerce

DATA-DRIVEN (Recommended):
→ Algorithmic based on data
→ Most accurate
→ Requires sufficient data
→ Google Ads default now

GOOGLE ADS ATTRIBUTION:

CURRENT DEFAULT: Data-Driven
→ Uses machine learning
→ Considers all touchpoints
→ Cross-device tracking
→ Most accurate for bidding

VIEW-THROUGH CONVERSIONS:

DISPLAY/YOUTUBE:
→ 30-day view-through window
→ User sees ad, doesn't click
→ Converts within 30 days
→ Important for upper funnel

SHOPPING/SEARCH:
→ Typically click-based
→ View-through less common
→ Still track for completeness
```
