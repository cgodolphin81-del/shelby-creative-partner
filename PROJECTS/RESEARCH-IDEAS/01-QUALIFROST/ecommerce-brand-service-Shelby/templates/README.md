## Brand Identity Templates

### Brand Positioning Statement

```
BRAND POSITIONING STATEMENT
===========================

For [target customer],
who [customer need or opportunity],
[Brand Name] is a [category]
that [key benefit or reason to buy].

Unlike [primary competitor],
we [primary differentiation].

Example:
For health-conscious professionals,
who want convenient nutrition,
Huel is a meal replacement
that provides complete nutrition in minutes.

Unlike protein shakes,
we offer balanced meals with 26 essential nutrients.
```

## Customer Service Templates

### Return/Refund Decision Tree

```
RETURN/REFUND DECISION TREE
===========================

Customer Request: Return/Refund

├─ Is item within return window (30 days)?
│  ├─ YES → Proceed
│  └─ NO → Offer store credit or exception
│
├─ Is item in original condition?
│  ├─ YES → Full refund
│  └─ NO → Assess damage, offer partial refund
│
├─ Is item defective/damaged?
│  ├─ YES → Full refund + replacement offered
│  └─ NO → Standard return process
│
├─ Is customer VIP (3+ orders)?
│  ├─ YES → Expedite, consider exceptions
│  └─ NO → Standard process
│
└─ Resolution:
   □ Full refund
   □ Partial refund ($_____)
   □ Store credit ($_____)
   □ Exchange
   □ Replacement sent
```

### Escalation Protocol

```
ESCALATION PROTOCOL
===================

LEVEL 1: Front-Line Support
- Standard issues (order status, basic returns, product questions)
- Authority: Up to $50 refund/credit
- Timeline: Respond within 4 hours

LEVEL 2: Senior Support
- Complex issues, unhappy customers, policy exceptions
- Authority: Up to $200 refund/credit
- Timeline: Respond within 2 hours
- Escalate when: Customer requests supervisor, issue exceeds $50

LEVEL 3: Customer Experience Manager
- Serious complaints, legal threats, PR risks
- Authority: Unlimited refund/credit
- Timeline: Respond within 1 hour
- Escalate when: Customer threatens legal action, media, or >$200

LEVEL 4: Founder/CEO
- Major PR issues, key customer at risk, policy changes
- Authority: Any resolution
- Timeline: Immediate
- Escalate when: Potential brand damage, VIP customer loss
```
