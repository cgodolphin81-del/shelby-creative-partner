## Help Desk Setup Guide

### Choosing a Help Desk

| Platform | Best For | Starting Price | Key Features |
|----------|----------|----------------|--------------|
| **Gorgias** | E-commerce (Shopify) | $60/mo | Shopify integration, automation |
| **Zendesk** | All-purpose | $19/mo | Robust features, scalable |
| **Help Scout** | Small teams | $20/mo | Simple, email-focused |
| **Intercom** | Chat-first | $74/mo | Live chat, bots |
| **Freshdesk** | Budget-friendly | $15/mo | Good value, solid features |

### Gorgias Setup (Recommended for Shopify)

**Step 1: Create Account**
- Visit: gorgias.com
- Start free trial
- Connect Shopify store

**Step 2: Configure Channels**
- Email (support@brand.com)
- Live chat (website widget)
- Social media (Facebook, Instagram)
- SMS (if applicable)

**Step 3: Set Up Automations**

**Common Automations:**
```
1. Order Status Auto-Reply
   Trigger: Customer asks "where is my order"
   Action: Reply with order status + tracking
   Condition: Order exists in Shopify

2. Return Request
   Trigger: Customer mentions "return"
   Action: Send return instructions + label
   Condition: Within return window

3. VIP Customer
   Trigger: Customer is VIP (3+ orders)
   Action: Tag as VIP, notify team
   Condition: Customer LTV > $X

4. Negative Sentiment
   Trigger: Negative sentiment detected
   Action: Flag for manager review
   Condition: Sentiment score < threshold
```

**Step 4: Create Macros (Canned Responses)**
- Save all templates above as macros
- Organize by category (Order, Shipping, Returns, etc.)
- Include personalization variables

**Step 5: Set Up SLAs**
```
First Response Time:
- VIP customers: 1 hour
- Regular customers: 4 hours
- Social media: 2 hours

Resolution Time:
- Simple issues: 24 hours
- Complex issues: 48 hours
- Escalations: 72 hours
```

### Help Desk Best Practices

**Response Time Goals:**
- First response: <4 hours (business hours)
- Resolution: <24 hours for simple issues
- Follow-up: Within 48 hours if waiting on customer

**Quality Standards:**
- Personalize every response (use customer name)
- Show empathy (acknowledge frustration)
- Provide clear next steps
- Close the loop (follow up until resolved)

**Metrics to Track:**
```
- First Response Time (target: <4 hours)
- Resolution Time (target: <24 hours)
- Customer Satisfaction (CSAT) Score (target: 90%+)
- Tickets per Agent per Day (target: 40-60)
- First Contact Resolution (target: 70%+)
```

## CS Metrics & KPIs

### Key Metrics

| Metric | Formula | Target |
|--------|---------|--------|
| **CSAT** | (Positive Responses / Total Responses) × 100 | 90%+ |
| **NPS** | % Promoters - % Detractors | 50+ |
| **First Response Time** | Average time to first reply | <4 hours |
| **Resolution Time** | Average time to resolve | <24 hours |
| **First Contact Resolution** | % resolved in first interaction | 70%+ |
| **Tickets per Day** | Total tickets / business days | 40-60/agent |
| **Customer Effort Score** | Average ease of resolution (1-7) | 6+ |

### Customer Satisfaction (CSAT) Survey

**Timing:** After ticket resolution
**Question:** "How satisfied were you with your support experience?"
**Scale:** 1-5 (Very Dissatisfied to Very Satisfied)

**Follow-up Question (if 1-3):**
"What could we have done better?"

### Net Promoter Score (NPS) Survey

**Timing:** 30 days after purchase
**Question:** "How likely are you to recommend [Brand] to a friend or colleague?"
**Scale:** 0-10

**Scoring:**
- Promoters (9-10): Loyal enthusiasts
- Passives (7-8): Satisfied but unenthusiastic
- Detractors (0-6): Unhappy customers

**NPS = % Promoters - % Detractors**
