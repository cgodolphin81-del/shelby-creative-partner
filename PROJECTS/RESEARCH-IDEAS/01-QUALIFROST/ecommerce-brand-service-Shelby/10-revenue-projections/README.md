## Unit Economics Calculator

### Key Metrics

| Metric | Formula | Year 1 | Year 2 | Year 3 |
|--------|---------|--------|--------|--------|
| **Average Order Value (AOV)** | Revenue / Orders | $75 | $80 | $85 |
| **Cost of Goods Sold (COGS)** | Product + Shipping | $26.25 | $25.60 | $25.50 |
| **Gross Profit per Order** | AOV - COGS | $48.75 | $54.40 | $59.50 |
| **Gross Margin %** | Gross Profit / AOV | 65% | 68% | 70% |
| **Customer Acquisition Cost (CAC)** | Ad Spend / New Customers | $35 | $30 | $28 |
| **Orders per Customer** | Total Orders / Unique Customers | 1.33 | 1.54 | 1.82 |
| **Lifetime Value (LTV)** | AOV × Orders per Customer | $100 | $123 | $155 |
| **LTV:CAC Ratio** | LTV / CAC | 2.9:1 | 4.1:1 | 5.5:1 |

### Unit Economics Breakdown

**Per Order Economics (Year 1):**
```
Revenue:                    $75.00
- COGS (Product):          -$20.00  (26.7%)
- COGS (Shipping):          -$6.25  (8.3%)
= Gross Profit:             $48.75  (65%)

- Marketing (CAC):         -$35.00  (46.7%)
- Fulfillment:              -$3.00  (4%)
- Payment Processing:       -$2.25  (3%)
= Contribution Margin:      $8.50   (11.3%)
```

**Per Order Economics (Year 3):**
```
Revenue:                    $85.00
- COGS (Product):          -$19.00  (22.4%)
- COGS (Shipping):          -$6.50  (7.6%)
= Gross Profit:             $59.50  (70%)

- Marketing (CAC):         -$28.00  (32.9%)
- Fulfillment:              -$2.50  (2.9%)
- Payment Processing:       -$2.55  (3%)
= Contribution Margin:      $26.45  (31.1%)
```

### Break-Even Analysis

**Monthly Break-Even Point:**

```
Fixed Costs (Monthly):
- Salaries: $15,000
- Software/Tools: $2,000
- Rent/Overhead: $3,000
- Other: $2,000
= Total Fixed: $22,000

Contribution Margin per Order: $8.50

Break-Even Orders = Fixed Costs / Contribution Margin
Break-Even Orders = $22,000 / $8.50 = 2,588 orders/month

Break-Even Revenue = 2,588 × $75 = $194,100/month
```

**Improving Break-Even:**
- Increase AOV (bundles, upsells)
- Reduce CAC (better ad performance)
- Reduce fixed costs (outsource vs. hire)
- Improve gross margins (negotiate with suppliers)

### Cash Flow Considerations

**Working Capital Needs:**
```
Inventory: 60 days of COGS = $29,000
Accounts Receivable: 0 days (e-commerce, immediate payment)
Accounts Payable: 30 days = $14,500

Working Capital = Inventory + AR - AP
Working Capital = $29,000 + $0 - $14,500 = $14,500

Recommended Cash Reserve: 3 months OpEx = $75,000
Total Funding Need (Year 1): ~$90,000
```

**Cash Flow Tips:**
- Negotiate net-30/60 terms with suppliers
- Keep inventory lean (just-in-time ordering)
- Collect payment immediately (standard for e-commerce)
- Maintain 3-6 months operating cash reserve
- Plan for seasonality (Q4 is typically 30-40% of annual revenue)

### Key Formulas

**Revenue Projection:**
```
Revenue = Traffic × Conversion Rate × AOV
```

**Customer Lifetime Value:**
```
LTV = AOV × Purchase Frequency × Gross Margin % × Customer Lifespan
```

**Customer Acquisition Cost:**
```
CAC = Total Marketing Spend / New Customers Acquired
```

**Break-Even Point:**
```
Break-Even Units = Fixed Costs / (Price - Variable Cost per Unit)
```

**Cash Runway:**
```
Runway (months) = Current Cash / Monthly Burn Rate
```
