# SEO Audit System

Complete framework for conducting professional SEO audits that convert to clients.

---

## Audit Types

### 1. Quick Audit (Free/Lead Magnet)
- 15-minute automated scan
- Top 5 issues identified
- Used for lead generation
- Conversion tool to paid audit

### 2. Standard Audit ($2,500-$5,000)
- Comprehensive site analysis
- 50-100 page report
- Prioritized recommendations
- 60-minute strategy call

### 3. Enterprise Audit ($10,000-$25,000)
- Deep-dive analysis
- 200+ page report
- Developer-ready tickets
- Implementation support
- 90-day roadmap

---

## Audit Framework

### Phase 1: Discovery (Day 1-2)

**Information Gathering:**
- [ ] Client goals and KPIs
- [ ] Target markets/regions
- [ ] Current pain points
- [ ] Previous SEO work
- [ ] Access requirements (GA, GSC, CMS, etc.)
- [ ] Competitor list (3-5)
- [ ] Target keywords (if any)

**Tools Setup:**
- [ ] Site crawl (Screaming Frog/Sitebulb)
- [ ] Google Analytics access
- [ ] Google Search Console access
- [ ] Ahrefs/SEMrush project
- [ ] PageSpeed Insights baseline
- [ ] Mobile-friendly test

---

### Phase 2: Technical Audit (Day 2-4)

#### Crawlability & Indexation

**Checklist:**
- [ ] robots.txt analysis
- [ ] XML sitemap review
- [ ] Index coverage (GSC)
- [ ] Orphan pages
- [ ] Redirect chains/loops
- [ ] Canonical tags
- [ ] Noindex/nofollow usage
- [ ] Pagination handling
- [ ] Faceted navigation
- [ ] JavaScript rendering

**Metrics to Capture:**
- Total URLs crawled
- Indexed vs. non-indexed
- Crawl errors (4xx, 5xx)
- Redirect count
- Orphan page count

---

#### Site Architecture

**Checklist:**
- [ ] URL structure analysis
- [ ] Internal linking depth
- [ ] Category/page hierarchy
- [ ] Breadcrumb implementation
- [ ] Navigation structure
- [ ] Silo structure (if applicable)
- [ ] Hub/spoke model

**Metrics to Capture:**
- Max click depth
- Average internal links per page
- Top linked pages
- Link equity distribution

---

#### Page Speed & Core Web Vitals

**Checklist:**
- [ ] LCP (Largest Contentful Paint)
- [ ] FID (First Input Delay) / INP
- [ ] CLS (Cumulative Layout Shift)
- [ ] Mobile speed score
- [ ] Desktop speed score
- [ ] Image optimization
- [ ] CSS/JS minification
- [ ] Server response time
- [ ] CDN usage
- [ ] Caching implementation

**Tools:**
- PageSpeed Insights
- GTmetrix
- WebPageTest
- Chrome DevTools

**Metrics to Capture:**
- CWV pass rate (%)
- Average load time
- Time to First Byte (TTFB)
- Total page size

---

#### Schema & Structured Data

**Checklist:**
- [ ] Organization schema
- [ ] LocalBusiness schema (if local)
- [ ] Product schema (if e-commerce)
- [ ] Article/BlogPosting schema
- [ ] FAQ schema
- [ ] Review/Rating schema
- [ ] Breadcrumb schema
- [ ] Valid markup (Rich Results Test)

**Tools:**
- Google Rich Results Test
- Schema Markup Validator
- Screaming Frog (schema extraction)

---

#### Mobile Optimization

**Checklist:**
- [ ] Mobile-friendly test
- [ ] Responsive design
- [ ] Mobile page speed
- [ ] Touch element spacing
- [ ] Font size readability
- [ ] Viewport configuration
- [ ] Mobile UX issues

---

#### Security & HTTPS

**Checklist:**
- [ ] HTTPS implementation
- [ ] Mixed content issues
- [ ] SSL certificate validity
- [ ] HSTS implementation
- [ ] Security headers

---

### Phase 3: Content Audit (Day 4-6)

#### On-Page SEO

**For Top 50 Pages:**
- [ ] Title tag optimization
- [ ] Meta description
- [ ] H1 tag (single, keyword-rich)
- [ ] H2-H6 hierarchy
- [ ] Keyword usage (natural)
- [ ] Content length
- [ ] Content quality/uniqueness
- [ ] Image optimization (alt text)
- [ ] Internal links
- [ ] External links
- [ ] URL structure
- [ ] Readability score

**Tools:**
- Screaming Frog (bulk extraction)
- Surfer SEO / ClearScope
- Hemingway App

---

#### Content Gap Analysis

**Process:**
1. Export top 100 keywords (GSC)
2. Export competitor keywords (Ahrefs/SEMrush)
3. Identify gaps (they rank, you don't)
4. Categorize by intent
5. Prioritize by opportunity

**Metrics:**
- Keyword gap count
- Estimated traffic opportunity
- Difficulty score average
- Content pieces needed

---

#### E-E-A-T Assessment

**Checklist:**
- [ ] Author bios
- [ ] About page quality
- [ ] Contact information
- [ ] Expertise signals
- [ ] Citations/references
- [ ] Original research/data
- [ ] Trust signals (certifications, awards)
- [ ] Review/testimonial presence

---

### Phase 4: Competitor Analysis (Day 6-7)

#### Competitor Selection
- Direct competitors (3-5)
- SERP competitors (who ranks for target keywords)
- Industry leaders (aspirational)

#### Analysis Framework

**For Each Competitor:**
- [ ] Domain Rating/Authority
- [ ] Estimated organic traffic
- [ ] Top 10 keywords
- [ ] Content volume
- [ ] Publishing frequency
- [ ] Backlink profile (quantity/quality)
- [ ] Top linked pages
- [ ] Content formats used
- [ ] Schema implementation
- [ ] Page speed scores

**Tools:**
- Ahrefs Site Explorer
- SEMrush Domain Overview
- SimilarWeb

---

### Phase 5: Backlink Analysis (Day 7-8)

#### Backlink Profile Audit

**Checklist:**
- [ ] Total backlinks
- [ ] Referring domains
- [ ] Domain Rating distribution
- [ ] Anchor text distribution
- [ ] Link types (dofollow/nofollow)
- [ ] Toxic links (spam score)
- [ ] Link velocity
- [ ] Competitor link gap

**Metrics:**
- DR/DA trend (12 months)
- New vs. lost links
- Top 20 linking domains
- Anchor text diversity

**Tools:**
- Ahrefs Backlink Profile
- SEMrush Backlink Audit
- Google Disavow Tool (if needed)

---

### Phase 6: Reporting & Recommendations (Day 8-10)

#### Report Structure

**1. Executive Summary (2-3 pages)**
- Current state overview
- Top 3 opportunities
- Top 3 risks
- Quick wins (30 days)
- Investment required
- Expected outcomes

**2. Technical Findings (10-20 pages)**
- Crawlability issues
- Site speed analysis
- Mobile optimization
- Schema opportunities
- Priority matrix

**3. Content Findings (10-15 pages)**
- On-page issues
- Content gaps
- E-E-A-T gaps
- Keyword opportunities
- Content calendar recommendation

**4. Competitor Analysis (5-10 pages)**
- Competitive landscape
- Gap analysis
- Opportunity areas

**5. Backlink Analysis (5-10 pages)**
- Current profile health
- Toxic link assessment
- Link building opportunities

**6. 90-Day Roadmap (5 pages)**
- Phase 1 (Days 1-30): Quick wins
- Phase 2 (Days 31-60): Foundation
- Phase 3 (Days 61-90): Scale
- Resource requirements
- Success metrics

## Conversion Tips

1. **Lead with value** - Give 2-3 actionable insights in the pitch
2. **Show, don't tell** - Use visuals and data
3. **Tie to revenue** - Connect SEO issues to business impact
4. **Create urgency** - Highlight competitive threats
5. **Offer payment plans** - Make it accessible
