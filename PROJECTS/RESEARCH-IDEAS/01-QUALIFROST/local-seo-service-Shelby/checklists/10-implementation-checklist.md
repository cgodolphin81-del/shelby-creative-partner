# Local SEO Implementation Checklist
## Week-by-Week Execution Plan (90-Day Launch)

---

## 📅 OVERVIEW

This 90-day implementation plan provides a week-by-week roadmap for launching and executing a complete local SEO campaign. Follow this checklist to ensure all foundational elements are in place and momentum is built systematically.

**Timeline:** 12 weeks (90 days)  
**Phases:** Foundation → Optimization → Growth → Maintenance  
**Review Points:** Week 4, Week 8, Week 12

---

## 🏗️ PHASE 1: FOUNDATION (Weeks 1-4)

### WEEK 1: Discovery & Setup

#### Day 1-2: Client Onboarding
- [ ] Send and receive completed client onboarding form
- [ ] Schedule kickoff call (30-60 minutes)
- [ ] Collect all login credentials (secure transfer)
- [ ] Collect brand assets (logo, photos, etc.)
- [ ] Review current online presence
- [ ] Set up project management board (Trello/Asana/ClickUp)
- [ ] Create shared folder for assets and reports

#### Day 3-4: Audit & Analysis
- [ ] Complete 50-point Local SEO Audit
- [ ] Run citation audit (check top 50 directories)
- [ ] Analyze current Google Business Profile
- [ ] Review website SEO (technical audit)
- [ ] Analyze competitor profiles (top 3-5)
- [ ] Identify quick wins and critical issues
- [ ] Document baseline metrics

#### Day 5: Strategy Development
- [ ] Develop keyword research list (25-50 keywords)
- [ ] Create content calendar (first 90 days)
- [ ] Identify priority citations to build
- [ ] Plan review generation strategy
- [ ] Set 90-day goals and KPIs
- [ ] Prepare Week 1-4 action plan

**Week 1 Deliverables:**
- [ ] Completed audit report
- [ ] Keyword research document
- [ ] Competitor analysis summary
- [ ] 90-day strategy document

---

### WEEK 2: Google Business Profile Optimization

#### Day 1-2: Profile Setup/Claim
- [ ] Claim or create Google Business Profile
- [ ] Verify business (request postcard if needed)
- [ ] Complete all profile sections 100%
- [ ] Set accurate business hours
- [ ] Add service areas (if applicable)

#### Day 3-4: Profile Optimization
- [ ] Write optimized business description (750 chars)
- [ ] Select primary and secondary categories
- [ ] Add all relevant attributes
- [ ] Upload 15+ high-quality photos
  - [ ] Logo (profile photo)
  - [ ] Cover photo
  - [ ] Interior photos (5+)
  - [ ] Exterior photos (5+)
  - [ ] Team photos (3+)
  - [ ] Product/service photos (5+)
- [ ] Add products with descriptions and prices
- [ ] Populate services section

#### Day 5: Profile Activation
- [ ] Create first GBP post (announcement)
- [ ] Add Q&A (5 common questions)
- [ ] Generate review request link
- [ ] Create QR code for reviews
- [ ] Set up GBP messaging (if applicable)
- [ ] Test all profile elements

**Week 2 Deliverables:**
- [ ] Fully optimized Google Business Profile
- [ ] 15+ photos uploaded
- [ ] First post published
- [ ] Review generation system ready

### WEEK 3: Website On-Page SEO

#### Day 1-2: NAP & Schema
- [ ] Audit NAP consistency across website
- [ ] Update NAP on homepage (above the fold)
- [ ] Add NAP to footer (site-wide)
- [ ] Update contact page with complete NAP
- [ ] Implement LocalBusiness schema markup
- [ ] Test schema with Google Rich Results Test
- [ ] Add click-to-call functionality

#### Day 3-4: Page Optimization
- [ ] Optimize homepage:
  - [ ] Title tag (with city + primary keyword)
  - [ ] Meta description
  - [ ] H1 tag
  - [ ] Local content additions
- [ ] Optimize service pages (top 3):
  - [ ] Title tags
  - [ ] Meta descriptions
  - [ ] H1/H2 structure
  - [ ] Local keyword integration
- [ ] Optimize About page:
  - [ ] Local history and community ties
  - [ ] Service area mentions
  - [ ] Team bios

#### Day 5: Technical SEO
- [ ] Check mobile responsiveness
- [ ] Test page speed (target: <3 seconds)
- [ ] Fix broken links
- [ ] Optimize images (compress, alt text)
- [ ] Ensure SSL certificate active
- [ ] Submit XML sitemap to Google
- [ ] Verify Google Search Console setup

**Week 3 Deliverables:**
- [ ] NAP consistent across website
- [ ] Schema markup implemented
- [ ] Homepage optimized
- [ ] Top 3 service pages optimized
- [ ] Technical SEO issues addressed
