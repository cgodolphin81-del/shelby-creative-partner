# Google Business Profile Optimization Checklist
## Complete Setup & Optimization Guide

---

## 🚀 PHASE 1: INITIAL SETUP (Day 1-3)

### 1.1 Claim or Create Your Profile
- [ ] Go to google.com/business
- [ ] Search for existing business listing
- [ ] If exists: Click "Claim this business"
- [ ] If doesn't exist: Click "Add your business"
- [ ] Verify business ownership (postcard, phone, or email)

### 1.2 Business Information - Core Details
- [ ] **Business Name**: Use exact legal name (NO keyword stuffing)
  - ✅ Correct: "Smith Plumbing Co."
  - ❌ Wrong: "Smith Plumbing - Best Plumber in Denver 24/7 Emergency"
  
- [ ] **Business Category**: 
  - [ ] Select primary category (most specific match)
  - [ ] Add up to 9 additional categories
  - [ ] Research competitor categories for ideas

- [ ] **Location**:
  - [ ] Enter complete street address
  - [ ] Pin location accurately on map
  - [ ] Mark service areas if you serve customers at their locations
  - [ ] Add service radius (if applicable)

- [ ] **Contact Information**:
  - [ ] Local phone number (avoid toll-free for local SEO)
  - [ ] Website URL (use UTM parameters for tracking)
  - [ ] Email address (monitored regularly)

### 1.3 Business Hours
- [ ] Set regular weekly hours
- [ ] Add special hours for holidays
- [ ] Mark as "Open 24 hours" if applicable
- [ ] Update for seasonal changes
- [ ] Add "Temporarily closed" if needed

## 📝 PHASE 2: PROFILE COMPLETION (Day 4-7)

### 2.1 Business Description
- [ ] Write 750-character description
- [ ] Include primary keywords naturally
- [ ] Mention service areas
- [ ] Highlight unique selling propositions
- [ ] Include call-to-action
- [ ] Avoid promotional language or ALL CAPS

**Template:**
```
[Business Name] has been serving [City/Region] since [Year]. 
We specialize in [primary services] for [target customers]. 
Our team of [credibility markers] is committed to [value proposition]. 
Contact us today for [call-to-action]. Serving [list key areas].
```

### 2.2 Attributes
Select ALL that apply:
- [ ] Women-led
- [ ] Veteran-led
- [ ] Black-owned
- [ ] LGBTQ+ owned
- [ ] Disability owned
- [ ] Senior-led
- [ ] Locally owned
- [ ] Family-owned
- [ ] Wheelchair accessible
- [ ] Free Wi-Fi
- [ ] Accepts credit cards
- [ ] Accepts mobile payments
- [ ] Outdoor seating (restaurants)
- [ ] Delivery available
- [ ] Curbside pickup
- [ ] In-store shopping
- [ ] Same-day delivery
- [ ] Appointment required
- [ ] Emergency services

### 2.3 Photos & Videos
**Minimum Requirements:**
- [ ] Profile photo (logo) - 720x720px minimum
- [ ] Cover photo - 1080x608px recommended
- [ ] 10+ interior photos
- [ ] 10+ exterior photos
- [ ] 5+ team/staff photos
- [ ] 5+ product photos
- [ ] 3+ work-in-progress photos
- [ ] 1+ video (30 seconds max, auto-plays)

**Photo Best Practices:**
- [ ] High resolution (minimum 720px)
- [ ] Well-lit and in focus
- [ ] Show actual business (no stock photos)
- [ ] Geotagged with location data
- [ ] Upload new photos monthly
- [ ] Add captions with keywords

## 🛍️ PHASE 3: PRODUCTS & SERVICES (Week 2)

### 3.1 Products Section
- [ ] Add individual products with photos
- [ ] Include prices (or price ranges)
- [ ] Write detailed descriptions
- [ ] Organize into categories
- [ ] Highlight bestsellers
- [ ] Update seasonally

### 3.2 Services Section
- [ ] List all core services
- [ ] Add service descriptions
- [ ] Include pricing (or "Contact for quote")

```
Perfect for [target customer].

```

### 3.3 Booking & Appointments
- [ ] Enable booking button (if applicable)

- [ ] Set available time slots
- [ ] Configure appointment types
- [ ] Test booking flow

## 💬 PHASE 4: REVIEWS & ENGAGEMENT (Ongoing)

### 4.1 Review Generation
- [ ] Generate review request link
- [ ] Add review link to email signatures
- [ ] Create QR code for in-person requests
- [ ] Send review requests after purchases
- [ ] Follow up with satisfied customers
- [ ] Target: 5+ new reviews per month

**Review Request Template:**
```
Subject: How did we do, [Customer Name]?

Hi [Name],

Thanks for choosing [Business Name]! We hope you're 
happy with your [service/product].

Would you take 2 minutes to share your experience? 
Your review helps us serve [City] better!

[Review Link]

Thanks,
[Your Name]
```

### 4.2 Review Response
- [ ] Respond to ALL reviews (positive and negative)
- [ ] Respond within 48 hours
- [ ] Personalize each response
- [ ] Thank reviewers by name
- [ ] Address concerns professionally
- [ ] Take conversations offline when needed

**Positive Review Response Template:**
```
Hi [Name], thank you for the wonderful review! 
We're thrilled you enjoyed [specific detail they mentioned]. 
Our team works hard to [value proposition]. 
We look forward to serving you again soon!
- [Your Name], [Title]
```

**Negative Review Response Template:**
```
Hi [Name], we're sorry to hear about your experience. 
This doesn't reflect our usual standards. 
Please contact us at [phone/email] so we can make this right. 
We'd appreciate the opportunity to resolve this.
- [Your Name], [Title]
```

### 4.3 Q&A Section
- [ ] Add common questions proactively
- [ ] Write helpful, detailed answers
- [ ] Monitor for new questions daily
- [ ] Respond within 24 hours
- [ ] Upvote helpful customer answers

## 📊 PHASE 5: POSTS & UPDATES (Weekly)

### 5.1 Post Types
Create weekly posts:
- [ ] **What's New**: General updates
- [ ] **Events**: Upcoming events or promotions
- [ ] **Offers**: Special deals with expiry dates
- [ ] **Products**: Featured products/services

### 5.2 Post Best Practices
- [ ] Include high-quality photo (minimum 720px)
- [ ] Write 100-300 characters
- [ ] Include call-to-action button
- [ ] Add relevant keywords naturally
- [ ] Post at least once per week
- [ ] Use event/offer expiry dates

**Post Template:**
```
🎉 [Announcement]!

[2-3 sentences about the update/offer/event]

✅ [Benefit 1]
✅ [Benefit 2]

[Call-to-action: Call, Book, Learn More, etc.]

#LocalBusiness #[City] #[Service]
```
