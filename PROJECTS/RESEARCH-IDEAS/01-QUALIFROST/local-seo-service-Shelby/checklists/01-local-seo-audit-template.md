## 📍 SECTION 1: GOOGLE BUSINESS PROFILE (15 points)

### Profile Completeness
- [ ] **1.1** Business name matches real-world signage (no keyword stuffing)
- [ ] **1.2** Primary category accurately reflects core business
- [ ] **1.3** Secondary categories added (all relevant categories)
- [ ] **1.4** Business hours are accurate and up-to-date
- [ ] **1.5** Special hours set for holidays/events
- [ ] **1.6** Phone number is local (not toll-free) and clickable
- [ ] **1.7** Website URL uses UTM tracking parameters
- [ ] **1.8** Business description optimized with keywords (750 chars)
- [ ] **1.9** Attributes selected (all applicable: women-led, veteran, etc.)
- [ ] **1.10** Products/services section populated

### Visual Content
- [ ] **1.11** Profile photo uploaded (logo, 720x720px minimum)
- [ ] **1.12** Cover photo uploaded (1080x608px recommended)
- [ ] **1.13** Minimum 10 interior/exterior photos
- [ ] **1.14** Team/staff photos included
- [ ] **1.15** Photos geotagged with location data

## 📋 SECTION 2: CITATIONS & NAP CONSISTENCY (12 points)

### Core Citations
- [ ] **2.1** Google Business Profile claimed & verified
- [ ] **2.2** Apple Business Connect profile created
- [ ] **2.3** Bing Places profile created
- [ ] **2.4** Facebook Business page active
- [ ] **2.5** Yelp business listing claimed

### Industry-Specific Directories
- [ ] **2.6** YellowPages listing exists
- [ ] **2.7** Manta listing exists
- [ ] **2.8** Foursquare/Swarm listing exists
- [ ] **2.9** Better Business Bureau profile (if applicable)
- [ ] **2.10** Industry-specific directories (e.g., Healthgrades, Avvo, Houzz)

### NAP Consistency Check
- [ ] **2.11** Business name identical across all platforms
- [ ] **2.12** Address format consistent (St. vs Street, Suite vs Ste)
- [ ] **2.13** Phone number format consistent
- [ ] **2.14** Website URL consistent (www vs non-www, https)

## 🖥️ SECTION 3: ON-PAGE SEO (12 points)

### Technical Foundation
- [ ] **3.1** SSL certificate installed (HTTPS)
- [ ] **3.2** Mobile-responsive design
- [ ] **3.3** Page load speed under 3 seconds
- [ ] **3.4** XML sitemap submitted to Google Search Console
- [ ] **3.5** Robots.txt properly configured

### Local SEO Elements
- [ ] **3.6** NAP displayed prominently on homepage
- [ ] **3.7** NAP in website footer
- [ ] **3.8** Embedded Google Map on contact/location page
- [ ] **3.9** Local schema markup implemented (LocalBusiness)
- [ ] **3.10** Location pages for each service area (if multi-location)

### Content Optimization
- [ ] **3.11** Title tags include city/region + primary keyword
- [ ] **3.12** Meta descriptions compelling with local intent
- [ ] **3.13** H1 tags include location modifiers
- [ ] **3.14** Local content/blog posts published (minimum 4/year)
- [ ] **3.15** About page mentions service areas and local history
- [ ] **3.16** Service pages optimized for local keywords

---

## ⭐ SECTION 4: REVIEWS & REPUTATION (8 points)

### Review Presence
- [ ] **4.1** Minimum 25 Google reviews
- [ ] **4.2** Average rating 4.5 stars or higher
- [ ] **4.3** Reviews received in last 90 days (minimum 3)
- [ ] **4.4** Reviews across multiple platforms (Google, Yelp, Facebook)

### Review Management
- [ ] **4.5** All reviews responded to (positive and negative)
- [ ] **4.6** Responses personalized (not template copies)
- [ ] **4.7** Response time under 48 hours
- [ ] **4.8** Review generation system in place (email/SMS requests)
