- Use simple, clear language
- Write in active voice
- Address reader as "you"
- Be specific and concrete
- Show personality

**Don't:**
- Use jargon or buzzwords
- Write in passive voice
- Be overly formal or stiff
- Make vague claims
- Sound like a robot

## 8. Dos & Don'ts

### Logo

| Do | Don't |
|----|-------|
| ✅ Use approved colors | ❌ Recolor logo |
| ✅ Maintain clear space | ❌ Crowd the logo |
| ✅ Use high-resolution files | ❌ Use pixelated logos |
| ✅ Keep proportions | ❌ Stretch or distort |

### Colors

| Do | Don't |
|----|-------|
| ✅ Use palette consistently | ❌ Add new colors |
| ✅ Follow 60-30-10 rule | ❌ Overuse accent colors |
| ✅ Check accessibility | ❌ Ignore contrast |
| ✅ Use correct formats | ❌ Approximate colors |

### Typography

| Do | Don't |
|----|-------|
| ✅ Use approved fonts | ❌ Add new fonts |
| ✅ Follow hierarchy | ❌ Use all caps excessively |
| ✅ Maintain readability | ❌ Use decorative fonts for body |
| ✅ Check spacing | ❌ Tight or loose kerning |

### Imagery

| Do | Don't |
|----|-------|
| ✅ Use high-quality photos | ❌ Use low-resolution images |
| ✅ Maintain consistent style | ❌ Mix conflicting styles |
| ✅ Show diversity | ❌ Use only stock photos |
| ✅ Get proper licenses | ❌ Use copyrighted images |
