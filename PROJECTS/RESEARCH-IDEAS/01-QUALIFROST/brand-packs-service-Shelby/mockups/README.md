## 🔧 How to Use Mockups

### Using PSD Mockups (Photoshop)

**Step 1: Open the File**
```
1. Open Photoshop
2. File → Open → Select mockup.psd
3. Wait for file to load
```

**Step 2: Find Smart Object**
```
1. Open Layers panel (Window → Layers)
2. Look for layer named "Your Design Here" 
   or "Smart Object"
3. Double-click the layer thumbnail
```

**Step 3: Add Your Design**
```
1. New document opens
2. Place your design (File → Place Embedded)
3. Resize to fit
4. Save (Cmd/Ctrl + S)
5. Close smart object document
```

**Step 4: Adjust Settings**
```
1. Back in main mockup
2. Adjust blending modes if needed
3. Adjust opacity
4. Add shadows/highlights
5. File → Export → Export As
```

**Step 5: Export**
```
1. File → Export → Export As
2. Choose format (PNG for web, JPG for photos)
3. Set quality (80-100%)
4. Click Export
5. Save to desired location
```

### Using PNG Mockups

**Step 1: Open in Any Editor**
- Photoshop, Canva, Photopea (free)

**Step 2: Add Design Layer**
```
1. Open mockup.png
2. Place your design on new layer
3. Position over mockup
4. Use blending modes if needed
```

**Step 3: Export**
```
1. Flatten or merge layers
2. Export as PNG or JPG
3. Save for web or print
```

### Using Figma Mockups

**Step 1: Open in Figma**
```
1. Go to figma.com
2. Click "Import" or drag .fig file
3. Wait for file to load
```

**Step 2: Replace Image**
```
1. Select the image frame
2. Right-click → "Choose Image"
3. Select your design
4. Adjust fit (Fill, Fit, Crop)
```

**Step 3: Export**
```
1. Select frame
2. Export section in right panel
3. Choose format (PNG, JPG, PDF)
4. Click Export
```

## ✅ Best Practices

### Mockup Selection

**Choose the Right Mockup:**
- Match your product type
- Consider your audience
- Select appropriate context
- Ensure quality matches brand

**Quality Standards:**
- Minimum 300 DPI for print
- Minimum 72 DPI for web
- Clean, professional appearance
- Proper lighting and shadows

### Design Placement

**Sizing:**
- Fill the designated area
- Don't stretch or distort
- Maintain aspect ratio
- Leave appropriate margins

**Alignment:**
- Center when appropriate
- Follow natural lines
- Consider focal points
- Check from multiple angles

### Realism

**Lighting:**
- Match light direction
- Add appropriate shadows
- Consider reflections
- Adjust brightness/contrast

**Textures:**
- Preserve mockup textures
- Use appropriate blending modes
- Don't make design too perfect
- Add subtle wear if appropriate

### Consistency

**Across Mockups:**
- Use same design versions
- Maintain color accuracy
- Keep style consistent
- Use similar angles/lighting

**Brand Consistency:**
- Include logo where appropriate
- Use brand colors
- Maintain brand style
- Keep messaging consistent

   - Match light direction
   - Add realistic shadows

**Common Issues:**

**Q: Smart object won't update**
A: Make sure you save (Cmd/Ctrl + S) the smart object document, not the main mockup.

**Q: Design looks pixelated**
A: Use higher resolution design files. Minimum 300 DPI for print, 72 DPI for web.

**Q: Colors look different**
A: Ensure color profiles match. Use sRGB for web, CMYK for print.
