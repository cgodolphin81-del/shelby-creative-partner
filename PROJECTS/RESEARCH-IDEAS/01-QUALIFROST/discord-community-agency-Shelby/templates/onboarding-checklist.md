# Client Onboarding Checklist

## Complete Onboarding Process

---

## 📋 Pre-Kickoff (Before First Meeting)

### Documentation & Setup
- [ ] Send welcome email with kickoff details
- [ ] Create shared project folder (Notion/Google Drive)
- [ ] Set up project management board (Asana/Trello/ClickUp)
- [ ] Create communication channels (Slack/Discord)
- [ ] Prepare kickoff presentation template
- [ ] Gather client contact information
- [ ] Set up time tracking (if applicable)

### Technical Requirements
- [ ] List server access requirements
- [ ] Identify bot permissions needed
- [ ] Prepare access request template
- [ ] Document brand assets needed
- [ ] List integrations to configure
- [ ] Prepare API key request form

---

## 🎯 Kickoff Call Agenda

### Introduction (10 min)
- [ ] Team introductions
- [ ] Project overview
- [ ] Goals and expectations
- [ ] Timeline review

### Discovery (20 min)
- [ ] Current state assessment
- [ ] Pain points & challenges
- [ ] Success metrics discussion
- [ ] Decision-maker identification
- [ ] Communication preferences

### Next Steps (10 min)
- [ ] Immediate action items
- [ ] Access & permissions needed
- [ ] Information to provide
- [ ] Timeline confirmation
- [ ] Q&A

**Duration:** 40-45 minutes  
**Attendees:** Project lead (both sides), key stakeholders
