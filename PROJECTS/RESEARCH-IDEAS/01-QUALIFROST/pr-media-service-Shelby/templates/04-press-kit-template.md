# Press Kit Template

## Overview

A press kit (media kit) provides journalists with everything they need to write about your company. This template includes all essential components.

### Executive Headshot Guidelines

```
Requirements:
• Resolution: Minimum 300 DPI, 2000px on longest side
• Format: High-resolution JPG or PNG
• Background: Neutral (white, gray, or blurred)
• Lighting: Professional, even lighting
• Attire: Business casual or industry-appropriate
• Expression: Approachable, confident
• Orientation: Both portrait and landscape versions
• File naming: firstname-lastname-headshot.jpg

Avoid:
✗ Selfies or casual photos
✗ Busy backgrounds
✗ Sunglasses or hats
✗ Heavy filters
✗ Group photos (crop to individual)
```

## Component 4: Product Assets

### Screenshot Guidelines

```
Requirements:
• Resolution: Minimum 1920x1080
• Format: PNG (for UI) or JPG (for photos)
• Context: Show product in use, not just login screens
• Variety: Dashboard, key features, mobile views
• Annotations: Optional callouts for key features

File Naming:
• product-dashboard.png
• product-feature-x.png
• product-mobile-view.png
• product-integration-y.png
```

### Demo Video Guidelines

```
Specifications:
• Length: 2-3 minutes maximum
• Format: MP4 (H.264 codec)
• Resolution: 1080p minimum
• Audio: Clear narration or text overlays
• Content: Problem → Solution → Key features → Call to action

Structure:
0:00-0:15 - Hook (problem statement)
0:15-0:45 - Product overview
0:45-1:45 - Key features demo
1:45-2:15 - Customer benefit/outcome
2:15-2:30 - Call to action

Hosting:
• Upload to YouTube (unlisted) or Vimeo
• Embed on press kit page
• Provide download link for media
```

## Press Kit Best Practices

### DO:
✓ Keep everything up-to-date (review quarterly)
✓ Make downloads easy (no gatekeeping)
✓ Include high-resolution assets (300 DPI for print)
✓ Provide multiple file formats (PNG, JPG, SVG)
✓ Mobile-optimize the press kit page
✓ Include clear contact information
✓ Update within 24 hours of major news

### DON'T:
✗ Require email capture for basic assets
✗ Use low-resolution images
✗ Hide contact information
✗ Let bios/facts become outdated
✗ Make journalists search for information
✗ Forget to include social media links

### Hosting Options:
1. **Dedicated Press Page:** yourcompany.com/press
2. **Press Kit Services:** PressKit.com, Newswire.com
3. **Cloud Folder:** Google Drive, Dropbox (public link)
4. **GitHub Pages:** Free, version-controlled
