## Overview

This guide covers the top 4 webinar platforms with setup instructions, pricing, pros/cons, and best use cases.

## 1️⃣ ZOOM WEBINARS

### Best For: Interactive training, internal webinars, Q&A-heavy sessions

### Pricing
- **Webinar Add-on**: $79/mo (up to 500 attendees)
- **Webinar Add-on**: $149/mo (up to 1,000 attendees)
- **Enterprise**: Custom pricing

### Setup Instructions

#### Step 1: Account Setup
```
1. Go to zoom.us and sign up/log in
2. Navigate to "Webinars" in left sidebar
3. Click "Schedule a Webinar"
4. Fill in webinar details:
   - Topic
   - Description
   - Date & Time
   - Duration
   - Time Zone
5. Configure registration (required for lead capture)
6. Click "Schedule"
```

#### Step 2: Registration Page Setup
```
1. After scheduling, click on your webinar
2. Go to "Registration" tab
3. Customize registration form:
   - Add custom questions
   - Set approval method (automatic/manual)
   - Add custom branding
4. Copy registration link for promotion
```

#### Step 3: Email Configuration
```
1. Go to "Email" tab in webinar settings
2. Customize confirmation email:
   - Subject line
   - Message body
   - Add calendar invite
3. Set up reminder emails:
   - 24 hours before
   - 1 hour before
   - Custom additional reminders
4. Customize confirmation email for after registration
```

#### Step 4: Webinar Room Setup
```
1. Join webinar 30 minutes early
2. Configure video/audio:
   - Test microphone
   - Test camera
   - Check lighting
3. Set up screen sharing:
   - Have slides ready
   - Close unnecessary tabs
   - Test video playback
4. Configure panelists:
   - Add co-hosts
   - Assign roles
   - Test their audio/video
5. Enable features:
   - Chat (on/off/moderated)
   - Q&A
   - Polls
   - Closed captioning
```

#### Step 5: During the Webinar
```
CHECKLIST:
□ Start recording (cloud or local)
□ Admit attendees from waiting room
□ Welcome people as they join
□ Launch polls at planned times
□ Monitor chat for questions
□ Share screen when needed
□ Manage Q&A queue
□ Keep an eye on time
```

#### Step 6: Post-Webinar
```
1. Stop recording
2. End webinar for all
3. Download recording (if local)
4. Access analytics:
   - Registration count
   - Attendance rate
   - Engagement metrics
   - Poll results
5. Send follow-up email:
   - Thank attendees
   - Share replay
   - Include offer
6. Export attendee list for CRM
```

### Zoom Pros
✅ Universally recognized (people trust it)
✅ Reliable, rarely has technical issues
✅ Good video/audio quality
✅ Built-in breakout rooms
✅ Strong security features
✅ Polls and Q&A built-in
✅ Mobile app works well

### Zoom Cons
❌ Registration pages are basic
❌ Limited marketing automation
❌ No built-in sales features
❌ Expensive for large audiences
❌ Webinar features require add-on
❌ Can feel "corporate" vs. engaging

### Best Practices for Zoom
- Use waiting room to control entry
- Have a co-host to manage chat/Q&A
- Enable "Allow participants to join before host" for early arrivals
- Use virtual background if your space isn't professional
- Test internet speed before going live (minimum 10 Mbps upload)

## 2️⃣ WEBINARJAM

### Best For: Marketing webinars, sales funnels, automated evergreen webinars

### Pricing
- **Starter**: $499/year (500 attendees, 2 hosts)
- **Professional**: $699/year (1,000 attendees, 4 hosts)
- **Enterprise**: $999/year (5,000 attendees, 6 hosts)

### Setup Instructions

#### Step 1: Account & First Webinar
```
1. Go to webinarjam.com and sign up
2. Complete onboarding wizard
3. Click "Create New Webinar"
4. Choose webinar type:
   - Live
   - Series (recurring)
   - Evergreen (automated)
   - Hybrid
5. Name your webinar
6. Set date/time (or evergreen schedule)
```

#### Step 2: Registration Page Builder
```
1. Go to "Pages" tab
2. Choose a template or start from scratch
3. Customize:
   - Headline
   - Subheadline
   - Bullet points
   - Host photo/bio
   - Date/time display
   - Registration form fields
4. Set up thank you page:
   - Confirmation message
   - Calendar add buttons
   - Social share buttons
5. Connect email provider (or use built-in)
6. Publish and get your URL
```

#### Step 3: Email Automation
```
1. Go to "Emails" tab
2. Set up email sequence:
   - Immediate confirmation
   - 24-hour reminder
   - 1-hour reminder
   - 10-minute reminder
   - Replay email (post-webinar)
   - Follow-up sequence
3. Customize each email:
   - Subject line
   - Body content
   - Personalization tags
   - CTA buttons
4. Set timing for each email
5. Test emails to yourself
```

#### Step 4: Webinar Room Configuration
```
1. Go to "Broadcast" tab
2. Configure room settings:
   - Welcome message
   - Chat settings
   - Poll creation
   - Handout uploads
   - Offer timing
3. Set up "Special Offer" page:
   - Create offer page within WebinarJam
   - Add product details
   - Set pricing
   - Connect payment processor
   - Set urgency (timer, spots)
4. Upload slides/video
5. Test all features
```

#### Step 5: Go Live Checklist
```
30 MINUTES BEFORE:
□ Log in to broadcast room
□ Test audio/video
□ Upload any last-minute slides
□ Set chat welcome message
□ Test special offer page
□ Confirm email automation is active

15 MINUTES BEFORE:
□ Start "waiting room" stream
□ Welcome early arrivals
□ Test screen sharing
□ Have co-host monitor chat

WEBINAR START:
□ Begin recording
□ Start presentation
□ Launch polls as planned
□ Present offer at planned time
□ Monitor registration/conversion dashboard
```

#### Step 6: Post-Webinar
```
1. End broadcast
2. Access analytics dashboard:
   - Registrations
   - Attendance rate
   - Peak attendance
   - Offer views
   - Sales/conversions
   - Revenue
3. Download attendee list
4. Replay automatically sent (if configured)
5. Review email sequence performance
6. Export data for CRM
```

### WebinarJam Pros
✅ Built specifically for marketing webinars
✅ Excellent registration page templates
✅ Full email automation included
✅ Built-in offer pages with urgency
✅ Evergreen webinar capability
✅ Detailed conversion analytics
✅ No transaction fees
✅ Great customer support

### WebinarJam Cons
❌ More expensive upfront (annual pricing)
❌ Steeper learning curve than Zoom
❌ Video quality slightly lower than Zoom
❌ Interface can feel cluttered
❌ Less ideal for training/education focus

### Best Practices for WebinarJam
- Use their templates for registration pages (they're optimized)
- Set up the evergreen option after running live successfully
- Use the built-in offer pages for seamless checkout
- Leverage the "panic button" for technical issues (switches to backup)
- A/B test your registration pages

## 3️⃣ STREAMYARD

### Best For: Live streaming to multiple platforms, interview-style webinars, podcasts

### Pricing
- **Free**: Basic features, 20 hours/month, StreamYard branding
- **Professional**: $20/mo (no branding, 1080p, recording)
- **Growth**: $39/mo (custom branding, multi-stream, analytics)
- **Professional Plus**: $99/mo (1080p, priority support)

### Setup Instructions

#### Step 1: Account & Studio Setup
```
1. Go to streamyard.com and sign up
2. Complete profile setup
3. Create your "Brand Kit":
   - Upload logo
   - Set brand colors
   - Choose fonts
   - Upload intro/outro video (optional)
4. Connect destinations:
   - YouTube
   - Facebook
   - LinkedIn
   - Twitter
   - Twitch
   - Custom RTMP
```

#### Step 2: Create a Broadcast
```
1. Click "Create" → "Broadcast"
2. Choose broadcast type:
   - Stream (live to platforms)
   - Record (record only)
   - Schedule (plan for later)
3. Fill in details:
   - Title
   - Description
   - Thumbnail image
   - Schedule date/time
4. Select destinations (where to stream)
5. Invite guests (up to 10 on paid plans):
   - Send email invites
   - Share guest link
   - Guests join via browser (no download)
```

#### Step 3: Customize Your Studio
```
1. Choose layout:
   - Single speaker
   - Side by side
   - Grid view
   - Screen share focus
2. Add overlays:
   - Lower thirds (name tags)
   - Banner messages
   - Logo placement
3. Set up screen sharing:
   - Full screen
   - Window
   - Tab
4. Configure audio:
   - Select microphone
   - Select speakers
   - Test levels
   - Enable noise cancellation
```

#### Step 4: Go Live
```
PRE-BROADCAST:
□ Join 15 minutes early
□ Test all guests' audio/video
□ Review run of show
□ Have backup plan for tech issues

GOING LIVE:
□ Click "Go Live"
□ Confirm destinations
□ Start with welcome/intro
□ Monitor comments from all platforms
□ Engage with audience
□ Follow your script/outline

ENDING:
□ Give closing remarks
□ Thank guests and audience
□ Click "End Broadcast"
□ Recording saves automatically
```

#### Step 5: Post-Broadcast
```
1. Access recording in "Content" tab
2. Download or edit:
   - Trim start/end
   - Add intro/outro
   - Export in desired format
3. Download chat/comments
4. Review analytics:
   - View count per platform
   - Peak viewers
   - Engagement metrics
   - Watch time
5. Repurpose content:
   - Upload to YouTube
   - Create clips
   - Transcribe for blog/email
```

### StreamYard Pros
✅ Extremely easy to use
✅ Multi-platform streaming
✅ Guests join via browser (no software)
✅ Good free tier
✅ Built-in branding options
✅ Reliable streaming quality
✅ Great for interviews/panels
✅ Automatic recording

### StreamYard Cons
❌ No built-in registration pages
❌ No email automation
❌ No sales/offer features
❌ Limited analytics on lower plans
❌ Not ideal for pure webinar funnels
❌ No evergreen capability

### Best Practices for StreamYard
- Use for live events, not automated funnels
- Pair with external registration (Eventbrite, Zoom, etc.)
- Create clips for social media promotion
- Use the "highlight" feature to mark key moments
- Have a producer manage comments while you present

## 4️⃣ RIVERSIDE

### Best For: High-quality recordings, podcast-style webinars, content repurposing

### Pricing
- **Standard**: $15/mo (5 hours recording, 720p)
- **Pro**: $24/mo (15 hours, 4K, AI tools)
- **Business**: $49/mo (50 hours, priority, team)
- **Enterprise**: Custom

### Setup Instructions

#### Step 1: Account & Studio
```
1. Go to riverside.fm and sign up
2. Create your studio:
   - Name your studio
   - Set default settings
3. Configure recording settings:
   - Video quality (up to 4K)
   - Audio quality (up to 48kHz)
   - Local recording (each track separate)
4. Set up branding:
   - Logo
   - Background
   - Custom waiting room
```

#### Step 2: Schedule a Recording
```
1. Click "Record" → "Schedule"
2. Fill in details:
   - Event name
   - Date & time
   - Duration
   - Time zone
3. Add guests:
   - Enter email addresses
   - Add custom message
   - Set guest permissions
4. Customize event page:
   - Add description
   - Upload thumbnail
   - Set public/private
5. Send invitations
```

#### Step 3: Pre-Recording Check
```
GUESTS RECEIVE:
- Email with join link
- Tech check page
- Instructions for optimal setup

HOST SHOULD:
□ Send reminder 24 hours before
□ Ask guests to test their setup
□ Have backup communication (text/phone)
□ Prepare recording outline
```

#### Step 4: Recording Session
```
BEFORE STARTING:
□ Join 10 minutes early
□ Greet guests as they arrive
□ Run audio/video checks
□ Review format and flow
□ Start recording

DURING RECORDING:
□ Monitor audio levels
□ Watch for connection issues
□ Follow your outline
□ Record separate tracks (automatic)
□ Use markers for key moments

ENDING:
□ Thank guests
□ Stop recording
□ Files upload automatically
□ Guests can download their tracks
```

#### Step 5: Post-Production
```
1. Access studio dashboard
2. Review recordings:
   - Separate video tracks
   - Separate audio tracks
   - Chat transcript
3. Use AI tools:
   - Auto-transcription
   - Magic Clips (auto-highlights)
   - Text-based editing
4. Export options:
   - Full recording
   - Individual tracks
   - Audio only (for podcast)
   - Video clips
5. Download or publish directly to:
   - YouTube
   - Podcast platforms
   - Social media
```

### Riverside Pros
✅ Highest quality recording (local 4K)
✅ Separate tracks for each speaker
✅ Excellent for content repurposing
✅ AI-powered editing tools
✅ Reliable even with poor internet
✅ Professional results
✅ Great for podcast + webinar hybrid
✅ Text-based video editing

### Riverside Cons
❌ Not designed for live audience interaction
❌ No registration pages
❌ No email automation
❌ No sales features
❌ Primarily for recording, not live streaming
❌ Limited audience engagement features

### Best Practices for Riverside
- Use for pre-recorded "webinars" that feel live
- Edit out mistakes before publishing
- Create multiple content pieces from one recording
- Use for high-value evergreen content
- Pair with external marketing tools for promotion

## 🔧 TECHNICAL REQUIREMENTS (ALL PLATFORMS)

### Internet Connection
- **Minimum**: 10 Mbps upload, 25 Mbps download
- **Recommended**: 25+ Mbps upload, 50+ Mbps download
- **Test**: speedtest.net before going live
- **Backup**: Have mobile hotspot ready

### Hardware
- **Computer**: Modern laptop/desktop (less than 5 years old)
- **Microphone**: USB condenser mic (Blue Yeti, Audio-Technica)
- **Camera**: 1080p webcam or DSLR (Logitech C920 minimum)
- **Lighting**: Ring light or natural light in front of you
- **Headphones**: Wired headphones (prevent echo)

### Software
- **Browser**: Chrome or Firefox (latest version)
- **Updates**: All software updated 24 hours before
- **Tabs**: Close unnecessary tabs/applications
- **Notifications**: Enable Do Not Disturb mode

### Environment
- **Background**: Clean, professional, or virtual background
- **Noise**: Quiet space, notify household members
- **Power**: Plugged in (not on battery)
- **Temperature**: Room not too warm (prevent sweating)

## 🚨 TROUBLESHOOTING GUIDE

### Audio Issues
```
PROBLEM: Echo or feedback
SOLUTION: Use headphones, not speakers

PROBLEM: Too quiet
SOLUTION: Check mic input level, move closer to mic

PROBLEM: Robotic/choppy audio
SOLUTION: Check internet speed, close other apps

PROBLEM: No audio
SOLUTION: Check correct mic selected in settings
```

### Video Issues
```
PROBLEM: Pixelated video
SOLUTION: Reduce video quality, check upload speed

PROBLEM: Frozen video
SOLUTION: Restart stream, check internet

PROBLEM: Dark/poor lighting
SOLUTION: Add light source in front of you

PROBLEM: Wrong camera
SOLUTION: Select correct camera in settings
```

### Connection Issues
```
PROBLEM: Dropped connection
SOLUTION: Switch to mobile hotspot, restart router

PROBLEM: Can't join webinar
SOLUTION: Clear browser cache, try incognito mode

PROBLEM: Lag/delay
SOLUTION: Close other bandwidth-heavy apps
```

### Platform-Specific Issues
```
ZOOM: Use "Optimize for video sharing" when showing video
WEBINARJAM: Use panic button for instant backup stream
STREAMYARD: Have co-host ready to take over if needed
RIVERSIDE: Record locally as backup to cloud
```
