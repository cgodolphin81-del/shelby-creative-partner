# Team Building

**Building a High-Performance Web3 Agency Team**

---

## Overview

This guide covers building, managing, and scaling a Web3 agency team from solo founder to full-service operation.

**Team Growth Stages:**
- Stage 1: Solo/Founder-led (0-6 months)
- Stage 2: Core Team (6-18 months)
- Stage 3: Scaled Agency (18+ months)

---

## Stage 1: Solo/Founder-Led

### Reality Check

**You'll need to wear all hats:**
- Sales/BD
- Project Management
- Strategy
- Some execution
- Client communication

**What to Outsource First:**
1. Specialized technical work (smart contracts)
2. Design/creative
3. Content writing
4. Community moderation

### Contractor Network

**Build relationships with:**

| Role | Rate | Where to Find |
|------|------|---------------|
| Smart Contract Dev | $100-$250/hr | Twitter, Discord, GitHub |
| Full-Stack Dev | $75-$150/hr | Upwork, Toptal, Twitter |
| UI/UX Designer | $50-$150/hr | Dribbble, Behance, Twitter |
| Content Writer | $50-$150/hr | Twitter, Crypto media |
| Community Mod | $15-$30/hr | Discord, Telegram |
| Video Editor | $40-$100/hr | YouTube, Fiverr Pro |

### Managing Contractors

**Best Practices:**
- Clear scope of work
- Fixed price when possible
- Milestone-based payments
- Written communication
- Portfolio review before hiring
- Trial project first

**Tools:**
- Notion/Asana for project management
- Slack/Discord for communication
- GitHub for code
- Figma for design
- Loom for async updates

## Stage 2: Core Team (5-15 people)

**Priority Order:**

1. **Head of Delivery/PM** ($80K-$120K)

   - Ensures quality & timelines

2. **Senior Smart Contract Dev** ($120K-$200K)

3. **Community Manager** ($50K-$80K)

### Scaling Considerations

**When to Hire:**
- Revenue can support 6+ months of salary
- Consistent pipeline of work
- Founder is bottleneck
- Quality is suffering

**Growth Rate:**
- Sustainable: 2-5 hires/quarter
- Aggressive: 5-10 hires/quarter
- Don't grow faster than revenue

**Maintaining Culture:**
- Regular all-hands meetings
- Clear values document
- Onboarding program
- Team offsites (quarterly/annually)
- Recognition programs

## Hiring Process

### Standard Process

**1. Application Review (1-2 days)**
- Resume/portfolio review
- Initial screening criteria

**2. First Interview (30 min)**
- Culture fit
- Background review
- Role expectations
- Salary expectations

**3. Skills Assessment (3-5 days)**
- Technical test (devs)
- Writing sample (writers)
- Design challenge (designers)
- Case study (PMs, marketers)

**4. Final Interview (45-60 min)**
- Deep dive on assessment
- Team fit
- Scenario questions
- Candidate questions

**5. Reference Check (1-2 days)**
- 2-3 professional references
- Previous manager preferred

**6. Offer (1 day)**
- Written offer
- Negotiation
- Acceptance

**Total Timeline:** 1-2 weeks

### Assessment Examples

**Smart Contract Dev:**
```
Task: Write an ERC-721 contract with:
- Allowlist minting
- Public minting
- Royalty enforcement
- Pause functionality

Deliver: GitHub repo with tests
Time: 4-6 hours
Compensation: $500-$1,000
```

**Designer:**
```
Task: Design landing page for NFT project
Provide: Brand guidelines, copy
Deliver: Figma file
Time: 4-8 hours
Compensation: $300-$800
```

**Writer:**
```
Task: Write 500-word article on [Web3 topic]
Provide: Topic, key points
Deliver: Google Doc
Time: 2-4 hours
Compensation: $150-$400
```

## Remote Team Management

### Communication Tools

| Purpose | Tools |
|---------|-------|
| Async Chat | Slack, Discord |
| Video Calls | Zoom, Google Meet |
| Project Management | Notion, Asana, Linear |
| Documentation | Notion, Google Docs |
| Design | Figma |
| Code | GitHub, GitLab |
| Time Tracking | Toggl, Harvest |

### Best Practices

**Daily:**
- Standup updates (async or video)
- Slack/Discord availability
- Quick response times (<2 hours)

**Weekly:**
- Team sync (30-60 min)
- 1:1s with direct reports
- Progress reviews

**Monthly:**
- All-hands meeting
- Company updates
- Recognition

**Quarterly:**
- Performance reviews
- Goal setting
- Team offsite (if possible)

### Timezone Management

**Overlap Requirements:**
- Minimum 4 hours overlap for core team
- Document everything async
- Record all meetings
- Clear handoff processes

**Meeting Times:**
- Rotate meeting times for fairness
- Record for those who can't attend
- Keep meetings concise

## Budget Planning

### Team Cost Examples

**5-Person Team:**
```
CEO/Founder: $150K
Smart Contract Lead: $160K
Full-Stack Dev: $130K
Designer: $100K
Community Manager: $65K
Total Salaries: $605K
+ Benefits (20%): $121K
+ Tools/Overhead: $50K
Total: ~$776K/year
```

**15-Person Team:**
```
Leadership (3): $450K
Engineering (5): $700K
Marketing (3): $300K
Community (2): $130K
Design (1): $100K
Operations (1): $80K
Total Salaries: $1.76M
+ Benefits (20%): $352K
+ Tools/Overhead: $150K
Total: ~$2.26M/year
```

**30-Person Team:**
```
Leadership (5): $1M
Engineering (12): $1.7M
Marketing (5): $500K
Community (5): $350K
Design (2): $200K
Operations (1): $100K
Total Salaries: $3.85M
+ Benefits (20%): $770K
```

### Revenue Requirements

**Rule of Thumb:**
- Team costs should be 40-60% of revenue
- Aim for 20-30% profit margin
- Reinvest 20-30% in growth

**Revenue Targets:**
```
5-person team: $1.5M-$2M/year revenue
15-person team: $4M-$6M/year revenue
30-person team: $8M-$12M/year revenue
```
