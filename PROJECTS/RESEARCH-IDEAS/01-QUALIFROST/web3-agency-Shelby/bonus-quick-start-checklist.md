# Quick-Start Checklist

**90-Day Launch Plan for Your Web3 Agency**

---

## Phase 1: Foundation (Days 1-15)

### Legal & Admin

- [ ] Choose business name
- [ ] Register business entity (LLC, etc.)
- [ ] Get EIN/tax ID
- [ ] Open business bank account
- [ ] Set up accounting software (QuickBooks, Xero)
- [ ] Get business insurance
- [ ] Create contract templates
- [ ] Set up payment processing (crypto + fiat)

### Brand & Presence

- [ ] Design logo and brand assets
- [ ] Build agency website (1-3 pages minimum)
- [ ] Set up professional email
- [ ] Create Twitter/X account
- [ ] Create LinkedIn company page
- [ ] Set up Discord server (internal)
- [ ] Create pitch deck template

### Service Definition

- [ ] Define core service offerings
- [ ] Set pricing for each service
- [ ] Create service packages
- [ ] Build proposal template
- [ ] Define ideal client profile

---

## Phase 2: Capability Building (Days 16-45)

### Team & Network

- [ ] Identify skills you have in-house
- [ ] Recruit 3-5 trusted contractors:
  - [ ] Smart contract developer
  - [ ] Full-stack developer
  - [ ] Designer
  - [ ] Content writer
  - [ ] Community manager
- [ ] Vet and test each contractor
- [ ] Create contractor agreements
- [ ] Build partner network:
  - [ ] Audit firms (2-3)
  - [ ] Law firms (1-2)
  - [ ] Other agencies (for referrals)

### Portfolio Development

- [ ] Create 2-3 detailed case studies
  - [ ] If no clients: do pro bono/discounted work
  - [ ] Or: document your own projects
- [ ] Build portfolio page on website
- [ ] Get testimonials (even from beta work)
- [ ] Create capability statement PDF

### Systems & Tools

- [ ] Project management: Notion/Asana
- [ ] CRM: HubSpot/Pipedrive/Spreadsheet
- [ ] Communication: Slack/Discord
- [ ] Design: Figma
- [ ] Development: GitHub
- [ ] Video calls: Zoom/Google Meet
- [ ] E-signature: DocuSign/PandaDoc
- [ ] Time tracking: Toggl (optional)

---

## Phase 3: Lead Generation (Days 46-75)

### Inbound Setup

- [ ] Start Twitter content strategy
  - [ ] Post 3-5x per day
  - [ ] Engage with 20+ accounts daily
  - [ ] Create 2 threads per week
- [ ] Start LinkedIn presence
  - [ ] Post 3x per week
  - [ ] Connect with 20 prospects/week
- [ ] Write 2-3 blog posts for SEO
- [ ] Create lead magnet (guide, checklist, template)
- [ ] Set up email capture on website

### Outbound Campaign

- [ ] Build target list of 100 prospects
  - [ ] Use Crunchbase for funded projects
  - [ ] Twitter for active projects
  - [ ] Discord for community projects
- [ ] Create outreach templates
- [ ] Start outreach: 20-50 per week
- [ ] Track responses in CRM
- [ ] Schedule discovery calls

### Networking

- [ ] Join 5-10 relevant Discords
- [ ] Attend 2-4 virtual events
- [ ] Apply to speak at 1-2 events
- [ ] Schedule 5-10 coffee chats
- [ ] Reach out to 10 potential partners

---

## Phase 4: First Clients (Days 76-90)

### Sales Process

- [ ] Conduct 10+ discovery calls
- [ ] Send 5+ proposals
- [ ] Close 2-3 initial clients
- [ ] Negotiate terms and contracts
- [ ] Collect deposits

### Delivery Setup

- [ ] Onboard first clients
- [ ] Set up project workspaces
- [ ] Schedule kickoff meetings
- [ ] Assign team/contractors
- [ ] Create project plans
- [ ] Set up communication cadence

### Delivery Excellence

- [ ] Deliver first milestones on time
- [ ] Over-communicate progress
- [ ] Get early feedback
- [ ] Document everything
- [ ] Request testimonials mid-project

## Common Pitfalls to Avoid

### ❌ Don't:

- Underprice your services
- Work without a contract
- Skip the discovery call
- Promise unrealistic timelines
- Take every client (be selective)
- Work without deposits
- Ignore cash flow
- Try to do everything yourself
- Skip portfolio building
- Give up too early

### ✅ Do:

- Price based on value, not hours
- Always use contracts
- Qualify clients thoroughly
- Under-promise, over-deliver
- Fire bad clients quickly
- Require 30%+ upfront
- Monitor finances weekly
- Build a team early
- Document everything
- Stay persistent (6-12 months to traction)

## After 90 Days

### If You Hit Targets:

- Double down on what's working
- Hire first full-time employee
- Raise prices 20-30%
- Build retainer relationships
- Consider raising funds (optional)

### If You Missed Targets:

- Analyze what didn't work
- Talk to prospects who said no
- Adjust positioning/pricing
- Increase outreach volume
- Consider pivot in services
- Don't give up - most agencies take 6-12 months

## Final Advice

1. **Start before you're ready** - You'll figure it out
2. **Charge what you're worth** - Don't compete on price
3. **Over-deliver early** - Build reputation fast
4. **Niche down** - Become known for something specific
5. **Build in public** - Share your journey on Twitter
6. **Network genuinely** - Help others without expecting returns
7. **Stay liquid** - Keep 6 months runway
8. **Fire bad clients** - Life's too short
9. **Invest in yourself** - Continuous learning
10. **Be patient** - This is a marathon, not a sprint

**Good luck! You've got this. 🚀**
