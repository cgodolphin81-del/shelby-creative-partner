# Community Building

**Complete Guide to Building & Managing Web3 Communities**

---

## Overview

Community is the foundation of every successful Web3 project. This guide covers building, growing, and managing communities across Discord, Telegram, Twitter, and governance platforms.

**Timeline:** Ongoing (start 4-8 weeks before launch)
**Budget Range:** $10K-$50K/month (ongoing)
**Key Metric:** Engaged members, not just total count

---

### Security Best Practices

**Essential Security Measures:**
- [ ] Enable 2FA for all mods/admins
- [ ] Set up verification gate
- [ ] Disable DMs from server members
- [ ] Auto-delete invite links
- [ ] Scam link detection
- [ ] Raid protection (Wick bot)
- [ ] Regular audit of permissions
- [ ] Backup admin accounts

**Common Scams to Watch:**
- Fake airdrop links
- Impersonation of team members
- "Support" DMs
- Fake minting sites
- Phishing links

## Telegram Management

### Channel Types

**Announcement Channel (Read-Only):**
- Official updates only
- No member messages
- Clean information source
- 1-3 posts per day

**Community Group (Open Chat):**
- Member discussion
- Active moderation needed
- Support questions
- Community building

### Best Practices

**Posting Schedule:**
- Morning update (UTC)
- Mid-day engagement
- Evening recap (optional)
- Breaking news as needed

**Moderation Requirements:**
- 24/7 coverage (rotate mods across timezones)
- Quick response to questions (<30 min)
- Scam link removal (<5 min)
- Clear rules enforcement

**Telegram-Specific Security:**
- Enable slow mode (30-60 seconds)
- Restrict media sharing
- Verify new members (captcha bot)
- Block known scam accounts

## Twitter/X Strategy

### Profile Optimization

**Essential Elements:**
- Professional profile picture (logo)
- Engaging banner (project art)
- Clear bio with value prop
- Link tree (website, Discord, Telegram)
- Pinned tweet (latest news/thread)

### Content Strategy

**Content Mix:**
```
40% - Educational/Value
25% - Project Updates
20% - Engagement/Community
10% - Promotional
5%  - Behind-the-scenes
```

**Posting Schedule:**
- Minimum: 3-5 tweets per day
- Optimal: 8-15 tweets per day
- Include: 2-3 threads per week
- Engage: 30+ replies per day

**Content Types:**
1. **Threads** (high engagement)
   - Project explainers
   - Industry insights
   - Tutorial/how-to

2. **Visual Content** (high shareability)
   - Infographics
   - Memes
   - Art reveals
   - Progress updates

3. **Engagement Posts** (community building)
   - Questions
   - Polls
   - Contests
   - Shoutouts

4. **News/Updates** (information)
   - Milestones
   - Partnerships
   - Listings
   - AMAs

### Growth Tactics

**Organic Growth:**
- Reply to top accounts in niche
- Engage with community posts
- Create viral threads
- Collaborate with other projects
- Use trending hashtags strategically

**Paid Growth:**
- Twitter Ads ($1K-$20K/month)
- Influencer partnerships ($2K-$50K)
- Thread sponsorships ($500-$5K)

**Key Metrics:**
- Followers (growth rate)
- Engagement rate (target: 3-10%)
- Impressions
- Profile clicks
- Link clicks

### Twitter Spaces

**Regular Spaces Strategy:**
- Weekly community call
- Monthly AMA with team
- Special event spaces (launches, partnerships)
- Co-hosted spaces with partners

**Best Practices:**
- Promote 24-48 hours in advance
- Have clear agenda
- Engage with listeners
- Record for later sharing
- Follow up with key points thread

## Community Programs

### Ambassador Program

**Structure:**
```
Tier 1: Ambassador (10-20 people)
- Monthly stipend: $500-$2,000
- Token allocation: 0.1-0.5%
- Requirements: 20+ hrs/week

Tier 2: Moderator (20-50 people)
- Monthly stipend: $200-$500
- Token allocation: 0.05-0.1%
- Requirements: 10+ hrs/week

Tier 3: Helper (50-100 people)
- Token rewards only
- Performance-based
- Requirements: 5+ hrs/week
```

**Responsibilities:**
- Community moderation
- Content creation
- Event hosting
- Translation/localization
- Partnership outreach

**Application Process:**
1. Application form
2. Interview
3. Trial period (2-4 weeks)
4. Onboarding
5. Regular performance reviews

### Referral Program

**Structure:**
- Unique referral codes
- Tiered rewards
- Both referrer and referee benefit
- Track via dashboard

**Example Rewards:**
```
1-5 referrals: 100 tokens each
6-20 referrals: 200 tokens each
21-50 referrals: 500 tokens each
50+ referrals: 1000 tokens + NFT
```

### Contests & Giveaways

**Types:**
- Art contests
- Meme contests
- Content creation
- Trading competitions
- Referral competitions

**Best Practices:**
- Clear rules and judging criteria
- Reasonable entry barriers
- Attractive prizes
- Fair judging
- Public announcement of winners

**Budget Guidelines:**
- Small contest: $500-$2K
- Medium contest: $2K-$10K
- Large contest: $10K-$50K

## Governance & DAO

### Governance Platforms

| Platform | Best For | Cost |
|----------|----------|------|
| Snapshot | Off-chain voting | Free |
| Tally | On-chain voting | Free |
| Discourse | Discussion forum | Free-$500/mo |
| Commonwealth | Full DAO suite | $500-$5K/mo |
| Boardroom | Governance dashboard | Free |

### Governance Structure

**Proposal Types:**
1. **Temperature Check** (informal)
   - Community sentiment
   - No binding vote
   - Discord/Twitter poll

2. **Signal Proposal** (Snapshot)
   - Off-chain vote
   - Token-weighted
   - Non-binding

3. **Executable Proposal** (Tally)
   - On-chain vote
   - Token-weighted
   - Binding execution

**Voting Parameters:**
```
- Quorum: 5-20% of supply
- Threshold: >50% or >66%
- Voting Period: 3-7 days
- Timelock: 24-72 hours
```

### Community Treasury

**Management Options:**
- Multi-sig wallet (Gnosis Safe)
- DAO-controlled treasury
- Foundation structure

**Typical Allocation:**
- Community initiatives: 30%
- Development grants: 25%
- Marketing: 20%
- Partnerships: 15%
- Reserve: 10%

## Crisis Management

### Common Crises

**Price Crash:**
- Acknowledge community concerns
- Reiterate long-term vision
- Share progress updates
- Avoid price speculation

**Security Incident:**
- Immediate transparent communication
- Action plan with timeline
- Regular updates
- Post-mortem report

**Team Controversy:**
- Swift response
- Clear statement
- Appropriate action
- Move forward decisively

**Delayed Roadmap:**
- Early communication
- Explain reasons
- New timeline
- Make it up to community

### Crisis Communication Template

```
1. Acknowledge the issue
2. Take responsibility (if applicable)
3. Explain what happened
4. Share action plan
5. Provide timeline
6. Commit to updates
7. Thank community for patience
```
