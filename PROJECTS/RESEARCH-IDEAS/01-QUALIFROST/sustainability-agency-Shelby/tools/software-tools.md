## Implementation & Selection

### Selection Criteria

#### Functional Requirements

| Category | Questions to Ask |
|----------|------------------|
| **Accounting** | Does it handle all scopes? Which standards? |
| **Data Collection** | How is data collected? Integrations? |
| **Calculations** | Emission factors? Methodology transparency? |
| **Reporting** | What reports? Frameworks supported? |
| **Supply Chain** | Supplier engagement? Scope 3 depth? |
| **Assurance** | Audit trail? Third-party verification? |
| **Integration** | ERP, HR, travel systems? API? |
| **Usability** | User-friendly? Training required? |
| **Support** | Implementation support? Ongoing? |
| **Security** | Data security? Certifications? |

---

#### Technical Requirements

- Cloud-based vs. on-premise
- API availability
- Data export capabilities
- User access controls
- Audit logs
- Backup and recovery
- Uptime SLA
- Performance

---

#### Commercial Requirements

- Total cost of ownership
- Implementation costs
- Ongoing costs
- Contract terms
- Exit provisions
- Scalability pricing
- Support included

---

### Implementation Process

#### Phase 1: Selection (4-8 weeks)

**Activities:**
- Requirements gathering
- Market research
- RFP process
- Demos and evaluations
- Reference checks
- Selection

**Deliverables:**
- Requirements document
- Vendor shortlist
- Evaluation matrix
- Selected vendor

---

#### Phase 2: Implementation (8-16 weeks)

**Activities:**
- Project kickoff
- Configuration
- Data migration
- Integrations
- Testing
- Training

**Deliverables:**
- Configured system
- Data loaded
- Integrations working
- Trained users
- Go-live

---

#### Phase 3: Optimization (Ongoing)

**Activities:**
- Usage monitoring
- Continuous improvement
- Feature adoption
- User support
- Annual review

**Deliverables:**
- Optimization roadmap
- Adoption metrics
- Value realization

---

### Implementation Best Practices

вң… **Do:**
- Involve all stakeholders early

вқҢ **Don't:**
- Boil the ocean (start too big)

- Skip training

- Go live without testing

## Technology Stack Recommendations

### Startup (<$10M revenue)

**Budget:** $5K-20K/year

**Stack:**
- Carbon accounting: Greenly or Carbon Better
- Spreadsheets: Excel/Google Sheets
- Reporting: Canva + PDF
- Offset procurement: Patch or Climate Neutral

**Total:** $10K-25K/year

---

### Growth Company ($10M-$100M revenue)

**Budget:** $30K-100K/year

**Stack:**
- Carbon accounting: Normative or Plan A
- ESG reporting: GRI Digital or Workiva
- Supply chain: EcoVadis (if applicable)
- Visualization: Tableau/Power BI
- Offset procurement: Direct or broker

**Total:** $50K-150K/year

---

### Enterprise ($100M+ revenue)

**Budget:** $100K-500K+/year

**Stack:**
- Carbon accounting: Persefoni or Watershed
- ESG reporting: MSCI, Sustainalytics, or Diligent
- Supply chain: SupplyShift or EcoVadis
- Integration: ERP, HR, travel systems
- Assurance: Big 4 or specialist
- Offset procurement: Direct + portfolio

**Total:** $200K-750K+/year

- [ ] Market position
- [ ] Roadmap alignment

**Product:**
- [ ] Feature completeness
- [ ] Usability
- [ ] Performance
- [ ] Scalability
- [ ] Security certifications

**Support:**
- [ ] Implementation support
- [ ] Training programs
- [ ] Ongoing support
- [ ] SLA terms

**Commercial:**
- [ ] Pricing transparency
- [ ] Contract terms
- [ ] Exit provisions
- [ ] Data ownership
- [ ] Price increase caps

---

### Contract Key Terms

**Service Level Agreement (SLA):**
- Uptime guarantee (99.5%+ typical)
- Support response times
- Performance metrics
- Credits for non-performance

**Data Rights:**
- Data ownership (you own your data)
- Data export rights
- Data deletion on termination
- Data usage restrictions

**Security:**
- SOC 2 Type II
- ISO 27001
- GDPR compliance
- Data residency
- Breach notification

**Termination:**
- Termination for cause
- Termination for convenience
- Transition assistance
- Data return/deletion
