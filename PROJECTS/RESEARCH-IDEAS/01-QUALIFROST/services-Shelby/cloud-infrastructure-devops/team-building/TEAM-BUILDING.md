## 2. Role Definitions

### Cloud Architect

```markdown
## Cloud Architect

### Level: Senior / Principal
### Reports to: CTO / VP Engineering
### Compensation: $180K-$250K + equity

### Role Overview
Design cloud infrastructure solutions for clients, provide technical 
leadership on migrations and transformations, and mentor engineering teams.

### Key Responsibilities
- Design cloud architectures for client engagements
- Create technical proposals and statements of work
- Lead architecture reviews and design sessions
- Mentor junior engineers
- Stay current with cloud provider updates and best practices
- Contribute to best practices and reusable patterns
- Support sales with technical discovery and proposals

### Required Skills
- 8+ years infrastructure/engineering experience
- 5+ years with cloud platforms (AWS/Azure/GCP)
- 3+ professional certifications
- Strong communication and presentation skills
- Experience with infrastructure as code
- Understanding of security and compliance

### Preferred Skills
- Multiple cloud platform expertise
- Kubernetes and container orchestration
- Security certifications (CCSP, CSSLP)
- Previous consulting experience
- Public speaking or technical writing

### Career Path
Cloud Architect → Principal Architect → CTO / VP Engineering
```

### DevOps Engineer

```markdown
## DevOps Engineer

### Level: Mid / Senior
### Reports to: Delivery Manager / CTO
### Compensation: $130K-$200K + equity

### Role Overview
Implement and maintain cloud infrastructure, CI/CD pipelines, and 
monitoring solutions for client engagements.

### Key Responsibilities
- Implement cloud infrastructure using IaC
- Build and maintain CI/CD pipelines
- Configure monitoring and alerting
- Support migration projects
- Troubleshoot and resolve issues
- Document configurations and processes
- Participate in on-call rotation (for managed services)

### Required Skills
- 3+ years DevOps or infrastructure experience
- 2+ years with cloud platforms
- Proficiency with Terraform or CloudFormation
- Experience with CI/CD tools (GitHub Actions, Jenkins, GitLab CI)
- Scripting skills (Python, Bash, PowerShell)
- Understanding of networking and security

### Preferred Skills
- Kubernetes administration
- Multiple cloud platforms
- Database administration
- Security best practices
- Previous consulting experience

### Career Path
DevOps Engineer → Senior DevOps Engineer → Solutions Architect / Delivery Manager
```

### Security Engineer

```markdown
## Security Engineer

### Level: Senior
### Reports to: CTO / VP Engineering
### Compensation: $160K-$220K + equity

### Role Overview
Ensure security best practices across all client engagements, conduct 
security assessments, and implement security controls.

### Key Responsibilities
- Conduct security assessments and audits
- Design and implement security architectures
- Implement compliance frameworks (SOC2, HIPAA, PCI-DSS)
- Review and approve security configurations
- Respond to security incidents
- Train team on security best practices
- Stay current with security threats and mitigations

### Required Skills
- 5+ years security engineering experience
- 3+ years cloud security experience
- Security certifications (CISSP, CCSP, Security+)
- Experience with security tools (WAF, SIEM, vulnerability scanning)
- Understanding of compliance frameworks
- Strong documentation skills

### Preferred Skills
- Penetration testing experience
- Incident response experience
- Multiple cloud platform security
- Security automation
- Previous consulting experience

### Career Path
Security Engineer → Principal Security Engineer → Security Practice Lead
```

### Site Reliability Engineer (SRE)

```markdown
## Site Reliability Engineer

### Level: Senior
### Reports to: Delivery Manager / CTO
### Compensation: $150K-$210K + equity

### Role Overview
Ensure reliability and performance of client systems, implement SRE 
practices, and manage incident response.

### Key Responsibilities
- Define and monitor SLOs/SLIs
- Implement observability solutions
- Lead incident response and post-mortems
- Automate operational tasks
- Capacity planning and performance optimization
- On-call rotation for managed services clients
- Implement chaos engineering practices

### Required Skills
- 5+ years operations/engineering experience
- Strong programming skills (Go, Python, or similar)
- Experience with monitoring tools (Prometheus, Grafana, Datadog)
- Understanding of distributed systems
- Incident management experience
- Automation mindset

### Preferred Skills
- Kubernetes operations
- Database reliability engineering
- Performance optimization
- Chaos engineering
- Previous SRE role at scale

### Career Path
SRE → Senior SRE → SRE Practice Lead / VP Engineering
```

```markdown
### Level: Senior / Principal
### Reports to: COO / VP Operations

### Preferred Skills
- Technical background

- Sales or account management experience
- Previous consulting experience

### Career Path
Delivery Manager → Senior Delivery Manager → VP Delivery / COO
```

```markdown
### Compensation: $120K-$180K base + commission (OTE $200K-$350K)

### Role Overview
Drive new business acquisition, manage sales pipeline, and close
```

### Sales Development Representative (SDR)

```markdown
## Sales Development Representative

### Level: Entry / Mid
### Reports to: Sales Director
### Compensation: $60K-$80K base + commission (OTE $90K-$120K)

### Role Overview
Generate qualified leads, conduct initial outreach, and book 
discovery calls for Account Executives.

### Key Responsibilities
- Prospect and identify target accounts
- Conduct outbound outreach (email, LinkedIn, phone)
- Qualify inbound leads
- Book discovery calls for AEs
- Maintain CRM data
- Meet activity and pipeline targets

### Required Skills
- 1+ years sales or SDR experience
- Excellent communication skills
- Resilient and goal-oriented
- CRM proficiency
- Social selling skills

### Preferred Skills
- Technical aptitude
- Interest in cloud/technology
- Previous startup experience
- Marketing or content experience

### Career Path
SDR → Senior SDR → Account Executive
```

```markdown
## First 30 Days: Learn

### Week 1: Orientation
- Company overview and culture
- Tools and systems setup
- Meet the team
- Review service offerings
- Security and compliance training

### Week 2-3: Training
- Cloud platform deep-dive
- Internal tools training
- Methodology and processes
- Shadow client calls
- Complete certifications (if needed)

### Week 4: Initial Contribution
- First client assignment (shadow)
- Internal project contribution
- 30-day check-in with manager

## Days 31-60: Contribute

### Week 5-7: Independent Work
- Own small client deliverables
- Participate in client meetings
- Begin certification exams
- Contribute to internal initiatives

### Week 8: Review
- 60-day performance review
- Adjust goals as needed
- Identify development areas

## Days 61-90: Excel

- Lead client workstreams
- Mentor new hires (if applicable)
- Contribute to best practices
- 90-day review and goal setting
```

```markdown
## Performance Review Schedule

### Weekly

- Focus: Current work, blockers, support needed

- Focus: Monthly objectives, development

- Formal quarterly review
- Focus: Performance against goals, feedback
```
