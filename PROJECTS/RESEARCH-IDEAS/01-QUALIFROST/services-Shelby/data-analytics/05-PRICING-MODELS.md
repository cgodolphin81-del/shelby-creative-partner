# Pricing Models

## Comprehensive Pricing Strategy

---

## Pricing Philosophy

**Value-Based, Not Hourly:** We price based on business impact delivered, not hours worked. This aligns our success with client success.

**Transparent & Predictable:** Clients should understand what they're paying for and why. No hidden fees or surprise charges.

**Flexible Engagement Models:** Different clients have different needs. We offer project, retainer, and hybrid options.

## Retainer Pricing

### Basic Support Retainer ($2,500 - $5,000/month)

**Best For:** Maintenance, minor updates, ongoing support

**Includes:**
```
├─ Email support (48-hour response)
├─ Monthly health check
├─ Bug fixes
├─ Minor dashboard updates (up to 5 hours/month)
├─ Pipeline monitoring
├─ Monthly usage report
└─ Quarterly business review
```

**Contract:** 12-month minimum, month-to-month after

## Hybrid Models

### Project + Retainer (Most Popular)

**Structure:**
```
Phase 1: Build (Project Fee)
├─ Discovery & assessment
├─ Implementation
├─ Training
└─ Go-live support

Phase 2: Grow (Retainer)
├─ Ongoing support
├─ Continuous improvement
├─ New features
└─ Strategic guidance

Example:
├─ Initial Project: $75,000
└─ Ongoing Retainer: $10,000/month
```

**Benefits:**
- Clear initial investment
- Predictable ongoing costs
- Aligned long-term partnership
- Continuous value delivery

---

### Success-Based Pricing

**Structure:**
```
Base Fee + Performance Bonus

Base Fee: Covers costs and standard delivery
Performance Bonus: Tied to measurable outcomes

Example:
├─ Base Project Fee: $50,000
├─ Bonus: $25,000 if KPIs achieved
│  ├─ Dashboard adoption >80%
│  ├─ Reported time savings >20 hours/week
│  └─ Client satisfaction >4.5/5
└─ Total Potential: $75,000
```

**Best For:** Clients comfortable with outcome-based partnerships
