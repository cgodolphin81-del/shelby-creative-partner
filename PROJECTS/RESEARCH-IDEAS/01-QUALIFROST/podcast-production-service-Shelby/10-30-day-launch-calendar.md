# 30-Day Launch Calendar

**Day-by-day countdown to a successful podcast launch**

---

## 📅 Overview

```
WEEK 1              WEEK 2              WEEK 3              WEEK 4
Foundation          Content             Promotion           LAUNCH!
Days 1-7            Days 8-14           Days 15-21          Days 22-30
```

**Goal:** Launch with 5 episodes, maximum visibility, and strong initial momentum.

---

## WEEK 1: Foundation (Days 1-7)

### Day 1: Vision & Strategy

**Morning (2 hours):**
- [ ] Define your "why" – write 1 page on why this podcast matters
- [ ] Set 90-day goals (downloads, reviews, email subscribers)
- [ ] Set 1-year goals
- [ ] Identify your unique angle/perspective

**Afternoon (2 hours):**
- [ ] Define target listener avatar (be specific)
- [ ] Research 5 competing podcasts in your niche
- [ ] Note what they do well and gaps you can fill
- [ ] Choose your podcast name (verify it's not taken)

**Evening (30 min):**
- [ ] Write down 20 potential episode topics
- [ ] Select top 5 for launch

**End of Day 1 Deliverables:**
- Goals document
- Listener avatar
- Competitor analysis
- Episode topic list

---

### Day 2: Branding & Assets

**Morning (2 hours):**
- [ ] Finalize podcast name
- [ ] Write short description (150 characters)
- [ ] Write long description (2000+ characters)
- [ ] Choose category and subcategory

**Afternoon (3 hours):**
- [ ] Commission cover art OR create in Canva
- [ ] Cover art specs: 3000x3000px, RGB, JPG/PNG
- [ ] Create social media graphics template
- [ ] Set up social media accounts (if needed)

**Evening (1 hour):**
- [ ] Choose theme music (Epidemic Sound, Artlist, etc.)
- [ ] Purchase/license music
- [ ] Download intro/outro music files

**End of Day 2 Deliverables:**
- Cover art (final)
- Show descriptions
- Social accounts created
- Music licensed

---

### Day 3: Technical Setup

**Morning (2 hours):**
- [ ] Choose podcast hosting platform (Buzzsprout recommended)
- [ ] Create account
- [ ] Set up show profile (add art, description, etc.)
- [ ] Get your RSS feed URL

**Afternoon (2 hours):**
- [ ] Set up recording space
- [ ] Test microphone and headphones
- [ ] Install recording software (Descript, Audition, etc.)
- [ ] Do test recording, check quality

**Evening (1 hour):**
- [ ] Create folder structure for project files
- [ ] Set up cloud backup (Google Drive, Dropbox)
- [ ] Create episode numbering system (EP001, EP002, etc.)

**End of Day 3 Deliverables:**
- Hosting account active
- RSS feed generated
- Recording setup tested
- File organization complete

---

### Day 4: Platform Submissions

**Morning (2 hours):**
- [ ] Submit to Apple Podcasts (podcastconnect.apple.com)
- [ ] Verify email ownership
- [ ] Submit to Spotify (podcasters.spotify.com)
- [ ] Verify email ownership

**Afternoon (2 hours):**
- [ ] Submit to Google Podcasts/YouTube Music
- [ ] Submit to Amazon Music
- [ ] Submit to iHeartRadio
- [ ] Claim show on Podchaser

**Evening (1 hour):**
- [ ] Create tracking spreadsheet for all platforms
- [ ] Note submission dates and approval status
- [ ] Set reminder to check approval status in 48-72 hours

**End of Day 4 Deliverables:**
- All major platforms submitted
- Tracking spreadsheet
- Verification emails confirmed

---

### Day 5: Trailer Production

**Morning (2 hours):**
- [ ] Write trailer script (see template in Launch Strategy guide)
- [ ] Keep it under 3 minutes
- [ ] Include: what, who, why, when, CTA to subscribe

**Afternoon (2 hours):**
- [ ] Record trailer
- [ ] Edit trailer (add music, clean up audio)
- [ ] Export as WAV and MP3

**Evening (1 hour):**
- [ ] Upload trailer to hosting platform
- [ ] Mark as "Trailer" episode type
- [ ] Write show notes for trailer
- [ ] Publish trailer (get it live now!)

**End of Day 5 Deliverables:**
- Trailer recorded and edited
- Trailer published live
- Show notes complete

---

### Day 6: Episode 1 Production

**Morning (2 hours):**
- [ ] Create detailed outline for Episode 1
- [ ] Gather any research/resources needed
- [ ] Prepare any questions (if interview)

**Afternoon (3 hours):**
- [ ] Record Episode 1
- [ ] Don't worry about mistakes – keep rolling
- [ ] Record room tone (10 seconds of silence)

**Evening (1 hour):**
- [ ] Backup recording files (cloud + local)
- [ ] Note any sections that need re-recording
- [ ] Rest your voice!

**End of Day 6 Deliverables:**
- Episode 1 recorded
- Files backed up
- Notes for editing

---

### Day 7: Rest & Review

**Morning (1 hour):**
- [ ] Review Week 1 progress
- [ ] Check platform approval status
- [ ] Adjust Week 2 plan if needed

**Afternoon:**
- [ ] REST – don't burn out
- [ ] Listen to other podcasts for inspiration
- [ ] Light topic research for future episodes

**Evening (1 hour):**
- [ ] Prep for Week 2 (gather materials, clear schedule)
- [ ] Set intentions for the week
- [ ] Early night!

**End of Day 7 Deliverables:**
- Week 1 review complete
- Week 2 plan finalized
- Mentally refreshed

## WEEK 2: Content Creation (Days 8-14)

### Day 8: Episode 1 Editing

**Morning (3 hours):**
- [ ] Import Episode 1 into editing software
- [ ] Rough cut: remove major mistakes, long pauses
- [ ] Listen at 1.5x speed for efficiency

**Afternoon (3 hours):**
- [ ] Fine cut: remove filler words, tighten pauses
- [ ] Noise reduction and audio cleanup
- [ ] Add intro/outro music

**Evening (1 hour):**
- [ ] Export first draft
- [ ] Listen to full episode on different device
- [ ] Note any changes needed

**End of Day 8 Deliverables:**
- Episode 1 first edit complete
- Change list for revisions

---

### Day 9: Episode 1 Finalization

**Morning (2 hours):**
- [ ] Make revisions from Day 8 notes
- [ ] Final audio polish (EQ, compression, limiting)
- [ ] Normalize to -16 LUFS

**Afternoon (2 hours):**
- [ ] Write show notes for Episode 1
- [ ] Include: summary, timestamps, links, CTA
- [ ] Create chapter markers

**Evening (2 hours):**
- [ ] Export final files (WAV master + MP3 distribution)
- [ ] Upload to hosting platform (set to private/scheduled)
- [ ] Add all metadata (title, description, artwork, etc.)

**End of Day 9 Deliverables:**
- Episode 1 fully edited
- Show notes complete
- Episode uploaded and scheduled
