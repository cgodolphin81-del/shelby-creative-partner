## Pre-Publish Checklist

**Script Review:**
- [ ] Hook grabs attention in first 5 seconds
- [ ] No fluff or rambling
- [ ] Conversational tone (read aloud test)
- [ ] Keywords included naturally
- [ ] Clear structure
- [ ] Strong CTA
- [ ] Within target word count

**Read Aloud Test:**
- [ ] Read script out loud
- [ ] Time it (should match target length)
- [ ] Note awkward phrases
- [ ] Fix any tongue-twisters
