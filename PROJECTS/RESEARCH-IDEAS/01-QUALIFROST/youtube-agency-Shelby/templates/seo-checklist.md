# 🔍 YouTube SEO Checklist

**Optimize every video for maximum discoverability**

---

## Pre-Upload

### Keyword Research
- [ ] Primary keyword identified
- [ ] 5-10 secondary keywords found
- [ ] Search volume validated (VidIQ/TubeBuddy)
- [ ] Competition level assessed
- [ ] Keyword intent matches content

### Title Optimization
- [ ] Primary keyword in first 50 characters
- [ ] Under 60 characters (avoid truncation)
- [ ] Includes benefit/outcome
- [ ] Creates curiosity
- [ ] No clickbait (deliver on promise)
- [ ] Year included if relevant (2026)
- [ ] 3 title variants created for testing

## Video Settings

### Basic Info
- [ ] Category selected (most relevant)
- [ ] Language set
- [ ] Video language set
- [ ] Captions uploaded/verified
- [ ] Age restriction: OFF (unless required)

### Advanced Settings
- [ ] Comments: ON
- [ ] Allow ratings: ON
- [ ] Show in subscriptions: ON
- [ ] Publish to subscriptions feed: ON
- [ ] Paid promotion: Disclosed (if applicable)

### Thumbnail
- [ ] Custom thumbnail uploaded
- [ ] Text readable at small size
- [ ] High contrast
- [ ] On-brand
- [ ] 2-3 variants ready for A/B test
- [ ] No important elements in corners

### End Screens (Last 20 seconds)
- [ ] Element 1: Video (most relevant)
- [ ] Element 2: Video or Playlist
- [ ] Element 3: Subscribe button
- [ ] Element 4: Playlist or Video
- [ ] Verbal CTA recorded

### Cards (Throughout video)
- [ ] Card 1 at _____:_____ (relevant moment)
- [ ] Card 2 at _____:_____ (relevant moment)
- [ ] Card 3 at _____:_____ (relevant moment)
- [ ] Cards link to related content
- [ ] Not too many (max 5)

### Playlist
- [ ] Added to relevant playlist
- [ ] Playlist is optimized (title, description)
- [ ] Video is in correct position

## SEO Audit (Weekly)

### For Each Video Published:

| Metric | Target | Actual | Action Needed |
|--------|--------|--------|---------------|
| CTR | 8%+ | | |
| Retention | 50%+ | | |
| Search impressions | Increasing | | |
| Search CTR | 5%+ | | |
| Ranking (primary keyword) | Page 1 | | |

### Optimization Actions:

**If CTR low (<5%):**
- [ ] Test new thumbnail
- [ ] Test new title
- [ ] Check if thumbnail matches title

**If retention low (<40%):**
- [ ] Review hook (first 30 seconds)
- [ ] Check for boring sections
- [ ] Add pattern interrupts
- [ ] Improve pacing in edit

**If search impressions low:**
- [ ] Improve title SEO
- [ ] Add more keywords to description
- [ ] Create more content on topic
- [ ] Build playlist around topic
