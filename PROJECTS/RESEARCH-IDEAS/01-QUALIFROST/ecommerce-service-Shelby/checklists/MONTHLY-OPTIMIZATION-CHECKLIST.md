## FACEBOOK/INSTAGRAM OPTIMIZATION

### Account-Level
- [ ] Review account spending limit
- [ ] Check account quality score
- [ ] Review policy violations
- [ ] Clean up unused audiences
- [ ] Clean up old campaigns

### Campaign-Level
- [ ] Pause campaigns with ROAS <1.5 (after $500+ spend)
- [ ] Scale campaigns with ROAS >3.0 (increase 20-30%)
- [ ] Review campaign budget allocation
- [ ] Test new campaign objectives
- [ ] Review attribution window performance

### Ad Set-Level
- [ ] Review audience performance
- [ ] Pause audiences with CPA > target
- [ ] Test new lookalike percentages
- [ ] Test new interest audiences
- [ ] Review placement performance
- [ ] Test Advantage+ placements vs. manual

### Ad-Level
- [ ] Review all active ads
- [ ] Pause ads with CTR <0.8%
- [ ] Pause ads with ROAS <1.5 (after $200+ spend)
- [ ] Identify top 3 performing ads
- [ ] Analyze why winners won
- [ ] Create new variations of winners
- [ ] Test new hooks and angles
- [ ] Refresh creative (minimum 4-8 new/month)

### Creative Analysis
- [ ] Review video retention rates
- [ ] Identify best performing hooks
- [ ] Analyze thumb-stop ratio
- [ ] Review creative fatigue (frequency >3.0)
- [ ] Document learnings for next creative brief
- [ ] Update creative best practices doc
