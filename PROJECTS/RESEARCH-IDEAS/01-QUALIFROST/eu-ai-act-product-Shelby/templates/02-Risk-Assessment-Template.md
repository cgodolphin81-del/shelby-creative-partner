**Risk Matrix:**

| Almost Certain | | | | | |
| Likely | | | | | |

| Unlikely | | | | | |

**Likelihood Definitions:**
- **Almost Certain:** >90% probability
- **Likely:** 60-90% probability
- **Possible:** 30-60% probability
- **Unlikely:** 10-30% probability
- **Rare:** <10% probability

**Impact Definitions:**
- **Insignificant:** No harm
- **Minor:** Minor inconvenience
- **Moderate:** Temporary harm, reversible
- **Major:** Serious harm, long-lasting effects
- **Catastrophic:** Severe harm, irreversible, life-threatening
