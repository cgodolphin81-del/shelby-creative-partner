# Email Signature Guide

## Overview

Professional email signatures that reinforce brand identity and provide essential contact information.

## Specifications

### Dimensions

**Overall Width:** 600-700 pixels maximum

**Logo Size:** 150x50 pixels (or proportional)

**Headshot:** 100x100 pixels (circle or square)

**Banner:** 600x150 pixels maximum

**Icons:** 24x24 pixels or 32x32 pixels

---

### File Formats

**Logo:** PNG with transparent background

**Icons:** PNG with transparent background

**Banner:** JPG or PNG (optimized for web)

**Headshot:** JPG (high quality, compressed)

---

### File Sizes

**Total Signature:** Under 50KB

**Images:** Each under 20KB

**Optimization:** Use TinyPNG or similar

---

## Typography

### Font Choices

**Primary:** [Brand Primary Font] or web-safe alternative

**Fallbacks:** Arial, Helvetica, sans-serif

**Size:**
- Name: 14-16pt

## Setup Instructions

### Gmail

1. Click Settings (gear icon) > See all settings
2. Scroll to "Signature" section
3. Click "Create new"
4. Name your signature
5. Use the editor to format or paste HTML
6. Insert images (logo, icons)
7. Set defaults (new emails, replies)
8. Scroll down and click "Save Changes"

**HTML Method:**
1. Create HTML file
2. In Gmail signature editor, paste HTML
3. Gmail will render it

---

### Outlook (Desktop)

1. File > Options > Mail > Signatures
2. Click "New" and name signature
3. Use editor to format or paste HTML
4. Insert images
5. Set as default for new messages/replies
6. Click "OK"

---

### Outlook (Web)

1. Settings (gear icon) > View all Outlook settings
2. Mail > Compose and reply
3. Create signature in editor
4. Toggle "Automatically include my signature"
5. Click "Save"

---

### Apple Mail

1. Mail > Preferences > Signatures
2. Select account
3. Click "+" to add signature
4. Format or paste HTML
5. Insert images
6. Close preferences (auto-saves)

---

### Mobile (iOS/Android)

**Note:** Mobile signatures often don't support images

**Option 1:** Use plain text signature
**Option 2:** Use signature app (e.g., Exclaimer, CodeTwo)
**Option 3:** Accept limitation (no signature on mobile)

## Best Practices

### DO:

✓ Keep it simple and scannable
✓ Use web-safe fonts
✓ Optimize images for fast loading
✓ Include only essential information
✓ Use consistent branding
✓ Test on multiple devices
✓ Update when info changes
✓ Keep under 600px wide

### DON'T:

✗ Include too much information
✗ Use large images
✗ Include quotes or inspirational messages
✗ Use multiple colors
✗ Add animated GIFs
✗ Include personal information
✗ Use low-resolution images
✗ Make it too tall (4-5 lines max for text)

## Legal Requirements

### Industry-Specific Disclaimers

**Financial Services:**
> "This email and any attachments are confidential and may be privileged. If you are not the intended recipient, please delete it and notify the sender."

**Legal:**
> "This communication is attorney-client privileged and confidential..."

**Healthcare (HIPAA):**
> "This email may contain protected health information (PHI). If you are not the intended recipient..."

**General:**
> "Confidentiality Notice: This email and any attachments are for the exclusive use of the intended recipient..."

**Placement:** Below signature, smaller font (10pt), gray color
