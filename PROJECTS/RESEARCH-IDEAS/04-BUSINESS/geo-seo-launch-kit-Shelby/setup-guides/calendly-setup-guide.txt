CALENDLY SETUP GUIDE
====================

Step-by-step instructions for setting up your booking system

---

STEP 1: CREATE YOUR CALENDLY ACCOUNT
------------------------------------

1. Go to calendly.com
2. Click "Sign Up Free"
3. Choose your plan:
   - Free: 1 event type, basic features
   - Essentials ($10/mo): 6 event types, reminders
   - Professional ($15/mo): Unlimited event types, workflows
   - Teams ($20/mo/user): Team scheduling

   RECOMMENDATION: Start with Professional plan

4. Connect your calendar (Google, Outlook, or iCloud)
5. Set your availability:
   - Timezone: {{YourTimezone}}
   - Working hours: {{YourWorkingHours}}
   - Buffer time: 15 minutes between meetings

---

STEP 5: INTEGRATE WITH YOUR TOOLS
---------------------------------

GOOGLE CALENDAR:
- Already connected in Step 1
- Set working hours and buffers

ZOOM/GOOGLE MEET:
- Settings → Integrations → Connect
- Auto-generate meeting links

SLACK:
- Settings → Integrations → Slack
- Get notified when calls are booked

CRM (HUBSPOT, PIPEDRIVE, ETC.):
- Use Zapier or native integration
- Auto-create contacts and deals

EMAIL MARKETING:
- Add invitees to your email list (with consent)
- Tag by event type for segmentation

---

STEP 6: EMBED ON YOUR WEBSITE
-----------------------------

1. Go to Share → Add to Website
2. Choose embedding option:
   - Inline: Embeds calendar directly on page
   - Popup: Calendar opens in popup window
   - Text link: Simple "Book Now" link

3. Copy the embed code

4. Paste into your website:
   - WordPress: Use Custom HTML block
   - Webflow: Use Embed element
   - Squarespace: Use Code block
   - Custom: Paste in HTML where you want calendar

5. Test the embedding

STEP 8: ANALYTICS & OPTIMIZATION
--------------------------------

Track these metrics monthly:

1. Booking rate: Bookings / Page views
2. No-show rate: No-shows / Bookings
3. Conversion rate: Clients / Strategy calls
4. Average time to close: First booking to signed contract

Optimize based on data:
- Low booking rate? Improve landing page copy
- High no-show rate? Add more reminders
- Low conversion? Refine qualification questions
