# Notion CRM Template - Mockup Specifications

## Required Images

### 1. Main Product Shot (Cover Image)
- **Size:** 1280×720px
- **Content:** Dashboard view with pipeline, contacts, and tasks visible
- **Style:** Clean, professional, Notion-native aesthetic
- **Text overlay:** "Notion CRM Template" + "Close More Deals"

### 2. Pipeline Kanban View
- **Size:** 1920×1080px
- **Content:** Full kanban board showing deal stages
- **Focus:** Visual pipeline management
- **Annotation:** Arrow pointing to drag-and-drop functionality

### 3. Contact Database View
- **Size:** 1920×1080px
- **Content:** Contact list with filters and properties
- **Focus:** Organization and searchability

### 4. Contact Detail Page
- **Size:** 1920×1080px
- **Content:** Single contact with all related info
- **Focus:** Complete contact profile view

### 5. Mobile View
- **Size:** 1080×1920px (vertical)
- **Content:** CRM on mobile device
- **Focus:** Responsive and mobile-friendly

### 6. Features Overview
- **Size:** 1920×1080px
- **Content:** Collage of 4-6 key features
- **Text:** Callouts for each feature

## Mockup Best Practices

1. **Use real data** (not "Lorem Ipsum")
2. **Show, don't tell** - demonstrate features visually
3. **Highlight benefits** with text overlays
4. **Consistent styling** across all images
5. **Include people** in some mockups (lifestyle shots)

## Color Palette

Match Notion's aesthetic:
- Background: #FFFFFF (white) or #F7F7F7 (light gray)
- Text: #37352F (dark gray)
- Accent: Notion blue #2EAADC or your brand color
- Success: #57C389 (green)
- Warning: #E3813A (orange)

## Typography

- **Headlines:** Inter, -apple-system, or similar sans-serif
- **Body:** Same as Notion default
- **Size:** Large enough to read on mobile

## Quick Mockup Workflow

1. Take screenshots of your Notion template
2. Import to Canva
3. Add to device mockup frames
4. Add text overlays for key benefits
5. Export as JPG (high quality)
6. Upload to Gumroad

## Pro Tips

- Use **consistent lighting** in all mockups
- Show the template **in use** (with sample data)
- Include **before/after** if possible
- Add **social proof** (testimonials) to one image
- Create a **GIF** showing template in action (optional)
