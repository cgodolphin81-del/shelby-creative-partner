## 🛠️ CANVA SETUP INSTRUCTIONS

### Creating the Templates

**Step 1: Set Up Brand Kit**
1. Go to Brand Hub in Canva
2. Add your brand colors (all 5 palettes)
3. Upload your logo
4. Add your brand fonts

**Step 2: Create Each Template**
1. Click "Create Design"
2. Enter custom dimensions
3. Build using the specs above
4. Save as template

**Step 3: Create Variations**
1. Duplicate each template
2. Apply different color palettes
3. Adjust text colors for contrast
4. Save each variation

**Step 4: Create Template Links**
1. Click Share → Template Link
2. Copy link for each template
3. Organize in a document

### Delivering to Customers

**Option 1: Template Links**
- Share Canva template links
- Customers click to edit
- No download needed

**Option 2: PDF Guide**
- Create PDF with all links
- Include usage instructions
- Add design tips

**Option 3: Notion Page**
- Create Notion database
- Embed template links
- Add video tutorials
