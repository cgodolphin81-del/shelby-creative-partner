## 🔍 Keyword Research Prompts

### Primary Keyword Research

```
You are an SEO expert. Analyze the keyword "[KEYWORD]" and provide:

1. Search Volume (estimate)
2. Keyword Difficulty (0-100)
3. CPC (cost per click)
4. Search Intent (informational, commercial, transactional, navigational)
5. Related keywords (5-10)
6. Question keywords (5-10)
7. Long-tail variations (10-15)
8. Top 3 competing pages
9. Content gaps we can exploit

Format as a structured brief.
```

### SERP Analysis

```
Analyze the search results for "[KEYWORD]":

1. What type of content ranks? (blog posts, product pages, videos, etc.)
2. Average word count of top 10 results
3. Common topics covered
4. Missing topics or angles
5. Content format preferences (lists, guides, comparisons)
6. Featured snippet opportunities
7. People Also Ask questions
8. Image pack presence
9. Video carousel presence

Provide actionable insights for creating better content.
```

## 📝 Outline Creation Prompts

### Article Outline

```
Create a comprehensive outline for an article about "[TOPIC]"

Target keyword: [KEYWORD]
Word count target: [1500-2500]
Audience: [beginner/intermediate/advanced]
Tone: [conversational/authoritative/friendly]

Include:
- Compelling introduction with hook
- 4-6 main H2 sections
- 2-3 H3 subsections per H2
- FAQ section with 5 questions
- Conclusion with CTA
- Notes for images/tables/diagrams

- External link opportunities

Make it scannable with bullet points.
```

This will be a comprehensive guide (4000-6000 words) that covers:
- Complete overview of the topic

- Table of contents

- Comparison tables
- Step-by-step guides
- FAQs
- Resource recommendations

Make it the definitive guide on this topic.

## ✍️ Content Writing Prompts

### Introduction Writing

```
Write an introduction for an article about "[TOPIC]"

Requirements:
- Start with a compelling hook (statistic, question, or story)
- Address the reader's problem
- Promise what they'll learn
- Include keyword naturally in first 100 words
- End with transition to main content
- Word count: 150-200 words
- Tone: [conversational/authoritative]

Don't use phrases like "In this article" or "Let's dive in."
```

### Section Writing

```
Write the "[SECTION TITLE]" section of an article about "[TOPIC]"

Context:
- Target keyword: [KEYWORD]
- Audience: [DESCRIPTION]
- Key points to cover:
  1. 
  2. 
  3. 

Requirements:
- 300-500 words
- Use short paragraphs (2-3 sentences)
- Include bullet points or numbered lists
- Add examples where relevant
- Include keyword naturally
- End with transition to next section
```

### Comparison Table

```
Create a comparison table for [PRODUCT A] vs [PRODUCT B] vs [PRODUCT C]

Include these criteria:
- Price
- Key features
- Pros
- Cons
- Best for
- Rating (1-5 stars)

Format as markdown table with clear headers.
```

### FAQ Section

```
Write 5 FAQ questions and answers for an article about "[TOPIC]"

Questions should:
- Address common concerns
- Include long-tail keywords
- Be specific and actionable
- Provide clear, concise answers
- Be 50-100 words each

Format as:
## Question 1
Answer to question 1

## Question 2
Answer to question 2
```

### Conclusion Writing

```
Write a conclusion for an article about "[TOPIC]"

Include:
- Brief summary of key points
- Final recommendation
- Call-to-action (affiliate/product/email)
- Encouraging closing statement
- Word count: 100-150 words

Make it actionable and motivating.
```

## 🎯 Humanization Prompts

### Voice & Tone Adjustment

```
Rewrite this section to sound more human and conversational:

[PASTE TEXT]

Make it:
- Sound like a real person talking
- Add personal opinions or preferences
- Use contractions (you'll, don't, can't)
- Vary sentence length
- Remove robotic phrases
- Keep the same information
- Add a friendly tone
```

### Personal Anecdote Addition

```
Add a brief personal anecdote to this section about "[TOPIC]":

[PASTE SECTION TEXT]

Include:
- A short story or experience (50-100 words)
- What I learned from it
- How it relates to the reader
- Keep it authentic and relatable
```

### Opinion Injection

```
Add my personal opinion to this product review section:

[PASTE TEXT]

Include:
- My preference between the options
- Why I recommend one over others
- Any drawbacks I've experienced
- Keep it balanced and honest
```

## 🔧 SEO Optimization Prompts

### Meta Tag Generation

```
Generate SEO meta tags for an article about "[TOPIC]"

Target keyword: [KEYWORD]
Secondary keywords: [LIST]

Create:
1. Title tag (under 60 characters, keyword front-loaded)
2. Meta description (150-160 characters, includes CTA)
3. URL slug (short, descriptive, includes keyword)
4. Focus keyphrase
5. SEO score prediction (1-10)
```

### Internal Linking Suggestions

```
Suggest internal linking opportunities for this article:

[PASTE ARTICLE TEXT]

My site structure includes:
- [Pillar Page URL] - [Topic]
- [Cluster Page URL] - [Topic]
- [Article URL] - [Topic]

Suggest:
1. 3-5 internal links with anchor text
2. Where to place each link in the content
3. Why each link is relevant
```

### Schema Markup Generation

```
Generate FAQ schema markup for these questions and answers:

[LIST QUESTIONS AND ANSWERS]

Format as JSON-LD for Google rich snippets.
Include:
- @context
- @type: FAQPage
- mainEntity array with Question and Answer objects
```

## 🎯 Prompt Best Practices

### Tips for Better Results

1. **Be Specific**: More context = better output
2. **Iterate**: Don't accept first result, refine
3. **Combine**: Mix multiple prompts for complex tasks
4. **Humanize**: Always edit AI output before publishing
5. **Test**: A/B test different approaches
6. **Track**: Monitor which prompts work best

### Prompt Variables

Use these variables to customize prompts:
- `[TOPIC]` - Main subject
- `[KEYWORD]` - Target keyword
- `[WORD COUNT]` - Article length
- `[AUDIENCE]` - Target reader
- `[TONE]` - Writing style
- `[PRODUCTS]` - Products to review
- `[URL]` - Link to include
