# 🎯 Niche Selection Framework

## Selection Criteria

Score each niche 1-10 on these factors:

| Factor | Weight | Description |
|--------|--------|-------------|
| **Profitability** | 25% | Affiliate commissions, product prices |
| **Competition** | 20% | Domain authority of top results |
| **Search Volume** | 20% | Total monthly searches |
| **Content Depth** | 15% | How much can you write? |
| **Monetization** | 10% | Multiple revenue streams |
| **Evergreen** | 10% | Long-term relevance |

**Target Score: 7.5+**

## 📊 Competition Analysis Framework

### Quick Assessment (15 minutes)

```bash
# Search top 3 keywords for your niche
# Analyze first page results:

1. Domain Authority (use MozBar extension)
   - DA < 40: Easy opportunity
   - DA 40-60: Medium competition
   - DA 60+: Hard but possible

2. Content Quality
   - Thin content? → Opportunity
   - Comprehensive guides? → Need differentiation
   - User-generated? → Easy to beat

3. Monetization
   - Heavy ads? → User experience angle
   - Affiliate-heavy? → Authenticity angle
   - E-commerce? → Content depth angle

4. Freshness
   - Old content (2+ years)? → Update opportunity
   - Recently updated? → Need unique angle
```
