### Step-by-Step: Create Mockup in Canva

**1. Choose Template Size**
```
Etsy Listing: 2000x2000px (square)
Etsy Banner: 3360x840px
Social Media: 1080x1080px or 1080x1350px
```

**2. Search for Mockup Templates**
```
Search terms:
- "product mockup"
- "printable mockup"
- "digital download mockup"
- "frame mockup"
- "lifestyle mockup"
```

**3. Upload Your Product**
```
Steps:
1. Click "Uploads" → "Upload files"
2. Select your product image
3. Drag onto mockup frame
4. Adjust to fit perfectly
```

**4. Customize Colors & Text**
```
- Match your brand colors
- Add shop name (subtle)
- Include key benefits (optional)
- Keep text minimal
```

**5. Download Settings**
```
Format: PNG (best quality)
Quality: High
Transparent BG: No (unless needed)
```

## 🎯 Mockup Best Practices

### Image Composition

**Rule of Thirds:**
```
Place your product at intersection points
Use grid lines in Canva to help
```

**White Space:**
```
Leave 20-30% empty space
Makes product stand out
Looks professional
```

**Focal Point:**
```
One clear focal point per image
Eye should flow naturally
Avoid clutter
```

### Color Theory

**Complementary Colors:**
```
Product color + opposite on color wheel
Creates visual pop
```

**Analogous Colors:**
```
Colors next to each other on wheel
Creates harmony
```

**Brand Consistency:**
```
Use your brand palette
3-4 colors max per mockup
```

### Lighting & Shadows

**Natural Light:**
```
- Morning/afternoon sunlight best
- Avoid harsh noon light
- Use window light when possible
```

**Artificial Light:**
```
- Softbox or ring light
- Avoid direct flash
- Multiple light sources
```

**Shadows:**
```
- Soft, natural shadows
- Match real-world lighting
- Don't over-edit
```

## 🎨 Mockup Creation Workflow

### Phase 1: Planning (30 min)

**1. Define Mockup Goals**
```
- What's the main benefit to show?
- What problem does it solve?
- Who is the target buyer?
- What emotions should it evoke?
```

**2. Research Competitors**
```
- Top 5 competitors in your niche
- Note their mockup styles
- Identify gaps you can fill
- Find unique angles
```

**3. Create Mockup List**
```
For each product, list:
- [ ] Hero mockup
- [ ] Detail shot
- [ ] Size reference
- [ ] Lifestyle shot
- [ ] Inside view
- [ ] Variation shot
```

### Phase 2: Creation (2-3 hours per 10 products)

**Batch Process:**
```
1. Set up Canva/SmartMockups project
2. Create master template with brand colors
3. Upload all products at once
4. Apply templates to each product
5. Customize individual mockups
6. Export all at once
```

### Phase 3: Quality Check (15 min)

**Checklist:**
- [ ] Images are 2000px minimum
- [ ] Colors match product accurately
- [ ] Text is readable (if included)
- [ ] No watermarks (unless free tool)
- [ ] Consistent style across all
- [ ] Mobile preview looks good
- [ ] File names are organized

## 📱 Mobile Optimization

**Mobile viewing is 60%+ of Etsy traffic!**

**Checklist:**
- [ ] Text is readable on phone screen
- [ ] Product is visible without zooming
- [ ] Important details are clear
- [ ] Images load quickly (optimize file size)
- [ ] Mockup works in square crop

**Testing:**
```
1. Open mockup on phone
2. Pinch to zoom - is it clear?
3. Scroll through - does it load fast?
4. Compare with desktop - consistent?
```

## 🎯 Mockup A/B Testing

**Test These Variables:**

1. **Background Color**
   - White vs. colored vs. lifestyle scene
   - Track conversion rate

2. **Product Angle**
   - Front view vs. angled vs. flat lay
   - See what resonates

3. **Props**
   - Minimal vs. detailed
   - Relevant vs. decorative

4. **Text Overlay**
   - No text vs. key benefit
   - Short vs. long copy

**Test Duration:** 2-4 weeks minimum
**Sample Size:** 100+ views per variant
