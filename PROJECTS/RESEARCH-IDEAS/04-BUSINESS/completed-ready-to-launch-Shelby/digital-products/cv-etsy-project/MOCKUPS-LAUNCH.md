# Mockup Images & Launch Announcement

## Part 1: Mockup Image Specifications

### Etsy Image Requirements
- **Minimum:** 1000 x 1000 pixels
- **Recommended:** 2000 x 2000 pixels (or 2000px wide for portrait)
- **Format:** JPG or PNG
- **First image:** Most important - shows full template clearly
- **Total images per listing:** Up to 10 (recommend 5-7)

---

### Mockup Template Structure (Create in Canva)

**Canvas Size:** 2000 x 2000px (square) or 2000 x 2600px (portrait)

**Layout for Each Template Mockup:**

```
┌─────────────────────────────────────┐
│                                     │
│         TEMPLATE PREVIEW            │
│         (Full CV on clean           │
│          background, angled         │
│          or flat lay)               │
│                                     │

│  ✨ Editable Word Doc               │
│  ✨ A4 + US Letter                  │
│  ⚡ Instant Download                │
├─────────────────────────────────────┤
│         [SHOP NAME/LOGO]            │
└─────────────────────────────────────┘
```

---

### Image Sequence Per Listing (5-7 images)

**Image 1: Hero Shot**
- Full template preview on clean background
- Show entire CV clearly
- Add subtle shadow for depth
- Text overlay: Template name
- Purpose: Main thumbnail, first impression

**Image 2: Close-up Detail**
- Zoomed view of header/contact section
- Show typography and formatting quality
- Purpose: Show attention to detail

**Image 3: Features Callout**
- Show template with callout boxes highlighting:
  - "ATS-Friendly Design"
  - "Easy to Edit"
  - "Professional Layout"
  - "Instant Download"
- Purpose: Communicate key benefits

**Image 4: Before/After or Comparison**
- Show template vs. plain text
- Or show different color variations
- Purpose: Demonstrate value add

**Image 5: What's Included**
- Show all 4 files (PDF A4, PDF Letter, DOCX A4, DOCX Letter)
- Icons for each file type
- Purpose: Set expectations

**Image 6: How It Works**
- Simple 3-step graphic:
  1. Download instantly
  2. Edit in Word/Google Docs
  3. Save as PDF & apply!
- Purpose: Reduce purchase anxiety

**Image 7: Shop Branding**
- Your shop logo
- Link to shop (etsy.com/shop/YourShopName)
- "Thank You" message
- Purpose: Brand recognition, drive to shop

---

### Mockup Background Options

**Option A: Clean Minimal**
- White or very light gray background (#F5F5F5)
- Subtle shadow under CV
- Professional, distraction-free

**Option B: Lifestyle Desk**
- CV on desk with laptop, coffee, plant
- Use Canva's mockup templates
- Relatable, aspirational

**Option C: Colored Background**
- Soft gradient matching template accent color
- Modern, eye-catching
- Ensure CV still reads clearly

**RECOMMENDATION:** Use Option A (Clean Minimal) for consistency across all 15 listings.

---

### Canva Mockup Creation Steps

1. **Open Canva** → Create design → Custom size (2000 x 2000px)

2. **Import your CV template** (export from your template design as PNG first)

3. **Add background:**
   - Solid color or subtle gradient
   - Or search "mockup" in Canva templates

4. **Position CV:**
   - Center or slightly angled (5-10 degrees)
   - Add shadow: Effects → Shadow → Adjust opacity to 30-40%

5. **Add text overlays:**
   - Template name (large, bold)
   - Key features (bullet points)
   - Keep text minimal and readable

6. **Add branding:**
   - Shop logo in corner
   - Consistent placement across all mockups

7. **Export:**
   - PNG format (highest quality)
   - 2000px minimum width


---

### Quick Mockup Tips

✅ **DO:**
- Use consistent styling across all 15 templates
- Ensure text is readable at thumbnail size
- Show the actual template clearly
- Use high contrast for readability
- Include your shop name/logo

❌ **DON'T:**
- Overcrowd with text
- Use busy backgrounds
- Make CV too small to read
- Forget mobile optimization (check how it looks small)
- Use low-resolution images
