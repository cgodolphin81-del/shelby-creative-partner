## 🔄 ADJUSTMENT TRIGGERS

### If no sales after 1 week:
- Lower prices by 20%
- Change main listing photos
- Add more tags
- Increase social media promotion
- Consider Etsy Ads ($1-2/day test)

### If low views:
- Improve SEO (titles, tags)
- Post more on social media
- Join more Facebook groups
- Try Pinterest (great for digital products)

### If high views but no sales:
- Review pricing (too high?)
- Improve listing photos
- Add more details to description
- Consider running a sale

### If good sales but low profit:
- Increase prices gradually
- Create higher-value bundles
- Reduce Etsy Ads if not profitable
