# CV Template 2: Creative Bold

**Style:** Unique, memorable, design-forward
**Best For:** Creative industries, marketing, design, startups
**Experience Level:** 2-12 years
**Page Count:** 1-2 pages

---

## 📐 CANVA DESIGN SPECIFICATIONS

### Document Setup
```
Page Size: US Letter (8.5" x 11") or A4 (210 x 297 mm)
Orientation: Portrait
Margins: 0.5" all sides (12.7 mm) - creative can break rules
Bleed: None needed (digital only)
Color Mode: RGB (for digital)
```


## 🎨 CANVA BUILD INSTRUCTIONS

### Step 1: Set Up Two-Column Layout
1. Create custom design: 8.5" x 11"
2. Add vertical line at 2.5" (left margin + sidebar width)
3. Add rectangle for sidebar background (2.5" x 11")
4. Set sidebar fill to light accent color (#F7FAFC or similar)

### Step 2: Create Header Band
1. Add rectangle at top (8.5" x 2")
2. Fill with accent color (#DD6B20 or #805AD5)
3. Add name text (white, Poppins Bold 34pt)
4. Add title text (white with transparency or light color)

### Step 3: Build Sidebar
1. Add section headers (Poppins SemiBold 12pt, UPPERCASE)
2. Add divider lines (2pt, accent color)
3. Add body text (Lato 9-10pt)
4. For skills: Create progress bar elements
   - Background bar: Light gray, rounded corners
   - Fill bar: Accent color, matching width to percentage

### Step 4: Build Main Content
1. Add section headers (Poppins SemiBold 14pt)
2. Add horizontal dividers (2pt, accent color)
3. Add job entries with consistent spacing
4. Use custom bullets (✨ or colored dots)

### Step 5: Add Visual Interest
1. Icons for contact info (download from Canva elements)
2. Subtle pattern in sidebar (optional, low opacity)
3. Colored tags for skills (rounded rectangles)
4. Profile photo placeholder (optional, circle frame)

### Step 6: ATS Considerations
⚠️ **IMPORTANT:** This creative template has ATS trade-offs

**For ATS Applications:**
- Create a simplified version (remove sidebar, use standard format)
- Or use this for direct email/human review only
- Always have a plain version ready

**What to Avoid for ATS:**
- Two-column layouts (some ATS scramble reading order)
- Graphics and icons (can't be parsed)
- Unusual section headers (use standard ones)
- Text in images (not readable)

### Step 7: Export
1. Download as Microsoft Word (.docx)
2. Also export PDF for visual reference
3. Create ATS-friendly simplified version
4. Include both in download package

## 📋 CUSTOMIZATION INSTRUCTIONS (For Buyers)

### How to Edit
1. Open in Microsoft Word or Google Docs
2. Replace all bracketed placeholders
3. Customize colors to match your personal brand
4. Add/remove sections based on your background
5. For creative fields: Link to portfolio prominently

### Design Tips
- Keep it readable (don't go too wild with colors)
- Ensure print quality (test print before sending)
- Save as PDF to preserve formatting
- Have a simplified ATS version ready

✓ Creative industry applications

✓ Portfolio submissions

### When to Use a Traditional Template

✓ Government positions
