# CV Template 1: Modern Professional

**Style:** Clean, contemporary, versatile
**Best For:** Business, tech, marketing, general corporate roles
**Experience Level:** 3-15 years
**Page Count:** 1-2 pages

---

## 📐 CANVA DESIGN SPECIFICATIONS

### Document Setup
```
Page Size: US Letter (8.5" x 11") or A4 (210 x 297 mm)
Orientation: Portrait
Margins: 0.75" all sides (19 mm)
Bleed: None needed (digital only)
Color Mode: RGB (for digital) / CMYK (if printing)
```

### Color Palette
```
Primary Text: #2D3748 (Charcoal Gray)
Secondary Text: #718096 (Medium Gray)
Accent Color: #3182CE (Professional Blue)
Background: #FFFFFF (White)
Divider Lines: #E2E8F0 (Light Gray)
```

### Typography
```
Name: Montserrat Bold, 24-28pt, #2D3748
Job Title: Montserrat Regular, 14-16pt, #3182CE
Section Headers: Montserrat SemiBold, 12-14pt, #2D3748, UPPERCASE
Body Text: Open Sans Regular, 10-11pt, #2D3748
Bullet Points: Open Sans Regular, 10-11pt, #2D3748
Contact Info: Open Sans Regular, 9-10pt, #718096
```

### Layout Grid
```
Header Section: Top 2.5"
  - Name centered or left-aligned
  - Title below name
  - Contact info in single line or 2-column grid

Main Content: Below header
  - Single column layout (ATS-friendly)
  - Section spacing: 0.5" between sections
  - Line spacing: 1.15 for body text

Column Structure:
  - Left margin: 0.75"
  - Right margin: 0.75"
  - Content width: 7" (for US Letter)
```

## 🎨 CANVA BUILD INSTRUCTIONS

### Step 1: Set Up Document
1. Open Canva, click "Create a design"
2. Select "Custom size" → 8.5" x 11" (or A4)
3. Name it "CV Template - Modern Professional"

### Step 2: Add Guides
1. Click File → View settings → Show guides
2. Add margin guides at 0.75" on all sides
3. Add horizontal guide at 2.5" (header bottom)

### Step 3: Build Header
1. Add text box for name (Montserrat Bold, 26pt)
2. Add text box for title (Montserrat Regular, 15pt, #3182CE)
3. Add text box for contact (Open Sans, 9pt, #718096)
4. Add line element below (1pt, #E2E8F0)

### Step 4: Build Sections
1. Create section header style (Montserrat SemiBold 13pt, UPPERCASE)
2. Add line element below each header (1pt, #3182CE, 1" width)
3. Create body text style (Open Sans 10.5pt, #2D3748)
4. Use bullet element or type • (Alt+0149)

### Step 5: Create Placeholder Text
1. Use bracketed placeholders: [Your Name], [Job Title]
2. Add example bullet points with quantified achievements
3. Include instructional notes in light gray italics

### Step 6: ATS Optimization Check
1. ✓ No text boxes (use Canva text elements, export as Word)
2. ✓ No tables or columns
3. ✓ Standard section headers
4. ✓ Simple bullets (no special characters)
5. ✓ Export as .docx for editing

### Step 7: Export Settings
1. Click Share → Download
2. Select "Microsoft Word (.docx)"
3. Also export PDF for reference
4. Name files: "Modern-Professional-CV-Template.docx"

## 📋 CUSTOMIZATION INSTRUCTIONS (For Buyers)

### How to Edit
1. Open in Microsoft Word (or Google Docs)
2. Replace all bracketed text [like this] with your info
3. Delete example bullet points, add your own
4. Adjust dates, titles, company names
5. Save as PDF when ready to submit

### Formatting Tips
- Keep to 1 page if under 10 years experience
- Use 2 pages if you have extensive relevant experience
- Maintain consistent date format (e.g., "Jan 2020 – Present")
- Use action verbs to start each bullet point
- Quantify achievements wherever possible

### ATS Tips
- Save as .docx when applying through company portals
- Use standard section headers (Experience, Education, Skills)
- Include keywords from the job description
- Avoid headers/footers (some ATS can't read them)
- Use simple bullets (•) not special characters
