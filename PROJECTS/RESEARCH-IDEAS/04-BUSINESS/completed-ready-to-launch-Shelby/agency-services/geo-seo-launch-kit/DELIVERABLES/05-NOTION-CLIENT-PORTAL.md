## DATABASE TEMPLATES

### Tasks Database
```
Properties:
- Task (Title)
- Status (Select: Backlog, In Progress, Review, Complete)
- Priority (Select: P0, P1, P2, P3)
- Assigned To (Person)
- Due Date (Date)
- Category (Select: Technical, Content, Citations, Reporting, Client Request)
- Effort (Select: Low, Medium, High)
- Notes (Text)
- Attachments (Files)
```

### Meetings Database
```
Properties:
- Meeting Title (Title)
- Date (Date)
- Type (Select: Kickoff, Weekly, Monthly Review, QBR, Ad-hoc)
- Attendees (Multi-select)
- Recording (URL)
- Notes (Text)
- Action Items (Relation to Tasks)
```

### Leads Database
```
Properties:
- Lead/Company (Title)
- Date (Date)
- Source (Select: ChatGPT, Google AI, Claude, Perplexity, Other)
- Value (Number)
- Status (Select: New, Contacted, Qualified, Won, Lost)
- Notes (Text)
```
