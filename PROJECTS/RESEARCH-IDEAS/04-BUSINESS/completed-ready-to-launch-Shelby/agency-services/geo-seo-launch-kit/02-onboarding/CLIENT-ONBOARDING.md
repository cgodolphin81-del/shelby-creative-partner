## PART 1: ONBOARDING CHECKLIST (Internal Use)

### PRE-KICKOFF (Before First Call)
- [ ] Contract signed and returned
- [ ] First payment received
- [ ] Welcome email sent with questionnaire link
- [ ] Internal project folder created
- [ ] Project manager assigned
- [ ] Kickoff call scheduled

### KICKOFF CALL (Week 1, Day 1-2)
- [ ] Introductions and team overview
- [ ] Review onboarding questionnaire responses
- [ ] Confirm business goals and KPIs
- [ ] Discuss timeline and expectations
- [ ] Identify key stakeholders
- [ ] Set up communication channels

### ACCESS COLLECTION (Week 1, Day 2-5)
- [ ] Website CMS access (or coordinate with dev team)
- [ ] Google Analytics access
- [ ] Google Search Console access
- [ ] Google Business Profile access (if local)
- [ ] Social media profiles list
- [ ] Brand assets (logos, guidelines)
- [ ] Existing content inventory

### TECHNICAL SETUP (Week 1, Day 3-7)
- [ ] Domain verification
- [ ] robots.txt audit and update
- [ ] llms.txt creation
- [ ] Schema markup baseline audit
- [ ] Site speed baseline measurement
- [ ] Mobile-friendliness check
- [ ] Crawlability assessment

### AUDIT PHASE (Week 1-2)
- [ ] AI visibility scan (7 platforms)
- [ ] Competitor analysis
- [ ] Content gap analysis
- [ ] Brand mention audit
- [ ] Technical SEO audit
- [ ] Compile audit report
- [ ] Internal quality review

### AUDIT PRESENTATION (Week 2)
- [ ] Schedule presentation call
- [ ] Present findings and recommendations
- [ ] Prioritize action items with client
- [ ] Get approval on implementation plan
- [ ] Set timeline for Phase 1

### IMPLEMENTATION PHASE 1 (Week 2-4)
- [ ] Technical fixes (robots.txt, llms.txt, schema)
- [ ] Content optimization (first batch)
- [ ] Citation building (first batch)
- [ ] Weekly progress updates
- [ ] Mid-point check-in call

### FIRST MONTHLY REPORT (Week 4)
- [ ] Compile month 1 data
- [ ] Create monthly report
- [ ] Schedule review call
- [ ] Present results and Month 2 plan
- [ ] Collect client feedback

### ONGOING (Month 2+)
- [ ] Monthly content optimization
- [ ] Monthly citation building
- [ ] Weekly monitoring
- [ ] Monthly reporting
- [ ] Regular strategy calls
- [ ] Quarterly business reviews (Enterprise)
