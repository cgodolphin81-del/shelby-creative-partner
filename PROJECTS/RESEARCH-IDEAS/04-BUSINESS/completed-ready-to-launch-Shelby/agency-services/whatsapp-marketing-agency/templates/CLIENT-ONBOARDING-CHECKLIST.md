# Client Onboarding Checklist

## 📋 Pre-Onboarding (Before First Payment)

### Account Setup
- [ ] Contract signed
- [ ] First payment received
- [ ] Access credentials collected
- [ ] BSP account created/connected
- [ ] Phone number verified
- [ ] Business profile completed

### Information Gathering
- [ ] Business overview completed
- [ ] Target audience defined
- [ ] Current marketing channels documented
- [ ] Key metrics collected (revenue, orders, etc.)
- [ ] Pain points identified
- [ ] Goals & KPIs defined
- [ ] Brand voice & tone documented
- [ ] Competitors identified

### Technical Setup
- [ ] CRM integration configured
- [ ] E-commerce platform connected
- [ ] Webhooks tested
- [ ] API access granted
- [ ] Team accounts created
- [ ] Permissions set

---

## 🚀 Week 1: Discovery & Strategy

### Deep-Dive Sessions
- [ ] Kickoff call completed
- [ ] Customer journey mapped
- [ ] Use cases identified
- [ ] Campaign calendar planned
- [ ] Automation flows designed
- [ ] Message templates outlined
- [ ] Segmentation strategy defined

### Content & Creative
- [ ] Brand assets collected
- [ ] Logo & images received

- [ ] Creative requirements defined

### Compliance
- [ ] Privacy policy reviewed

- [ ] Data processing agreement signed

- [ ] End-to-end flows tested

- [ ] Performance validated

## 🧪 Week 3: Testing & Optimization

### Quality Assurance
- [ ] All message templates tested
- [ ] All automation flows tested
- [ ] All integrations tested
- [ ] Mobile responsiveness checked
- [ ] Accessibility verified
- [ ] Opt-out mechanism tested

### Small-Scale Launch
- [ ] Internal team test group
- [ ] 5% of audience test
- [ ] Performance monitored
- [ ] Issues identified & fixed
- [ ] Client approval received

### Optimization
- [ ] A/B test variants created
- [ ] Send times optimized
- [ ] Message copy refined
- [ ] CTAs tested
- [ ] Landing pages optimized

## ✅ Onboarding Complete

**Sign-off Requirements:**
- [ ] All systems operational
- [ ] First campaign successful
- [ ] Client satisfied
- [ ] KPIs being tracked
- [ ] Ongoing management plan confirmed

**Handoff to Account Management:**
- [ ] All documentation complete
- [ ] Team trained
- [ ] Access granted
- [ ] Success criteria defined
- [ ] Next review scheduled

---

*Onboarding typically takes 2-4 weeks depending on complexity. Rushed onboarding = implementation issues later.*
