# Campaign Launch Checklist

## Pre-Launch Verification for Ad Campaigns

---

## STRATEGY & PLANNING

### Campaign Objectives
- [ ] Primary goal defined (Sales, Leads, Traffic)
- [ ] KPIs established (ROAS, CPA, CTR targets)
- [ ] Budget allocated
- [ ] Timeline defined
- [ ] Success criteria clear

### Target Audience
- [ ] Audience personas defined
- [ ] Interest audiences created
- [ ] Lookalike audiences created (if data available)
- [ ] Retargeting audiences created
- [ ] Exclusion audiences set up
- [ ] Audience sizes appropriate (1M+ for prospecting)

### Offer & Messaging
- [ ] Offer defined (discount, free shipping, etc.)
- [ ] Key messaging pillars identified
- [ ] Unique value propositions clear
- [ ] Pain points addressed
- [ ] Call-to-action defined

## FACEBOOK/INSTAGRAM CAMPAIGNS

### Account Setup
- [ ] Business Manager verified
- [ ] Ad account created and active
- [ ] Payment method added
- [ ] Spending limit set
- [ ] Facebook Page connected
- [ ] Instagram account connected
- [ ] Domain verified
- [ ] Pixel installed and active
- [ ] Conversions API configured (recommended)
- [ ] Aggregated Event Measurement configured

### Campaign Structure
- [ ] Campaign objective: Sales/Conversions
- [ ] Campaign budget optimization (CBO) or ABO decided
- [ ] Campaign naming convention followed
- [ ] Special ad categories reviewed (if applicable)

### Ad Sets
- [ ] Ad set names clear and descriptive
- [ ] Conversion event selected (Purchase)
- [ ] Attribution window set (7-day click, 1-day view)
- [ ] Audiences selected (3-5 per campaign)
- [ ] Placements selected (Advantage+ or manual)
- [ ] Budget set per ad set
- [ ] Schedule set (continuous or date range)
- [ ] Optimization for ad delivery: Conversions

### Ads
- [ ] 3-5 ads per ad set
- [ ] Ad names descriptive
- [ ] Creative uploaded (videos/images)
- [ ] Creative specs correct (ratio, size, length)
- [ ] Primary text written (multiple variations)
- [ ] Headlines written (multiple variations)
- [ ] Descriptions written (optional)
- [ ] Call-to-action button selected
- [ ] Landing page URL correct
- [ ] UTM parameters added
- [ ] Preview on all placements
- [ ] No policy violations

### Creative Checklist
- [ ] Hook in first 3 seconds (video)
- [ ] Captions on all videos
- [ ] Text on image <20%
- [ ] Branding visible
- [ ] Clear call-to-action
- [ ] Mobile-optimized
- [ ] Multiple formats (feed, stories, reels)
- [ ] No prohibited content

## OPTIMIZATION PLAN

### Day 1-3: Learning Phase
- [ ] Minimal changes (let algorithm learn)
- [ ] Monitor for disapprovals
- [ ] Check tracking accuracy
- [ ] Document initial benchmarks

### Day 4-7: Initial Optimization
- [ ] Pause ads with no purchases after $50-100 spend
- [ ] Pause ads with CTR <0.5%
- [ ] Increase budget on winners (20-30%)
- [ ] Review search terms, add negatives
- [ ] Check audience performance

### Week 2: Scale or Iterate
- [ ] Analyze full week data
- [ ] Scale winning campaigns
- [ ] Launch new creative tests
- [ ] Expand audiences
- [ ] Test new offers

## EMERGENCY PROCEDURES

### If Campaign Disapproved
1. Review policy violation
2. Edit creative/copy if minor
3. Appeal if believe error
4. Launch alternative creative
5. Document for future reference

### If Tracking Breaks
1. Pause campaigns immediately
2. Diagnose tracking issue
3. Fix tracking
4. Test conversions
5. Resume campaigns
6. Note data gap in reporting

### If Performance Tanks
1. Check for technical issues
2. Review recent changes
3. Check competitor activity
4. Review audience fatigue
5. Launch fresh creative
6. Consider audience refresh

### If Budget Exhausted Too Quickly
1. Check for unusual spike
2. Review bids
3. Check for click fraud
4. Adjust budget/bids
5. Review targeting breadth
