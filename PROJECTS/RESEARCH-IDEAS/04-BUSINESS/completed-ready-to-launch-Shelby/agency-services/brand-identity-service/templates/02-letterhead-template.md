# Letterhead Template

## Overview

- Top: 1" (25.4mm)

## Footer Design

### Standard Footer

**Content:**
- Company name
- Full address
- Phone number
- Email address
- Website
- Social media icons (optional)
- Registration number (if applicable)
- Privacy statement (if applicable)

**Format:**
```
[Company Name] | [Street Address], [City], [State] [ZIP]
[Phone] | [Email] | [Website]
[Social Icons]
```

---

### Minimal Footer

**Content:**
- Website
- Email
- Phone

**Format:**
```
[Website] | [Email] | [Phone]
```

---

### Legal Footer

**Content:**
- Company name
- Address
- Registration info
- Privacy notice
- Confidentiality notice

**Format:**
```
[Company Name], [Registration Number]
[Address] | [Phone] | [Email] | [Website]

CONFIDENTIALITY NOTICE: This message contains confidential information...
```

## Letter Structure

### Formal Letter Format

```
[Date]

[Recipient Name]
[Recipient Title]
[Company Name]
[Address]
[City, State ZIP]

Dear [Name],

[Opening paragraph - purpose of letter]

[Body paragraphs - details, context, supporting information]

[Closing paragraph - call to action, next steps]

Sincerely,

[Signature]

[Typed Name]
[Title]
[Company Name]

Enclosures: [List if applicable]
cc: [List if applicable]
```

---

### Email Letter Format

```
[Date]

[Recipient Name],

[Opening]

[Body]

[Closing]

Best regards,

[Name]
[Title]
[Company Name]
[Phone]
[Email]
[Website]
```

## Printing Guidelines

### Paper Options

**Standard:**
- 24lb Bond (good quality)
- 32lb Bond (better quality)

**Premium:**
- 100lb Text (luxury feel)
- Cotton paper (high-end)
- Recycled paper (eco-friendly)

**Finish:**
- Smooth (standard)
- Linen (textured)
- Eggshell (subtle texture)
- Gloss (shiny - not recommended)

---

### Printing Options

**Local Print Shop:**
- Pros: Quick turnaround, personal service
- Cons: May be more expensive

**Online Printers:**
- Pros: Competitive pricing, bulk discounts
- Cons: Shipping time, can't see before printing

**In-House:**
- Pros: On-demand, cost-effective for small runs
- Cons: Quality varies, limited paper options

---

### Recommended Quantities

**Small Business:** 100-250 sheets

**Medium Business:** 500-1000 sheets

**Large Business:** 2000+ sheets

**Tip:** Order 20% extra for mistakes and growth

## Customization Instructions

### To Customize in Word:

1. Open `Letterhead_US_Letter.docx`
2. Insert > Header > Edit Header
3. Replace logo (Insert > Pictures)
4. Update text fields
5. Save as template (.dotx)
6. Use for all new documents

### To Customize in InDesign:

1. Open `Letterhead_US_Letter.indd`
2. Update logo (File > Place)
3. Edit text fields
4. Update colors (Swatches panel)
5. Export as PDF (File > Export)

### To Customize in Illustrator:

1. Open `Letterhead_US_Letter.ai`
2. Update logo
3. Edit text
4. Save copy for use
5. Export as PDF
