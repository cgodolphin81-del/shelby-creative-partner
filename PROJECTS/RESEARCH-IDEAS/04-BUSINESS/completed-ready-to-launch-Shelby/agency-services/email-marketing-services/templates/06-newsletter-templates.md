## TEMPLATE IMPLEMENTATION GUIDE

### Design Best Practices (All Templates)

1. **Mobile-First Design**
   - Single column layout
   - 14px+ font size
   - 44px+ button height
   - Images scale properly

2. **Accessibility**
   - Alt text on all images
   - Sufficient color contrast
   - Clear hierarchy
   - Descriptive link text

3. **Brand Consistency**
   - Logo in header
   - Brand colors throughout
   - Consistent voice/tone
   - Standard footer format

4. **Performance**
   - Keep under 100KB
   - Optimize images
   - Minimal code
   - Test across clients

### Testing Checklist

Before sending any template:
- [ ] Mobile rendering test
- [ ] Desktop rendering test
- [ ] Dark mode test
- [ ] All links working
- [ ] Images loading
- [ ] Spam score check
- [ ] Unsubscribe working
- [ ] Physical address included
