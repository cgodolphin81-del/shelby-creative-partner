### 2.2 Stage Details

#### Stage 1: Idea Generation (2 days)
**Owner:** Content Strategist

**Inputs:**
- [ ] Keyword research
- [ ] Customer questions
- [ ] Sales team input
- [ ] Industry trends
- [ ] Competitor analysis
- [ ] Performance data

**Output:** Idea backlog with priority scores

**Idea Scoring:**
```
Score = (Search Volume Г— 0.3) + (Business Value Г— 0.4) + (Feasibility Г— 0.3)

Business Value: 1-10 (alignment with goals)
Feasibility: 1-10 (resources, expertise available)
```

#### Stage 2: Brief Creation (1 day)
**Owner:** Content Strategist

**Brief Must Include:**
- [ ] Working title
- [ ] Target keyword
- [ ] Search intent
- [ ] Target audience
- [ ] Word count
- [ ] Key points to cover
- [ ] Competitor URLs
- [ ] Internal links to include
- [ ] CTA
- [ ] Due date

#### Stage 3: Assignment (Same day)
**Owner:** Content Manager

**Considerations:**
- Writer expertise/interest
- Current workload
- Deadline
- Complexity match

#### Stage 4: Writing (2-5 days)
**Owner:** Writer

**Deliverables:**
- [ ] First draft in Google Docs
- [ ] Meta title & description
- [ ] Suggested images/graphics
- [ ] Internal link suggestions
- [ ] Sources cited

#### Stage 5: Approval Process (1-2 days)
**Owner:** Editor вҶ’ Content Strategist

**Review Checklist:**
```
EDITOR REVIEW:
в–Ў Grammar & spelling
в–Ў Brand voice consistency
в–Ў Clarity & flow
в–Ў Fact-checking
в–Ў CTA included
в–Ў Follows brief

SEO REVIEW:
в–Ў Keyword usage (natural)
в–Ў Header structure
в–Ў Meta tags
в–Ў Internal links
в–Ў Image alt text
в–Ў URL slug

STRATEGIST FINAL:
в–Ў Aligns with strategy
в–Ў Quality standard met
в–Ў Ready for publication
```

#### Stage 6: Publishing (Same day)
**Owner:** Content Manager

**Pre-Publish Checklist:**
- [ ] Formatted in CMS
- [ ] Images optimized & uploaded
- [ ] Links working
- [ ] Categories/tags assigned
- [ ] Featured image set
- [ ] Schema markup added
- [ ] Preview checked (desktop & mobile)
- [ ] Publish date scheduled

#### Stage 7: Distribution (1-2 days)
**Owner:** Social Media Manager

**Distribution Checklist:**
- [ ] Social posts created (all platforms)
- [ ] Email newsletter included
- [ ] Community shares (where appropriate)
- [ ] Paid boost (if budget allows)
- [ ] Influencer outreach (if applicable)

#### Stage 8: Performance Analysis (Ongoing)
**Owner:** Content Strategist

**Review Cadence:**
- **Week 1:** Initial traction check
- **Month 1:** Full performance review
- **Quarter:** Update/refresh decision

**For Approvers:**
```
вң“ DO:
  - Be specific about what needs changing
  - Explain the "why" behind feedback
  - Provide examples when helpful
  - Give feedback within SLA
  - Acknowledge what's working well

вң— DON'T:
  - Give vague feedback ("make it better")
  - Rewrite without explanation
  - Miss deadlines without communication
  - Add new requirements mid-review
  - Be harsh or personal
```

## 6. Quality Standards

### 6.1 Content Quality Checklist

| Criteria | Excellent (5) | Good (3) | Needs Work (1) |
|----------|---------------|-----------|----------------|
| **Clarity** | Crystal clear, easy to understand | Mostly clear, minor confusion | Confusing, hard to follow |
| **Accuracy** | Fact-checked, sourced | Minor issues | Errors, unsourced claims |
| **Depth** | Comprehensive, original insights | Adequate coverage | Surface-level |
| **Engagement** | Compelling, hard to put down | Readable | Dry, boring |
| **SEO** | Fully optimized | Basic optimization | Not optimized |
| **Brand Voice** | Perfect match | Mostly on-brand | Off-brand |
| **Formatting** | Clean, scannable, visual | Readable | Dense, hard to scan |
| **CTA** | Clear, compelling | Present | Missing or weak |

**Quality Threshold:** Average score of 4.0+ to publish

### 6.2 Style Guide Essentials

**Voice & Tone:**
```
We are: [Adjective 1], [Adjective 2], [Adjective 3]
We are not: [Adjective 1], [Adjective 2], [Adjective 3]

Examples:
вң“ "Here's how to solve this problem"
вң— "One must consider the following methodology"
```

**Grammar & Usage:**
- [ ] Oxford comma: Yes/No
- [ ] Contractions: Yes/No
- [ ] Active voice preferred
- [ ] Sentence length: Aim for 15-20 words avg
- [ ] Paragraph length: 2-4 sentences

**Formatting:**
- [ ] Use H2 for main sections
- [ ] Use H3 for subsections
- [ ] Bullet points for lists
- [ ] Bold for emphasis (not italics)
- [ ] Short paragraphs (2-4 sentences)
