## Template Usage Guidelines

### Version Control
- All templates have version numbers
- Review date included
- Owner identified
- Change log maintained

### Customization
- Templates are starting points
- Customize for each engagement
- Legal review required for contracts
- Compliance review for healthcare-specific content

### Storage
- Master templates in this repository
- Client-specific versions in secure client folders
- Executed contracts in contract management system

---

## Template Standards

### Formatting
- Consistent header/footer
- Company branding
- Version control
- Confidentiality markings

### Content
- Clear instructions for use
- Placeholders clearly marked [LIKE THIS]
- Examples provided where helpful
- Cross-references to related templates

### Review Cycle
- Quarterly template review
- Annual major updates
- Immediate updates for regulatory changes
- Feedback loop from users

## Template Request Process

Need a new template or update to existing?

1. Submit request to: templates@healthaiconsulting.com
2. Include:
   - Template name/purpose
   - Intended users
   - Key sections needed
   - Priority/timeline
3. Template team reviews within 5 business days
4. Development and review (2-4 weeks)
5. Publication and training
