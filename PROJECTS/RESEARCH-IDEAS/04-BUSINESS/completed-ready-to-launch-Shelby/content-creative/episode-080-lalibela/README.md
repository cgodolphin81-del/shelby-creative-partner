## 🎯 Quick Reference

### Key Facts
| Detail | Information |
|--------|-------------|
| **Location** | Lalibela, Amhara Region, Ethiopia |
| **Altitude** | 2,480 meters (8,136 ft) |
| **Construction** | Late 12th - Early 13th century |
| **Builder** | King Gebre Meskel Lalibela |
| **Churches** | 11 monolithic structures |
| **UNESCO** | 1978 (one of first African sites) |
| **Entrance Fee** | $50 USD (valid 5 days) |
| **Best Time** | October - March (dry season) |
| **Major Festivals** | Genna (Jan 7), Timkat (Jan 19-20), Meskel (Sep 27) |

### 11 Churches
**Northern Group:**
1. Biete Medhane Alem (largest monolithic church)
2. Biete Maryam
3. Biete Maskal
4. Biete Denagel
5. Biete Golgotha Mikael (King Lalibela's tomb, women cannot enter)

**Southern Group:**
6. Biete Amanuel
7. Biete Qeddus Mercoreus
8. Biete Abba Libanos
9. Biete Gabriel-Rufael
10. Biete Lehem

**Western Group:**
11. Biete Ghiorgis (most famous, cross-shaped)

### Pronunciation Guide
- **Lalibela:** la-lee-BEH-la
- **Biete Ghiorgis:** BYEH-teh GHEE-or-gis
- **Biete Medhane Alem:** BYEH-teh med-HAH-neh AH-lem
- **Ge'ez:** GHEE-ez
- **Genna:** GEN-na
- **Timkat:** TIM-kat
