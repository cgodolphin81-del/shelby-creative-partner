## Usage Notes

- All prompts optimized for Midjourney v6
- Adjust `--ar` (aspect ratio) as needed for your format
- Add `--stylize` parameter (0-1000) for artistic interpretation
- Use `--v 6` for latest Midjourney version
- For DALL-E 3: Remove parameters, keep descriptive text only
- For Stable Diffusion: Add negative prompts for quality control

**Recommended Negative Prompts (for SD):**
```
blurry, low quality, distorted, oversaturated, text, watermark, signature, cartoon, anime, 3d render, plastic looking
```
