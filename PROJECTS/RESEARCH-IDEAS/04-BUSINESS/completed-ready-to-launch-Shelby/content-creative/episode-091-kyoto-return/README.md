## Episode Summary

**Title:** The Middle Kingdom Return  
**Location:** Kyoto, Japan  
**Theme:** Return, stillness, impermanence, temples as memory  

**Logline:** After years of wandering, Shelby returns to Kyoto—Japan's ancient capital—to discover that the temples, bamboo groves, and cherry blossoms have been teaching all along. This is not a travel episode. It's a homecoming.

**Key Locations:**
- Fushimi Inari Taisha (10,000 vermilion torii gates)
- Kinkaku-ji (Golden Pavilion)
- Arashiyama Bamboo Grove
- Gion District (geisha quarter)
- Kiyomizu-dera (wooden stage temple)
- Philosopher's Path (cherry blossoms)

**Philosophical Themes:**
- Mono no aware (the pathos of things, impermanence)
- Shokunin (the craftsman's spirit, devotion)
- Return as going deeper, not going back
- Temples as memory and legacy
- Stillness as a frequency, not a destination
