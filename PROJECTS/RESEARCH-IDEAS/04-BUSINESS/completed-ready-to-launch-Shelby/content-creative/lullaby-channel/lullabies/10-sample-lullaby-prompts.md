## Generation Tips

### For Suno AI:
1. Use Custom Mode for full control
2. Generate multiple versions and select the best
3. Use "Instrumental" tag if vocals aren't working
4. Extend tracks if they end too soon
5. Download highest quality available

### For Udio:
1. Use "High Quality" generation setting
2. Generate 2-4 variations per prompt
3. Use "Extend" feature for longer tracks
4. Adjust "Creativity" slider lower for more consistent results
5. Download WAV files for best quality

### Post-Processing (Optional):
- Normalize volume across all tracks
- Add 2-second fade in/out
- Apply gentle high-pass filter (remove rumble below 40Hz)
- Ensure all tracks are similar loudness (-18 to -14 LUFS)
