## 🚀 QUICK START GUIDE

### Step 1: Set Up Your Etsy Shop (If Not Done)
1. Go to Etsy.com/sell
2. Create your shop name
3. Set up payment/billing
4. Configure shop preferences

### Step 2: Customize Shop Content
1. **Shop Announcement** → Edit `shop/announcement.md`
   - Add your shop name
   - Set launch discount code and dates
   - Add your personal story

2. **About Page** → Edit `shop/about.md`
   - Add your name and background
   - Customize the story
   - Add your photo (for Etsy)

3. **Policies** → Edit `shop/policies.md`
   - Review and adjust as needed
   - Add your contact info
   - Set your response time expectations

### Step 3: Create Your Listings
You have 15 complete listing packages. Start with 3-5 to launch:

**Recommended Launch Mix:**
1. Professional CV Template (broad appeal)
2. Modern Resume Template (popular)
3. Student Resume Template (high volume)
4. Executive CV Template (premium pricing)
5. Resume Bundle (value option)

**For Each Listing:**
1. Copy title from `listings/seo-titles.md`
2. Copy description from `listings/descriptions.md`
3. Copy tags from `listings/tags.md`
4. Create listing photos (use Canva templates from `designs/`)
5. Set your price
6. Publish!

### Step 4: Build Your Templates
Use the design specs in `designs/` folder:

1. Open Canva (free or Pro)
2. Follow the Canva build instructions in each template file
3. Create the template with placeholder text
4. Export as .docx (for editing) and PDF (for reference)
5. Test edit in Word/Google Docs
6. Package files for download

**Template Files to Create:**
- Template 1: Modern Professional (Entry-level pricing: $8-12)
- Template 2: Creative Bold (Mid-tier pricing: $12-18)
- Template 3: Minimalist Clean (Premium pricing: $15-25)

### Step 5: Set Up Customer Systems
1. **Thank You Messages** → Copy from `customer/thank-you-card.md`
   - Set up Etsy's automatic "Message to Buyers"
   - Or send manually after each purchase

2. **Customer Service** → Review `customer/service-scripts.md`
   - Familiarize yourself with common questions
   - Copy/paste and customize responses as needed

### Step 6: Launch Marketing
**Week 1: Launch**
- Day 1: Send Email 1 (Launch Announcement) to your network
- Day 1: Post Social Post 1 (Launch) on all platforms
- Day 2: Post Social Post 2 (Problem/Solution)
- Day 3: Post Social Post 3 (Testimonial/Social Proof)
- Day 4: Post Social Post 4 (Behind the Scenes)
- Day 5: Post Social Post 5 (Value Tips)
- Day 6-7: Engage, respond, share

**Week 2: Momentum**
- Send Email 4 (Follow-up) to non-openers
- Post Social Post 6 (Product Showcase)
- Post Social Post 7 (Personal Story)
- Post Social Post 8 (Urgency - if running limited promo)

**Week 3: Social Proof**
- Post Social Post 9 (FAQ)
- Post Social Post 10 (Gratitude/Milestone)
- Send Email 5 (Success Story) if you have one

## 📸 LISTING PHOTO GUIDELINES

### Required Photos (Etsy allows 10)
1. **Main Image:** Template preview (clean, professional)
2. **Lifestyle Shot:** Template on laptop/desk
3. **Before/After:** Old resume vs. your template
4. **Features:** Callout key features (ATS-friendly, editable, etc.)
5. **What's Included:** List files they receive
6. **Testimonial:** Customer review graphic
7. **Color Options:** Show variations (if applicable)
8. **Instructions Preview:** Show how easy it is to edit
9. **Bundle Preview:** If selling bundles
10. **Shop Logo/Branding:** Your shop identity

### Photo Specs
- **Size:** Minimum 2000px on shortest side
- **Format:** JPG or PNG
- **Background:** Clean, professional
- **Text:** Minimal, large enough to read on mobile

### Canva Templates to Create

## 🎯 SEO BEST PRACTICES

### Titles
- Use all 140 characters Etsy allows
- Front-load important keywords
- Include: template type, format, key benefit
- Example: "Professional CV Template Resume Template Word Download | Modern Clean Resume Design | ATS Friendly Curriculum Vitae | Instant Download"

### Tags (13 per listing)
- Use all 13 tags
- Mix of broad and specific
- Include: resume, CV, template, industry, experience level
- Tags are plural and singular (resume/resumes)

### Categories
- Primary: Resume Templates
- Secondary: Career & Interview Prep

### Attributes
- Fill out all applicable attributes
- File type: DOCX, PDF
- Occasion: Job Interview, Career Change
- Style: Professional, Modern, etc.

## ⚠️ COMMON MISTAKES TO AVOID

### Listing Mistakes
❌ Using stock photos that don't show YOUR template
❌ Vague titles without keywords
❌ Not using all 13 tags
❌ Pricing too low (undervalues your work)
❌ Pricing too high (without social proof)

### Customer Service Mistakes
❌ Slow response times (over 24 hours)
❌ Robotic, impersonal responses
❌ Not following up on issues
❌ Arguing with unhappy customers publicly

### Marketing Mistakes
❌ Only posting "buy my stuff" content
❌ Not engaging with comments/messages
❌ Giving up after week 1 (this is a marathon)
❌ Not asking for reviews

### Template Mistakes
❌ Not testing in Word/Google Docs before selling
❌ Forgetting to include instructions
❌ Making it too complex to edit
❌ Not considering ATS compatibility
