**Style:** Understated, elegant, timeless
**Best For:** Executives, academics, consultants, lawyers, senior professionals
**Experience Level:** 8+ years
**Page Count:** 1-2 pages


**Design Notes:**
- Name: Garamond Bold 28pt, centered, pure black
- Title: Garamond Italic 15pt, centered, dark gray
- Contact: Garamond Regular 10pt, centered, medium gray
- Separators: Bullet point (•) with spaces
- Spacing: 0.5" above and below name block
- Divider: 0.5pt line, 6" width, centered, light gray

## 🎨 CANVA BUILD INSTRUCTIONS

### Step 1: Set Up Document
1. Create custom design: 8.5" x 11"
2. Set margins to 1" on all sides (use guides)
3. Name: "CV Template - Minimalist Clean"

### Step 2: Create Header
1. Add text box for name (Garamond Bold 28pt, centered)
2. Add text box for title (Garamond Italic 15pt, centered, gray)
3. Add text box for contact (Garamond 10pt, centered, gray)
4. Add line element (0.5pt, 6" width, centered, light gray)
5. Space: 0.5" between each element

### Step 3: Build Section Headers
1. Create master section header style (Garamond SemiBold 12pt)
2. Set to small caps (or UPPERCASE if using sans-serif)
3. Center all headers
4. Add subtle divider below (optional, 0.5pt, 2" width)

### Step 4: Build Content Blocks
1. Use consistent left-alignment for body content
2. Maintain generous spacing between sections (0.5")
3. Use standard bullets (•)
4. Keep line spacing at 1.15-1.25

### Step 5: White Space Strategy
1. Don't fill every inch—white space = luxury
2. If content runs short, increase spacing
3. If content runs long, edit ruthlessly (not shrink font)
4. Two pages is fine for senior executives

### Step 6: Typography Hierarchy
```
Name: 28pt Bold
Title: 15pt Italic
Section Headers: 12pt SemiBold
Job Title/Company: 11pt Regular/Bold
Body/Bullets: 11pt Regular
Contact/Dates: 10pt Regular
```

### Step 7: Export
1. Download as Microsoft Word (.docx)
2. Also export PDF for reference
3. Ensure fonts are embedded or use system fonts
4. Test print to verify spacing

## 📋 CUSTOMIZATION INSTRUCTIONS (For Buyers)

### How to Edit
1. Open in Microsoft Word
2. Replace all bracketed text with your information
3. Adjust section order based on your strengths
4. Delete sections that don't apply (e.g., Publications)
5. Save as PDF for submission

### Content Tips for Senior Roles
- Lead with impact, not responsibilities
- Quantify everything (revenue, team size, savings)
- Show progression and increasing scope
- Include board/advisory roles (shows stature)
- Mention notable companies/brands by name

### Formatting Tips
- Keep consistent date format throughout
- Use parallel structure in bullet points
- Bold metrics sparingly (only the impressive ones)
- Two pages is expected at senior levels
- Print on quality paper if handing in person
