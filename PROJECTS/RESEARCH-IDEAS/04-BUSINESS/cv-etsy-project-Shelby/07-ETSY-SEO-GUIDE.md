## 📚 SECTION 1: ETSY SEO BASICS

### How Etsy Search Works

Etsy's algorithm considers 3 main factors:

1. **QUERY MATCHING** (Most Important)
   - Does your listing match what the buyer searched?
   - Titles, tags, categories, attributes

2. **RELEVANCE SCORE**
   - How well does your listing match the search?
   - Complete listings rank higher

3. **CUSTOMER EXPERIENCE**
   - Click-through rate
   - Conversion rate
   - Reviews
   - Shipping time (less relevant for digital)

### Key Ranking Factors (In Order of Importance)

1. ✅ **Title Keywords** (Most important!)
2. ✅ **Tags** (13 tags - use ALL of them)
3. ✅ **Category** (Choose the most specific)
4. ✅ **Attributes** (Fill out all that apply)
5. ✅ **Listing Quality Score** (Clicks, favorites, sales)
6. ✅ **Recency** (New listings get a boost)
7. ✅ **Customer Reviews** (5-star reviews help)
8. ✅ **Shop Performance** (Response time, policies)

## 📝 SECTION 3: TITLE OPTIMIZATION

### Title Formula

```
[Main Keyword] + [Style/Type] + [Format] + [Benefit] + [Audience/Use Case]
```

### Good vs. Bad Titles

**❌ BAD (Too vague):**
```
CV Template Download
```

**❌ BAD (Keyword stuffing):**
```
CV Template Resume Template ATS Template Canva Template Word Template Download Template Professional Template
```

**✅ GOOD (Optimized):**
```
ATS Friendly CV Template Professional Resume Template Canva Editable Word DOCX Download Career Business Corporate Job Application
```

### Title Best Practices

1. **Front-Load Important Keywords**
   - First 40 characters are most visible in search
   - Put main keywords first

2. **Use All 140 Characters**

   - But keep it readable!

3. **Include Variations**

   - "Template" AND "Design"
   - "Canva" AND "Word"

4. **Add Specific Use Cases**
   - "Career Change"
   - "Recent Graduate"
   - "Executive Level"
   - "Tech Industry"

5. **Mention Key Features**
   - "ATS Friendly"
   - "Editable"
   - "Instant Download"
   - "A4 Letter"

## 📊 SECTION 7: ANALYTICS & OPTIMIZATION

### Etsy Stats to Monitor

**In Etsy App → Stats:**

1. **Views**
   - How many people see your listing
   - Low views = SEO problem

2. **Visits**
   - How many click through to your shop
   - Low visits = Main photo problem

3. **Orders**
   - How many buy
   - Low orders = Price/description problem

4. **Conversion Rate**
   - Orders ÷ Visits
   - Average: 1-3%
   - Good: 3-5%
   - Excellent: 5%+

### When to Optimize

**Low Views (<100/week):**
- Improve title keywords
- Change tags
- Add more specific long-tail keywords
- Consider Etsy Ads ($1-2/day test)

**Low Click-Through (<2%):**
- Change main photo
- Improve first image
- Adjust price (if too high)

**Low Conversion (<1%):**
- Improve description
- Add more photos
- Add reviews/social proof
- Consider lowering price

### A/B Testing Process

**Week 1-2: Baseline**
- List with current optimization
- Track views, clicks, sales

**Week 3-4: Test Change**
- Change ONE thing (main photo OR title)
- Track results

**Compare:**
- Did views increase?
- Did clicks increase?
- Did sales increase?

**Keep what works, change what doesn't!**

- Renewing = small SEO boost
- Renew 1 listing per day (not all at once)
- Best time: Morning (9-11am) when buyers are active

### Tip 2: Seasonal Optimization

**January:**
- Add "New Year career change" keywords

**May-June:**
- Add "Graduation" keywords
- "Student CV template"

**September:**

- "Fall job search"

### Tip 3: Long-Tail Keyword Pages

Create listings for specific niches:
- "Nurse CV Template"
- "Teacher Resume Template"
- "Software Engineer CV"

These have less competition and higher conversion!

### Tip 4: Use Etsy Ads Wisely

**When to Use:**
- After you have 5+ reviews
- When conversion rate is 2%+
- When you have profit margin for ads

**Budget:**
- Start: $1-2/day
- Good ROI: Increase to $5/day
- Bad ROI: Pause and optimize listings first

**Expected Results:**
- Cost per click: $0.10-0.50
- Conversion: 1-3%
- Break even: 3-5 sales per $10 ad spend

### Tip 5: External Traffic Bonus

Etsy LOVES when you bring external traffic!

- Share on social media

Etsy may boost your ranking for bringing in buyers!

## 🔧 SECTION 12: TROUBLESHOOTING

### "No Views After 1 Week"

**Problem:** SEO or visibility issue

**Solutions:**
1. Check title - are main keywords in first 40 characters?
2. Verify all 13 tags are filled
3. Share on social media (external traffic helps)
4. Consider $1/day Etsy Ads test
5. Renew listing (gives small boost)

### "Lots of Views, No Sales"

**Problem:** Conversion issue

**Solutions:**
1. Check price vs. competitors
2. Improve main photo (is it clear?)
3. Add more photos (show what's included)
4. Improve description (address objections)
5. Add reviews (ask happy customers)

### "Good Views, Low Click-Through"

**Problem:** Main photo or price

**Solutions:**
1. Change main photo (make it clearer)
2. Add text overlay to photo
3. Adjust price (test lower price)
4. Check if you're showing up for right keywords
