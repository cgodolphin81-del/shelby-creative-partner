### Step 3: Make It ATS-Compliant (CRITICAL!)

**For Each Template, Do This:**

1. **Remove any:**
   - ❌ Photos/headshots
   - ❌ Icons or graphics
   - ❌ Tables
   - ❌ Text boxes (use regular text)
   - ❌ Columns (make single column)
   - ❌ Headers/footers

2. **Adjust Fonts:**
   - Name: 24-32pt, Bold
   - Headings: 13-16pt, Bold
   - Body: 10-12pt, Regular
   - Use ONLY: Arial, Calibri, Helvetica, Times New Roman, Georgia, Garamond

3. **Set Colors:**
   - Background: Pure white (#FFFFFF) or off-white (#FAFAFA for Executive)
   - Text: Black (#000000)
   - Accent: Use template-specific color from specs (headings only)

4. **Set Margins:**
   - All sides: 0.75" (19mm)
   - In Canva: File → Page setup → Custom margins

5. **Sections to Include:**
   - Contact Info (top, no header)
   - Professional Summary/About
   - Experience (reverse chronological)
   - Education
   - Skills
   - + Template-specific sections (see TEMPLATE-SPECS.md)

### Step 4: Create Both Sizes

**For EACH template, create TWO versions:**

1. **A4 Version:**
   - File → Page setup → Custom size
   - Width: 210mm, Height: 297mm
   - Save as: `CV_Template##_Name_A4`

2. **US Letter Version:**
   - File → Page setup → Custom size
   - Width: 8.5in, Height: 11in
   - Save as: `CV_Template##_Name_Letter`

### Step 5: Export Files

**For each template size:**

1. **PDF (Print-Ready):**
   - Share → Download → PDF Print
   - Name: `CV_Template01_ClassicProfessional_A4.pdf`

2. **DOCX (Editable):**
   - Share → Download → Microsoft Word (.docx)
   - Name: `CV_Template01_ClassicProfessional_A4.docx`

3. **Preview Image:**
   - Share → Download → PNG
   - Scale: 2x or 3x
   - Name: `Preview_Template01_A4.png`

### Step 6: Create Template Share Link (FOR ETSY DELIVERY)

**This is how customers get editable access:**

1. Open your completed template in Canva
2. Click **Share** (top right)
3. Click **"Template link"** (NOT "View link")
4. Copy the link
5. This link lets customers make their own copy

**Example Template Link Format:**
```
https://www.canva.com/design/DAFxxxxxx/edit?utm_content=DAFxxxxx
```

## 💡 CANVA PRO TIPS

### Free vs Pro:
- **Free works fine** for this project
- **Pro gives you:** More fonts, premium templates, background remover
- **Recommendation:** Start with free, upgrade if needed

### Keyboard Shortcuts:
- `Ctrl/Cmd + D`: Duplicate element
- `Ctrl/Cmd + G`: Group elements
- `Ctrl/Cmd + Shift + C`: Copy color
- `Ctrl/Cmd + Shift + V`: Paste color

### Brand Kit (Pro Feature):
- Save your 15 accent colors for quick access
- Saves time switching between templates

### Folders:
- Create folder: "CV Templates - Etsy Shop"
- Subfolders: "01-Classic", "02-Modern", etc.
- Keeps everything organized

## 🔧 TROUBLESHOOTING

### "Can't export as DOCX"
- Solution: File → Download → Microsoft Word (.docx)
- If grayed out: Some elements may not be compatible
- Fix: Remove any non-standard elements

### "Template link not working"
- Make sure you clicked "Template link" not "View link"
- Template link allows others to make a copy
- View link only shows preview

### "Colors look different in PDF"
- Canva sometimes shifts colors slightly
- Always check exported PDF before listing
- Use standard hex codes for consistency

### "Fonts changed in DOCX"
- This is normal - Word substitutes similar fonts
- In instructions, tell customers: "If font looks different, select text and change to Arial/Calibri"
