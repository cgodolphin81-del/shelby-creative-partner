## 🆘 TROUBLESHOOTING

### "Can't list digital items"
- Make sure you selected "Digital" as the item type
- Complete all required shop setup steps first
- Some countries have restrictions on digital sales

### "Files won't upload"
- Check file size (max 20MB per file)
- Compress PDFs if needed (use ilovepdf.com)
- Upload files one at a time

### "Template link not working for customers"
- Make sure you used "Template link" not "View link"
- Test link in incognito browser
- Contact Canva support if persistent issue

### "No sales after launch"
- This is normal! Etsy SEO takes time
- Share listings on social media
- Consider Etsy Ads ($1-2/day to start)
- Optimize listings based on Etsy search analytics
