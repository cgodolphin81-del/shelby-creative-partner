## DAY 1: GET STARTED

### Morning (2 hours)
- [ ] Read `EXECUTIVE_SUMMARY.md` (15 min)
- [ ] Register business name (LLC) — $100-300

- [ ] Open business bank account

### Afternoon (2 hours)
- [ ] Purchase domain: theunicornstable.com — ~$15/year
- [ ] Secure social handles: @theunicornstable (IG, TikTok, Pinterest, FB)
- [ ] Create Shopify account — $39/month
- [ ] Install Dawn theme (free)

### Evening (1 hour)
- [ ] Read `DESIGN_PROMPTS.md`
- [ ] Generate first 10 designs in Midjourney
- [ ] Save best results to folder

**Day 1 Budget Spent:** ~$500
**Day 1 Wins:** Business exists, domain secured, first designs created

## DAYS 2-3: BRAND FOUNDATION

### Logo & Branding
- [ ] Create logo in Canva (or use AI: "ornate unicorn hoof in high heel, baroque frame, gold and purple")
- [ ] Define color palette (use Brand Bible hex codes)
- [ ] Create brand assets folder (logo, colors, fonts)

### Shopify Setup
- [ ] Customize theme with brand colors
- [ ] Create navigation: Home, Shop, Collections, About, FAQ, Contact
- [ ] Set up payment gateways (Shopify Payments + PayPal)
- [ ] Configure shipping rates (free over $100)

### POD Integration
- [ ] Create Printful account — Free
- [ ] Connect Printful to Shopify
- [ ] Set up first 5 product templates

**Days 2-3 Budget Spent:** ~$100 (Canva Pro if needed)
**Days 2-3 Wins:** Brand visuals ready, store framework built

---

## DAYS 4-7: PRODUCT CREATION

### Design Generation
- [ ] Generate 50+ designs using `DESIGN_PROMPTS.md`
- [ ] Select best 20-25 for launch
- [ ] Refine in Photoshop/Canva
- [ ] Upload to Printful products

### Product Pages
- [ ] Write product descriptions (use Brand Bible voice)
- [ ] Create mockups for each product
- [ ] Set pricing (60% margin target)
- [ ] Organize into 7 collections

### Order Samples
- [ ] Order 1 sample from each collection (7-10 pairs)
- [ ] Budget: $500-800
- [ ] Timeline: 7-14 days delivery

**Days 4-7 Budget Spent:** ~$800 (samples)
**Days 4-7 Wins:** 20+ products ready, samples ordered

## COMMON PITFALLS TO AVOID

❌ **Not ordering samples** → Always test quality first
❌ **Underpricing** → Factor in ALL costs
❌ **Launching with <20 products** → Need critical mass
❌ **Ignoring email list** → Start building Day 1
❌ **Perfectionism** → Launch at 80%, improve after
❌ **Not testing checkout** → Buy from yourself multiple times
❌ **Giving up too soon** → First month is just the beginning
