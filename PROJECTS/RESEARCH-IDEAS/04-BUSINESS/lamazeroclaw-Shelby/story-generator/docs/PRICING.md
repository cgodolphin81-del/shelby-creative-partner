## Pricing Strategy

### Psychological Pricing

```
✓ Charm pricing: $14.99 instead of $15
✓ Anchoring: Show annual price first ($149.99), then monthly
✓ Decoy: Commercial tier makes Pro look reasonable
✓ Free tier: Removes friction for trial
```

### Discount Strategy

```
Annual Discount: 17% off (2 months free)
- Pro Annual: $149.99 (vs $179.88 monthly)
- Commercial Annual: $499.99 (vs $599.88 monthly)

Launch Promotion (First 1000 Pro users):
- 50% off first 3 months
- Lock in rate for life

Student Discount:
- 50% off Pro tier
- Requires .edu email

Non-Profit Discount:
- 50% off all tiers
- Verification required
```

### Price Increase Strategy

```
Grandfathering:
- Existing users keep current price
- New users get new price

Communication:
- 60 days notice
- Clear value justification
- Option to lock in current price with annual

Timing:
- After major feature releases
- Annual review (Q4)
- In line with market (not first mover)
```

## Upgrade/Downgrade Flow

### Upgrade Path

```
FREE → PRO
- Immediate access to Pro features
- Prorated charge for remainder of month
- Reset story limit immediately

FREE → COMMERCIAL
- Immediate access to Commercial features
- Prorated charge
- Onboarding call scheduled

PRO → COMMERCIAL
- Immediate access to Commercial features
- Prorated credit for unused Pro time
- API keys generated
- Team setup assistance
```

### Downgrade Path

```
COMMERCIAL → PRO
- Effective at end of billing cycle
- Retain Commercial features until cycle ends
- API access continues until cycle ends
- Team seats reduced to 1

PRO → FREE
- Effective at end of billing cycle
- Retain Pro features until cycle ends
- Stories over limit: Read-only access
- Export access continues
```

### Cancellation Flow

```
1. User requests cancellation
2. Offer retention discount (20% off 3 months)
3. If declined, confirm cancellation
4. Set cancellation date (end of cycle)
5. Send confirmation email
6. Export reminder 7 days before cancellation
7. Account becomes read-only after cancellation
8. Data retained for 90 days
9. Delete data after 90 days (with notice)
```
