# Export Options

## Overview
StoryForge AI supports multiple export formats for publishing stories across different platforms and media.

---

## Supported Formats

### 1. PDF Export

Professional print-ready documents with customizable layouts.

```typescript
interface PDFExportOptions {

    bottom: number;
    left: number;
    right: number;
  };
  
  // Typography
  fontFamily: string;
  fontSize: number;
  lineHeight: number;
  chapterHeadingFont: string;
  chapterHeadingSize: number;
  
  // Layout
  columns: 1 | 2;
  dropCap: boolean;
  pageNumbers: boolean;
  pageNumberPosition: 'header' | 'footer';
  
  // Content
  includeTitlePage: boolean;
  includeTableOfContents: boolean;
  includeAboutAuthor: boolean;
  includeCopyright: boolean;
  
  // Styling
  theme: 'classic' | 'modern' | 'minimal' | 'ornate';
  chapterDecorations: boolean;
  sceneBreaks: string; // '*', '***', custom
  
  // Advanced
  bleed: number; // For print
  cropMarks: boolean;
  embedFonts: boolean;
  compression: 'none' | 'fast' | 'best';
}
```

### 2. ePub Export

Industry-standard eBook format compatible with all major readers.

```typescript
interface ePubExportOptions {
  // Metadata
  title: string;
  author: string;
  language: string;
  publisher: string;
  isbn?: string;
  description?: string;
  categories: string[];
  
  // Version
  version: 'epub2' | 'epub3';
  
  // Features
  fixedLayout: boolean; // For illustrated books
  mediaOverlay: boolean; // Read-aloud
  accessibility: boolean; // WCAG compliance
  
  // Navigation
  toc: 'auto' | 'manual';
  landmarks: boolean;
  pageList: boolean;
  
  // Styling
  css: string; // Custom stylesheet
  theme: 'default' | 'dark' | 'sepia' | 'custom';
  
  // Images
  imageQuality: number; // 0.5 - 1.0
  maxImageWidth: number;
  coverImage: string; // Path to cover
  
  // DRM
  drm: boolean;
  drmProvider?: 'adobe' | 'apple' | 'amazon';
}
```

### 4. Audio Narration (Audiobook)

AI-generated narration with multiple voice options.

```typescript
interface AudioExportOptions {
  // Voice
  voice: string; // Voice ID from ElevenLabs, Amazon Polly, etc.
  voiceStyle: 'narrative' | 'dramatic' | 'conversational';
  speakingRate: number; // 0.5 - 2.0
  pitch: number; // -20 to 20
  
  // Characters
  characterVoices: {
    [characterName: string]: string; // Voice ID
  };
  differentiateNarrator: boolean;
  
  // Audio
  format: 'mp3' | 'wav' | 'flac' | 'm4b';
  bitrate: number; // kbps
  sampleRate: number; // Hz
  
  // Chapters
  chapterMarkers: boolean;
  chapterIntro: string; // Template for chapter intros
  chapterOutro: string;
  
  // Effects
  backgroundMusic: boolean;
  musicVolume: number; // 0 - 1
  soundEffects: boolean;
  pauseBetweenScenes: number; // seconds
  
  // Metadata
  includeISRC: boolean;
  acxCompliant: boolean; // For Audible submission
}
```

### Audio Quality Standards (ACX)

```
✓ Peak levels: -3dB to 0dB
✓ RMS average: -18dB to -23dB
✓ Noise floor: -60dB or lower
✓ Format: MP3 or WAV
✓ Sample rate: 44.1kHz
✓ Bit rate: 192kbps or higher
✓ Consistent volume across chapters
✓ No background noise
✓ No mouth clicks or plosives
```

### 5. Print-Ready (InDesign/Canva)

Files ready for professional printing.

```typescript
interface PrintExportOptions {
  // Trim Size
  size: '5x8' | '6x9' | '5.5x8.5' | 'A5' | 'custom';
  customSize?: { width: number; height: number };
  
  // Bleed
  bleed: number; // inches
  safeZone: number; // inches
  
  // Color
  colorMode: 'grayscale' | 'color';
  colorProfile: string; // 'FOGRA39', 'USWebCoatedSWOP', etc.
  
  // Binding
  binding: 'perfect' | 'saddle-stitch' | 'spiral' | 'hardcover';
  spineWidth: number; // Calculated from page count
  
  // Paper
  paperType: 'white' | 'cream' | 'recycled';
  paperWeight: number; // gsm
  
  // Output
  format: 'pdf-x1a' | 'pdf-x4' | 'indd' | 'canva';
  includeProof: boolean;
  
  // Print Run
  printer: 'kdp' | 'ingramspark' | 'lulu' | 'custom';
}
```

### Print Specifications by Platform

```
Amazon KDP (Kindle Direct Publishing)
─────────────────────────────────────
• Trim sizes: 5"×8" to 8.5"×11"
• Bleed: 0.125" on all sides
• Margins: 0.25" minimum (0.375" for >400 pages)
• Format: PDF (print-ready)
• Color: CMYK or grayscale
• Resolution: 300 DPI minimum

IngramSpark
───────────
• Trim sizes: 4.25"×6.87" to 8.5"×11"
• Bleed: 0.125" on outside edges
• Margins: 0.5" inner, 0.25" outer
• Format: PDF/X-1a
• Color: CMYK
• Resolution: 300 DPI

Lulu
────
• Trim sizes: Various
• Bleed: 0.125"
• Margins: 0.5" minimum
• Format: PDF
• Color: CMYK or RGB
• Resolution: 300 DPI
```

### 6. Serialized Format

For publishing chapter-by-chapter on platforms like Royal Road, Wattpad, etc.

```typescript
interface SerializedExportOptions {
  // Platform
  platform: 'royalroad' | 'wattpad' | 'scribblehub' | 'webnovel' | 'custom';
  
  // Scheduling
  schedule: {
    frequency: 'daily' | 'weekly' | 'biweekly';
    dayOfWeek?: number; // 0-6
    time?: string; // HH:MM
    timezone: string;
  };
  
  // Formatting
  chapterLength: number; // Target words per chapter
  cliffhangers: boolean;
  authorNotes: boolean;
  authorNotesPosition: 'top' | 'bottom';
  
  // Engagement
  endOfChapterCTA: string;
  patreonLink?: string;
  discordLink?: string;
  
  // SEO
  tags: string[];
  contentWarnings: string[];
  rating: 'G' | 'PG' | 'PG-13' | 'R' | 'NC-17';
  
  // Monetization
  earlyAccessChapters: number; // For Patreon
  paidChapters: boolean;
}
```

## Export API

### Create Export Job

```typescript
POST /api/v1/exports

Request:
{
  "storyId": "uuid",
  "format": "pdf" | "epub" | "web" | "audio" | "print" | "screenplay",
  "options": { /* Format-specific options */ },
  "priority": "normal" | "high",
  "notifyOnComplete": true,
  "email": "user@example.com"
}

Response:
{
  "exportId": "uuid",
  "status": "queued" | "processing" | "completed" | "failed",
  "estimatedTime": 120, // seconds
  "progressUrl": "/api/v1/exports/:exportId"
}
```

### Check Export Status

```typescript
GET /api/v1/exports/:exportId

Response:
{
  "exportId": "uuid",
  "storyId": "uuid",
  "format": "pdf",
  "status": "completed",
  "progress": 100,
  "createdAt": "2024-01-15T10:30:00Z",
  "completedAt": "2024-01-15T10:32:15Z",
  "downloadUrl": "https://cdn.storyforge.ai/exports/abc123.pdf",
  "expiresAt": "2024-01-22T10:32:15Z",
  "fileSize": 2456789, // bytes
  "metadata": {
    "pages": 245,
    "words": 85000,
    "chapters": 24
  }
}
```

### Download Export

```typescript
GET /api/v1/exports/:exportId/download

Headers:
Content-Type: application/pdf
Content-Disposition: attachment; filename="story-title.pdf"
```

| Format    | Best For                    | File Size | Compatibility     |

| Audio     | Audiobooks, podcasts        | Large     | Audio players     |
| Print     | Professional printing       | Large     | Printers          |
