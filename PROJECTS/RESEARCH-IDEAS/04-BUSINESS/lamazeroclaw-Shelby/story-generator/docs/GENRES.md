# Genre Library - 20+ Genres

## Overview
StoryForge AI includes 20+ curated genre templates, each with specific tropes, conventions, character archetypes, and plot structures.

---

## 1. FANTASY

### Subgenres
- High Fantasy
- Urban Fantasy
- Dark Fantasy
- Epic Fantasy
- Sword & Sorcery

### Core Tropes
- Chosen one / prophecy
- Magic systems (hard/soft)
- Mythical creatures
- Ancient evils
- Quest narratives
- World-building depth

### Character Archetypes
- The Hero (reluctant or destined)
- The Mentor (wise guide)
- The Dark Lord
- The Companion (loyal friend)
- The Trickster

### Typical Structure
- Act 1: Ordinary world → Call to adventure
- Act 2: Training/allies → Trials → Midpoint revelation
- Act 3: Final confrontation → Resolution

### Word Count Range
- Short story: 3,000-7,500
- Novella: 17,500-40,000
- Novel: 80,000-150,000+

### Example Prompts
```
"Generate a high fantasy opening where a farm boy discovers 
he's the last dragon rider"

"Create an urban fantasy detective who can see ghosts"
```

## Genre Selection API

```typescript
interface Genre {
  id: string;
  name: string;
  subgenres: string[];
  tropes: string[];
  archetypes: string[];
  wordCountRange: {
    min: number;
    max: number;
  };
  structure: {
    acts: number;
    beats: string[];
  };
  prompts: string[];
}

// GET /api/v1/genres
// Returns all available genres

// GET /api/v1/genres/:id
// Returns detailed genre information

// POST /api/v1/genres/:id/generate
// Body: { subgenre?: string, tropes?: string[], customBeats?: string[] }
// Returns: Generated story outline
```
