# Quality Control

Editing processes, brand voice guidelines, fact-checking, and approval workflows to ensure consistent, high-quality output.

---

## Overview

Quality is what separates premium agencies from content mills. Our QC system ensures every piece meets our standards before it reaches the client.

**QC Principles:**
1. **Systematic** - Every piece goes through the same process
2. **Documented** - Standards are written, not assumed
3. **Measured** - Quality is tracked and improved
4. **Owned** - Someone is accountable for every piece

## Stage 1: Self-Edit (Writer)

**Goal**: Catch obvious issues before human review.

### Writer Self-Edit Checklist

**Content:**
- [ ] Follows the brief completely
- [ ] All required sections included
- [ ] Word count within ±10% of target
- [ ] Claims are supported with data/sources
- [ ] Examples are relevant and clear
- [ ] CTA is included and appropriate

**Writing Quality:**
- [ ] No spelling errors (ran spell check)
- [ ] Grammar is correct
- [ ] Sentences are clear and concise
- [ ] Paragraphs are 2-4 sentences
- [ ] Active voice used 80%+ of time
- [ ] Reading level is appropriate (Grade 8-10)

**SEO:**
- [ ] Primary keyword in title
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in 1-2 H2s
- [ ] Secondary keywords included naturally
- [ ] Internal links added (3-5)
- [ ] External links to authoritative sources (2-5)

**Formatting:**
- [ ] Headers are hierarchical (H1 → H2 → H3)
- [ ] Lists are used where appropriate
- [ ] Images have alt text
- [ ] Links are formatted correctly

**Before Submission:**
- [ ] Read aloud (catches awkward phrasing)
- [ ] Reviewed on mobile (if applicable)
- [ ] All placeholder text removed
- [ ] File named correctly

### Self-Edit Time Allocation

| Content Type | Self-Edit Time |
|--------------|----------------|
| Blog Post (1,000 words) | 30 minutes |
| Blog Post (2,000 words) | 45 minutes |
| Whitepaper | 2-3 hours |
| Case Study | 1 hour |
| Ebook Chapter | 1-2 hours |

## Stage 2: Developmental Edit

**Goal**: Ensure structure, argument, and strategic alignment.

### Developmental Edit Checklist

**Strategic Alignment:**
- [ ] Content supports the stated objective
- [ ] Appropriate for target funnel stage
- [ ] Aligned to content pillar
- [ ] Matches target persona's knowledge level

**Structure:**
- [ ] Introduction hooks effectively
- [ ] Logical flow between sections
- [ ] Each section supports main point
- [ ] No redundant sections
- [ ] Conclusion summarizes key points
- [ ] CTA is clear and appropriate

**Argument Strength:**
- [ ] Main thesis is clear
- [ ] Claims are supported
- [ ] Counterarguments addressed (if relevant)
- [ ] Examples illustrate points
- [ ] Data is relevant and recent

**Completeness:**
- [ ] All brief requirements met
- [ ] No gaps in logic or information
- [ ] Reader questions anticipated and answered
- [ ] Nothing important missing

### Developmental Edit Feedback Format

**Provide feedback in this structure:**

```
## Overall Assessment
[2-3 sentences on overall quality and main strengths]

## Major Revisions Required
1. [Specific issue + location + suggested fix]
2. [Specific issue + location + suggested fix]

## Minor Suggestions
1. [Nice-to-have improvements]
2. [Optional enhancements]

## Questions
1. [Clarifications needed from writer]
```

**Good Feedback Example:**
> "The introduction doesn't hook the reader. Currently it starts with a definition, which is dry. Suggest opening with the stat about 73% of companies struggling with content - that's more compelling. Also, the section on 'Tools' feels disconnected from the main argument about strategy. Either tie it back or move to an appendix."

**Bad Feedback Example:**
> "Intro needs work. Section 3 is confusing."

|--------------|------|
| Blog Post | 30-45 minutes |
| Whitepaper | 2-3 hours |
| Case Study | 1 hour |
| Ebook | 3-4 hours |
| Video Script | 45 minutes |

## Stage 3: Line Edit

**Goal**: Improve clarity, flow, and engagement.

### Line Edit Checklist

**Clarity:**
- [ ] Every sentence is easy to understand
- [ ] Jargon is explained or removed
- [ ] Acronyms are defined on first use
- [ ] No ambiguous statements

**Flow:**
- [ ] Transitions between paragraphs are smooth
- [ ] Transitions between sections are clear
- [ ] Ideas build logically
- [ ] No jarring topic shifts

**Engagement:**
- [ ] Opening paragraph hooks the reader
- [ ] Content maintains interest throughout
- [ ] Examples are compelling
- [ ] Voice is consistent and appropriate
- [ ] Tone matches brand guidelines

**Conciseness:**
- [ ] No wordy phrases (see below)
- [ ] Redundant words removed
- [ ] Filler words eliminated
- [ ] Every word earns its place

### Common Wordy Phrases to Fix

| Instead Of | Use |
|------------|-----|
| In order to | To |
| Due to the fact that | Because |
| At this point in time | Now |
| In the near future | Soon |
| For the purpose of | To |
| In the event that | If |
| With regard to | About |
| In close proximity to | Near |
| A large number of | Many |
| In the process of | Currently |

## Stage 4: Copy Edit

**Goal**: Eliminate errors and ensure consistency.

### Copy Edit Checklist

**Grammar & Mechanics:**
- [ ] No spelling errors
- [ ] Grammar is correct
- [ ] Punctuation is correct
- [ ] Subject-verb agreement
- [ ] Pronoun usage is correct
- [ ] Tense is consistent

**Style Consistency:**
- [ ] Following style guide (AP, Chicago, or house)
- [ ] Consistent capitalization
- [ ] Consistent number formatting (digits vs. words)
- [ ] Consistent date formatting
- [ ] Consistent heading format

**Fact-Checking:**
- [ ] All statistics are accurate
- [ ] All quotes are accurate and attributed
- [ ] All links work
- [ ] Company names are correct
- [ ] Product names are correct
- [ ] Dates are accurate

**Formatting:**
- [ ] Consistent font usage
- [ ] Consistent spacing
- [ ] Lists are formatted consistently
- [ ] Images are properly sized
- [ ] Tables are formatted correctly

### Fact-Checking Process

**For Every Claim:**

1. **Identify the claim**: "73% of companies struggle with content marketing"
2. **Find the source**: Link or citation provided?
3. **Verify the source**: 
   - Is it authoritative?
   - Is it recent (within 2 years)?
   - Does it actually say what we claim?
4. **Document the source**: Keep a source list
5. **Flag if unverifiable**: Mark for client review

**Sources to Trust:**
- Government websites (.gov)
- Educational institutions (.edu)
- Established industry publications
- Peer-reviewed research
- Reputable news organizations

**Sources to Question:**
- Unfounded blog posts
- Outdated sources (>3 years for fast-moving industries)
- Sources with clear bias
- Circular citations (A cites B, B cites A)

## Stage 5: SEO Review

**Goal**: Ensure content is optimized for search without sacrificing quality.

### SEO Review Checklist

**On-Page Elements:**
- [ ] Title tag: 50-60 characters, keyword front-loaded
- [ ] Meta description: 140-155 characters, includes keyword and CTA
- [ ] URL slug: Clean, includes primary keyword
- [ ] H1: Includes primary keyword, compelling
- [ ] H2s: Include secondary keywords where natural

**Keyword Optimization:**
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in 1-2 H2s
- [ ] Primary keyword density: 1-2% (natural)
- [ ] Secondary keywords included (2-3)
- [ ] LSI/related keywords included
- [ ] No keyword stuffing

**Content Quality:**
- [ ] Content is comprehensive (covers topic fully)
- [ ] Content is original (not thin or duplicate)
- [ ] Readability score is appropriate
- [ ] Internal links included (3-5)
- [ ] External links to authoritative sources (2-5)

**Technical:**
- [ ] Images have descriptive alt text
- [ ] Schema markup added (if applicable)
- [ ] Mobile-friendly formatting
- [ ] No broken links

## Stage 6: Client Review

**Goal**: Get client approval while managing revisions efficiently.

### Client Review Guidelines

**What to Send:**
- Google Doc with comment access (allows easy feedback)
- Clear deadline for feedback (3-5 business days)
- Specific questions if you need input
- Summary of what was delivered

**Email Template:**
```
Subject: [Content Title] - Ready for Review

Hi [Client Name],

Great news - [content title] is ready for your review!

📄 Review Link: [Google Doc URL]
📅 Feedback Due: [Date - 3-5 business days]

What's Included:
- [Brief description of deliverable]
- [Word count / duration]
- [Key highlights]

Review Process:
1. Add comments directly in the document
2. Use "Suggesting" mode for edit suggestions
3. We include 2 rounds of revisions

Questions? Just reply to this email.

Looking forward to your feedback!

Best,
[Your name]
```

### Managing Revisions

**Revision Rounds:**
- **Round 1**: Client provides all feedback
- **Round 2**: Writer addresses feedback, client approves
- **Round 3+**: Only for critical issues (may incur additional fees)

**Revision Guidelines for Clients:**
```
To keep things efficient, please:
✓ Provide all feedback in one round
✓ Be specific about changes needed
✓ Use comments/suggestions in the doc
✓ Respond within 3-5 business days

Please avoid:
✗ Sending feedback piecemeal
✗ Vague requests ("make it pop")
✗ New requirements not in the brief
✗ Extending timeline without notice
```

### Common Revision Scenarios

| Scenario | Handling |
|----------|----------|
| Minor tweaks (typos, small edits) | Quick turnaround, no pushback |
| Tone adjustments | Clarify brand voice, make changes |
| Structural changes | Assess if brief was followed; charge if scope changed |
| New requirements | "Happy to add this. This is outside original scope - estimate is X. Proceed?" |
| Endless revisions | "We've completed 3 rounds. Let's schedule a call to align on final changes." |

### Revision Tracking

| Content | Round 1 | Round 2 | Round 3 | Status |
|---------|---------|---------|---------|--------|
| Blog Post 1 | ✅ | ✅ | - | Approved |
| Blog Post 2 | ✅ | 🔄 | - | In Review |
| Whitepaper | ✅ | ✅ | 🔄 | Final Review |

## Brand Voice Guidelines

**Goal**: Ensure consistent voice across all content.

### Voice Dimensions

Define your brand voice on these spectrums:

| Dimension | End 1 | End 2 | Your Brand |
|-----------|-------|-------|------------|
| Formality | Casual | Formal | [Position] |
| Enthusiasm | Reserved | Enthusiastic | [Position] |
| Respect | Irreverent | Respectful | [Position] |
| Practicality | Emotional | Practical | [Position] |

### Voice Attributes

**Example Brand Voice:**

```
OUR VOICE IS:

✓ Professional but approachable
  - We know our stuff, but we don't show off
  - We use plain language, not jargon

✓ Direct and clear
  - We say what we mean
  - We don't use filler or fluff

✓ Helpful and actionable
  - We focus on what readers can DO
  - We provide examples and specifics

✓ Confident but not arrogant
  - We stand behind our recommendations
  - We acknowledge limitations and nuance

OUR VOICE IS NOT:

✗ Academic or stuffy
✗ Overly casual or slang-heavy
✗ Vague or wishy-washy
✗ Salesy or promotional
```

### Voice Examples

**Good (Matches Voice):**
> "Most content strategies fail because they're built on assumptions, not data. Here's how to fix that in three steps."

**Bad (Doesn't Match Voice):**
> "In today's digital landscape, it is imperative that organizations leverage data-driven methodologies to optimize their content strategic frameworks."

### Voice Training

**For New Writers:**
1. Provide voice guidelines document
2. Share 3-5 example pieces that exemplify voice
3. Share 2-3 pieces that miss the mark (and why)
4. First 3 pieces get extra voice feedback
5. Regular voice check-ins

**Voice Audit (Quarterly):**
- Review 10 random pieces
- Score each on voice consistency (1-5)
- Identify drift patterns
- Retrain team if needed

## Quality Metrics

**Goal**: Measure and improve quality over time.

### Metrics to Track

| Metric | Target | How to Measure |
|--------|--------|----------------|
| First-pass approval rate | 80%+ | % of pieces approved in Round 1 |
| Revision rounds per piece | <2.0 | Average revisions before approval |
| Error rate | <1% | Errors found post-publish |
| Client satisfaction | 4.5/5+ | Monthly CSAT survey |
| On-time delivery | 95%+ | % delivered by deadline |
| Internal QC pass rate | 90%+ | % passing internal QC first time |
