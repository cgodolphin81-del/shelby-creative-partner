## 1. Blog Posts

**Primary Use**: SEO traffic, thought leadership, funnel nurturing

### Specifications

| Attribute | Standard | Premium |
|-----------|----------|---------|
| Word Count | 800-1,200 | 1,500-2,500 |
| Research Level | Secondary sources | Original data, expert interviews |
| Images | 1-2 stock | 3-5 custom graphics |
| SEO Optimization | Basic keywords | Full optimization + schema |
| Internal Links | 2-3 | 5-8 |
| External Sources | 2-3 | 5-10 authoritative |

### Blog Post Structure

```
1. HEADLINE (5+ options provided)
   - Include primary keyword
   - Promise clear value
   - Create curiosity or urgency

2. INTRODUCTION (100-150 words)
   - Hook: stat, question, or bold statement
   - Context: why this matters
   - Promise: what reader will learn

3. BODY (organized with H2/H3 headers)
   - Logical flow
   - Scannable sections
   - Examples and data points
   - Visual breaks every 300 words

4. KEY TAKEAWAYS
   - 3-5 bullet points
   - Actionable summary

5. CALL-TO-ACTION
   - Aligned to content stage
   - Clear next step
   - Low friction

6. META DATA
   - Title tag (60 chars)
   - Meta description (155 chars)
   - URL slug
```

### Blog Categories We Write

| Category | Description | Example Topics |
|----------|-------------|----------------|
| How-To Guides | Step-by-step tutorials | "How to Build a Content Calendar in 7 Steps" |
| List Posts | Curated lists | "15 Tools Every Content Marketer Needs" |
| Thought Leadership | Industry insights | "Why Most Content Strategies Fail in Year 2" |
| Case Studies | Client success stories | "How X Company 3X'd Traffic in 90 Days" |
| Trend Analysis | Industry developments | "The State of AI in Content Marketing 2026" |
| Comparison Posts | Tool/service comparisons | "HubSpot vs. Marketo: Which is Right for You?" |
| Opinion/Commentary | Bold takes | "Stop Chasing Viral Content. Do This Instead." |

### SEO Requirements

- Primary keyword in: title, H1, first 100 words, one H2, meta
- Secondary keywords: 2-3 naturally integrated
- Readability: Grade 8-10 (Hemingway)
- Internal links: 3-5 to relevant content
- External links: 2-5 authoritative sources
- Image alt text: descriptive, keyword-appropriate

## 2. Whitepapers

**Primary Use**: Lead generation, authority building, complex topic exploration

### Specifications

| Attribute | Standard |
|-----------|----------|
| Length | 8-15 pages (2,500-5,000 words) |
| Format | PDF, gated landing page |
| Design | Professional, branded |
| Research | Original + secondary |
| Timeline | 3-4 weeks |

### Whitepaper Structure

```
1. EXECUTIVE SUMMARY (1 page)
   - Problem statement
   - Key findings
   - Recommendations

2. INTRODUCTION (1-2 pages)
   - Context and background
   - Why this matters now
   - Methodology (if original research)

3. PROBLEM/CHALLENGE (2-3 pages)
   - Current state
   - Pain points
   - Cost of inaction

4. SOLUTION FRAMEWORK (3-5 pages)
   - Proposed approach
   - Components explained
   - Implementation considerations

5. CASE EXAMPLES (1-2 pages)
   - Real-world applications
   - Results achieved

6. RECOMMENDATIONS (1-2 pages)
   - Actionable next steps
   - Prioritized roadmap

7. CONCLUSION (0.5 page)
   - Summary
   - CTA

8. APPENDIX (optional)
   - Additional data
   - Methodology details
   - Resources
```

### Production Process

1. **Topic Selection** - High-value, complex topic worthy of deep dive
2. **Research** - 2-3 weeks for data collection
3. **Outline Approval** - Client signs off on structure
4. **First Draft** - 1 week
5. **Design** - 1 week (parallel with revisions)
6. **Final Review** - 3-5 days
7. **Landing Page** - Optional, for gating

### Pricing: $8,000 - $20,000

## 3. Case Studies

**Primary Use**: Sales enablement, social proof, conversion optimization

### Specifications

| Attribute | Standard |
|-----------|----------|
| Length | 1,500-2,500 words |
| Format | PDF + web version |
| Design | Branded template |
| Timeline | 2-3 weeks |
| Includes | Client interview, data verification |

### Case Study Structure (STAR Method)

```
1. HEADLINE
   - Client name + result
   - Example: "How Acme Corp Increased Leads by 247% in 6 Months"

2. CLIENT OVERVIEW (100-150 words)
   - Company name, industry, size
   - Context about their business

3. THE CHALLENGE (300-400 words)
   - What problem were they facing?
   - Why was it urgent?
   - What had they tried before?

4. THE SOLUTION (400-600 words)
   - What did we do?
   - Specific strategies implemented
   - Timeline of implementation

5. THE RESULTS (300-400 words)
   - Quantifiable outcomes
   - Before/after metrics
   - Client quote

6. KEY TAKEAWAYS (100-150 words)
   - 3-5 lessons for readers
   - Applicable insights

7. ABOUT THE CLIENT (50 words)
   - Brief company description
   - Website link

8. CTA
   - "Want similar results?"
   - Contact information
```

### Required Assets

- [ ] Client interview (30-45 min)
- [ ] Before/after metrics (verified)
- [ ] Client quote (written or recorded)
- [ ] Logo usage permission
- [ ] Screenshots/visuals (if applicable)

### Pricing: $3,000 - $7,000 per case study

| Word Count | 8,000-15,000 | 15,000-25,000 |

### Ebook Structure

```
1. COVER
   - Compelling title
   - Subtitle with benefit
   - Professional design

2. TITLE PAGE

3. TABLE OF CONTENTS

4. INTRODUCTION (2-3 pages)
   - Hook
   - What's inside
   - How to use this guide

5. CHAPTERS (5-8 chapters, 4-8 pages each)
   - Clear progression
   - Actionable content
   - Examples and visuals
   - Chapter summaries

6. CONCLUSION (2-3 pages)
   - Key takeaways
   - Implementation roadmap
   - CTA

7. ABOUT THE AUTHOR/COMPANY (1-2 pages)

8. RESOURCES/APPENDIX (optional)
   - Templates
   - Checklists
   - Further reading
```

### Production Timeline

| Phase | Duration |
|-------|----------|
| Topic & Outline | 1 week |
| Research | 1-2 weeks |
| Writing | 2-3 weeks |
| Design | 1-2 weeks |
| Revisions | 1 week |
| **Total** | **6-9 weeks** |

### Pricing: $10,000 - $30,000

- Script writing
- Professional recording (remote or on-location)
- Editing (color, audio, cuts)
- Basic motion graphics
- Captions/subtitles
- Thumbnail design
- 2 revision rounds

```

BODY (varies)
- Main points (3-5 max)
- Visual support for each
- Transitions between points

CTA (last 10-15 sec)
- Clear next step
- URL or action
```

**Primary Use**: Authority building, audience engagement, repurposing hub

| Format | Length | Frequency | Best For |
|--------|--------|-----------|----------|
| Solo Commentary | 10-20 min | Weekly | Thought leadership |
| Interview | 30-60 min | Weekly/Bi-weekly | Network building |

### Episode Structure

```
COLD OPEN (0-30 sec)
- Hook from later in episode
- Create curiosity

INTRO (30-60 sec)
- Show branding
- Episode topic teaser
- Host introduction

MAIN CONTENT
- Interview, monologue, or discussion
```

## Content Quality Standards

All content types must meet:

- [ ] **Accuracy**: Facts verified, sources cited
- [ ] **Clarity**: Grade 8-10 readability
- [ ] **Originality**: No plagiarism, unique insights
- [ ] **Brand Alignment**: Voice and tone match
- [ ] **SEO**: Keywords naturally integrated
- [ ] **Accessibility**: Alt text, captions, readable formatting
- [ ] **CTA**: Clear next step included
