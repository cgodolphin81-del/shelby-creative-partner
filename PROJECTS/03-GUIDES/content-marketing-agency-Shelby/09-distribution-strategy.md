# Distribution Strategy

Comprehensive plan for amplifying content reach through SEO, social, email, paid promotion, and syndication.

---

## Overview

**The Reality:** Creating great content is only 20% of the battle. Distribution is the other 80%.

**Our Distribution Philosophy:**
1. **Owned First** - Maximize channels you control
2. **Earned Second** - Build organic reach and authority
3. **Paid Last** - Amplify what's already working

**Distribution Timeline:**
```
PUBLISH DAY → DAY 1-3 → DAY 7 → DAY 14 → DAY 30 → ONGOING
     ↓           ↓         ↓        ↓        ↓        ↓
  Launch     Social    Refresh  Email   Paid    Evergreen
             Push               Follow-up
```

## Channel 1: SEO Distribution

**Goal:** Maximize organic search visibility and traffic.

### On-Page Optimization

**Per-Page Checklist:**

| Element | Best Practice |
|---------|---------------|
| Title Tag | 50-60 chars, keyword front-loaded, compelling |
| Meta Description | 140-155 chars, keyword, CTA |
| URL Slug | Short, clean, includes keyword |
| H1 | One per page, includes keyword |
| H2-H6 | Hierarchical, includes related keywords |
| First 100 Words | Include primary keyword naturally |
| Image Alt Text | Descriptive, includes keyword when relevant |
| Internal Links | 3-5 links to related content |
| External Links | 2-5 links to authoritative sources |
| Content Length | Comprehensive (1,500+ words for competitive topics) |
| Readability | Grade 8-10 level |

### Internal Linking Strategy

**Why It Matters:**
- Distributes page authority
- Helps search engines discover content
- Keeps visitors on your site longer
- Supports keyword rankings

**Internal Linking Process:**

1. **Identify Pillar Pages** - 5-10 comprehensive guides
2. **Map Cluster Content** - Related blog posts to each pillar
3. **Link Strategically**:
   - Cluster posts → Pillar page
   - Pillar page → Relevant cluster posts
   - Related cluster posts → Each other

**Internal Link Audit (Monthly):**
- [ ] Identify orphan pages (no internal links)
- [ ] Find linking opportunities (related content)
- [ ] Update old posts with links to new content
- [ ] Fix broken internal links

### Technical SEO

**Site Health Checklist:**

| Factor | Target | Tool |
|--------|--------|------|
| Page Speed | <3 seconds | PageSpeed Insights |
| Mobile-Friendly | 100% score | Mobile-Friendly Test |
| Core Web Vitals | All green | Search Console |
| Crawl Errors | 0 | Search Console |
| XML Sitemap | Submitted | Search Console |
| Robots.txt | Configured | Manual check |
| HTTPS | 100% of pages | Manual check |
| Schema Markup | Implemented | Schema Validator |

### Content Optimization Workflow

**For New Content:**
1. Keyword research (before writing)
2. On-page optimization (during writing)
3. SEO review (before publishing)
4. Index request (after publishing)
5. Performance monitoring (ongoing)

**For Existing Content:**
1. Identify underperforming pages
2. Update with fresh information
3. Improve on-page SEO
4. Add internal links
5. Resubmit for indexing

## Channel 2: Social Media Distribution

**Goal:** Drive engagement, traffic, and brand awareness through social platforms.

### Platform Strategy

| Platform | Best For | Content Types | Frequency |
|----------|----------|---------------|-----------|
| LinkedIn | B2B, professional services | Articles, carousels, video, thought leadership | 3-5x/week |
| Twitter/X | Tech, media, real-time | Threads, hot takes, engagement | 1-3x/day |
| Instagram | B2C, visual brands | Reels, carousels, stories | 3-5x/week |
| TikTok | Gen Z, consumer brands | Short-form video | 3-5x/week |
| YouTube | Education, entertainment | Long-form video, Shorts | 1-2x/week |
| Facebook | Local business, 35+ demo | Video, community, events | 3-5x/week |

### Content Distribution Per Piece

**For Every Blog Post:**

| Platform | Content | Timing |
|----------|---------|--------|
| LinkedIn | Post with key insight + link | Day 0, 3, 7 |
| Twitter | Thread summarizing key points | Day 0 |
| Twitter | Single tweet with quote | Day 2, 5 |
| Instagram | Carousel with key takeaways | Day 1 |
| Facebook | Post with summary + link | Day 0, 4 |
| Email | Newsletter feature | Next send |

### LinkedIn Strategy

**Profile Optimization:**
- Headline: Clear value prop + keywords
- About: Story + credibility + CTA
- Featured: Best content, case studies
- Activity: Consistent posting and commenting

**Post Types That Work:**
- **Text-only posts** - Personal insights, stories
- **Carousels** - Step-by-step guides, lists
- **Video** - Talking head, screen shares
- **Polls** - Engagement, market research
- **Articles** - Long-form thought leadership

**Posting Best Practices:**
- Post when audience is active (8-10am, 12-1pm, 5-6pm)
- Engage with comments within 2 hours
- Tag relevant people/companies (sparingly)
- Use 3-5 relevant hashtags
- Include clear CTA

**LinkedIn Content Formula:**
```
HOOK (1-2 lines)
- Surprising stat
- Bold statement
- Relatable problem

BODY (3-10 paragraphs)
- Story or insight
- Supporting points
- Examples

TAKEAWAY (1-2 lines)
- Key lesson
- Actionable advice

CTA (1 line)
- Question for engagement
- Link to learn more
```

### Twitter/X Strategy

**Thread Formula:**
```
Tweet 1: Hook + promise
- "Here's how to [achieve result] in [timeframe]:"
- Or: "[Number] lessons from [experience]:"

Tweets 2-8: Main content
- One point per tweet
- Use numbers, bullets, bold text
- Include examples

Final Tweet: Summary + CTA
- Key takeaways
- Link to full content
- Ask for retweet/follow
```

**Engagement Strategy:**
- Reply to 10-20 relevant tweets daily
- Quote tweet with added insight
- Participate in relevant conversations
- Build relationships with industry voices

## Channel 3: Email Distribution

**Goal:** Drive engaged traffic and nurture leads through email.

### Email Types for Content Distribution

| Type | Purpose | Frequency |
|------|---------|-----------|
| Newsletter | Regular content roundup | Weekly/Bi-weekly |
| Dedicated Send | Single piece of content | As needed |
| Drip Campaign | Nurture sequence | Automated |
| Segmented Send | Targeted to specific audience | As needed |

### Newsletter Structure

```
SUBJECT LINE
- Clear and compelling
- 40-50 characters
- Creates curiosity or promises value

PREHEADER
- Supports subject line
- 80-100 characters
- Additional context

OPENING
- Personal greeting
- Brief intro/hook
- What's inside

MAIN CONTENT
- Featured article (with image)
- 2-3 secondary articles
- Clear CTAs for each

CLOSING
- Personal sign-off
- Social links
- Unsubscribe option
```

### Email Best Practices

**Subject Lines That Work:**
- "How [Company] 3X'd traffic in 90 days"
- "The content mistake 80% of companies make"
- "5 lessons from [recent event]"
- "[Name], your weekly content digest"

**Content Selection:**
- Lead with your best piece
- Include variety (blog, video, resource)
- Keep descriptions concise (1-2 sentences)
- Use clear, action-oriented CTAs

**Timing:**
- B2B: Tuesday-Thursday, 10am-2pm
- B2C: Varies by audience
- Test and optimize for your list

**Segmentation:**
- By industry
- By funnel stage
- By engagement level
- By content preferences

## Channel 4: Paid Promotion

**Goal:** Amplify top-performing content to targeted audiences.

### When to Use Paid

**Good Candidates for Paid:**
- High-performing organic content
- Gated assets (whitepapers, ebooks)
- Webinar/event promotion
- Product launches
- Retargeting website visitors

**Not Good Candidates:**
- Unproven content (test organically first)
- Thin or promotional content
- Content without clear CTA

### Platform Selection

| Platform | Best For | Avg. CPC |
|----------|----------|----------|
| LinkedIn Ads | B2B, professional services | $5-12 |
| Google Ads | Intent-based search | $2-8 |
| Facebook/Instagram | B2C, broad targeting | $1-4 |
| Twitter Ads | Tech, media, real-time | $1-3 |
| Native (Taboola/Outbrain) | Content discovery | $0.50-2 |
| YouTube Ads | Video content | $0.10-0.30/view |

### Paid Promotion Framework

**Phase 1: Test (Week 1-2)**
- Budget: $500-1,000
- 3-5 ad variations
- Broad targeting
- Goal: Identify winners

**Phase 2: Optimize (Week 3-4)**
- Budget: $1,000-3,000
- Scale winning ads
- Refine targeting
- Goal: Improve CPA

**Phase 3: Scale (Month 2+)**
- Budget: $3,000-10,000+
- Multiple ad sets
- Lookalike audiences
- Goal: Maximize ROI

### Ad Creative Best Practices

**LinkedIn:**
- Use images (1200x627px)
- Clear headline
- Benefit-focused copy
- Professional tone

**Facebook/Instagram:**
- Eye-catching visuals
- Short, punchy copy
- Clear CTA
- Mobile-optimized

**Google Search:**
- Match keyword intent
- Include keyword in ad
- Strong value prop
- Relevant landing page

### Retargeting Strategy

**Audiences to Target:**
- Website visitors (last 30 days)
- Blog readers (specific pages)
- Content downloaders
- Email subscribers
- Video viewers (50%+)

**Messaging:**
- Acknowledge previous engagement
- Offer next logical step
- Address common objections
- Include social proof

## Channel 5: Syndication

**Goal:** Extend reach through third-party publications.

### Syndication Options

| Type | Examples | Process |
|------|----------|---------|
| Guest Posts | Industry blogs, publications | Pitch, write, publish |
| Content Networks | Medium, LinkedIn Articles | Republish with canonical |
| Industry Publications | Trade magazines, news sites | Pitch to editors |
| Content Aggregators | Hacker News, Reddit, Quora | Share strategically |

### Guest Posting Strategy

**Target Identification:**
- Domain Authority 40+
- Relevant audience
- Accepts guest posts
- Allows author bio with link

**Pitch Template:**
```
Subject: Guest Post Pitch: [Topic]

Hi [Editor Name],

I'm a fan of [Publication] - especially [specific article].

I'd like to contribute a guest post on [topic]. Here are 
three headline options:

1. [Headline 1]
2. [Headline 2]
3. [Headline 3]

The article would cover:
- Key point 1
- Key point 2
- Key point 3

I'm [credentials] and have written for [publications].

Sample of my work: [Link]

Would this be a fit for [Publication]?

Best,
[Your name]
```

**Content Guidelines:**
- Original content (not published elsewhere)
- 1,500-2,500 words
- Actionable and valuable
- Includes examples and data
- Follows publication style

### Medium/LinkedIn Articles

**Republishing Strategy:**
1. Publish on your blog first
2. Wait 7+ days
3. Republish on Medium/LinkedIn
4. Add canonical link to original
5. Modify intro/outro for platform

**Platform-Specific Optimization:**
- Medium: Use publications for reach
- LinkedIn: Tag relevant people/companies
- Both: Engage with comments promptly

### Content Aggregators

**Where to Share:**

| Platform | Best For | Guidelines |
|----------|----------|------------|
| Reddit | Niche communities | Follow subreddit rules, no spam |
| Hacker News | Tech, startups | High-quality, substantive |
| Quora | Answer questions | Genuine helpfulness |
| Indie Hackers | Startup community | Transparent, helpful |

**Best Practices:**
- Be a community member first
- Share only when genuinely relevant
- Add value in comments, not just links
- Disclose affiliation when required

## Channel 6: Influencer & Partner Distribution

**Goal:** Leverage others' audiences for amplification.

### Influencer Outreach

**Identification:**
- 10K-100K followers (micro-influencers often better)
- Relevant audience
- High engagement rate
- Aligned values

**Outreach Approach:**
```
Hi [Name],

I've been following your work on [topic] - especially 
loved [specific content].

I just published [content title] that I think your 
audience would find valuable. It covers [brief summary].

No ask here - just thought you might find it interesting!

[Link]

Best,
[Your name]
```

**Collaboration Ideas:**
- Co-created content
- Expert quotes for articles
- Podcast guest appearances
- Social media takeovers
- Joint webinars

### Partner Co-Marketing

**Ideal Partners:**
- Complementary services (not competitors)
- Similar audience
- Similar company size
- Good relationship

**Co-Marketing Ideas:**
- Co-authored research report
- Joint webinar
- Guest blog exchange
- Social media collaboration
- Bundle offerings
