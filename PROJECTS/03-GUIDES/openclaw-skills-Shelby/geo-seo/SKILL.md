## Commands

| Command | What It Does |
|---------|-------------|
| `geo audit <url>` | Full GEO + SEO audit with scoring |
| `geo quick <url>` | 60-second AI visibility snapshot |
| `geo citability <url>` | Score content for AI citation readiness (0-100) |
| `geo crawlers <url>` | Check AI crawler access (robots.txt) |
| `geo report <url>` | Generate client-ready markdown report |

## Quick Start

### Run a Full Audit
```
geo audit https://example.com
```

### Quick Visibility Check
```
geo quick https://example.com
```

### Generate Client Report
```
geo report https://example.com
```
