## 📅 Day 1: Prospect Research & Outreach

### Morning (9:00 AM - 12:00 PM)

**Prospect Research (2 hours)**
- [ ] Identify 30 potential affiliates
- [ ] Focus on: AI tool reviewers, content creators, business coaches, side hustle influencers
- [ ] Minimum criteria: 1,000+ engaged followers/subscribers
- [ ] Review each prospect's content
- [ ] Note specific posts/videos you genuinely liked
- [ ] Categorize by tier:
  - Tier 1: 10,000+ followers (power affiliates)
  - Tier 2: 3,000-10,000 followers (growth affiliates)
  - Tier 3: 1,000-3,000 followers (emerging affiliates)

**Prepare Outreach (1 hour)**
- [ ] Draft personalized cold email for each prospect
- [ ] Choose appropriate variation (A: personal, B: data-driven, C: mutual benefit)
- [ ] Include specific reference to their content
- [ ] Add affiliate program one-pager link
- [ ] Add commission structure link
- [ ] Add calendar link for 15-min chat

### Afternoon (1:00 PM - 5:00 PM)

**Send Outreach (2 hours)**
- [ ] Send 10-15 personalized emails
- [ ] Track sent emails in spreadsheet
- [ ] Include: prospect name, platform, tier, email variant used, sent date
- [ ] Send via personal email or warm email account (not sales email)

**Follow Up on Social (1 hour)**
- [ ] Like 3-5 of each prospect's recent posts
- [ ] Leave genuine comments (not sales-y)
- [ ] Share their content if relevant (with credit)
- [ ] Follow them if not already

### Evening Wrap-up
- [ ] Review outreach spreadsheet
- [ ] Note which emails went well
- [ ] Plan follow-ups for tomorrow

---

## 📅 Day 2: More Outreach & Follow-ups

### Morning (9:00 AM - 12:00 PM)

**Send More Outreach (2 hours)**
- [ ] Send 10-15 more personalized emails
- [ ] Aim for 20-30 total sent by end of day
- [ ] Mix of Tier 1, 2, and 3 prospects
- [ ] Ensure personalization in each email

**Engage on Social (1 hour)**
- [ ] Reply to comments on their posts
- [ ] Share relevant content
- [ ] Continue building rapport before pitching

### Afternoon (1:00 PM - 5:00 PM)

**Monitor Responses (1 hour)**
- [ ] Check email for replies
- [ ] Respond to any interest within 1 hour
- [ ] Schedule calls for interested prospects
- [ ] Send follow-up emails if needed

**Prepare for Calls (1 hour)**
- [ ] Review notes from prospect research
- [ ] Prepare talking points for 15-min calls
- [ ] Have commission structure ready to share
- [ ] Have demo link ready
- [ ] Have application link ready

### Evening Wrap-up
- [ ] Update outreach spreadsheet with responses
- [ ] Schedule calls for interested prospects
- [ ] Note any objections or concerns

---

## 📅 Day 3: Calls & Applications

### Morning (9:00 AM - 12:00 PM)

**Discovery Calls (2 hours)**
- [ ] Conduct 15-min discovery calls with interested prospects
- [ ] Focus on: their audience, promotion style, past affiliate experience
- [ ] Share value proposition (50% launch week commission)
- [ ] Show commission structure and earning potential
- [ ] Address any concerns
- [ ] Invite them to apply

**Follow Up from Calls (1 hour)**
- [ ] Send personalized follow-up email after each call
- [ ] Include: next steps, application link, timeline
- [ ] Answer any questions they had

### Afternoon (1:00 PM - 5:00 PM)

**Review Applications (1 hour)**
- [ ] Check for new applications
- [ ] Review each application within 2 hours
- [ ] Approve qualified applicants (70-80% approval rate target)
- [ ] Request more info for borderline cases
- [ ] Reject unqualified applicants (polite, offer future)

**Send Approvals (1 hour)**
- [ ] Send welcome email to approved affiliates
- [ ] Include: onboarding guide, commission structure, swipe file
- [ ] Add affiliate dashboard access
- [ ] Set up payment method reminder
- [ ] Invite to affiliate community/Discord

### Evening Wrap-up
- [ ] Track applications received vs. outreach sent
- [ ] Note conversion rate
- [ ] Plan tomorrow's follow-ups

---

## 📅 Day 4: Onboarding & First Promotions

### Morning (9:00 AM - 12:00 PM)

**Onboarding New Affiliates (1 hour)**

- [ ] Answer questions via email
- [ ] Offer 15-min onboarding call if needed

**Review First Promotions (1 hour)**
- [ ] Check if any affiliates made their first promotion

- [ ] Encourage more to get started

### Afternoon (1:00 PM - 5:00 PM)

- [ ] Send follow-up email to non-responders (3rd touch)
- [ ] Personalize based on previous interactions
- [ ] Include new value (testimonial, case study)
- [ ] Keep it light and helpful

**Continue Engagement (1 hour)**
- [ ] Engage with prospects on social
- [ ] Share affiliate success stories
- [ ] Build FOMO (others are joining)
- [ ] Respond to any new interest

### Evening Wrap-up
- [ ] Track applications and approvals
- [ ] Review affiliate activation rate
- [ ] Note any patterns in responses

## 🚨 Troubleshooting Common Issues

### Issue: Low Response Rate
**Solutions:**
- [ ] Improve email personalization
- [ ] Test different subject lines
- [ ] Follow up more consistently
- [ ] Target higher-quality prospects

### Issue: High Application, Low Approval
**Solutions:**
- [ ] Clarify audience requirements
- [ ] Improve application process
- [ ] Review prospect quality
- [ ] Be more selective

### Issue: Approved Affiliates Not Active
**Solutions:**
- [ ] Send reminder emails
- [ ] Offer 1:1 onboarding calls
- [ ] Share success stories from other affiliates
- [ ] Provide more guidance/swipes

### Issue: Low Conversion from Affiliate Links
**Solutions:**
- [ ] Review affiliate content quality
- [ ] Provide A/B test suggestions
- [ ] Share top-performing examples
- [ ] Offer performance coaching
