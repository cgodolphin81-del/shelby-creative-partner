## 3. The 1→10+ Workflow

### Step 1: Create the Pillar (Day 1-3)

**Choose Your Hero Content:**
- [ ] Comprehensive blog post (3000+ words)
- [ ] Long-form video (15+ minutes)
- [ ] Webinar/live event
- [ ] Original research/report
- [ ] Podcast episode (30+ minutes)
- [ ] Ebook/ultimate guide

**Quality Checklist:**
- [ ] Solves a real problem
- [ ] Unique insights/data
- [ ] Well-researched
- [ ] Engaging format
- [ ] Clear CTA

### Step 2: Extract Key Moments (Day 4)

**For Written Content:**
```
□ Identify 5-10 key quotes
□ Extract 3-5 statistics
□ Note 3-5 actionable tips
□ Find 2-3 compelling stories
□ Highlight 1-3 controversial takes
```

**For Video/Audio Content:**
```
□ Transcribe full content
□ Timestamp key moments
□ Identify quotable sections (15-60 sec)
□ Note visual moments for clips
□ Extract main takeaways
```

### Step 3: Create Derivative Assets (Day 5-7)

**Priority Order:**
1. **High Impact** (Do First)
   - Blog post (if video source)
   - Email newsletter
   - Social media posts (platform-specific)

2. **Medium Impact** (Do Second)
   - Quote graphics
   - Short-form video clips
   - Thread/carousel

3. **Nice to Have** (Do When Time Allows)
   - Infographic
   - Podcast version
   - Presentation deck

## 4. Platform-Specific Adaptation

### 4.1 LinkedIn (Professional/B2B)

| Asset Type | Format | Best Practices |
|------------|--------|----------------|
| **Article** | 1000-2000 words | Professional tone, data-driven, tag relevant people |
| **Post** | 150-300 words | Hook in first line, add value, question at end |
| **Carousel** | 5-10 slides | One idea per slide, bold visuals, CTA on last |
| **Video** | 1-5 minutes | Captions, professional setting, clear audio |
| **Document** | PDF upload | Guides, checklists, frameworks |

**Content Adaptation Example:**
```
Original Blog: "10 Ways to Improve Your Marketing ROI"

LinkedIn Post:
"Most marketers are wasting 40% of their budget.

After analyzing 500+ campaigns, we found the top performers 
all do these 3 things differently:

1. [Insight]
2. [Insight]  
3. [Insight]

The full breakdown with data is in the comments. 👇

What's your biggest ROI challenge?"
```

### 4.2 Twitter/X (Real-time/Conversational)

| Asset Type | Format | Best Practices |
|------------|--------|----------------|
| **Tweet** | 280 characters | Hook, value, engagement ask |
| **Thread** | 5-15 tweets | Numbered, clear flow, CTA at end |
| **Quote Tweet** | Add commentary | Add value to original, don't just RT |

**Thread Structure:**
```
Tweet 1: Hook + promise
"Most people get [topic] wrong.

Here's the right way (a thread):"

Tweets 2-8: Main content
- One idea per tweet
- Use numbers, data, examples
- Build on each other

Tweet 9-10: Summary + CTA
"To recap:
• Point 1
• Point 2
• Point 3

Want more? [CTA]"
```

### 4.3 Instagram (Visual/Brand)

| Asset Type | Format | Best Practices |
|------------|--------|----------------|
| **Feed Post** | Square/vertical | High-quality image, engaging caption |
| **Carousel** | 3-10 slides | Educational, save-worthy |
| **Reel** | 15-90 seconds | Hook in 3 sec, trending audio, captions |
| **Story** | 15 sec each | Behind-scenes, polls, Q&A |

**Caption Formula:**
```
[Hook - first line visible before "more"]

[Story/Context - 2-3 paragraphs]

[Value/Takeaway]

[CTA - Comment, Save, Share, Link in Bio]

[3-5 relevant hashtags]
```

### 4.4 Facebook (Community)

| Asset Type | Format | Best Practices |
|------------|--------|----------------|
| **Post** | Text + image/video | Conversational, ask questions |
| **Video** | 1-5 minutes | Native upload, captions |
| **Live** | 10-30 minutes | Announce ahead, engage comments |

### 4.5 TikTok/YouTube Shorts (Short Video)

| Asset Type | Format | Best Practices |
|------------|--------|----------------|
| **Short** | 15-60 seconds | Hook immediately, vertical, trending sounds |
| **Tutorial** | 60-90 seconds | Quick, actionable, text overlays |

**Short Video Script Template:**
```
0-3 sec:  Hook ("Stop doing X wrong")
3-15 sec: Problem/Context
15-45 sec: Solution/Steps
45-60 sec: CTA ("Follow for more")
```

### 4.6 Email (Direct/Personal)

| Asset Type | Format | Best Practices |
|------------|--------|----------------|
| **Newsletter** | 300-800 words | Personal tone, one main idea, clear CTA |
| **Sequence** | 3-7 emails | Educational → Value → Offer |

**Newsletter Structure:**
```
Subject: [Intriguing + relevant]

Hey [Name],

[Personal opener - 1-2 sentences]

[Main content - the repurposed piece]

[Key takeaway/action item]

[CTA]

Cheers,
[Name]

P.S. [Bonus tip or secondary CTA]
```

## 10. Repurposing Best Practices

### Do's ✅
- Start with high-quality pillar content
- Adapt (don't just copy) for each platform
- Maintain consistent messaging
- Link related content together
- Track what works and double down
- Batch similar tasks together
- Use templates for efficiency

### Don'ts ❌
- Don't create low-quality derivatives
- Don't post identical content everywhere
- Don't forget platform-specific best practices
- Don't neglect engagement on derivative posts
- Don't repurpose outdated content
- Don't skip the quality check
- Don't forget to update old derivatives
