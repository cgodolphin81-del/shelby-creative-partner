# Local Business Website Service

Complete turnkey website service for local businesses. This package includes everything needed to build, launch, and maintain professional websites for local service businesses.

## 📁 Package Contents

| Folder | Description |
|--------|-------------|
| `industry-templates/` | 20+ industry-specific website templates |
| `copy-templates/` | Homepage, services, about, contact, FAQ copy |
| `local-seo/` | GBP optimization, citations, local keywords |
| `lead-capture/` | Forms, CTAs, chatbots, call tracking |
| `hosting-maintenance/` | Updates, backups, security, support docs |
| `pricing/` | Package pricing and proposals |
| `onboarding/` | Client intake forms and processes |
| `portfolio/` | Before/after templates, case studies |
| `sales-scripts/` | Discovery calls, proposals, closing |
| `calendar/` | 30-day build calendar |

1. Review `sales-scripts/discovery-call.md` for initial client conversation
