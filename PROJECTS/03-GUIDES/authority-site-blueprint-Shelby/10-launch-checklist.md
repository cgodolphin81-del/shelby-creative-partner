## Pre-Launch Checklist

### Domain & Hosting

- [ ] **Domain registered** (Namecheap, Porkbun, or similar)
  - [ ] .com preferred
  - [ ] Brandable name (not exact-match keyword)
  - [ ] No trademark conflicts
  - [ ] Social handles available

- [ ] **Hosting purchased** (Cloudways recommended)
  - [ ] Server launched
  - [ ] WordPress installed
  - [ ] SSL certificate active (HTTPS)

- [ ] **Domain connected**
  - [ ] DNS configured
  - [ ] Domain verified in hosting
  - [ ] WWW vs. non-WWW decided (pick one, stick with it)

---

### WordPress Setup

- [ ] **Basic Settings**
  - [ ] Site title and tagline set
  - [ ] Permalinks set to "Post name"
  - [ ] Timezone configured
  - [ ] Admin email verified

- [ ] **Theme Installed**
  - [ ] Lightweight theme (GeneratePress, Kadence, or Astra)
  - [ ] Child theme created (if customizing)
  - [ ] Theme options configured

- [ ] **Essential Plugins Installed**
  - [ ] SEO: Rank Math or Yoast
  - [ ] Caching: WP Rocket or similar
  - [ ] Security: Wordfence or Sucuri
  - [ ] Backup: UpdraftPlus
  - [ ] Image optimization: ShortPixel
  - [ ] Table of Contents: TOC Plus
  - [ ] Analytics: Site Kit by Google

- [ ] **Plugin Configuration**
  - [ ] SEO plugin configured (sitemap, titles, etc.)
  - [ ] Caching enabled and tested
  - [ ] Security scan completed
  - [ ] Backup schedule set (daily)
  - [ ] Image optimization settings configured

---

### Pages to Create

- [ ] **Homepage**
  - [ ] Clear value proposition
  - [ ] Featured/pillar content
  - [ ] Email signup opportunity
  - [ ] Navigation to key sections

- [ ] **About Page**
  - [ ] Your story/mission
  - [ ] Credentials/expertise
  - [ ] Photo (builds trust)
  - [ ] Contact information

- [ ] **Contact Page**
  - [ ] Contact form
  - [ ] Email address
  - [ ] Social media links
  - [ ] Response time expectation

- [ ] **Privacy Policy**
  - [ ] GDPR compliant (if EU visitors)
  - [ ] CCPA compliant (if California)
  - [ ] Cookie disclosure
  - [ ] Affiliate disclosure

- [ ] **Terms of Service**
  - [ ] Usage terms
  - [ ] Disclaimer (especially for YMYL niches)
  - [ ] Copyright information

- [ ] **Affiliate Disclosure**
  - [ ] Clear, conspicuous disclosure
  - [ ] On every page with affiliate links
  - [ ] FTC compliant language

- [ ] **404 Page**
  - [ ] Helpful navigation
  - [ ] Search bar
  - [ ] Friendly message

---

### Technical Setup

- [ ] **Google Tools**
  - [ ] Google Analytics 4 installed
  - [ ] Google Search Console verified
  - [ ] Google Tag Manager (optional)
  - [ ] XML sitemap submitted to GSC

- [ ] **Site Speed**
  - [ ] PageSpeed score 90+ (mobile)
  - [ ] Images optimized
  - [ ] Caching enabled
  - [ ] CDN configured (Cloudflare)

- [ ] **Mobile Optimization**
  - [ ] Responsive design tested
  - [ ] Mobile menu works
  - [ ] Touch targets adequate size
  - [ ] No horizontal scrolling

- [ ] **Security**
  - [ ] SSL certificate active
  - [ ] Admin username changed (not "admin")
  - [ ] Strong password set
  - [ ] Two-factor authentication enabled
  - [ ] Login attempt limiting enabled
  - [ ] XML-RPC disabled (if not needed)

---

### Content Preparation

- [ ] **Pillar Pages**
  - [ ] At least 1 pillar page published (3,000+ words)
  - [ ] Internal linking structure planned
  - [ ] Related cluster topics identified

- [ ] **Initial Content**
  - [ ] Minimum 10 articles published before launch
  - [ ] Mix of content types (guides, reviews, how-tos)
  - [ ] All content edited and optimized
  - [ ] Images added and optimized

- [ ] **Content Calendar**
  - [ ] 4-8 weeks of content planned
  - [ ] Keyword research completed
  - [ ] Publishing schedule set

---

### Monetization Setup

- [ ] **Google AdSense** (or alternative)
  - [ ] Account created
  - [ ] Site submitted for approval
  - [ ] Ad placements planned

- [ ] **Affiliate Programs**
  - [ ] Amazon Associates applied (if applicable)
  - [ ] Niche-specific affiliate programs identified
  - [ ] Affiliate links ready to insert
  - [ ] Disclosure language prepared

- [ ] **Email Marketing**
  - [ ] Email service provider chosen (ConvertKit, etc.)
  - [ ] Signup form created
  - [ ] Welcome email sequence written
  - [ ] Lead magnet created (optional but recommended)

---

### Legal & Compliance

- [ ] **Privacy Policy** - Covers data collection, cookies, GDPR
- [ ] **Terms of Service** - Usage terms, disclaimers
- [ ] **Affiliate Disclosure** - FTC compliant
- [ ] **Cookie Consent** - If EU visitors expected
- [ ] **Medical/Legal Disclaimer** - If YMYL niche
- [ ] **Copyright Notice** - Footer copyright

## Launch Week Checklist

### Day 1: Soft Launch

- [ ] Site accessible via direct URL
- [ ] All pages load correctly
- [ ] No broken links
- [ ] Forms working (test contact form, email signup)
- [ ] Analytics tracking verified (real-time data showing)

---

### Day 2-3: Testing

- [ ] **Cross-browser testing**
  - [ ] Chrome
  - [ ] Firefox
  - [ ] Safari
  - [ ] Edge

- [ ] **Device testing**
  - [ ] Desktop
  - [ ] Tablet
  - [ ] Mobile (iOS and Android)

- [ ] **Speed testing**
  - [ ] PageSpeed Insights
  - [ ] GTmetrix
  - [ ] WebPageTest

- [ ] **Fix any issues found**

---

### Day 4-5: Indexing

- [ ] **Request indexing** in Google Search Console
- [ ] **Submit sitemap** (if not auto-submitted)
- [ ] **Check robots.txt** - Ensure not blocking important pages
- [ ] **Verify noindex tags** - Remove from pages that should be indexed

---

### Day 6-7: Initial Promotion

- [ ] **Social media announcement**
  - [ ] Twitter/X post
  - [ ] LinkedIn post (if B2B)
  - [ ] Facebook page post
  - [ ] Instagram post (if visual niche)

- [ ] **Personal network**
  - [ ] Email friends/colleagues
  - [ ] Share in relevant communities (if allowed)
  - [ ] Ask for feedback

- [ ] **First backlinks**
  - [ ] Add to personal social profiles
  - [ ] Add to email signature
  - [ ] Submit to relevant directories (quality only)

---

## Post-Launch Checklist

### Week 1-2

- [ ] Monitor Google Search Console for errors
- [ ] Check analytics for any tracking issues
- [ ] Respond to any comments/emails
- [ ] Publish 2-3 new articles
- [ ] Begin basic link building (social profiles, directories)

---

### Week 3-4

- [ ] Review initial traffic data
- [ ] Identify top-performing content
- [ ] Begin outreach for guest posts
- [ ] Apply for additional affiliate programs
- [ ] Continue publishing 2-4 articles/week

---

### Month 2

- [ ] First content audit (update any outdated info)
- [ ] Apply for Ezoic (if not using AdSense)
- [ ] Build 5-10 quality backlinks
- [ ] Review and optimize underperforming content
- [ ] Plan content for month 3

---

### Month 3

- [ ] Quarterly review of goals vs. actual
- [ ] Assess monetization performance
- [ ] Adjust strategy based on data
- [ ] Consider hiring help (editor, writer)
- [ ] Apply for Mediavine Journey (if eligible)

---

## Outreach Scripts

### Guest Post Pitch Template

```
Subject: Guest Post Idea for [Site Name]

Hi [First Name],

I've been following [Site Name] for a while and really enjoyed your 
recent article on [specific article title] - especially your point 
about [specific detail].

I'm reaching out because I have a guest post idea that I think would 
resonate with your audience:

**Proposed Topic:** [Specific, unique angle]

**Why It Fits:** [Explain relevance to their audience]

**Brief Outline:**
- [Point 1]
- [Point 2]
- [Point 3]
- [Point 4]

**About Me:** I'm [your role] at [your site], where I write about 
[your niche]. My work has been featured on [publications, if any].

Would you be open to a guest post on this topic? I can have a draft 
to you within [timeframe].

Best regards,
[Your Name]
[Your Site]
[Your Email]
```
