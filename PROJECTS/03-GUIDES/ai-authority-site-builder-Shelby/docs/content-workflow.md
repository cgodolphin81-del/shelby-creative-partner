# 🤖 AI Content Workflow

## The 6-Step Pipeline

```
RESEARCH → OUTLINE → DRAFT → HUMANIZE → OPTIMIZE → PUBLISH
   │          │         │         │          │          │
   ▼          ▼         ▼         ▼          ▼          ▼
Keywords   Structure  AI Write   Voice     SEO       CMS
```

### AI Research Prompt

```
You are an SEO researcher. Analyze this topic: [TOPIC]

Find:
1. Primary keyword with search volume and difficulty
```

## Step 2: OUTLINE (10-15 min/article)

### Outline Template

```markdown
# [Article Title]

## Introduction (150 words)
- Hook: [statistic/question/story]
- Problem: [what reader struggles with]
- Promise: [what they'll learn]
- CTA: [preview main points]

## H2: [Main Point 1]
### H3: [Subpoint 1a]
- Key fact/stat
- Example
- Affiliate product mention

### H3: [Subpoint 1b]
- Key fact/stat
- Example

## H2: [Main Point 2]
### H3: [Subpoint 2a]
- Step-by-step instruction
- Pro tip
- Visual/diagram opportunity

## H2: [Main Point 3]
- Comparison table
- Pros/cons list
- Recommendation

## H2: [FAQ Section]
- Question 1 + answer
- Question 2 + answer
- Question 3 + answer

## Conclusion (100 words)
- Summary of key points
- Final recommendation
- CTA (affiliate/newsletter/comment)

## Internal Links to Include
- [Pillar page]
- [Related cluster 1]
- [Related cluster 2]
```

### AI Outline Prompt

```
Create a detailed outline for an article about: [TOPIC]

Target keyword: [KEYWORD]
Word count: [1500-2500]
Audience: [beginner/intermediate/advanced]

Include:
- Compelling introduction with hook
- 4-6 main H2 sections
- 2-3 H3 subsections per H2
- FAQ section with 5 questions
- Conclusion with CTA
- Notes for images/tables/diagrams
- Internal linking opportunities

Make it scannable with bullet points and numbered lists.
```

## Step 3: DRAFT (AI Writing - 5-10 min)

### AI Draft Prompt (Section by Section)

```
Write the [INTRODUCTION] section of an article about [TOPIC].

Context:
- Target keyword: [KEYWORD]
- Audience: [DESCRIPTION]
- Tone: [conversational/authoritative/friendly]
- Word count: [150-200]

Requirements:
- Start with a hook (stat/question/story)
- Address the reader's problem
- Promise what they'll learn
- Include keyword naturally in first 100 words
- End with transition to main content

Don't use phrases like "In this article" or "Let's dive in."
```

### Full Article Prompt (Alternative)

```
Write a comprehensive article about [TOPIC].

Specifications:
- Target keyword: [PRIMARY KEYWORD]
- Secondary keywords: [LIST]
- Word count: [2000]
- Tone: Conversational but authoritative
- Reading level: Grade 8-10

Structure:
[INSERT OUTLINE HERE]

Requirements:
- Include all outline sections
- Use short paragraphs (2-3 sentences)
- Add bullet points and numbered lists
- Include 2-3 comparison tables
- Write natural affiliate product mentions
- Add FAQ section with 5 questions
- Include internal link placeholders [LINK: page-name]

SEO:
- Use keyword in H1, first paragraph, 2-3 H2s, conclusion
- Include semantic keywords naturally
- Write meta description (150-160 characters)
```

## Step 4: HUMANIZE (20-30 min/article)

### Humanization Checklist

```markdown
## Voice & Personality
- [ ] Add personal anecdotes/experiences
- [ ] Include opinions and preferences
- [ ] Use contractions (you'll, don't, can't)
- [ ] Add humor where appropriate
- [ ] Write like talking to a friend

## Authenticity
- [ ] Verify all facts and statistics
- [ ] Add real product experience (if possible)
- [ ] Include pros AND cons (not just praise)
- [ ] Mention limitations and caveats
- [ ] Add "why I recommend" explanations

## Engagement
- [ ] Ask rhetorical questions
- [ ] Use "you" frequently
- [ ] Add callout boxes for key points
- [ ] Include actionable tips
- [ ] Create curiosity gaps

## AI Detection Avoidance
- [ ] Vary sentence length (mix short/long)
- [ ] Use transition words naturally
- [ ] Remove repetitive phrases
- [ ] Add unique insights not in AI output
- [ ] Include current events/recent data
```

### Humanization Prompts

```
Rewrite this paragraph to sound more human and conversational:

[AI-GENERATED TEXT]

Make it:
- Sound like a real person talking
- Add a personal touch or opinion
- Vary sentence structure
- Remove robotic phrases
- Keep the same information
```

```
Add a personal anecdote to this section about [TOPIC]:

[SECTION TEXT]

Include:
- A brief story or experience
- What I learned from it
- How it relates to the reader
- Keep it under 100 words
```

## Step 5: OPTIMIZE (15-20 min/article)

### On-Page SEO Checklist

```markdown
## Title & Meta
- [ ] Title includes primary keyword (front-loaded)
- [ ] Title under 60 characters
- [ ] Meta description 150-160 characters
- [ ] Meta includes keyword and CTA

## Content
- [ ] Keyword in H1
- [ ] Keyword in first 100 words
- [ ] Keyword in 2-3 H2s
- [ ] Keyword in conclusion
- [ ] Keyword density 1-2% (natural)
- [ ] Semantic keywords included
- [ ] Word count meets target

## Technical
- [ ] URL includes keyword
- [ ] Image alt text includes keywords
- [ ] Internal links (3-5 per article)
- [ ] External links (2-3 authoritative)
- [ ] Schema markup added
- [ ] Mobile-friendly formatting

## Engagement
- [ ] Table of contents (for long articles)
- [ ] Images every 300 words
- [ ] At least 1 table or chart
- [ ] Bullet points and lists
- [ ] Bold key phrases
- [ ] Short paragraphs
```

### AI Optimization Prompt

```
Optimize this article for SEO:

[ARTICLE TEXT]

Target keyword: [KEYWORD]
Secondary keywords: [LIST]

Tasks:
1. Suggest title tag (under 60 chars, keyword front)
2. Write meta description (150-160 chars, include CTA)
3. Identify where to add internal links
4. Suggest 2-3 external authoritative links
5. Recommend image alt text for existing images
6. Generate FAQ schema markup
7. Identify keyword density issues
8. Suggest semantic keywords to add
```

---

## Step 6: PUBLISH (10 min/article)

### Pre-Publish Checklist

```markdown
## Content Review
- [ ] Spell check complete
- [ ] Grammar check complete
- [ ] All links working
- [ ] Images optimized (WebP, under 200KB)
- [ ] Tables formatted correctly
- [ ] Mobile preview looks good

## SEO
- [ ] Title tag set
- [ ] Meta description set
- [ ] URL slug optimized
- [ ] H1 tag correct
- [ ] Schema markup added
- [ ] Canonical URL set

## Monetization
- [ ] Affiliate links added (with disclosure)
- [ ] Ad placements configured
- [ ] Email opt-in included
- [ ] CTA buttons working

## Publishing
- [ ] Category assigned
- [ ] Tags added (3-5)
- [ ] Featured image set
- [ ] Social sharing image set
- [ ] Schedule/publish
```

### Social Media Snippets

```
Generate 3 social media posts for this article:

Key points: [LIST]

Platforms:
- Twitter (280 chars, 2-3 hashtags)
- LinkedIn (professional tone, 3-5 hashtags)

- Pinterest (description for pin)
```
