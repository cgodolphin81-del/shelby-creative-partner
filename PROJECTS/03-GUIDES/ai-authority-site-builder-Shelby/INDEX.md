## 🔧 Script Usage

### Deployment
```bash
# Deploy to GitHub Pages
./scripts/deploy/deploy-to-github-pages.sh \
  --repo your-username/site-name \
  --domain yoursite.com
```

### Keyword Research
```bash
# Run keyword research
./scripts/seo-automation/keyword-research.sh
```

### SEO Optimization
```bash
# Run on-page optimizer
python3 scripts/seo-automation/on-page-optimizer.py

# Generate schema
node scripts/seo-automation/schema-generator.js
```
