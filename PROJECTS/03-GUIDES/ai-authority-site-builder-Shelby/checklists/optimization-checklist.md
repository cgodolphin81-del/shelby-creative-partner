## Page-Level Optimization Checklist

### Before Publishing
- [ ] Target keyword in title
- [ ] Target keyword in first paragraph
- [ ] Target keyword in 2-3 H2s
- [ ] Word count meets target (1500-2500)
- [ ] Readability score (Grade 8-10)
- [ ] Images optimized (WebP, <200KB)
- [ ] Alt text on all images
- [ ] Internal links (3-5)
- [ ] External links (2-3)
- [ ] Meta description (150-160 chars)
- [ ] URL slug optimized
- [ ] Schema markup added
- [ ] Mobile preview checked

### After Publishing
- [ ] Submit to Google Search Console
- [ ] Share on social media
- [ ] Add to email newsletter
- [ ] Monitor rankings (1 week, 1 month)
- [ ] Update based on performance
- [ ] Add to relevant internal links

## SEO Optimization Checklist

### On-Page SEO
- [ ] Title tag (50-60 chars, keyword front)
- [ ] Meta description (150-160 chars, CTA)
- [ ] H1 tag (includes keyword)
- [ ] H2/H3 structure (logical hierarchy)
- [ ] Keyword density (1-2%, natural)
- [ ] Image optimization (compressed, alt text)
- [ ] Internal linking (3-5 links)
- [ ] External linking (2-3 authoritative)
- [ ] URL optimization (short, descriptive)
- [ ] Mobile-friendly
- [ ] Fast load time (<3s)
- [ ] Schema markup

- [ ] Anchor text diverse
- [ ] Domain authority increasing
- [ ] Brand mentions increasing
- [ ] Social signals present
- [ ] Guest posts published
- [ ] HARO responses
- [ ] Resource page links

## Performance Optimization Checklist

### Page Speed
