# 🎨 AI Image Generation Prompts

## Overview

This guide provides ready-to-use AI prompts for generating all types of images needed for authority sites:

- Featured images
- Product images
- Infographics
- Comparison charts
- Diagrams
- Icons and illustrations

---

## 🖼️ Featured Images

### General Structure

```
[Subject], [Style], [Color Palette], [Composition], [Quality]
```

### Prompt Templates

#### Professional Photo Style

```

shallow depth of field, bokeh background, high resolution, 
photorealistic, commercial quality, 8K --ar 16:9 --v 6

```

```
bokeh city background, high resolution, photorealistic, 
commercial quality, 8K --ar 16:9 --v 6

```

## 📋 Prompt Modifiers

### Styles

| Style | When to Use |
|-------|-------------|
| `photorealistic` | Real product photos, lifestyle |
| `flat design` | Icons, simple graphics |
| `vector graphics` | Infographics, diagrams |
| `3D render` | Product showcases |
| `watercolor` | Lifestyle, wellness |
| `minimalist` | Modern, clean aesthetics |
| `illustration` | Conceptual graphics |

### Aspect Ratios

| Ratio | Use Case |
|-------|----------|
| `--ar 16:9` | Featured images, blog headers |
| `--ar 4:3` | Social media, standard images |
| `--ar 1:1` | Profile pictures, product shots |
| `--ar 3:4` | Pinterest, mobile-friendly |
| `--ar 21:9` | Wide banners, hero images |

### Quality Settings

| Setting | Effect |
|---------|--------|
| `--v 6` | Latest version, best quality |
| `--q 2` | Higher quality, slower |
| `--style raw` | More photorealistic |
| `--s 750` | Higher artistic variance |

```bash
#!/bin/bash
# Generate multiple images from a list

INPUT_FILE="image-prompts.txt"
OUTPUT_DIR="generated-images"

while IFS= read -r prompt; do
    echo "Generating: $prompt"
    # Run your AI image generation tool here
    # Example: dall-e "$prompt" --output "$OUTPUT_DIR/$(date +%s).png"
done < "$INPUT_FILE"
```
